## 报告文件命名约定

```
S1-全量轻扫-R{N}-{YYYYMMDD-HHmm}.md     ← 每轮 S1 汇总
R{N}-{模块名}-{YYYYMMDD-HHmm}.md        ← 每轮 S2 深度审计
FINAL-SUMMARY-{session_id}.md            ← Session 最终报告
auto-fixes.log                            ← 自动修复日志（追加）
```

## 报告格式要求

- 所有 .md 文件可直接在 Obsidian 中打开阅读
- 文件路径使用 `[[wikilink]]` 格式引用设计文档
- 表格中对齐使用标准 Markdown 表格语法
- 中文正文，技术术语保留英文
- > 块引用用于元数据、警告、重要提示
