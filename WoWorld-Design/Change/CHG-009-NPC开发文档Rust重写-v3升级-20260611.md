## 〇、背景

NPC 活人感开发文档 ver1.01（~17000 行）和 ver1.01short（压缩版）是基于 GDScript 全栈假说编写的。CHG-007 和 CHG-008 之后，技术栈已演进为 Rust 模拟核心 + Godot 渲染客户端（016 v2.0 → 018 v3.0），设计上新增了半自动战斗、载具、天象感知、蓝图施工、L4 超远距 NPC 等系统。原文档在语言假设、技术架构、系统覆盖上全面过期。

本次变更是根本性重写，而非增量修改。

## 一、变更范围

### 1.1 GDScript → Rust 翻译

| 原 (ver1.01 GDScript) | 新 (ver2.0 Rust) |
|----------------------|-------------------|
| `Dictionary` 存储 NPCData | `#[derive(Serialize, Deserialize)] struct NpcData` |
| 信号总线 (`signal`) | `trait EventSubscriber` + 同步/异步队列 |
| 场景树 LOD 管理 (`_process`) | `SimulationScheduler` + `SimulationLevel` enum |
| JSON 存档 (`FileAccess`) | `bincode` + LMDB 流式 Chunk 存储 |
| Godot 插件推荐 (Voxel Tools/SQLite/NavigationServer) | 全部移除 — 自建 Rust 方案 |

### 1.2 v3 新系统植入

| # | v3 系统 | 文档位置 |
|---|--------|---------|
| 1 | L4 超远距统计 NPC | §3.1 (四层模拟), L4: 每周/月批量, 仅宏观指标 |
| 2 | 半自动战斗 AI | §5.1, trait CombatAI, CombatCommand 指令层 |
| 3 | 载具上 NPC 行为 | §5.2, VehicleState, 船员/乘客 GOAP |
| 4 | 天象感知 | §5.3, SkyPerception, SensoryProvider::perceive_sky |
| 5 | 蓝图集体施工 | §5.4, ConstructionJob, 施工角色分配 |
| 6 | 长途旅行者 | §5.5, TravelerGenerator, TravelerType |
| 7 | 海洋环境 NPC | §5.6, 港口/船上/海上遭遇/海岸居民行为 |
| 8 | 城市功能分区 | §2.7, UrbanDistrict, DistrictType |
