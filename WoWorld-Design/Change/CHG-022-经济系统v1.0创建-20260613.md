# CHG-022 — 经济系统 v1.0 创建

## 概述

WoWorld 经济系统从零建立——被 40+ 文件交叉引用但此前只有 3 行脑暴草稿的经济概念，正式成为独立设计模块。9 篇开发规格文档 + README，约 5,600 行设计规格。

**核心设计决策**：经济系统 = 客观事实层（价格从交易涌现、货币真实流动、订单簿撮合）。NPC 心智 = 主观决策层（人格×情绪×记忆×行为经济学偏差×认知能力）。经济系统**不告诉 NPC 该买什么**。

### 设计决策（9 项用户裁决）

1. **经济系统 = 客观事实层，NPC 心智 = 主观决策层** — 经济系统不告诉 NPC 该买什么，NPC 用自己的人格×情绪×记忆×认知做决策
2. **价格从限价订单簿撮合中涌现** — 不存在中央定价公式。分层定价：正常中间价 / 紧急偏移 / 薄市场协商 / 政府强制价
3. **市场 = 供需匹配机制（非物理地点）+ 店铺 = 交易原子单位** — 铁匠在自家铺子卖剑就是市场。远程订单、长期合同、中间人撮合都是市场
4. **商人身份从四条件涌现**（信息+资本+心理+物流）— 不指定"商人 NPC"
5. **同一套订单簿引擎 + MarketRegulations 参数 = 全部经济体制** — 自由市场到军国经济，参数连续可调
6. **借贷 = 特化交易** — 贷款是赊购铜币，欠条是物品系统中的 `ItemCategory::Financial` 物品。不需要独立银行系统
7. **权力是 PowerAtom 的集合** — 权力的原子化拆解。四种来源（正式/购买/事实垄断/暴力）。玩家和 NPC 走同一代码路径
8. **职业标签是基础原子的排列组合** — ~80-100 个原子标签，2-4 个排列组合覆盖全部可能职业。不预设"铁器商"——`[锻造+交易]` 自然涌现
9. **LLM 是经济解说员，非经济模拟器** — 无 LLM 时 100% 功能正常

## 新增跨模块接口契约

| 概念 | 权威 Owner | 消费方（引用权威） | 关键约定 |
|------|-----------|-------------------|---------|
| 价格形成 (Market.last_price/TradeRecord) | **经济系统** `002` | 全部模块 | 价格从订单簿撮合交易中涌现——不存在中央定价公式。last_price 是最近成交价的 EMA。**任何模块不得直接写入价格** |
| 市场机制 (Market/MarketScope/MarketListing) | **经济系统** `002` | NPC/物品/世界生成/载具 | Market ≠ 物理地点——是订单簿+范围+交易模式。Storefront 是交易原子单位——任何 NPC 在任何地点都可以开 |
| 交易执行 (TradeExecutor) | **经济系统** `002` | NPC/物品 | 玩家和 NPC 走同一代码路径。两阶段提交（钱包±库存）保证原子性 |
| NPC 经济子状态 (NpcEconomicState) | **经济系统** `001` | NPC | 通过 trait 注入 NpcData——不修改 NpcData 本体。包含钱包/已知价格表/经济认知/贸易偏好/商人状态 |
| 经济认知 (EconomicCognition) | **经济系统** `001`/`007` | NPC | 六维缓存（金融素养/市场理解/价格记忆精度/时间偏好率/搜索广度/满意阈值）——从人格×技能×经验派生，非独立存储 |
| 供需匹配 (MarketListing/SurplusNeedAssessment) | **经济系统** `002`/`004` | NPC/世界生成 | 供给侧=盈余驱动（持股>需求→挂单）。需求侧=匮乏驱动（生理/职业/社交）。**不指定"职业→供给品种类"** |
| 中间商涌现 (trader_score/ArbitragePlan) | **经济系统** `004` | NPC | 四条件涌现（信息+资本+心理+物流）。**不在 world gen 时"指定商人 NPC"** |
| 借贷 (IOU/LoanContract) | **经济系统** `004` | 物品系统/NPC | 借贷=特化交易（赊购铜币）。欠条=ItemCategory::Financial 的 ItemDefId。**无独立银行系统** |
| 经济体制 (MarketRegulations) | **经济系统** `005` | 法律系统(待)/政治系统(待) | 全部参数连续可调——同一套订单簿引擎适应自由市场至军国经济。四项预设为 [TUNING] 起点 |
| 市场权力 (PowerAtom/MarketAuthority) | **经济系统** `006` | 政治系统(待)/法律系统(待) | 权力 = PowerAtom 集合（~15 种原子操作）。四种来源（正式/购买/事实垄断/暴力）。玩家和 NPC 走同一 exercise_power() |
| 行为经济学映射 | **经济系统** `007` | NPC | 十个行为经济学概念全部从 NPC 已有字段派生——**不新增人格维度**。EconomicCognition 为计算缓存 |
| 货币总量 (MoneySupply) | **经济系统** `008` | 世界生成/物品系统 | 三条管道（铸币/消费回收/跨区流动）。五大自动稳定器。货币总量增速与商品总量增速对齐 |
| 职业标签 (ProfessionTag) | **世界生成**（初始分配）/ **NPC 身份系统**（运行时） | 经济系统（消费收入来源类型） | ~80-100 个原子标签——TOML 数据驱动+预留新增接口。proficiency 从技能系统派生。排列组合=职业涌现 |
| LLM 经济增强 | **经济系统** `009` | 语言表达系统 | LLM 不参与任何经济计算。结构化数据注入→自然语言包装。模板回退覆盖 100% 事件类型 |
