## 三、World Gen 005 迁移映射

| 005 中的概念 | → | 新位置 |
|-------------|---|--------|
| `BuildingData` struct | → | `Building` struct（[[建筑模块/002-建筑数据模型\|002]]） |
| `RoomData` struct | → | `Region` struct（[[建筑模块/002-建筑数据模型#五、Region\|002 §五]]） |
| `BuildingStyle` struct | → | **废弃**——以 [[文化系统]] `BuildingStylePreferences` 为准 |
| `CityWfcTile` enum | → | 留在 [[世界生成]]（城市级 WFC） |
| `WfcTemplate` struct | → | [[建筑模块/004-约束求解系统\|004]] WfcBuildingSolver 约束配置 |
| `RoomTile` enum | → | `WfcTile` struct（[[建筑模块/004-约束求解系统#2.3\|004 §2.3]]） |
| `ConstraintGenerator` trait | → | `BuildingGenerator` trait（[[建筑模块/007-建筑生成器族谱\|007]]） |
| WFC 两阶段流程 | → | 城市级→World Gen P5，建筑级→建筑模块 `WfcBuildingSolver` |
| `RoofStyle` / `WallMaterial` enum | → | **废弃**——文化系统已有 |
| 家具放置 | → | [[物品系统]]（家具从建筑模块剥离） |
