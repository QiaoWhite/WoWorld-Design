## 修正后的 Crate 结构

```
woworld/
├── Cargo.toml
├── crates/
│   ├── woworld_core/       # 零依赖：WorldPos, EntityId, SurfaceMaterial,
│   │                        #   四空间trait, AgentSnapshot struct,
│   │                        #   ProfessionTagId, StatusQuery trait
│   ├── woworld_spatial/    # → core: EntityIndex(8方法), VisibilityQuery, SpatialEventBus

```

## 修正后的 Phase 规划

| Phase | 内容 |
|-------|------|
| **Phase 1**（原型） | 双层噪声地形 + 5群系 + 海洋平面 + NPC L4 动物 + 旁观者视角 + 昼夜循环 |
| **Phase 2** | NPC L3（0.1Hz）+ 玩家夺舍 NPC + 基本需求接世界资源 + 物品拾取/丢弃 |
| **Phase 3** | NPC L2-L1 深化 + LMDB 存档集成 + 战斗系统雏形 |
| **Phase 4+** | 魔法·经济·文化·信仰·建筑·载具… 按涌现优先级逐步解锁 |

- ❌ LMDB 存档 / 正式 UI
- ❌ 海洋物理 / 正式美术资产
