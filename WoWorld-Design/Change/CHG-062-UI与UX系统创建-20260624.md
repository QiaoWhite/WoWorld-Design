## 关键设计约束

1. **Rust 不碰 UI**: 全部 UI 在 Godot Control 节点。Rust 只提供纯数据。
2. **沉浸式优先**: L0 常驻(准心) → L1 条件浮现(HP/MP/Bark) → L2 按需展开(面板) → L3 暂停层(设置)
3. **手柄=键鼠**: 完全平等的输入映射
4. **信息诚实**: UI 不泄露 NPC 不知道的信息

## 跨模块影响

- 无设计冲突——UI 系统是纯消费方，不修改任何已有模块接口
- 消费模块：生命/NPC/物品/技能/世界生成/历史/经济/权力/语言/音频/感官
- 接头总览需新增 UI/UX 模块槽位
