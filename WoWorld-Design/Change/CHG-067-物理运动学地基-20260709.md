## 动机

用户就"WoWorld 是否需要引用成熟物理引擎（Rapier/Jolt/Avian）"发起 grill 式深挖，并要求主动盘点全项目物理接触面、堵住盲区。盘点（参考文档 045）暴露：现有 `空间查询与物理/001` 有 COM 抛物体，但缺"上游"（质量、冲量来源）和多处"泛化"，导致 8 个盲区——尤其 **`mass` 字段根本不存在、击退距离无公式、投射物弹道横跨 3 模块全空白、掉落物 velocity 无人积分、爬墙/扒巨兽为零**。

## 决策

**不引入通用刚体物理引擎，扩展现有 Rust 运动学 + 空间查询**（重申并延续 CHG-033）。在 COM 抛物体之上补一层**「质量→冲量→单体运动学」涌现地基**，让击退/坠落/碰撞/投射/挂载/攀爬全部涌现自"质量 + 冲量 + 一个积分器"，而非逐招硬编。

新增文档 [[../Happy Game/开发阶段/模型动作与物理系统/运动学地基/001-质量冲量与涌现运动学|运动学地基/001]]，规格要点：

| # | 决策 | 出处 |
|---|------|------|
| Q1 | 不引擎、扩展自建（全项目取向本就无求解器） | 045/002 §Q1 |
| Q1.5 | 不做大规模刚体动力学（持久化/确定性/LOD 三冲突） | §Q1.5 |
| Q2 | 统一「质量→冲量→单体 COM」地基（`Mass`+复活 `Velocity`+冲量事件+三态机+快慢路分离），多体连锁排除 | §Q2 |
| Q3 | 玩家被击退统一走 COM，落地即还控 | §Q3 |
| Q4 | 冲量=真实动量涌现 + 分流（穿透/击退双系数）；一次读数喂伤害+击退 | §Q4 |
| Q5 | 飞行中撞实体=检测+离散非递归反应（递归深度上限 1） | §Q5 |
| Q6 | `SurfaceAnchor` 薄契约统一甲板/坐骑/巨兽；双层位姿；不触"载具≠坐骑"意志分界 | §Q6 |
| Q7 | 投射物复用 core 运动学内核 + CCD，轻量 SoA；填投射物/火球 AOE/掉落物三空白 | §Q7 |
| Q8 | 攀爬=movement_system 泛化到任意法线 + 握力涌现；规模化承重墙是导航+LOD 非物理 | §Q8 |

**架构约定**：纯运动学/碰撞数学（自由函数）在 `woworld_core`（仅 glam）；ECS Component（`Mass`/`Velocity`/`LocomotionMode`/`Projectile`/`AttachedTo`/`WalkableAnchor`/`Climbing`）在 `woworld_ecs`；**定向冲量走专用一次性冲量队列 `ImpulseQueue`**（确定性 drain），`SpatialEventBus` 另行承载 `WeaponImpact/Explosion` 作感知通知（现有 `WeaponImpact` 只带标量力+位置，不承载定向冲量）。**不引入 nalgebra/parry**（形状简单 + 地形是 SDF）。

## 明确延后 / 依赖不成熟模块

用户明确提示**魔法、战斗模块尚不成熟**。本 CHG 只定"物理地基自有部分"，消费方数值留插口不填：

- 火球发多大冲量、每招击退系数、投射物 payload 内容 → 魔法/战斗成熟后填（provisional 插口 P1/P2/P5）。
- 冲量分流系数、握力/体力曲线、体密度表、二次冲量封顶 → 待调参（N1-N5）。
- 玩家旋转甲板 / 玩家攀爬竖直面的 Godot 方案 → **需原型验证**（V1/V2/V3），不在文档层锁死。
- 壁面重力重定向（蜘蛛）动画/相机 → 参数极值，延后。
- **可攀爬边导航图 + A* 握力掩码** → 属导航/NPC 行动系统，非本模块，待那边立项（Q-A4）。
- 首冲刺（Q2-Q5 主链 + Mass）是否现在动工 → 待用户决定（Q-A2）。

以上开放项集中于 [[../参考文档/045-物理地基设计讨论-20260709/003-待验证与开放问题|参考文档 045/003]]。

## 接头总览更新

本次为模型动作与物理系统模块的**内部扩展**，owner 不变。新增对外契约（待各模块消费方接入时建正式接头）：

- **提供** `SurfaceAnchor` trait（woworld_core）← 载具系统 / 生命系统实现；
- **提供** `ImpulseQueue`（定向冲量入口）← 战斗 / 魔法命中时 push；`SpatialEvent::WeaponImpact/Explosion`（已在空间查询 §1.3）继续作感知通知，不承载定向冲量；
- **提供** `Mass` 消费契约 ← 生命 / 物品 / 载具在 spawn 时填；
- **消费** `OceanProvider`（船体波采样）、`WeatherQuery`（风偏/湿度→握力）、`vitals.fatigue`（握力衰减）、`surface_material_at`（握力/摩擦）；
- **待立项**（不在本模块）：导航/NPC 行动系统的"可攀爬边 + A* 握力掩码"。
