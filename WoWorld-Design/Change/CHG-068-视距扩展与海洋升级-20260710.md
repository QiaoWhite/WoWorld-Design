## 一、动机

当前 WoWorld 世界生成有四项限制：

1. **视距上限 15km** — LOD 7 clipmap ring 最远 15km，camera far plane 20km，海面 22km。站在 700m 山上也看不到地平线的壮阔感。
2. **世界尺度仅英国大小** (~500km × 500km) — 对异世界冒险游戏来说太小，不足以容纳几块主要大陆之间的汪洋冒险。
3. **海面是单层平面** — 22km PlaneMesh，200×200 细分，110m 顶点间距。6 个 Gerstner 波波长 3-18m 全部走样（Nyquist 需要 ≤1.5-9m 间距）。无法表现"此起彼伏"的戏剧性海浪。
4. **视距不可调节** — 硬编码，没有玩家可配置的图形选项。

## 三、关键技术决策

### 3.1 世界拓扑：双层海陆噪声

旧系统用单层 continent 噪声（100km 波长）+ 单阈值决定海陆。在 3200km 世界，此波长产生过度细碎的陆地。

新系统加一层 `super_continent`（~3333km 波长，全世界约 1 周期），与 `region`（~333km 波长，原名 continent）做 **AND**：两层都超过各自阈值才是陆地。

```
super_continent > 0.38 AND region > 0.35 → 陆地（40%）
否则 → 海洋（60%）
```

两种简单 Perlin 噪声的交集涌现出大陆腹地、半岛、大岛链、远洋孤岛、汪洋——不手写任何地理规则。换 seed 换世界，但宏观结构（几块大陆 + 群岛 + 汪洋）保持稳定。

旧 `sea_threshold` 字段被双层阈值**完全替代并删除**。海深改用加权 deficit 公式：`sea_factor = sc_deficit * 0.5 + rc_deficit * 0.5`。

### 3.2 LOD：8→10 层

新增 L8 (20-30km, SH 128m 间距, ~122K 顶点) 和 L9 (30-40km, SH 256m 间距, ~43K 顶点)。总 clipmap 顶点约 1,147K，VRAM ~37 MB。

远景 L8/L9 使用 `sample_height_coarse()`——仅采样 super_continent + region 两层，跳过 detail + mountain 高频噪声（这些频率在 128m+ 间距下完全走样）。

所有硬编码 LOD 边界（`terrain_chunk.rs` 中 10 处 `8`/`7` + `lod.rs` 中 4 处 `.min(7)` + `clipmap.rs` LEVELS 数组）改为从 `LEVELS.len()` 派生。

### 3.3 海洋：双 Tier + TOML 驱动

**Near tier** (0-5km)：512² 顶点，9.8m 间距，3 个 Gerstner 波（25/35/50m 波长）。Nyquist 限 19.6m，3 个波全部正确表示——这是"此起彼伏"的宏观涌浪。

**Far tier** (0-50km)：200² 顶点，250m 间距，无 Gerstner，仅 Fresnel 颜色渐变。

**关键变更**：波参数从 Rust `default_waves()` + GLSL `wave_a..wave_f` 两边硬编码 → `ocean_waves.toml` 单一真相源，Rust 和 GLSL 各自消费。`OceanProvider` trait 不变。`GerstnerWave` 从 `[GerstnerWave; 6]` 固定数组 → `Vec<GerstnerWave>` TOML 驱动。

短于 25m 的原 6 个波全部删除——它们在 110m 旧间距下已经是走样的，在 9.8m 新间距下仍然走样。微细节远期通过 normal map 补充。

### 3.4 大气：Shader 距离雾

terrain shader 新增 `fog_color` 和 `fog_density` uniform，指数距离雾。`fog_color` 取自 `ResolvedAtmosphere.sky_horizon`（已有 17 参数之一），`fog_density = 1.2 / view_distance_meters`。L8/L9 设 `is_distant_layer=true` 跳过 per-fragment normal 计算。

### 3.5 视距调节

`ViewDistance` 类型（`woworld_core`）作为单一真相源。Godot UI 连续滑块 1-40km 步长 1km，确认后通过 WorldDriver `#[export]` property 传播到 `Camera3D.far`、LOD 层可见性、shader fog。

## 四、破坏性变更

| 变更 | 影响 | 处理 |
|------|------|------|
| seed 数组 5→6 | 同 seed 产生不同世界 | 开发阶段接受 |
| `continent_scale` → `region_scale` 重命名 | `noise_params.toml` 格式变更 | 手动迁移 |
| `sea_threshold` 删除 | 旧 TOML 解析失败 | 新 TOML 格式替代 |
| `[GerstnerWave; 6]` → `Vec<GerstnerWave>` | `default_waves()` API 变更 | 改为 `from_toml_str()` |
