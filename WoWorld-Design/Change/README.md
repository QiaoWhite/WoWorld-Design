# Change 文件夹

本文件夹记录 WoWorld 设计文档的结构性变更。每份变更文档独立编号，追溯所有重大修改的动机、内容和影响。

## 命名规范

```
CHG-XXX-简短描述-YYYYMMDD.md
```

| 部分 | 说明 |
|------|------|
| `CHG` | 固定前缀（Change） |
| `XXX` | 三位数字序号，从 001 开始递增 |
| `简短描述` | 中文简要说明变更内容 |
| `YYYYMMDD` | 变更日期 |
