## 一、设计哲学：思考是背景层

### 1.1 核心原则

**思考不是"NPC在做什么"，而是"NPC在想什么"。** 两者同时发生，互不排斥。

- 铁匠可以一边打铁一边琢磨领主昨天说的话（背景思考+外部行动并行）
- 专心吃饭时可以突然停手想到亡妻（背景思考偶尔浮到表面）
- 对话中可以在回答之前有0.3秒的微思考（听→想→答）

**不专门占用时间和地点。** 对标已有感官管线的背景处理——认知加工是并行的认知底层流。

### 1.2 三层差异化设计

| 层 | 计算频率 | 产出 | 消费者 |
|----|---------|------|--------|
| CognitiveTide | 每决策周期 | 3个f32 | 决策引擎、偏误计算、触发检测 |
| ThoughtTrigger | 惰性事件驱动 | `Option<ThoughtTrigger>` | ThoughtFragment生成 |
| ThoughtFragment | 仅玩家15m内 | 文本+SurfacingType | Godot渲染、LifeTrace、DialogueIntent |

## 二、ThoughtTrigger：6类触发源

### 2.1 设计原则

**不主动轮询——被其他系统的正常事件自然携带。** 对标感官管线的显著性引擎。

每个check是纯函数，返回`Option<ThoughtTrigger>`。大部分时候返回None。

### 2.2 触发类型1：感知关联

```rust
/// 附着在感官管线的PerceptEntry处理中
/// NPC看到一个entity→自动检查是否匹配高impact记忆中的参与者
pub fn check_perceptual_trigger(
    percept: &PerceptEntry,
    memory: &MemoryStore,
    cognitive_style: &CognitiveStyle,
    tide: &CognitiveTide,
) -> Option<ThoughtTrigger> {
    // 仅处理Signals级别的实体（对标已有显著性引擎）
    if !percept.salience.is_signal() { return None; }
    if tide.cognitive_load > 0.7 { return None; }  // 太忙了没空想
    
    // 搜索记忆：该entity是否出现在高impact记忆中
    let matching_memories = memory.query_by_participant_or_location(
        percept.entity_id, percept.spatial.position
    );
    
    for mem in matching_memories.iter().take(3) {
        if mem.impact_score > 0.5 || mem.emotional_encoding.intensity() > 0.6 {
            let strength = mem.impact_score * (1.0 - mem.decay_factor);
            // 直觉型NPC更容易被触发
            let style_mult = (1.0 - cognitive_style.analytic_intuitive) * 0.3 + 0.7;
            
            if strength * style_mult > 0.3 {
                return Some(ThoughtTrigger::PerceptualAssociation {
                    percept_entity_id: percept.entity_id,
                    linked_memory_id: mem.id,
                    trigger_strength: strength,
                    association_type: if mem.emotional_encoding.valence() < 0.0 {
                        AssociationType::NegativeFlashback
                    } else {
                        AssociationType::Reminiscence
                    },
                });
            }
        }
    }
    None
}
```

### 2.3 触发类型2：地点触发

```rust
/// 附着在NPC位置更新中
pub fn check_location_trigger(
    position: &Position,
    memory: &MemoryStore,
    emotion: &EmotionState,
    cognitive_style: &CognitiveStyle,
) -> Option<ThoughtTrigger> {
    // 对标已有空间认知模型（NPC ver2.0 §2.7）
    let nearby_memories = memory.spatial_query(position, 50.0);  // 50m半径
    
    for mem in nearby_memories.iter().take(5) {
        let trigger_strength = mem.emotional_encoding.intensity() 
                             * (1.0 - mem.decay_factor);
        
        // 直觉型→被地点触发的情绪涌起更强
        let style_mult = (1.0 - cognitive_style.analytic_intuitive) * 0.5 + 0.5;
        
        if trigger_strength * style_mult > 0.3 {
            return Some(ThoughtTrigger::LocationMemory {
                memory_id: mem.id,
                location_id: mem.location_id,
                distance_m: compute_distance(position, &mem.location),
                nostalgic: mem.emotional_encoding.valence() > 0.0 && mem.age_days() > 365.0,
            });
        }
    }
    None
}
```

### 2.4 触发类型3：情绪波动

```rust
/// 附着在情绪引擎emotion_step()后
/// 当情绪发生显著变化时，自动检索同情绪的记忆
pub fn check_emotional_trigger(
    emotion_delta: f32,  // 从情绪引擎获取
    current_emotion: &EmotionState,
    memory: &MemoryStore,
    personality: &BigFive,
) -> Option<ThoughtTrigger> {
    if emotion_delta.abs() < 0.3 { return None; }
    
    // 情绪一致性检索——对标已有记忆系统的Emotional Congruence重建滤波器
    let resonant_memories = memory.retrieve_emotionally_congruent(
        current_emotion,
        5,  // 最多5条
    );
    
    for mem in &resonant_memories {
        // neuroticism高→情绪触发更敏感
        let threshold = 0.3 / (1.0 + personality.neuroticism * 1.5);
        if mem.impact_score > threshold {
            return Some(ThoughtTrigger::EmotionalResonance {
                linked_memory_id: mem.id,
                resonance_type: if emotion_delta > 0.0 { 
                    ResonanceType::PositiveSurge 
                } else { 
                    ResonanceType::NegativePlunge 
                },
                compounding: mem.access_count > 10,  // 反复想起→可能是反刍
            });
        }
    }
    None
}
```

### 2.5 触发类型4：认知失调

```rust
/// 附着在信息接收中（对标语言表达系统的信息传播）
pub fn check_dissonance_trigger(
    new_info: &InformationPayload,
    self_narrative: &SelfNarrative,
    existing_models: &[MentalModel],
    cognitive_style: &CognitiveStyle,
) -> Option<ThoughtTrigger> {
    // 检查新信息是否与已有信念冲突
    let max_conflict = existing_models.iter()
        .map(|m| compute_belief_conflict(new_info, m))
        .max_by(|a, b| a.partial_cmp(b).unwrap());
    
    // 也与core_values比较
    let value_conflict = check_value_alignment(new_info, &self_narrative.core_values);
    
    let total_dissonance = max_conflict.unwrap_or(0.0).max(value_conflict);
    
    if total_dissonance > 0.5 {
        Some(ThoughtTrigger::CognitiveDissonance {
            severity: total_dissonance,
            conflicting_models: conflicting_model_ids,
            emotional_impact: total_dissonance * (1.0 - cognitive_style.cognitive_dissonance_tolerance()),
        })
    } else {
        None
    }
}
```

### 2.6 触发类型5：空闲漫游

```rust
/// 对标已有的mind_wander_drive机制
/// 仅当cognitive_load<0.4时概率触发
pub fn check_wander_trigger(
    tide: &CognitiveTide,
    cognitive_style: &CognitiveStyle,
    recent_experience_narrowness: f32,
    time_since_last_sleep: Duration,
    rng: &mut Pcg64,
) -> Option<ThoughtTrigger> {
    let base_prob = tide.mind_wander_drive() * 0.15;  // 基础概率~0-15%
    
    // 经验窄→高漫游（对标"俄罗斯方块效应"）
    let narrowness_boost = recent_experience_narrowness * 0.3;
    
    // 长时间清醒→高漫游
    let wake_boost = sigmoid(time_since_last_sleep.as_hours() - 4.0) * 0.2;
    
    let final_prob = (base_prob + narrowness_boost + wake_boost).min(0.5);
    
    if rng.gen_bool(final_prob as f64) {
        Some(ThoughtTrigger::MindWander {
            direction: if cognitive_style.abstract_concrete > 0.5 {
                WanderDirection::FutureHypothetical  // 抽象→"如果...会怎样"
            } else if cognitive_style.analytic_intuitive < 0.3 {
                WanderDirection::PastFlashback  // 直觉→过去记忆
            } else {
                WanderDirection::PuzzleSolving  // 分析→当前问题
            },
            source_narrowness: recent_experience_narrowness,
        })
    } else {
        None
    }
}
```

### 2.7 触发类型6：Zeigarnik效应

```rust
/// 未消化的不确定信念→占用认知资源→频繁被拉入思绪
pub fn check_zeigarnik_trigger(
    models: &[MentalModel],
    emotion: &EmotionState,
    biases: &CognitiveBiases,
    rng: &mut Pcg64,
) -> Option<ThoughtTrigger> {
    let unresolved: Vec<_> = models.iter()
        .filter(|m| !m.abandoned)
        .filter(|m| m.confidence > 0.3 && m.confidence < 0.7)  // 不确定
        .filter(|m| m.counter_count > 0 && m.last_tested.elapsed_days() < 7)
        .collect();
    
    if unresolved.is_empty() { return None; }
    
    let zeigarnik_pressure = zeigarnik_contribution(models);
    
    // 反刍倾向高→Zeigarnik触发概率高
    if rng.gen_bool((zeigarnik_pressure * biases.rumination_tendency * 0.1) as f64) {
        let chosen = unresolved[rng.gen_range(0, unresolved.len())];
        Some(ThoughtTrigger::ZeigarnikPull {
            model_id: chosen.id,
            uncertainty: 1.0 - chosen.confidence.abs(),
            cycles_since_last_test: chosen.last_tested.elapsed_days() as u16,
        })
    } else {
        None
    }
}
```

## 三、ThoughtFragment：3级清晰度

### 3.1 数据结构

```rust
pub struct ThoughtFragment {
    pub content: ThoughtContent,
    pub clarity: f32,           // 0=完全模糊, 1=完全清晰
    pub intensity: f32,         // 这个想法有多"强烈"
    pub surfacing_type: SurfacingType,
    pub generated_at: GameInstant,
}

pub enum ThoughtContent {
    /// 模糊感觉：只有情绪色彩+"隐约觉得哪里不对""好像忘了什么"
    VagueFeeling {
        emotional_valence: f32,
        approximate_domain: MentalModelDomain,
        associated_image: Option<ImageSeed>,  // 心理意象而非文本
    },
    
    /// 半成形："如果...那么...不对，应该是..."
    HalfFormedThought {
        fragments: ArrayVec<String, 3>,
        direction: ThoughtDirection,
    },
    
    /// 清晰命题：可明确表达的完整思考
    ClearThought {
        proposition: String,
        supporting_elements: ArrayVec<ThoughtElement, 3>,
    },
}
```

### 3.2 从Trigger到Fragment的生成

```rust
pub fn generate_thought_fragment(
    trigger: &ThoughtTrigger,
    npc: &NpcData,
    rng: &mut Pcg64,
) -> ThoughtFragment {
    match trigger {
        ThoughtTrigger::PerceptualAssociation { linked_memory_id, .. } => {
            let memory = npc.memory.get(*linked_memory_id);
            // clarity取决于记忆的清晰度和认知风格
            let clarity = (1.0 - memory.decay_factor) 
                        * npc.cognitive_style.analytic_intuitive * 0.5 + 0.3;
            
            if clarity < 0.4 {
                ThoughtContent::VagueFeeling { /*...*/ }
            } else if clarity < 0.7 {
                ThoughtContent::HalfFormedThought { /*...*/ }
            } else {
                ThoughtContent::ClearThought {
                    proposition: compose_clear_thought(memory, npc),
                    // ...
                }
            }
        },
        // ...（其他5类触发同理）
    }
}
```

## 四、片段组合模型

### 4.1 设计原则

**对标语言表达系统的片段组合模型（~430片段·~86KB）。** 不使用LLM生成。

ThoughtTemplate ~200-300条模板，覆盖常见思考模式。`CognitiveStyle`选择表达模式。

### 4.2 表达模式差异

```rust
/// CognitiveStyle→思考表达模式
pub fn select_thought_template(
    trigger: &ThoughtTrigger,
    style: &CognitiveStyle,
) -> ThoughtPattern {
    match style.analytic_intuitive {
        // 分析型："如果A，那么B。但如果C，则D..."
        a if a > 0.7 => ThoughtPattern::Analytical,
        // 直觉型："总觉得...说不上为什么"
        a if a < 0.3 => ThoughtPattern::Intuitive,
        // 混合型
        _ => ThoughtPattern::Mixed,
    }
}
```

## 五、SurfacingModality：6种浮现方式

### 5.1 从内心到外部的渐变

```rust
pub enum SurfacingType {
    /// 闷在心里：仅Godot微表情渲染（皱眉/眼神失焦）
    InternalOnly,
    
    /// 微表情：皱眉/眼神失焦/嘴角微动→Godot动画层
    MicroExpression {
        expression_type: MicroExpressionType,
        duration_ms: u16,  // 200-5000ms
    },
    
    /// 自言自语：Godot渲染字幕（玩家3m内可见）
    Muttering {
        text: String,       // 3-8词短文本
        volume_db: f32,     // -40dB（几乎听不见）到-10dB（清晰自语）
        repeats: u8,        // 1-3次重复（反复念叨）
    },
    
    /// 动作暂停：当前动作暂停2-5秒
    ActionPause {
        duration_ms: u16,   // 2000-5000ms
        gaze_target: Option<Vec3>,  // 望向的方向（或虚空）
    },
    
    /// 产出冲动：产生"得写下来/画下来/弹出来"的冲动
    WritingImpulse {
        modality: SurfacingModality,  // 书写/绘画/音乐
        urgency: f32,                // 0=先存着以后写, 1=马上就得写
    },
    
    /// 说话冲动：产生"得说出来"的冲动→DialogueIntent
    SpeakingImpulse {
        target: Option<NpcId>,       // 最想对谁说
        dialogue_intent: DialogueWise, // 转入DialogueIntent::WisdomSharing
    },
}
```

### 5.2 WritingImpulse的媒介选择

```rust
/// 根据NPC的技能分布选择最佳表达媒介
pub fn select_surfacing_modality(
    impulse: &ThoughtSurfacing,
    skills: &SkillSet,
    cognitive_style: &CognitiveStyle,
) -> SurfacingModality {
    // 分析型→文字；直觉型→图像/音乐
    let writing_power = skills.get(SkillId::CREATIVE_WRITING).level as f32 
                      * (1.0 - cognitive_style.analytic_intuitive * 0.3);
    let painting_power = skills.get(SkillId::PAINTING).level as f32 
                       * cognitive_style.analytic_intuitive * 0.5;
    let music_power = skills.get(SkillId::MUSIC_COMPOSITION).level as f32 
                    * cognitive_style.analytic_intuitive * 0.5;
    
    weighted_random_select([
        (SurfacingModality::Writing, writing_power),
        (SurfacingModality::Painting, painting_power),
        (SurfacingModality::Music, music_power),
    ])
}
```

## 六、对话中的微思考

### 6.1 听→想→答三步骤

```rust
/// 嵌入对话系统的DialogueProcessor中
/// 对标已有digest_conversation_to_memory()
pub fn micro_think_during_dialogue(
    understanding: &UtteranceUnderstanding,
    listener: &NpcData,
    speaker: &NpcData,
    relationship: &Relationship,
    context: &DialogueContext,
) -> Option<DialogueThought> {
    // 大多数时候返回None——直接回应，不显式思考
    let think_prob = 0.05 + listener.cognitive_style.reflective_impulsive * 0.10;  // 5-15%
    
    if !rng.gen_bool(think_prob as f64) { return None; }
    
    // 1. 这话是否与我持有的MentalModel冲突？
    let belief_conflict = check_belief_conflict(understanding, &listener.mental_models);
    
    // 2. 这话是否让我想起了某条特定记忆？
    let memory_trigger = check_memory_resonance(understanding, &listener.memory);
    
    // 3. 说话者的可信度？（对标已有的DeceptionDetection）
    let credibility = relationship.trust * speaker.mental_attributes.charisma;
    
    if belief_conflict > 0.5 {
        Some(DialogueThought::DissonanceDetected { severity: belief_conflict })
    } else if let Some(memory) = memory_trigger {
        Some(DialogueThought::MemoryEvoked { memory_id: memory.id })
    } else if credibility < 0.3 {
        Some(DialogueThought::CredibilityDoubt)
    } else {
        None
    }
}
```

## 七、思考→情绪产出

### 7.1 思考本身产生情绪

```rust
/// 对标：情绪引擎的external_event_push机制
pub fn thought_emotional_output(
    trigger: &ThoughtTrigger,
    emotion: &mut EmotionState,
    cognitive_style: &CognitiveStyle,
) -> Option<EmotionEvent> {
    match trigger {
        // 认知失调→焦虑/困惑
        ThoughtTrigger::CognitiveDissonance { severity, .. } => {
            if *severity > 0.7 {
                Some(EmotionEvent::push(emotion, Emotion::Anxiety, severity * 0.3))
            } else {
                None
            }
        },
        // 重大领悟→兴奋/释然（仅高openness）
        ThoughtTrigger::CreativeLeap { novelty } => {
            if cognitive_style.abstract_concrete > 0.5 {
                Some(EmotionEvent::push(emotion, Emotion::Excitement, novelty * 0.25))
            } else {
                None
            }
        },
        // 反刍阻塞→挫折
        ThoughtTrigger::ZeigarnikPull { cycles_since_last_test, .. } => {
            if *cycles_since_last_test > 10 {
                Some(EmotionEvent::push(emotion, Emotion::Frustration, 0.2))
            } else {
                None
            }
        },
        // 大部分触发→不产生情绪
        _ => None,
    }
}
```

## 九、伤春悲秋：感性主导的思考

### 9.1 完整链条

```
环境感知 → 情绪唤醒 → 记忆共鸣 → 自我叙事激活 → 思考涌现 → 身体表达
```

此链条的每一环**在已有系统中均存在**，本模块只提供自动桥接。

### 9.2 桥接函数

```rust
/// 纯函数——从已有数据到思考片段的桥接
/// 仅当cognitive_load<0.4时惰性调用
pub fn aesthetic_contemplation_chain(
    scene: &VisualScene,
    emotion: &EmotionState,
    memory: &MemoryStore,
    self_narrative: &SelfNarrative,
    cognitive_style: &CognitiveStyle,
) -> Option<ThoughtFragment> {
    // 场景是否触发了足够的情绪？
    if scene.aesthetic_valence.abs() < 0.4 { return None; }
    
    // 认知风格是否倾向于此？
    if cognitive_style.analytic_intuitive > 0.7 { return None; }  // 纯分析型不伤春悲秋
    
    // 检索情感一致记忆
    let resonant = memory.retrieve_emotionally_congruent(emotion, 3);
    if resonant.is_empty() { return None; }
    
    // 关联SelfNarrative主题
    let theme_hook = self_narrative.find_theme_hook(&resonant[0]);
    
    // 组合片段
    let clarity = if cognitive_style.abstract_concrete > 0.5 {
        0.6  // 抽象思维→更强烈的文学性质思考
    } else {
        0.3  // 具象思维→模糊的感觉
    };
    
    Some(ThoughtFragment::compose(
        scene, emotion, &resonant[0], theme_hook, clarity
    ))
}
```
