## 一、边界情况

### 1.1 极端长寿

**情况**：一个 NPC 活到 max_lifespan × 1.2 以上。

**处理**：Gompertz 模型数学上允许，但概率极微。每多活一个月都是一个奇迹。当 NPC 超过 max_lifespan × 1.5（~105 人类岁）时，月死亡概率 > 99.9999%——几乎是确定的事实，但仍有一粒数学的可能性。

这不是 bug——这是"全世界最长寿的人"的涌现。这个 NPC 会被历史模块自动标记——他的墓碑会显示异常长的寿命。

### 1.2 不死族（liches/神灵/某些怪物）

**情况**：`max_lifespan = ∞`。

**处理**：Gompertz 从不触发——`age_pct` 永远是 0/∞ = 0。唯一死亡途径：外力摧毁精魄（spirit→0）、被战斗杀死、或被特定魔法消灭。LifeStage 始终为 Adult（不进入 Elder）。

### 1.3 跨种族不育

**情况**：`MixedRaceRule::Incompatible`。

**处理**：受孕概率恒为零。这不是"系统禁止"——是 `fertility_potential_pair()` 对该配对返回 0.0。NPC 的 GOAP 中仍然可能出现"想要孩子"的目标——但生物学概率永远不会满足。

### 1.4 青春期前生育

**情况**：NPC 在 Juvenile 晚期（约人类 12-14 岁）受孕。

**处理**：`fertility_potential` 在 Juvenile 阶段很可能仍 < 0.001，但数学上不是绝对零。Adolescent 晚期 fertility 开始上升。极少数极端边缘情况可发生——概率极低但可能。这在涌现系统中是接受的：不是系统鼓励的，但也不是系统绝对禁止的。文化模块可能对此有强烈谴责（非法/禁忌/污名化），这可能压制生育行为——但不改变生理概率。

### 1.5 近亲繁殖

**情况**：父女/母子/兄弟姐妹之间的受孕。

**处理**：系统不禁止。受孕概率同样按照生物学公式计算。**但**——绝大多数文化中的乱伦禁忌会被内化为 NPC 的 MentalModel（"这是绝对不可以的"），GOAP 中根本不会产生此类目标。在极端情况下（孤立环境、崩溃的文化禁忌、特定信仰系统允许神圣血统内的近亲繁殖），可能发生。

这不是 bug——是涌现中的极端案例。世界生成时的家族结构通常自然避免这种情况（父母和子女之间的正常关系值通常使此类 GOAP 目标不会产生）。

### 1.6 少女/少年怀孕

**情况**：约 14-16 岁的 Adolescent NPC 受孕。

**处理**：生物学上 fertility 在此阶段开始上升——概率偏低但非零。若发生，孕期风险较高（`age_factor` 对低龄母亲的修正）。文化中可能有严厉污名或可能接受（取决于 `IllegitimacyStance` 和婚姻制度）。

### 1.7 母亲死亡时妊娠终止

**情况**：`MaternalDeath` → GestationOutcome。

**处理**：胎儿随母亲一同死亡。`DeathCause` 记录为复合原因：母亲的死因 + `MaternalDeath`。在一些魔法设定中，精魄转移法术可能允许分离胎儿——但这是 mod 范畴。

### 1.8 全部亲属死亡——种群延续

**情况**：一个隔离聚落的所有成年 NPC 死亡（战争/瘟疫），只剩婴幼儿。

**处理**：婴幼儿无法自主生存——→全部死于饥饿（`DeathCause::Starvation`）。隔离聚落灭亡。这是涌现的悲剧。在宏观层面，世界生成 P9 的人口动力学可能在其他区域补充人口——但在微观层面，这个聚落结束了。

### 1.9 人口金字塔极端形态

**情况**：由于历史模拟中异常事件连锁反应——某个聚落 80% 的人口是 Elder。

**处理**：这不是需要"修复"的 bug——这是涌现的世界状态。这个聚落将在接下来几年内经历大规模人口更替——Gompertz 死亡 + 低生育率（几乎没有 YoungAdult 女性）。聚落可能自然消亡，或通过移民复兴。玩家发现这样的"鬼镇"是一种游戏内容。

### 1.10 两个玩家角色都死且无血亲

**情况**：两个角色同时或在短时间内死于无可继承的情况。

**处理**：结算界面。存档保留——下次读取时弹出大日志候选人选择。玩家永远可以继续——只是可能要从另一个 NPC 的角度重新开始。

```rust
    }
```


### 2.6 不开放 Mod 的领域

以下核心特性**不**通过 trait 开放替换——以确保涌现哲学不被破坏：

| 概念 | 不开放的理由 |
|------|------------|
| 年龄门控 | 零年龄门控是核心涌现原则——允许 mod 添加门控会破坏此原则 |
| 系统开关（"X 岁解锁 Y"） | 同上——零系统开关是设计基础 |
| 生育欲望专用通道 | 生育欲望走通用认知管线——允许 mod 添加专用通道会破坏平等性原则 |
| 中性事件原则 | 事件不可携带情感建议——允许 mod 添加情感预设会破坏涌现性 |

**如果 mod 想要这些效果**——他们可以通过正常路径实现：不是关闭"系统开关"，而是通过修改 NPC 的内部参数（学习率/认知能力/物理属性）来自然产生同样的结果。如果 mod 想要"X 岁解锁技能 Y"——他们可以通过在 X 岁突然提高学习率来实现类似效果，但这仍然是通过参数空间影响——不是通过门控。
