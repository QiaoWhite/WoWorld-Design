# NPC活人感系统——开发者文档（压缩版）

**版本**: 1.01short | **原版**: 1.01 | **引擎**: Godot 4.6.2 | **模式**: 单机优先, 联机搁置

> 📘 本文件为原文档的压缩版，保留所有数据结构、关键参数、阈值为Claude Code开发参考。完整阐述见原版。

---

## 核心设计原则

**只创造法则，不创造故事。** 开发者定义人格参数、行动可能性、情绪规则、文化传播率、概率权重因子；具体故事由NPC局部互动以概率和统计方式自然涌现。

---

## 总体架构

### 五层模型（由内向外）

1. **数据本体** — NPCData Resource，全局数据库中的唯一真相来源
2. **心智核心** — 情绪引擎、记忆系统、行为决策器(GOAP/概率/LLM)、社会认知、文化常识
3. **感官外壳** — 视觉/听觉/情绪感知，受物理限制，可能产生错觉
4. **物理躯体** — CharacterBody3D，动画/移动/痕迹生成/非语言表达
5. **社会投影** — 其他NPC记忆中的"他"、共识场中的"他"

### 三大决策环

1. **本地概率决策** — 基于文化习俗统计+个人习惯+情绪人格加权采样, 覆盖~90%日常行为
2. **GOAP安全网** — 饥饿/致命威胁/重大交易/仪式时，确定性规划确保合理
3. **LLM增强(可选)** — 复杂社交场景调用LLM，仅从预定义行动库选取，带安全网关

## 第1章: NPCData数据结构

### 设计原则
- 纯数据无逻辑、可序列化、全局唯一ID、字典化存储(消除多线程引用计数崩溃)
- 版本号[MONOR, MAJOR]支撑小版本自愈与大版本熔断

### 一级结构
```
NPCData (Dictionary)
├── identity          # 身份标识
├── personality       # 大五人格
├── physiology        # 生理状态
├── emotion           # 情绪状态
├── memory            # 记忆库 { events: [], spatial: [] }
├── relationships     # { npc_id: { affection, trust, familiarity, status } }
├── skills            # { skill_id: { level, experience, talent_bonus } }
├── combat            # 预留(战斗属性)
├── knowledge         # 已知地点/派系/传闻/信念
├── long_term_planning # { primary_goal, sub_goals, suspended_plan }
├── social_identity   # { self_labels, perceived_status, group_roles, cultural_identity }
├── factions          # [faction_id]
├── consensus_cache   # { faction_id: { entries, last_updated } }
├── behavior_stats    # { situation_id: { action_id: count } }
├── current_state     # { simulation_level, embodiment_state, current_goal, ... }
└── metadata          # { created_day, data_version: [MAJOR,MINOR], custom_tags }
```

### 身份标识
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 全局唯一, 如 "npc_7a3f_001" |
| name | String | 显示名称 |
| race | String | 种族ID |
| gender | String | "male"/"female"/"other" |
| age | float | 当前年龄(岁) |
| birth_day | int | 出生游戏天数 |
| is_dead | bool | 是否已死亡 |
| death_day | int | 死亡日期 |

### 大五人格 (0-1)
- **开放性(Openness)**: 高→好奇/尝试新事物; 低→保守/偏好熟悉
- **尽责性(Conscientiousness)**: 高→自律/守时/计划性强; 低→随性/易分心
- **外向性(Extraversion)**: 高→主动社交/群体愉悦; 低→回避大型社交
- **宜人性(Agreeableness)**: 高→信任/合作/易感染; 低→竞争/怀疑/交易激进
- **神经质(Neuroticism)**: 高→负面事件敏感/恢复慢; 低→情绪稳定/恢复快

人格在NPC生命周期极少变动，仅在极高冲击力创伤或长期环境冲突时微调。

### 生理状态 (0-1)
| 指标 | 安全区 | 警告区 | 危险区(GOAP强制) |
|------|--------|--------|------------------|
| hunger | <0.3 | 0.3-0.6(愉悦↓) | >0.85 → 进食 |
| thirst | <0.3 | 0.3-0.6 | >0.9 → 饮水 |
| fatigue | <0.5 | 0.5-0.75(觉醒↓) | >0.9 → 睡眠 |
| health | >0.5 | 0.3-0.5(掌控↓) | <0.3 → 治疗 |

体力区别于疲劳度(瞬时可用能量)，法力预留。

### 情绪状态
- 三维底层轴: `pleasure(-1~1)`, `arousal(0~1)`, `control(-1~1)`
- 8种基本情绪强度(0-1): joy, trust, fear, surprise, sadness, disgust, anger, anticipation
- 当前复合情绪标签: `active_label` + `label_intensity`
- 无聊累积值: `boredom_accumulated`

### 记忆库
**EventMemory**: id, timestamp, game_day, location_id, summary, participants[], participant_roles{}, emotional_encoding{ pleasure, arousal, control, dominant_emotion, dominant_intensity, composite_label }, impact_score(0-1), is_public, access_count, decay_factor, tags[]  
**SpatialMemory**: id, path_type(五级道路), waypoints[], landmarks[], confidence, source, use_count, is_validated

### NPCData大小估算: 10000 NPC × ~12KB = ~120MB

### 三层模拟等级
| 维度 | L1 全模拟 | L2 轻量 | L3 统计 |
|------|----------|---------|---------|
| 数量上限 | 20-50 | 200-500 | 10000+ |
| 距离玩家 | ≤50m | 50-150m | >150m |
| 场景节点 | 完整3D | 简化3D | 无 |

| 记忆系统 | 完整读写 | 只读 | 无 |

| 社会认知 | 完整B4+C5 | 休眠 | 无 |
| 更新频率 | 0.2s | 2-3次/s | 每日/周 |

### 具身化生命周期 (体素世界耦合)
```
GHOST(纯数据) → ANCHORED(分配坐标,等待地形) → MANIFESTING(节点实例化但无碰撞,等待碰撞烘焙) → EMBODIED(完全物理实体)
```
- 降级逆序: EMBODIED→MANIFESTING→ANCHORED→GHOST
- 关键信号: block_terrain_ready(体素网格就绪) → block_collision_baked(物理碰撞就绪) → embodiment_ready
- 安全射线检测机制防止NPC坠落

### 升降级机制
- 滞后区间: 升级50m / 降级55m + 3秒最小维持时间
- 紧急升级: 战斗触发 / 高冲击力事件目击 / 玩家主动交互
- 关键角色override: critical(永久L1) / high(L2+) / normal

## 第3章: 情绪引擎

### 普拉奇克情绪轮
| 基本情绪 | 进化功能 | 对立情绪 | 低强度 | 中强度 | 高强度 |
|----------|---------|---------|--------|--------|--------|
| Joy | 趋近有益刺激 | Sadness | Serenity | Joy | Ecstasy |
| Trust | 建立社会联结 | Disgust | Acceptance | Trust | Admiration |
| Fear | 逃避威胁 | Anger | Apprehension | Fear | Terror |
| Surprise | 重新定向注意 | Anticipation | Distraction | Surprise | Amazement |
| Sadness | 保存能量 | Joy | Pensiveness | Sadness | Grief |
| Disgust | 排斥有害物质 | Trust | Boredom | Disgust | Loathing |
| Anger | 移除障碍 | Fear | Annoyance | Anger | Rage |
| Anticipation | 计划与预期 | Surprise | Interest | Anticipation | Vigilance |

### 三维轴值与基本情绪映射
| 情绪 | Pleasure | Arousal | Control |
|------|----------|---------|---------|
| 狂喜 | 0.8-1.0 | 0.6-1.0 | 0.4-0.8 |
| 愤怒 | -1.0~-0.3 | 0.6-1.0 | 0.3-0.8 |
| 恐惧 | -1.0~-0.3 | 0.6-1.0 | -1.0~-0.3 |
| 悲伤 | -1.0~-0.2 | 0-0.3 | -1.0~-0.2 |
| 期待 | 0.2-0.7 | 0.3-0.6 | 0.2-0.6 |
| 信任 | 0.3-0.8 | 0-0.3 | 0.5-1.0 |

### 30种复合情绪标签 (速查)
**第一层(相邻组合)**: 喜爱、屈从、敬畏、失望、懊悔、蔑视、攻击性、乐观  
**第二层(隔位组合)**: 内疚、好奇、绝望、震惊、怨恨、挖苦、胜利狂喜、希望  
**第三层(三重复合)**: 敌意、焦虑、羞耻、兴奋、自罪感、嫉妒、自豪、同情、尴尬、怀旧、无聊、困惑、决心、宽慰

### 情绪动态机制
- **惯性**: 情绪非瞬间跳变，变化速率由觉醒度决定(高觉醒→变化快)
- **激变**: 冲击力>0.85时惯性覆盖，情绪瞬间跳变+强制GOAP重规划
- **性格稳态点**: 情绪总是回归由人格定义的基线(神经质→愉悦偏移;外向性→觉醒偏移;尽责性→掌控偏移)
- **生理拉扯**: 饥饿→愉悦↓; 疲劳→觉醒↓; 伤病→掌控↓

### 情绪感染 8重调节因子
1. 表达强度(基础)
2. 关系亲密度(配偶0.9-1.0 / 陌生人0.05-0.15 / 敌对-0.1~0.1)
3. 社会地位梯度(高位→低位×1.5 / 低位→高位×0.5)
4. 群体相似性(同族×1.3 / 同派系×1.2 / 异族×0.3)
5. 观察者人格(高宜人性×1.4 / 高神经质×1.3仅负面情绪)
6. 先前情绪(消沉→降阈调节积极情绪 / 攻击性→反向调节敌人情绪)
7. 动机权重(利益相关×1.3)
8. 文化调节(表达主义×1.2 / 抑制主义×0.7)

### 集体情绪
- **情绪级联**: 高地位→亲信→群众逐层扩散
- **情绪共振**: 同质群体相互感染正反馈
- **情绪极化**: 群体情绪强度超个体+方向一致化+社会身份激活
- **情绪免疫**: 低宜人性/对立身份/认知中介导致的冷静少数

## 第4章: 记忆系统

### 编码时机
- **必然编码**: 社交互动、目标完成/失败、地点发现、技能突破、关系里程碑、战斗参与
- **阈值编码**: 冲击力>0.1的日常观察/情绪波动/听闻消息
- **不编码**: 连续无变化状态、极低冲击力重复行为

### 冲击力计算
冲击力 = 影响范围(0-0.3) + 后果深度(0-0.3) + 道德权重(0-0.2) + 反转程度(0-0.2)

### 四种记忆重构过滤器
| 过滤器 | 触发条件 | 重构方式 |
|--------|---------|---------|
| 自我辩护 | 掌控<0 + 自身角色负面 + 尽责性>0.4 | 逃跑→"战术撤退"; 攻击→"正当防卫"; 背叛→"别无选择" |
| 效价扭曲 | 始终启用(强度随当前情绪) | 情绪好→正面细节夸大; 情绪差→负面细节放大 |
| 掌控感扭曲 | 当前掌控<-0.3或>0.5 | 低掌控+成功→"全靠我"; 低掌控+失败→"谁都没办法" |
| 情绪一致性 | 检索时自动生效 | 当前情绪与记忆情绪越近，越优先检索 |

重构不可逆：深度重构(自我辩护)会永久改写原始summary。

### 记忆容量与衰减
- 容量上限: 200条/人
- 衰减因子: `e^(-λ × 距今时间)`，λ受冲击力、觉醒度、回忆频率修正
- >0.5正常 / 0.2-0.5模糊 / <0.2遗忘边缘 / <0.05→压缩为摘要后移出活跃列表

### 社交传播衰减
- 每次传播冲击力衰减30%(传播系数0.7)
- 信任度修正: 高信任→衰减小(0.85); 低信任→衰减小(0.5)
- 传闻路线: 传播衰减系数0.5, 航点逐次丢失(每次20%)

## 第5章: 行为决策

### 本地概率决策 (日常行为~90%)
```
最终权重 = 文化习俗权重 × 个人习惯权重 × 人格偏移 × 情绪修正
```
- 加权随机采样, 选出后坚持执行直到完成或中断
- 文化习俗权重: 缓存于团体数据, 每游戏月更新
- 个人习惯权重: 行为完成后累积更新
- 情绪修正系数: 见3.5节修正表(0.1x-2.0x)

### GOAP安全网触发条件
| 条件 | 阈值 | 强制目标 |
|------|------|---------|
| 极饿 | hunger>0.85 | 进食 |
| 极渴 | thirst>0.9 | 饮水 |
| 极疲 | fatigue>0.9 | 睡眠 |
| 重伤 | health<0.3 | 治疗 |
| 威胁 | 检测到战斗信号/致命生物 | 逃离/自卫 |
| 高后果 | 重大财产/法律/仪式 | 规划执行 |

### 情绪→行动权重修正速查
| 行动类别 | Joy | Trust | Fear | Sadness | Anger | Anticipation |
|----------|-----|-------|------|---------|-------|-------------|
| 友好社交 | 1.4 | 1.6 | 0.3 | 0.4 | 0.2 | 1.1 |
| 物理攻击 | 0.1 | 0.1 | 0.2 | 0.4 | 1.9 | 0.5 |
| 逃避 | 0.1 | 0.3 | 2.0 | 0.8 | 0.2 | 0.4 |
| 探索 | 1.2 | 0.8 | 0.2 | 0.4 | 0.6 | 1.8 |
| 寻求慰藉 | 0.7 | 1.3 | 1.2 | 1.7 | 0.4 | 0.7 |
| 独处 | 0.5 | 0.7 | 1.3 | 1.8 | 0.8 | 0.5 |

### GOAP目标优先级情绪修正
| 目标类别 | Joy | Trust | Fear | Sadness | Anger |
|----------|-----|-------|------|---------|-------|
| 生存类 | 0 | 0 | 0 | 0 | 0 |
| 安全类 | -1 | -1 | +3 | 0 | -1 |
| 社交-亲和 | +2 | +2 | -2 | -2 | -2 |
| 社交-对抗 | -2 | -2 | -2 | -1 | +3 |

- 高神经质+低宜人性 → reckless

- 高宜人性+高外向性 → supportive

## 第19章: 性能要点

- 10000 NPC内存: ~150-200MB
- 存档序列化: 100-500ms(后台线程,不阻塞)
- 单NPC概率决策: <0.1ms(查表+采样)
- GOAP规划: 异步分帧, 每次规划<5ms
- 情绪感染: 仅L1 NPC的视野内计算(O(n×k), k<10)
- 导航: 认知优先+局部烘焙+体素A* 三层组合
