## 文档编写约定

- 版本号格式：`verX.Y`（MAJOR=重大重构, MINOR=增量修订）
- 状态标注（如 001-xxx.md 中）：概念草稿→开发规格→已验证
- 使用 `[[wikilink]]` 交叉引用
- Rust 伪代码为 LLM 辅助开发提供数据合同和接口边界
- 和 `开发阶段/技术栈方案/` 保持一致——技术决策以技术栈方案 v3.0 为准
