> **开发代号**: WoWorld (Wonder World)
> **引擎**: Godot 4.7
> **模块**: UI/UX 系统
> **上一级**: [[README|UI/UX 系统]]

## 设计哲学

WoWorld 的 UI 设计遵循一条核心原则：**界面是世界的窗，不是画在世界上的画**。这意味着：

1. **UI 不创造信息**——它只呈现模拟核心已经存在的数据
2. **UI 不替代感知**——玩家通过感官系统获取信息的规则和 NPC 一致
3. **UI 可消失**——关闭所有 UI 后，玩家仍可通过世界中的线索（NPC 对话、告示牌、书籍、环境声音）正常游玩

### 核心原则

| 原则 | 说明 |
|------|------|
| **沉浸优先** | 默认最小 UI。信息浮现，非驻留 |
| **玩家自主** | 玩家决定哪些信息可见、在哪里、多大 |
| **信息诚实** | UI 不显示 NPC 不知道的信息（除非玩家通过其他途径获知） |
| **设备平等** | 手柄和键鼠有完全相同的操作能力 |
| **可访问** | 色盲模式、字体缩放、语音输入——不因身体差异而无法游玩 |

## 信息层级

UI 信息按四个层级组织。层级越高，对沉浸的侵入越大。

### L0 — 常驻层（无 UI 感）

| 元素 | 说明 |
|------|------|
| **准心** | 屏幕中心十字准心（2px 白 + 1px 黑边）。交互提示时高亮 |
| **交互高亮** | 指向 NPC/物品时准心变色或出现图标：💬 对话 / 🖐️ 拾取 / ⚔️ 攻击 |

### L1 — 条件浮现层（短暂出现后消失）

| 元素 | 触发条件 | 消失条件 |
|------|---------|---------|
| **HP 条** | 受伤/治疗时浮现 | 3 秒无变化淡出 |
| **MP 条** | 施法/魔力变化时浮现 | 3 秒无变化淡出 |
| **Stamina 条** | 冲刺/跳跃消耗时浮现 | 3 秒无变化淡出 |
| **对话气泡** | NPC 说话时 | 2-5 秒（按文本长度） |
| **日志提示** | 新发现/任务更新 | 5 秒后淡出 |

### L2 — 按需展开层（玩家主动打开）

| 面板 | 快捷键 | 暂停游戏？ |
|------|--------|-----------|
| 物品栏 | Tab / I | 否 |
| 技能面板 | K | 否 |
| 地图 | M | 否 |
| 日志 | J / L | 否 |
| 角色信息 | C | 否 |
| 设置 | Esc | **是**（暂停模拟） |
| 蓝图编辑器 | B | 否 |

### L3 — 暂停层

仅设置菜单和主菜单属于此层。暂停整个模拟。

## 双输入设计

### 键鼠

| 操作 | 默认键位 |
|------|---------|
| 移动 | WASD |
| 环顾 | 鼠标移动 |
| 交互（对话/拾取） | E |
| 跳跃 | Space |
| 冲刺 | Shift |
| 攻击 | 鼠标左键 |
| 格挡/副手 | 鼠标右键 |
| 快捷栏选择 | 1-0 数字键 |
| 对话选项 | 1-4 数字键 |
| 按住说话 | V（按住） |
| 打开物品栏 | Tab / I |
| 打开地图 | M |
| 打开日志 | J |
| 暂停/设置 | Esc |

### 手柄

| 操作 | 默认键位 |
|------|---------|
| 移动 | 左摇杆 |
| 环顾 | 右摇杆 |
| 交互 | A (南键) |
| 跳跃 | B (东键) / 左摇杆按下 |
| 冲刺 | 左摇杆按下(按住) |
| 攻击 | RT (右扳机) |
| 格挡/副手 | LT (左扳机) |
| 快捷栏选择 | D-Pad 左右 + A 确认 |
| 对话选项 | D-Pad 上下 / 右摇杆方向 + A |
| 按住说话 | 方向键上(按住) |

| 地图 | Y (北键) |
| 暂停 | Menu 键(Start) |

## 玩家 UI 自定义

玩家可在设置面板中调整：

| 选项 | 默认 | 范围 |
|------|------|------|
| 状态条可见性 | 条件浮现 | 条件/常驻/隐藏 |
| 准心样式 | 十字 | 十字/点/圆圈/无 |
| 准心透明度 | 80% | 20%-100% |
| 小地图可见性 | 条件（按键显示） | 条件/常驻/隐藏 |
| 小地图位置 | 右上 | 左上/右上/左下/右下 |
| 对话气泡透明度 | 70% | 30%-100% |
| 字体缩放 | 100% | 80%-200% |
| 色盲模式 | 关闭 | 红绿色盲/蓝黄色盲/全色盲 |

---

## 可访问性

| 特性 | 说明 |
|------|------|
| **色盲模式** | 三套配色方案。HP 条从红/绿改为橙/蓝 |
| **字体缩放** | 80%-200%，独立于分辨率 |
| **语音输入** | 对话中可选语音转文本（系统 TTS 引擎反方向） |
| **字幕** | 所有 NPC 语音可选显示字幕（不仅是对话——环境语音也可选字幕） |
| **输入重映射** | 所有键位/手柄按钮完全可重映射 |
| **高对比度模式** | UI 元素增加 2px 黑边，提高可读性 |

## 信息诚实约定

> UI 不会告诉你"这个 NPC 对你的好感度是 73/100"。
> 你要从她的表情、语气、是否避开你、是否主动打招呼中自己判断。

| 显示 | 不显示 |
|------|--------|
| 自己的 HP/MP/Stamina | NPC 的 HP/MP（除非战斗评估技能） |
| 探索过的地图 | 未探索区域的内容 |
| 自己物品栏的内容 | NPC 物品栏（除非交易中对方展示） |
| 任务类对话气泡（"去找铁匠"） | NPC 内心独白 |
| 对话中标注声量/善意 | NPC 隐藏动机 |
