## 准心

### 默认十字准心
- 屏幕正中，2px 白线 + 1px 黑边，12px 直径
- 不随移动/转头偏移（始终中心）
- 透明度 80%

### 状态变化

| 状态 | 准心变化 |
|------|---------|
| 空闲 | 白色十字，80% 透明 |
| 指向 NPC（可对话） | 白色 + 💬 图标浮现 |
| 指向物品（可拾取） | 白色 + 🖐️ 图标浮现 |
| 指向敌人（可攻击） | 红色 + 轻微扩大 |
| 攻击命中 | 短暂红色闪烁（0.1s） |
| 施法中 | 蓝色光晕脉冲 |

## 状态条

### HP（生命值）— 红色
- 条件浮现：受伤时 0.3s 内从 0% 透明度 → 80%
- 3 秒无变化后 0.5s 淡出
- 显示：细条（屏幕左上角，200px × 8px）+ 数字（如 "85/100"）
- 低血量警告（< 25%）：红色脉冲 + 模糊边缘效果

### MP（魔力值）— 蓝色
- 条件浮现：同 HP 逻辑
- 显示：细条（HP 下方，200px × 6px）+ 数字

### Stamina（体力值）— 绿色
- 条件浮现：冲刺/跳跃消耗时立即浮现
- 显示：细条（HP 上方，200px × 4px）
- 耗尽时：黄色闪烁 + 角色喘气音效

### 玩家自定义
- 各状态条可独立设置：条件浮现 / 常驻 / 隐藏
- 常驻模式下透明度可调 20%-100%

## 小地图

### 外观
- 右上角圆形，直径 150px
- 半透明底色（黑 60%）
- 玩家位置：白色三角箭头（指向镜头方向）
- 已探索区域：地形颜色（草地绿/水面蓝/雪地白）
- 未探索区域：黑色（全黑，非半透明——不暗示任何信息）

### 行为
- 缩放：鼠标滚轮（50m ~ 5km 范围）
- 旋转模式：玩家可切换"北朝上"或"镜头朝上"
- 按 M 打开全屏大地图面板

### 标记
- 玩家可在地图上放置标记点（右键 → 选择图标 → 输入标签）
- NPC 可卖给玩家地图信息（标记自动出现）
- 已发现的聚落/地标会永久标记

### 知识迷雾
- 探索过的区域**永久可见**（不衰减）
- 未探索区域全黑——不透露地形/建筑/生物
- NPC 出售的地图信息将对应区域解除迷雾

## 快捷栏

### 外观
- 屏幕底部居中，10 个槽位（对应数字键 1-0）
- 每个槽位：48×48px 图标，物品/技能缩略图
- 当前选中槽位：白色边框高亮
- 冷却中的槽位：灰色叠加 + 扇形倒计时动画

### 内容
- 物品快捷（武器/药水/食物）
- 技能快捷（法术/战技）
- 空槽位显示 "+" 半透明提示

### 切换
- 数字键 1-0 直接选择
- 手柄 D-Pad 左右切换 + A 确认
- 鼠标滚轮循环切换
- 按住 Tab 展开 10 槽位的放大视图 + 物品栏快速拖放

## Bark 气泡

### 外观
- NPC 头顶上方浮现，白色半透明气泡（黑 80% 底 + 白字）
- 最大宽度 300px，自动换行
- 最大同时显示 5 个——超出时最旧的先淡出

### 内容类型

| 类型 | 示例 | 颜色 |
|------|------|------|
| 普通对话 | "今天天气不错" | 白字 |
| 任务相关 | "你能帮我去找铁匠吗？" | 金色 |
| 情绪表达 | "哈哈！" 😂 | 黄字 |
| 伤害反应 | "啊！" | 红字 |
| 环境评论 | "好冷…" | 蓝灰字 |

### 行为
- 距离衰减：50m 外不显示
- NPC 在玩家背后时气泡出现在屏幕边缘（指向 NPC 方向）
- 点击气泡 = 将注意力转向该 NPC

## 日志提示

### 外观
- 屏幕右侧，从右边缘滑入
- 180px × 80px，黑底 80%，白字
- 5 秒后淡出
- 最多同时 3 条——新的从底部推入

### 触发条件
- 新发现（探索到新地点、发现新物种、学到新配方）
- 任务状态更新（"找到铁匠" → 已完成）
- 技能升级（"锻造提升至 Lv.5"）
- 名声变化（"你在橡木镇的声誉变为'受尊敬'"）
