## 设计核心

玩家裁决（hand/011）确定了三条对话路径，均遵循**无强制聚焦、无特写、可打断、可自由移动**原则：

1. **指向对话** — 准心指向 NPC + E 键 → 选项浮现
2. **按住说话** — V 键按住语音 → 文本转译 → 广播
3. **托管对话** — 把人身自由托管给角色时，角色按自己意愿发起对话

---

## 路径 1: 指向对话

### 发起
1. 准心指向 NPC，准心变为 💬
2. 按 E 键（此按键行为不被 NPC 察觉——NPC 不会因为你按了键就有反应）
3. 游戏根据玩家角色的人格、记忆、与 NPC 的历史关系、当前场合 生成**高优先级备选对话列表**

### 选项呈现（可配置位置）

默认位置：**屏幕下方**，水平排列 4 个选项条。

玩家可在设置中改为：
- NPC 头顶上方浮现
- 屏幕左侧/右侧垂直排列
- 屏幕底部水平排列（默认）

### 每个选项标注

| 信息 | 显示方式 |
|------|---------|
| 说话内容 | 白色文字 |
| 声量大小 | 🔊 大喊 / 🔉 正常 / 🔇 耳语 |
| 善意/恶意 | 💚 友善 / ⚪ 中性 / 💀 威胁 |
| 所需物品 | 如果选项涉及物品(如"送礼物")，显示物品图标 |
| 技能关联 | 如果选项涉及说服/欺骗，标注关联技能和难度 |

### 选择方式
- 键盘 1-4 数字键
- 鼠标滚轮高亮 + 左键确认
- 手柄 D-Pad 上下 + A 确认

### 特殊选项

| 特殊选项 | 触发键 | 说明 |
|---------|--------|------|
| **自由输入** | T | 弹出文本输入框。玩家键入/语音输入任意内容 → 游戏将其匹配为可理解的交流内容。支持自然语言模糊匹配 |
| **日志引用** | L | 打开大日志浏览器，玩家选择其中一条记录作为对话论据（"你还记得三年前的今天吗？"） |
| **托管这句** | G | 不手动选择——角色按自己的性格自动生成一句最合理的回应 |

### 打断
- 玩家**任何移动**超过 2m，对话自动终止（无过渡动画）
- 玩家**受到攻击**时对话立即中断
- 第三方 NPC **插话**：如果周围 NPC 的注意焦点在对话双方身上，且性格/关系符合条件，可主动插话（打断玩家的对话选择）

---

## 路径 2: 按住说话

### 操作
1. 按住 V 键（可改键）
2. 同时**滑动鼠标滚轮**调节声量大小（耳语 → 大声喊叫）
3. 松开 V 键 = 说完
4. 游戏将语音转译为 NPC 世界观内可理解的文本
5. 按声音响度广播——范围内的 NPC 各自决定是否回应

### 反馈
- 按住期间屏幕下方显示音量条（滚轮位置）
- 转译完成后在玩家头顶短暂浮现对话气泡（显示说了什么——可供 NPC 回应）

---

## 路径 3: 托管对话

- 玩家在设置中开启"角色自主对话"
- 当准心指向 NPC 且玩家在可对话范围内停留 > 3 秒，角色自动发起对话
- 对话内容由角色的 BigFive + Memory + Relationship + Context 决定
- 玩家可随时按键接管（路径 1 覆盖路径 3）

## 交易界面

### 打开条件
- 对话中选择交易选项
- 直接走向店铺（有 Storefront 的 NPC）

### 布局
- 左侧：玩家物品栏（网格，30 格 + 容器展开）
- 右侧：NPC 物品栏（网格）
- 中间：交换确认区
- 底部：金币输入框 + 确认/取消按钮

### 交互
- 拖拽物品到中间交换区
- 可选加金币（输入数字或滑动条）
- NPC 基于价格认知 + BigFive + 关系 决定接受/拒绝/还价
- 确认后调用 `TradeExecutor`（Rust 侧两阶段提交）

---

## 交互轮盘（可选）

按住 Tab 键：围绕玩家浮现 4-8 个扇形选项：
- 对话 / 交易 / 检查 / 跟随 / 攻击 / 偷窃 / 赠送 / 邀请组队

仅用于快速访问常用交互——**不是取代对话**。
