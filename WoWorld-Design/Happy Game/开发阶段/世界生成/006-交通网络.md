## 二、陆地道路网络

### 2.1 最小生成树（MST）算法（保留）

```
节点 = 所有聚落 + 独立建筑（桥梁/哨站/神龛/陵墓）
边权重 = 欧氏距离 × 地形代价 × 0.7 + 已有道路利用 × 0.3
```

**地形通行代价**：

| 地形 | 代价系数 |
|------|---------|
| 平原 | 1.0 |
| 丘陵 | 1.5 |
| 森林 | 2.0 |
| 沼泽 | 4.0 |
| 山脉 | 10.0 |
| 水域 | ∞（不可通行——除非有桥） |

### 2.2 道路等级

| 等级 | 宽度 | 路面 | 连接对象 |
|------|------|------|---------|
| **主干道（驿道）** | 4-6m | 石板 | 城市↔城市 / 城市↔城堡 |
| **次级道路** | 2-3m | 碎石 | 城市↔小镇 / 小镇↔主干道 |
| **小路** | 1-1.5m | 土路 | 小镇↔村庄 / 村庄↔次级道路 |

### 2.3 桥梁生成

道路跨越水域时自动生成桥梁：
- 桥宽 = 道路宽度
- 材料 = 区域建材（石桥→山区；木桥→森林；砖桥→发达聚落）
- 大型桥梁（跨度>20m）仅在 3 级以上道路上生成

## 三、海上航线网络（新增）

### 3.1 航线节点与边权重

```
节点 = 港口城市 + 可停靠的岛屿聚落
边权重 = f(地理距离, 洋流助力/阻力, 季风方向, 中途岛屿补给, 暗礁/浅滩危险)
```

```rust
struct SeaRoute {
    from_port: SettlementId,
    to_port: SettlementId,
    /// 航线路径——考虑洋流修正的实际航行路径
    waypoints: Vec<Vec2>,
    /// 航行时间估算（游戏时间——天）
    travel_time_days: f32,
    /// 航线等级
    grade: SeaRouteGrade,
    /// 季节性变化——某些航线仅在特定季节可安全航行
    seasonality: Option<SeasonalRestriction>,
}

enum SeaRouteGrade {
    Coastal,      // 近海航线——沿海岸线，安全，短距
    CrossSea,     // 跨海航线——横跨海域，中等风险和距离
    Oceanic,      // 远洋航线——跨越大洋，高风险，最长距离
}
```

### 3.2 洋流与季风对航线的影响

```rust
fn compute_sailing_time(
    from: Vec2,
    to: Vec2,
    currents: &OceanCurrents,
    season: Season,
) -> f32 {
    let direct_distance = from.distance(to);
    let current_boost = estimate_current_assist(from, to, currents, season);
    let wind_factor = estimate_wind_assist(from, to, season);

    // 顺洋流可大幅缩短时间；逆洋流则绕道反而更快
    let effective_speed = BASE_SAILING_SPEED * (1.0 + current_boost) * wind_factor;

    // 如果逆流太强，尝试绕道路径
    if effective_speed < MIN_VIABLE_SPEED {
        return compute_detour_time(from, to, currents, season);
    }

    direct_distance / effective_speed
}
```

### 3.3 航线分级

| 等级 | 覆盖范围 | 航行时间 | 风险 | 典型船只 |
|------|---------|---------|------|---------|
| **近海航线** | <50km，沿大陆海岸 | 数小时-1天 | 低 | 小帆船、渔船 |
| **跨海航线** | 50-300km，跨海域 | 1-7天 | 中（风暴、海盗） | 大帆船、商船 |
| **远洋航线** | 300km+，跨越大洋 | 数周-数月 | 高（风暴、迷航、补给不足） | 大型远洋船 |

## 四、旅行者 NPC（新增）

在已有道路和航线上动态分布旅行者（v3.0 §8.3）。

```rust
fn traveler_density(road: &Road, region: &RegionState) -> f32 {
    let base = match road.grade {
        RoadGrade::Primary => 0.05,   // 每 km 0.05 人
        RoadGrade::Secondary => 0.02,
        RoadGrade::Tertiary => 0.005,
    };

    let safety_mod = region.lawfulness * 1.5;      // 治安好 → 更多旅行者
    let prosperity_mod = region.economic_prosperity * 1.2; // 繁荣 → 更多商旅

    base * safety_mod * prosperity_mod
}

enum TravelerType {
    Merchant,       // 商队（3-8 NPC + 马车/驮兽）
    Courier,        // 信使（1-2 NPC，快速移动）
    Pilgrim,        // 朝圣者（1-5 NPC）
    Wanderer,       // 流浪者（1 NPC）
    Adventurer,     // 冒险小队（2-6 NPC）
    Bandit,         // 盗贼（由低治安+经济崩溃涌现）
    Refugee,        // 难民（战争/灾害驱动）
    Patrol,         // 巡逻队（治安维持）
}
```
