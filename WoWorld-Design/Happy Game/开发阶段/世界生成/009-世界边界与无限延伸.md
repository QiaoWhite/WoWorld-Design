# 009-世界边界与无限延伸

> **状态**: 开发规格 v1.0
> **关联**: [[001-世界生成总流程]] · [[004-聚落生成]] · `技术栈方案 v3.0` §4.9

---

## 一、核心原则

WoWorld 的世界边界设计是**渐进的**——初步版有明确的界限，最终版拆除它。两者都必须让"世界尽头"在游戏中**自然地、有机地**被感知，而不是生硬的"无法前进"或隐形墙。

---

## 二、初步版（25 万 km²）——自然海洋边界

### 2.1 结构

```
          风暴带（无法航行）
        ┌─────────────────────┐
        │  ┌───────────────┐  │
        │  │   可航行海域    │  │
        │  │  ┌───────┐    │  │
        │  │  │ 大陆  │    │  │
        │  │  │ (陆域)│    │  │
        │  │  │       │    │  │
        │  │  └───────┘    │  │
        │  │   可航行海域    │  │
        │  └───────────────┘  │
        │    深海（危险航行）   │
        └─────────────────────┘
             风暴带——软边界
```

### 2.2 边界层次

| 层 | 距离大陆 | 特征 | 玩家体验 |
|----|---------|------|---------|
| **大陆** | 0km | 陆地/聚落/一切正常 | 正常游戏 |
| **近海** | 0-50km | 可航行，渔场，岛屿 | 正常航海 |
| **远海** | 50-200km | 海浪增大，岛屿稀少 | 大型船可以走但风险增大 |
| **深海危险区** | 200-400km | 巨浪、怪物、迷航概率↑ | 仅远洋级船只可尝试 |
| **风暴带（软边界）** | 400-500km+ | 持续的超级风暴、无补给、导航失效 | 任何进入的船只终将被摧毁/打回 |

### 2.3 软边界的游戏表现

```
玩家航行进入风暴带:
  → 浪高 ×5-8（远超正常暴风雨）
  → 船只结构持续损伤
  → 船员恐慌 + 技能检定失败 → 迷航（方向随机旋转）
  → NPC 船员恐惧值爆表 → 集体拒绝继续前进
  → 最终: 船只被打回边界内 或 被摧毁（幸存者漂回）
```

**这不是硬编码的"你不能过去"**——你可以尝试。但物理上不可能。

### 2.4 世界的文化感知

NPC 对世界边界的感知进入传说系统：

```
"艾尔尼亚的老水手说，在西方的尽头，大海不像是水了—— 
 船在那里不是在航行，而是在坠落。他说他还活着回来，
 是因为他掉头得早。"
```

NPC 的传说/信仰/恐惧中包含对"世界尽头"的各种解释——让边界本身成为世界观的一部分。

## 三、最终版（Minecraft 级）——种子无限延伸

### 3.1 核心机制

```
世界 = 无硬边界
任何位置 → 种子 → Chunk 可以无限生成
```

最终版的"边界"不再是地理边界——而是**内容密度边界**：
- 离起始点越远，内容密度越低（NPC/聚落/生物——自然衰减）
- 极远处的生成仅保留地形+海洋+基础生物——纯荒野
- 不存在"走到世界尽头"的体验——只有"走到荒野尽头"
- 任何位置都可以成为新的文明起点（玩家建造/NPC 迁徙）

### 3.2 与初步版的衔接

初步版的 25 万 km² + 风暴带 → 当实现最终版时：

```
1. 移除风暴带生成逻辑
2. 风暴带之外 = 从种子继续生成（海洋+岛屿+新大陆）
3. 原有大陆的 NPC 的传说中："世界的尽头打开了"
4. 触发世界事件："风暴带消散"——新旧大陆之间航道贯通
```

这个过渡本身可以成为游戏内的**重大历史事件**。

## 四、L4 超远距统计（v3.0 §8.1）

在 25 万 km² 尺度上，绝大多数 NPC 和聚落不在玩家视线内。L4 层提供"远方世界依然在运转"的统计感知。

| | L1 | L2 | L3 | **L4** |
|---|-----|-----|-----|--------|
| 数量 | ≤1,000 | ≤10,000 | ~89,000 | **~数百万** |
| 距玩家 | ≤50m | 50-150m | >150m | **>5km** |
| 更新 | 每 0.2s | 每 1-2s | 每日/周 | **每周/月（统计批量）** |
| 渲染 | 全模型 | 简化模型 | 无 | 无 |

L4 提供：
- **远方城市的人口基数**——"铁炉城有约 12,000 人"不是空话，统计系统确实在追踪
- **经济宏观指标**——远方大陆的小麦丰收/歉收会影响贸易品的全球价格
- **文明存在感**——即使玩家从未去过某个大陆，消息/传说/移民/难民会让它"存在"于世界的叙事中

```rust
struct L4RegionStats {
    total_population: u64,
    dominant_culture: CultureClusterId,
    economic_output: EconomicSummary,
    political_stability: f32,
    disaster_events: Vec<MajorEvent>,
    // 这些数据每周批量更新一次
}
```

## 五、世界边界的数据标记

```rust
struct WorldBoundary {
    /// 边界类型
    boundary_type: BoundaryType,
    /// 陆地硬边界——无（软边界）
    land_boundary: Option<HardBoundary>,
    /// 海洋软边界
    sea_soft_boundary: SoftBoundary,
}

enum BoundaryType {
    /// 初步版——自然海洋+风暴带软边界
    SoftOcean,
    /// 最终版——种子无限延伸
    InfiniteSeed,
}

struct SoftBoundary {
    /// 边界起始距离（距原点）
    start_distance_km: f64,     // ~400km
    /// 边界强度曲线
    intensity_curve: BoundaryCurve,
    /// 风暴带宽度
    storm_zone_width_km: f64,   // ~100km
}
```
