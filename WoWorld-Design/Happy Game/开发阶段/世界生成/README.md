## 定位

世界生成是从一个 `uint64` 种子出发，经过 **15 个阶段**（P0-P13 + P2.25）逐级细化的**确定性编排管线**。

**核心原则**：
- **种子决定一切**。山脉、海洋、聚落、NPC、文化——全部由噪声函数和生成规则产生。
- **体素 = 大地素材**。自建 Transvoxel（Rust 侧）→ Godot ArrayMesh。
- **海洋是内容，不是空地**。海:陆 ≈ 7:3，航海/岛屿/跨洋发现是核心体验。
- **涌现优先**。不写故事——只写规则。世界生成设定涌现的初始条件。
- **结构层全量，细节层渐进**。NPC/权力/经济/历史全量预生成。内饰/植物/动物按 Chunk 渐进。
- **协同生成 > 顺序生成**。NPC、家族、历史事件在同一过程中产生——因为它们互相引用。
- **Bootstrap ≠ 破坏涌现**。初始订单簿、初始权力边是"已运行千年世界的当前快照"。

## 生成管线速览

```
种子(uint64)
  → P0:  全球大陆架构 (海陆掩码7:3·造山带·热点)
  → P1:  海洋系统 (洋流·深度分级·海岸分类)
  → P2:  陆地自然基底 (L0-L6密度场·高程·水文·气候·群系参数场)
  → P2.25: ★ 植被覆盖 (香农熵优势种·多层噪声·L6.5密度场·木材可获得性)
  → P2.5: 文化与信仰种子 (消费植被→建筑风格偏好)
  → P3:  资源分布 (7大类·矿田生成)
  → P3.5: ★ 生命世界初始化 (动物·怪物·亡灵—★植物已移至P2.25, 亡灵后移至P9后)
  → P4:  聚落选址 (承载容量·评分·等级)
  → P5:  ★ 社会结构推导 (BuildingSpec·RoleTemplate·PowerSkeleton·户外场所)
  → P6:  建筑生成 (调用建筑模块)
  → P7:  交通网络 (MST道路+航线)
  → P8:  ★ 人口投影与家族协同 (四阶段·FamilyTree·NpcAnchors)
  → P9:  ★ 知识种子传播+历史事件 (两步走: P9a传播→P8技能天花板; P9b因果+参与者)
  → P10: ★ 权力 Bootstrap (模板→边实例化→Polity涌现)
  → P11: ★ 经济 Bootstrap (供需聚合→订单簿→钱包→EconomicRelation)
  → P12: 世界边界处理
  → P13: ★ 校验与终组 (引用·时间·人口·可玩性·Delta应用·墓碑·ModuleInit)
  → 输出: ✅ WorldState → LMDB
```

## 关键参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| **世界面积** | 25 万 km²（≈英国，约 500×500km） | 初步版 |
| **海陆比** | 7:3（可调） | `sea_threshold=0.70` |
| **高程范围** | ~1500m | 含 700m+ 山峦 |
| **体素单位** | 0.5m³/体素 | 地形精度 |
| **Chunk** | 32m×32m 水平，垂直稀疏 | 仅地表附近存体素 |
| **LOD** | 8 级 Clipmap | 最远 10km+ |
| **NPC 总数** | ~100K | 全量预生成 |
| **聚落总数** | ~800-3000 | 村庄60%·小镇25%·城市10%·城堡5% |
| **历史事件** | ~30K-40K | 0-2000 年 |
| **权力边** | ~500K | 初始预填充 |
| **生成时间** | 5-13s（加载画面） | 典型 ~8s |
| **初始存档** | ~140MB | LMDB 压缩 |
| **运行时内存** | ~700MB（Rust 预算 1.4GB） | 50% 使用率 |
