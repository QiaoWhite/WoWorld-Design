## 定位

大气与氛围系统是 WoWorld 第 26 个独立模块。它将群系、时间、天气、季节的相互作用编译为一组渲染参数（ResolvedAtmosphere），供 Godot 渲染管线消费。

本模块只产出数据——不渲染。所有 GPU 工作由 Godot 的 toon shader、WorldEnvironment 和全局 shader uniform 完成。

---

## 核心职责

- 定义 `BiomeAtmosphere` —— 群系基值（17 参数 × ~12 群系）
- 定义时间/天气/季节调制曲线（TOML 数据驱动）
- 每帧计算 `ResolvedAtmosphere`（四层叠加：biome × time × weather × season）
- 管理 `SkyEvent` 覆盖——叙事性天空事件（龙出现、神明显灵、末日）
- 处理水下深度分层氛围和室内光源派生氛围

## 关键原则

- **只创造视觉规则，不逐个调教场景** —— BiomeAtmosphere 定义规则，100K+ km² 自动受益
- **四层叠加，非替换** —— biome基调 × 时间光温 × 天气条件 × 季节偏移
- **标点即编码** —— 氛围参数是数字，Godot shader 解释数字。Rust 不知道"好看"
- **数据驱动** —— 全部 TOML。可 Mod

## ResolvedAtmosphere 参数速查 (17参数·68 bytes·Copy类型)

| 类别 | 参数 |
|------|------|
| **光色** (4) | 环境光RGB · 直射光RGB · 天顶色 · 地平线色 |
| **雾** (3) | 雾密度 · 雾颜色 · 雾起始距离 |
| **曝光** (2) | 曝光补偿 · 对比度 |
| **阴影** (2) | 阴影强度 · 阴影色偏 |
| **色散** (2) | 色散强度 · 色散角度 |
| **辉光** (2) | Bloom 阈值 · Bloom 强度 |
| **体积** (2) | 体积光密度 · 体积光散射方向 |

## 消费模块

| 模块 | 消费内容 |
|------|---------|
| Godot 渲染 | WorldEnvironment 全局 shader uniform (每帧推送) |
| NPC 活人感 | 情绪基线微调 (阴天→愉悦偏低, 金红色晚霞→愉悦偏高) |
| 音频 | 环境混响参数 (室外/室内/水下/洞穴) |
| 天气系统 | ← 本模块消费 WeatherQuery (双向引用已建立) |

## 性能

| 项目 | 数值 |
|------|------|
| 每帧计算 | <0.01ms (TOML 查表+四层乘加) |
| ResolvedAtmosphere | 68 bytes——栈上 Copy |
| TOML 加载 | ~12群系×17参数×4曲线 ≈ 一次性加载 < 10KB |
| Godot 推送 | 1次/帧 set_shader_uniform (17个 float/vec3) |
