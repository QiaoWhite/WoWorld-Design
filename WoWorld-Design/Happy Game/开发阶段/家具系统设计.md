# 家具系统设计

> **创建 2026-06-02**：详细设计家具的种类、功能、质量等级和文化变体  
> 关联：[[04-房间生成规则]]、[[06-体素设计决策]] §2.3、[[建筑风格]]

---

## 一、概述

家具是房间内部的基本构成元素。每个[[房间]]通过家具的组合获得其功能属性——没有灶台的房间不是厨房，没有床的房间不是卧室。

家具系统需要平衡**丰富度**（让不同文化、不同财富等级的房间有差异化）和**开发可行性**（独立开发者能完成的资产量）。

---

## 二、家具分类

### 2.1 按功能分类

#### 核心功能家具（定义房间用途）

这类家具**定义了房间的存在理由**。一个房间至少有一件核心功能家具。

| 家具 | 对应的房间功能 | 变体数量（最低） | 交互性 |
|------|--------------|----------------|--------|
| 床 | 卧室 | 5（单人/双人/大床 × 不同材料） | 可睡眠、可坐 |
| 灶台/炉灶 | 厨房 | 4（明火灶/石灶/砖炉/魔法炉） | 可烹饪 |
| 工作台/铁砧 | 工坊（铁匠） | 3 | 可锻造 |
| 织机 | 工坊（裁缝） | 2 | 可纺织 |
| 实验台 | 工坊（炼金） | 2 | 可炼药 |
| 附魔台 | 工坊（附魔） | 2 | 可附魔 |
| 神像/祭坛 | 神殿 | 按神祇类型（每个神至少1种变体） | 可祈祷 |
| 书桌 | 书房 | 3 | 可阅读/写作 |
| 柜台/货架 | 商铺 | 4 | 可交易 |
| 餐桌（大） | 宴会厅/食堂 | 3 | 可聚餐 |

#### 辅助功能家具（支撑房间功能）

| 家具 | 常见位置 | 功能 |
|------|---------|------|
| 椅子/凳子 | 全房间 | 可坐 |
| 桌子（小/中） | 客厅、卧室、书房 | 摆放物品、用餐 |
| 衣柜/衣架 | 卧室 | 存储服装 |
| 箱子/桶 | 全房间 | 存储物品（容器） |
| 书架 | 书房、客厅 | 存储书籍、装饰 |
| 水缸/水槽 | 厨房 | 储水（距水源远时必需） |
| 案板/备菜台 | 厨房 | 食材准备 |
| 调料架/餐具柜 | 厨房 | 存储调料和餐具 |
| 工具架/工具墙 | 工坊 | 存储和展示工具 |
| 淬火槽 | 铁匠铺 | 金属淬火 |
| 坩埚/蒸馏器 | 炼金工坊 | 炼金设备 |
| 跪垫/长椅 | 神殿 | 信徒使用的座位 |
| 烛台/灯架 | 全房间 | 照明 |
| 镜子 | 卧室、客厅 | 装饰、NPC 外观检查 |

#### 装饰性家具（非必需，纯粹美观或显示地位）

| 家具 | 说明 |
|------|------|
| 花瓶/花盆 | 装饰，可放置采摘的花 |
| 挂画/壁毯 | 墙面装饰，可描绘风景/人物/宗教场景 |
| 地毯/兽皮毯 | 地面装饰，也提供温暖 |
| 雕塑/小雕像 | 桌面或地面装饰 |
| 帘幔/屏风 | 空间分隔、隐私 |
| 战利品展示（武器架/头骨挂） | 战士/猎人的家中展示 |
| 乐器（竖琴/鲁特琴/鼓） | 可弹奏（基础音效） |
| 棋盘/游戏桌 | NPC 可进行休闲活动 |

### 2.2 按房间功能分类（家具套装）

每个房间功能类型对应一套**推荐家具清单**：

| 房间 | 核心家具 | 必备辅助 | 常见装饰 | 槽位估算 |
|------|---------|---------|---------|---------|
| **卧室（普通）** | 床 | 衣柜、椅子 | 烛台、镜子、地毯 | 4-8 槽位 |
| **卧室（豪华）** | 大床 | 衣柜×2、梳妆台、椅子×2 | 烛台×2、镜子、地毯、挂画、花瓶 | 10-18 槽位 |
| **厨房（普通）** | 灶台 | 水缸、案板、储物桶 | 烛台、调料架 | 5-8 槽位 |
| **厨房（豪华）** | 砖炉/魔法炉 | 水槽、备菜台、储物柜×2 | 烛台×2、调料架、餐具柜 | 10-16 槽位 |
| **客厅** | 餐桌+椅子 | 储物柜 | 烛台/灯架、地毯、挂画、花瓶 | 6-12 槽位 |
| **铁匠铺** | 锻炉+铁砧 | 淬火槽、工具架、煤炭堆 | 武器展示架 | 6-10 槽位 |
| **书房** | 书桌+椅子 | 书架×2-8 | 烛台、地毯、星图/挂画 | 5-15 槽位 |
| **神殿内室** | 神像/祭坛 | 跪垫/长椅×信徒数 | 烛台×N、香炉、供品台、圣物柜 | 5-20 槽位 |
| **商铺** | 柜台 | 货架×2-4、储物箱 | 烛台、招牌 | 5-12 槽位 |
| **酒馆大厅** | 吧台 | 桌子×3-8 + 椅子×若干、酒桶 | 烛台/灯架×N、壁毯、乐器 | 15-40 槽位 |

## 三、家具质量等级

家具与核心用具一样，分为 5 个质量等级：

| 等级 | 名称 | 材料 | 耐久度 | 使用效果加成 | 所有者财富水平 |
|------|------|------|--------|-------------|--------------|
| **0** | 粗制 | 未加工的自然材料（原木/粗石/茅草） | 100 | 基础功能 | 赤贫 |
| **1** | 普通 | 基础加工材料（木板/普通石材/铁件） | 200 | 基础功能 | 普通 |
| **2** | 良好 | 优质材料（硬木/优质石材/钢件） | 400 | +10% 效率 | 小康 |
| **3** | 精良 | 精选材料 + 工匠级工艺 | 800 | +20% 效率 | 富裕 |
| **4** | 卓越 | 稀有材料 + 大师级工艺 + 魔法强化 | 1600 | +35% 效率 | 富豪/贵族 |

> 注：使用效果加成仅适用于功能性家具（床→睡眠恢复速度、灶台→烹饪成功率和速度、工作台→锻造/制作效率等）。纯装饰家具不受此影响。

## 四、家具的文化变体

### 4.1 同一家具，不同文化的外观

家具的**功能相同**，但**外观（模型/材质/颜色）** 随文化变化：

| 家具 | 北方简约 | 东方精致 | 南方奢华 | 沙漠实用 |
|------|---------|---------|---------|---------|
| **床** | 厚重松木床架、毛皮被褥 | 竹制/红木床架、丝绸被褥 | 雕花大床、天鹅绒被褥 | 石台/棕榈纤维垫 |
| **桌子** | 厚橡木板、铁件加固 | 细木工、无钉榫卯 | 大理石桌面、镀金桌腿 | 矮石台/折叠木桌 |
| **椅子** | 高背木椅、简洁线条 | 曲木椅、座垫 | 软垫扶手椅、雕花 | 坐垫/矮凳 |
| **储物** | 铁皮箱、橡木柜 | 漆木柜、多层屉 | 雕花大衣柜、皮箱 | 陶罐、藤编篮 |
| **照明** | 铁烛台、牛油蜡烛 | 纸灯笼、油灯 | 水晶吊灯、蜜蜡蜡烛 | 陶灯、油灯 |

### 4.2 文化变体的实现策略

- **参数化变体**（推荐）：用一个基础模型 + 材质/纹理/颜色参数来生成文化变体，而不是为每种文化单独建模每种家具
- **模块化装饰**：家具主体不变，装饰部件（雕刻、镶嵌、涂装）按文化切换
- **文化覆盖规则**：`基础家具模型 × 文化材质映射表 × 质量等级附加装饰`

## 五、家具的交互属性

### 5.1 交互类型

| 交互类型 | 示例家具 | 交互效果 |
|----------|---------|---------|
| **使用** | 床、椅子、灶台、工作台 | NPC/玩家使用家具执行特定动作 |
| **存储** | 箱子、桶、衣柜、书架 | 打开容器界面，存取物品 |
| **观察** | 挂画、雕塑、战利品 | 查看详细信息（如有著名度关联） |
| **移动** | 椅子、小桌子、桶 | 玩家可拖拽/推动（NPC 一般不移家具） |
| **破坏** | 全部 | 可被破坏（战斗、灾害），掉落材料 |
| **制作** | 灶台、工作台、织机 | 打开制作界面 |

### 5.2 NPC 与家具的交互

NPC 使用家具的具体行为从其日常决策环中触发：

```
NPC 决策 → 选择行为 → 需要特定家具 → 前往该家具所在位置 → 播放交互动画
```

示例：
- 饥饿 → 决定做饭 → 去厨房用灶台 → 做饭动画
- 疲劳 → 决定睡觉 → 去卧室用床 → 睡觉动画
- 社交需求 → 决定招待客人 → 去客厅餐桌 → 社交动画

### 5.3 家具归属

家具归属于其所在[[建筑]]和[[房间]]的所有者。NPC 通常不会使用他人住宅中的家具（除非被邀请或闯入）。

## 六、家具的获取与制作

### 6.1 家具来源

| 来源 | 说明 |
|------|------|
| **世界生成** | 建筑生成时，根据房间功能自动放置家具 |
| **NPC 制作** | 木匠/铁匠 NPC 可以制作家具并出售 |
| **玩家制作** | 玩家通过[[技能系统]]中的木工/锻造技能制作 |
| **购买** | 从家具商 NPC 处购买 |
| **战利品/发现** | 在遗迹/地牢中发现特殊家具 |

### 6.2 家具制作需求（玩家）

| 家具 | 主要技能 | 次要技能 | 主要材料 |
|------|---------|---------|---------|
| 木制家具（床/桌/椅/柜/架） | 木工 | — | 木材 + 铁件 |
| 石制家具（石台/石像） | 石工 | — | 石材 |
| 金属家具（铁箱/铁砧/武器架） | 锻造 | — | 金属锭 |
| 纺织品（地毯/帘幔/坐垫） | 纺织 | — | 纤维/毛皮 |
| 魔法家具（魔法灯/附魔台） | 附魔 | 木工/锻造 | 对应材料 + 魔法水晶 |

## 七、性能考量

家具的数量级和优化：

| 聚落等级 | 建筑数 | 平均房间数 | 平均每房间家具数 | 总家具数 |
|----------|--------|-----------|----------------|---------|
| 村庄 | 15-40 | 2-4 | 3-8 | 90-1,280 |
| 小镇 | 50-150 | 3-6 | 4-12 | 600-10,800 |
| 城市 | 200-1500 | 4-10 | 5-15 | 4,000-225,000 |

**优化策略**（与房间生成相同）：
- 未进入玩家视野的建筑不实例化家具网格
- L2 距离用统计摘要代替（"此房间有 5 件家具，平均质量 2"）
- L3 距离仅保留房间功能标签，不保存家具数据
- 模板房间的家具使用共享网格实例（GPU Instancing）
