## 一、核心概念

### 1.1 组件族（ComponentFamily）

**组件族**是一个参数化的建筑组件的抽象模板。一个 `ComponentFamily` 定义「这是什么类型的组件」，而 `ComponentParams` 定义「这个具体实例的参数」。

关键设计决策：**组件族使用关联类型（associated type）**，每个族定义自己的 `Params` 类型。这支持 Mod 扩展——新族不需要修改核心 crate 的 enum。

**但关联类型使 trait 不可用作 trait object**（`dyn ComponentFamily` 编译不过）。因此采用**擦除模式**——双层 trait：

```rust
// ━━ 注册层：object-safe 的基 trait —— ComponentRegistry 存这个 ━━
pub trait ComponentFamilyBase: Send + Sync + 'static {
    fn family_id(&self) -> ComponentFamilyId;
    fn family_name(&self) -> &str;
    fn is_available(&self, ctx: &BuildContext) -> bool;
    fn params_schema(&self) -> ParamSchema;

    /// 返回擦除后的参数（不暴露具体类型）
    fn default_params_erased(&self, ctx: &BuildContext) -> Box<dyn ErasedParams>;

    /// 从 WFC tile → ComponentInstance
    /// `floor`: 此组件所在的楼层（0=地面, -1=地下, 1=二楼）
    /// `time`: 建造时间——世界生成时传入世界创建时间，运行时传入 GameTime::now()
    fn instance_from_tile(&self, tile: &WfcTile, ctx: &BuildContext, position: Vec3,
                          floor: i8, time: GameTime) -> ComponentInstance;

    /// 从 Blueprint TOML params → ComponentInstance
    fn instance_from_blueprint(&self, params: &toml::Value, ctx: &BuildContext,
                               position: Vec3, floor: i8, time: GameTime)
        -> Result<ComponentInstance, Vec<ParamError>>;

    /// 此族在给定方向上有无连接面（WFC 兼容性检查用）
    fn face_on_side(&self, tile: &WfcTile, side: Direction, ctx: &BuildContext)
        -> Option<ConnectionFace>;

    /// WFC tile 映射
    fn wfc_tiles(&self, ctx: &BuildContext) -> Vec<WfcTileVariant>;
}

// ━━ 实现层：完整 trait（非 object-safe——具体族 impl 这个） ━━
pub trait ComponentFamily: ComponentFamilyBase {
    type Params: ErasedParams + Clone;

    fn default_params(&self, ctx: &BuildContext) -> Self::Params;

    /// 从 WFC tile + ctx 反向推导参数——每个族必须实现
    fn derive_params_from_tile(&self, tile: &WfcTile, ctx: &BuildContext) -> Self::Params;

    fn connection_faces(&self, params: &Self::Params) -> Vec<ConnectionFace>;
    fn validate_params(&self, params: &Self::Params) -> Result<(), Vec<ParamError>>;
}

// ━━ blanket impl：自动桥接 ━━
impl<T: ComponentFamily> ComponentFamilyBase for T {
    fn default_params_erased(&self, ctx: &BuildContext) -> Box<dyn ErasedParams> {
        Box::new(self.default_params(ctx))
    }
    fn face_on_side(&self, tile: &WfcTile, side: Direction, ctx: &BuildContext)
        -> Option<ConnectionFace> {
        let params = self.derive_params_from_tile(tile, ctx);
        let world_face = tile_face_to_world(tile.face, side);
        self.connection_faces(&params).into_iter()
            .find(|f| f.face == world_face)
    }
    fn instance_from_tile(&self, tile: &WfcTile, ctx: &BuildContext, pos: Vec3,
                          floor: i8, time: GameTime) -> ComponentInstance {
        let params: T::Params = self.derive_params_from_tile(tile, ctx);
        let dim = params.dimensions();
        ComponentInstance {
            id: ComponentInstanceId::new(),
            hot: ComponentHot {
                spatial: Transform::from_translation(pos),
                bounds: dim.to_aabb(pos),
                mesh_key: params.mesh_key(),
                health: 1.0,
            },
            cold: ComponentCold {
                family: self.family_id(),
                placed_at: time,
                placed_by: None,
            },
            params: Box::new(params),
            floor,
            vertical_span: 1,
        }
    }
    fn instance_from_blueprint(&self, bp_params: &toml::Value, ctx: &BuildContext,
                               pos: Vec3, floor: i8, time: GameTime)
        -> Result<ComponentInstance, Vec<ParamError>> {
        let params: T::Params = toml::from_value(bp_params.clone())
            .map_err(|e| vec![ParamError::deserialize(e.to_string())])?;
        self.validate_params(&params)?;
        let dim = params.dimensions();
        Ok(ComponentInstance {
            id: ComponentInstanceId::new(),
            hot: ComponentHot {
                spatial: Transform::from_translation(pos),
                bounds: dim.to_aabb(pos),
                mesh_key: params.mesh_key(),
                health: 1.0,
            },
            cold: ComponentCold {
                family: self.family_id(),
                placed_at: time,
                placed_by: None,
            },
            params: Box::new(params),
            floor,
            vertical_span: 1,
        })
    }
    fn wfc_tiles(&self, ctx: &BuildContext) -> Vec<WfcTileVariant> {
        // 默认从 params 推导——族可以覆盖
        let default = self.default_params(ctx);
        vec![WfcTileVariant {
            kind: WfcTileKind::from_family(self.family_id()),
            material: default.material_classes().first().copied()
                .unwrap_or(MaterialClass::Stone),
            subtype: 0,
            weight: 1.0,
        }]
    }
}

/// 将 WFC tile 的 local face + 邻接方向 → 世界空间 FaceDir
fn tile_face_to_world(tile_face: Option<FaceDir>, side: Direction) -> FaceDir {
    match (tile_face, side) {
        (Some(f), _) => {
            // tile 的外立面朝向 f。若 side 与该面同向 → 返回 f 的反面
            // 若 side 是相邻方向 → 按 tile 旋转偏移
            if side.to_face_dir() == f { f.opposite() } else { side.to_face_dir() }
        }
        (None, _) => side.to_face_dir(),  // 无声明 face → 默认映射
    }
}

// ━━ 擦除后的参数 —— 只有需要具体类型时才 downcast ━━
pub trait ErasedParams: Send + Sync + 'static {
    fn dimensions(&self) -> Dimensions;
    fn material_classes(&self) -> Vec<MaterialClass>;
    fn mesh_key(&self) -> MeshKey;
    fn as_serialize(&self) -> &dyn erased_serde::Serialize;
    fn clone_box(&self) -> Box<dyn ErasedParams>;
}
```

### 1.2 网格系统

**网格定义对齐精度，不限制组件尺寸。**

| 层级 | 网格尺寸 | 用途 |
|------|----------|------|
| **主网格** | 1.0m | 墙、地板、天花板、柱子、门框——结构组件 |
| **子网格** | 0.25m | 薄墙、窗框、栏杆、楼梯台阶——精细结构组件 |
| **自由层** | 无网格 | 家具旋转、装饰品、灯具——非结构装饰 |

组件占位必须是主网格或子网格的整数倍。一个「石墙组件」可以是 1m×3m×0.5m（宽×高×厚），占据 4×12 个子网格。

**为什么参数化族而非离散变体**：核心结构组件（墙、地板、门、窗、柱、梁、楼梯）用参数化族——尺寸变化是连续的（受种族体型、文化规范影响），不可能枚举所有文化×种族组合。装饰/特色组件用离散定义。

### 1.3 连接面（ConnectionFace）

连接面是组件之间的物理连接点。不是六面全必须——每个族只声明实际使用的面。

```rust
pub struct ConnectionFace {
    /// 此面在组件局部坐标中的方位
    pub face: FaceDir,

    /// 面的尺寸
    pub area: (f32, f32),

    /// 连接类型
    pub connection_type: ConnectionType,

    /// 此面在局部坐标中的偏移
    pub offset: Vec3,

    /// ★ CHG-050 — 供 Pass A 胶水代码识别面用途。
    /// None = 纯结构连接面。Some(u16) = 用途标签（如 chimney_opening / builtin_shelf / wall_shrine）
    /// 标签注册由 WorkstationRegistry 统一管理。
    pub usage_tag: Option<u16>,
}

pub enum ConnectionType {
    /// 承重连接——墙接地板、梁接柱。载荷必须通过此面传播。
    Structural { load_bearing: f32 },

    /// 非承重连接——窗嵌入墙、装饰挂墙。不传播载荷。
    Attachment,

    /// 穿越连接——门在墙上、楼梯穿过楼层。
    PassThrough,
}

pub enum FaceDir {
    PosX, NegX,
    PosY, NegY,  // PosY = 顶面, NegY = 底面
    PosZ, NegZ,
}
impl FaceDir {
    pub fn opposite(self) -> Self {
        match self {
            PosX => NegX, NegX => PosX,
            PosY => NegY, NegY => PosY,
            PosZ => NegZ, NegZ => PosZ,
        }
    }
}

/// 空间尺寸——组件族的 dimensions() 返回值
pub struct Dimensions {
    pub width: f32,
    pub height: f32,
    pub depth: f32,
}
impl Dimensions {
    pub fn to_aabb(&self, origin: Vec3) -> Aabb {
        Aabb {
            min: origin,
            max: Vec3::new(origin.x + self.width, origin.y + self.height, origin.z + self.depth),
        }
    }
}

/// 从 BuildingMaterialProps 查询物理属性
/// 启动时由胶水代码从 NPC 物理原子层 MaterialProperties 填充
static MATERIAL_PROPS: OnceLock<HashMap<MaterialClass, BuildingMaterialProps>> = OnceLock::new();

struct MaterialPropsTable;
impl MaterialPropsTable {
    fn get(mat: MaterialClass) -> BuildingMaterialProps {
        let table = MATERIAL_PROPS.get()
            .expect("MaterialPropsTable 必须在启动时由胶水代码初始化");
        table.get(&mat).copied().unwrap_or(BuildingMaterialProps {
            compressive_strength: 5_000_000.0,  // 5 MPa 默认——松木级别
            shear_strength: 500_000.0,          // 0.5 MPa
            density: 500.0,                     // 500 kg/m³
            durability: 0.5,
            nailability: 0.5,
            min_thickness: 0.1,
            max_span: 3.0,
            thermal_insulation: 0.3,
        })
    }
}
```

**兼容性规则**：两个 ConnectionFace 能连接当且仅当：
1. `face_a == opposite(face_b)`（面相对）
2. `connection_type` 匹配（Structural ↔ Structural, Attachment ↔ Attachment, PassThrough 一个提供面一个接受）
3. 面积偏差 ≤ 20%
4. 位置偏差 ≤ 0.25m（子网格 snap 容差）

连接兼容性从 ComponentRegistry 自动推导为兼容矩阵——无需手写。

### 2.1 Wall（墙）——最基础的围护结构

```rust
pub struct WallParams {
    pub width: f32,        // 沿墙面的水平长度
    pub height: f32,       // 垂直高度
    pub thickness: f32,    // 墙面法线方向的厚度
    pub material: MaterialClass,
    pub wall_type: WallType,
    pub style: WallStyle,
}

pub enum WallType {
    Exterior,     // 外墙——一面朝外(Empty)，一面朝内(Floor)
    Interior,     // 内墙——两面都朝室内
    Retaining,    // 挡土墙——嵌入地形
}

pub enum WallStyle {
    PlankHorizontal, PlankVertical,
    BlockRough, BlockSmooth,
    BrickRunning, BrickFlemish,
    AdobeSmooth, WattleAndDaub,
    LogHorizontal,
}
impl WallStyle {
    pub fn from_u8(v: u8) -> Self {
        match v {
            0 => Self::PlankHorizontal, 1 => Self::PlankVertical,
            2 => Self::BlockRough, 3 => Self::BlockSmooth,
            4 => Self::BrickRunning, 5 => Self::BrickFlemish,
            6 => Self::AdobeSmooth, 7 => Self::WattleAndDaub,
            8 => Self::LogHorizontal,
            _ => Self::BlockRough,  // 默认回退
        }
    }
}
```

**连接面（以 ExteriorWall 为例）**：
- 底面（NegY）：Structural → 接地板/地基
- 顶面（PosY）：Structural → 接天花板/上层墙/屋顶梁
- 左面（NegX）：Structural → 接相邻墙或柱
- 右面（PosX）：Structural → 接相邻墙或柱
- 前面（NegZ 或 PosZ，取决于朝向）：无声明连接面（此面是外立面或内立面——可供 Attachment 嵌入窗/门/装饰）

**尺寸派生**（从 BuildContext）：
```rust
fn default_params(ctx: &BuildContext) -> Self::Params {
    WallParams {
        thickness: ctx.climate.wall_thickness(0, 1),  // 单层建筑壁厚
        material: ctx.culture.preferred_structural_material(),
        wall_type: WallType::Exterior,
        style: ctx.culture.wall_style_for(material),
        width: 1.0,   // WFC 后处理时合并相邻 cell 决定实际宽度
        height: ctx.race.ceiling_height(),  // 人类2.5m, 兽人3.5m, 矮人2.1m
    }
}
```

## 三、一族一例——WallFamily 完整实现

```rust
pub struct WallFamily {
    id: ComponentFamilyId,
}

impl ComponentFamily for WallFamily {
    type Params = WallParams;

    fn family_id(&self) -> ComponentFamilyId { self.id }
    fn family_name(&self) -> &str { "Wall" }

    fn default_params(&self, ctx: &BuildContext) -> Self::Params {
        let material = ctx.culture
            .map(|c| c.preferred_structural_material())
            .unwrap_or(MaterialClass::Stone);
        WallParams {
            width: 1.0,
            height: ctx.race.ceiling_height(),
            thickness: ctx.climate.wall_thickness(0, 1),
            material,
            wall_type: WallType::Exterior,
            style: ctx.culture.map(|c| c.wall_style_for(material))
                .unwrap_or(WallStyle::BlockRough),
        }
    }

    fn derive_params_from_tile(&self, tile: &WfcTile, ctx: &BuildContext) -> Self::Params {
        WallParams {
            width: 1.0,
            height: ctx.race.ceiling_height(),
            thickness: ctx.climate.wall_thickness(0, 1),
            material: tile.material,
            wall_type: match tile.family_tag {
                FamilyTag::ExteriorWall => WallType::Exterior,
                FamilyTag::InteriorWall => WallType::Interior,
                _ => WallType::Exterior,
            },
            style: WallStyle::from_u8(tile.subtype),
        }
    }

    fn connection_faces(&self, params: &Self::Params) -> Vec<ConnectionFace> {
        let half_w = params.width / 2.0;
        let half_t = params.thickness / 2.0;
        let h = params.height;
        // ★ 材料物理属性从 BuildingMaterialProps 表查询（见 003 §五），不调用 MaterialClass 方法
        let props = MaterialPropsTable::get(params.material);
        vec![
            ConnectionFace {
                face: FaceDir::NegY,
                area: (params.width, params.thickness),
                connection_type: ConnectionType::Structural {
                    load_bearing: props.compressive_strength * params.thickness * params.width,
                },
                offset: Vec3::new(0.0, 0.0, 0.0),
            },
            ConnectionFace {
                face: FaceDir::PosY,
                area: (params.width, params.thickness),
                connection_type: ConnectionType::Structural {
                    load_bearing: props.compressive_strength * params.thickness * params.width,
                },
                offset: Vec3::new(0.0, h, 0.0),
            },
            ConnectionFace {
                face: FaceDir::NegX,
                area: (params.height, params.thickness),
                connection_type: ConnectionType::Structural {
                    load_bearing: props.shear_strength * params.height * params.thickness,
                },
                offset: Vec3::new(-half_w, 0.0, 0.0),
            },
            ConnectionFace {
                face: FaceDir::PosX,
                area: (params.height, params.thickness),
                connection_type: ConnectionType::Structural {
                    load_bearing: props.shear_strength * params.height * params.thickness,
                },
                offset: Vec3::new(half_w, 0.0, 0.0),
            },
        ]
    }

    fn validate_params(&self, params: &Self::Params) -> Result<(), Vec<ParamError>> {
        let mut errors = Vec::new();
        let props = MaterialPropsTable::get(params.material);
        if params.thickness < props.min_thickness {
            errors.push(ParamError::out_of_range(
                "thickness", params.thickness, props.min_thickness, f32::MAX));
        }
        // max_wall_height 从抗压强度推导：Euler 屈曲公式近似
        let max_h = (props.compressive_strength / (props.density * 9.81 * params.thickness)).sqrt()
            * params.thickness * 2.0;
        if params.height > max_h {
            errors.push(ParamError::out_of_range("height", params.height, 0.0, max_h));
        }
        if errors.is_empty() { Ok(()) } else { Err(errors) }
    }
}

/// 参数错误——Blueprint 验证和运行时校验共用
pub enum ParamError {
    OutOfRange { field: String, value: f32, min: f32, max: f32 },
    Deserialize(String),
    Unsupported(String),
}
impl ParamError {
    pub fn out_of_range(field: &str, value: f32, min: f32, max: f32) -> Self {
        Self::OutOfRange { field: field.to_string(), value, min, max }
    }
    pub fn deserialize(msg: String) -> Self { Self::Deserialize(msg) }
}
```

## 四、ComponentRegistry

```rust
/// 组件族注册中心 —— 编译期构建，依赖注入
pub struct ComponentRegistry {
    /// 存 object-safe 的基 trait
    families: HashMap<ComponentFamilyId, Box<dyn ComponentFamilyBase>>,
    name_index: HashMap<String, ComponentFamilyId>,
    /// FamilyTag → ComponentFamilyId（核心结构族映射——编译期固定）
    tag_to_id: HashMap<FamilyTag, ComponentFamilyId>,
    /// 预计算的连接面兼容矩阵
    face_compatibility: FaceCompatibilityMatrix,
}

impl ComponentRegistry {
    pub fn builder() -> ComponentRegistryBuilder { ... }

    pub fn resolve(&self, id: ComponentFamilyId) -> Option<&dyn ComponentFamilyBase> { ... }
    pub fn resolve_by_name(&self, name: &str) -> Option<&dyn ComponentFamilyBase> { ... }
    /// FamilyTag 快速解析——用于 WFC 求解器
    pub fn resolve_tag(&self, tag: FamilyTag) -> &dyn ComponentFamilyBase {
        &*self.families[&self.tag_to_id[&tag]]
    }
    pub fn compatible(&self, face_a: &ConnectionFace, face_b: &ConnectionFace) -> bool { ... }
    pub fn all_families(&self) -> impl Iterator<Item = &dyn ComponentFamilyBase> { ... }
}

pub struct ComponentRegistryBuilder { ... }

impl ComponentRegistryBuilder {
    /// register 接受 impl ComponentFamily（自动 blanket → ComponentFamilyBase）
    pub fn register<F: ComponentFamily>(mut self, family: F) -> Self { ... }
    pub fn register_with_mod(mut self, mod_toml: &str) -> Result<Self, ModError> { ... }
    pub fn build(self) -> ComponentRegistry { ... }
}
```

**FamilyTag 不扩展**——它只覆盖 7 个核心结构族。Mod 注册的新族走 `ComponentFamilyId`（u64），不支持 WFC tile 映射（除非 Mod 在注册 TOML 中声明 `wfc_tag` 并提供 `wfc_tiles()` 实现）。

### 4.1 启动时构建

```rust
let registry = ComponentRegistry::builder()
    .register(WallFamily::new())
    .register(FloorFamily::new())
    .register(DoorFamily::new())
    .register(WindowFamily::new())
    .register(BeamFamily::new())
    .register(ColumnFamily::new())
    .register(StairsFamily::new())
    .register(FoundationFamily::new())
    .register(RoofFamily::new())
    // 可选文化扩展
    .register(ButtressFamily::new())         // 飞扶壁（哥特）
    .register(ArchFamily::new())             // 拱门（罗马/哥特）
    .register(WindCatcherFamily::new())      // 风塔（沙漠）
    .register(TatamiMatFamily::new())        // 榻榻米（和风）
    .register(LivingWoodFamily::new())       // 活木（精灵）
    // Mod 扩展
    .register_with_mod("dwarven_masonry.toml")?
    .register_with_mod("elven_sylvan.toml")?
    .build();
```

**不是全局单例**——`ComponentRegistry` 是普通 struct，通过引用传入所有需要它的地方。Mod 通过 TOML 声明注册，不是运行时 register()。
