## 二、核心约定

1. 感官系统 = 无状态纯函数管线。`perceive(params) → PerceptBatch`
2. 感官不辨识、不诊断、不整合跨模态、不定义事件边界
3. 同一份 PerceptBatch 同时供日常决策、战斗情报、审美判断、大日志消费
4. 感知退化是距离的连续函数。层级只是调度分桶
5. woworld_core 定义所有 trait。各 crate 实现。NPC crate 做组合

## 三、性能预算

| 层级 | NPC数 | 周期 | CPU/帧 |
|------|-------|------|--------|
| L1 (全感知) | ≤200 | 逐帧 | ≤0.8ms |
| L2 (降频) | ≤2000 | 0.5-2s | ≤0.3ms |
| L3 (统计) | ≤100K | 30s+ | ≤0.1ms |
| 射线预算 | 自适应——基于显著性优先 | L1 ≤120 rays/NPC | — |
