# 002-战斗发起与遭遇响应

## 一、核心原则：感知驱动，非距离驱动

系统知道附近在战斗 ≠ 每个 NPC 都知道。NPC 必须先通过感官外壳检测到威胁信号，才能进入"遭遇评估"。没有上帝视角。

---

## 二、战斗触发条件

任一条满足 → 该 NPC 进入"遭遇评估"阶段：

### 2.1 蓄意攻击

任何实体对 NPC 执行了"攻击"行动。先手判定：攻击者的敏捷 × 隐蔽度 vs 防御者的感知能力。成功 → 防御者提前察觉。失败 → 攻击者获得先手优势。

### 2.2 威胁升级

日常摩擦（争夺资源/社交对峙）持续 >15s 且双方情绪愤怒 >0.6，或一方做出了"攻击前兆"动作（拔剑/蓄力魔法/战吼）。

### 2.3 被卷入（感知驱动）

必须通过感官外壳检测到战斗信号：

- **听到**：武器碰撞声/爆炸声/呼救声/战吼（听觉感知，距离衰减）。戴着耳罩/被沉默 → 听不到 → 不触发
- **看到**：视野内队友/同派系成员正在战斗（视觉感知，遮挡检测）。在建筑物背面/被烟幕遮挡 → 看不到 → 不触发
- **直觉/魔法感知**：高阶能力，非所有人都有
- **被交流告知**：队友呼唤（需要队友主动传递 + 自身接收到）

### 2.4 主动宣战

对话中选择战斗触发选项。双方同时进入。

### 2.5 伏击

一方从潜行/隐藏状态发起 → 持续性先手检定 → 可能被提前发现。

---

## 三、遭遇响应——四选一

感知到威胁 → 进入"遭遇评估"（通常 2-5 秒）→ 选定响应策略：

### 3.1 战斗

评估后认为自己有胜算 / 有职责义务 / 情绪压倒了理智。

子目标：击杀 / 致残 / 击退 / 拖延（等援军）

### 3.2 逃离

实力悬殊 / 高神经质 / 不值得拼命。

子目标：立即逃跑 / 找机会脱身 / 把别人推向危险自己跑。

对方不追击 → 不进入战斗状态 → 只是"逃离状态"。

### 3.3 妥协

实力悬殊但逃不掉 / 高宜人性 / 有谈判筹码。

子目标：投降 / 谈判 / 虚张声势（唬住对方）。

对方接受 → 不进入战斗。对方拒绝 → 重新评估。

### 3.4 回避

还没被发现 / 可以躲起来 / 让其他人去处理。

子目标：隐藏 / 绕开 / 装作没看见。不进入战斗。

---

## 四、评估维度

```
评估 = f(
  实力对比 (我的战斗力 vs 对方——基于可用的战斗信息, 可能不准),
  代价收益 (打赢了得到什么? 打输了失去什么?),
  限定条件 (信仰/道德/法律/职业职责),
  人格偏移 (神经质→逃避↑, 宜人性→妥协↑, 尽责性→职责履行↑),
  当前情绪 (愤怒→战斗, 恐惧→逃避, 信任→谈判),
  已知支援 (附近有我认识的人吗? ——只基于"我知道的")
)
```

没有固定权重。所有因素同时在场，它们的"声音"或大或小地混在一起——最终的决定是混合物的产物。
