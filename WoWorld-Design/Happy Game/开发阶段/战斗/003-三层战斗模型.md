# 003-三层战斗模型

## 一、本能层——逐帧自动执行

不需要"决定"格挡——就像人不需要"决定"眨眼。只要当前姿态允许 + 来袭攻击在感知范围内 → 自动执行。

- **自动闪避**：检测到来袭攻击 → 侧身/下蹲/后仰。成功率 = f(技能熟练度, 疲劳, 专注值, 是否被围攻)
- **自动格挡/弹反**：防御姿态下 → 自动举盾/抬剑。精准弹反需要更高技能
- **自动反击**：完美格挡后 → 自动利用对方后摇出刀
- **本能化法术**：熟练度 500+ 次的法术 → 触发条件满足时身体自动施放
- **掩体本能**：被远程攻击时 → 自动微调位置进入掩体阴影

本能层的表现上限 = 技能熟练度 + 身体状态。一个老练剑客的手会自动找到剑柄——不需要人格，不需要知识。

---

## 二、节奏层——0.5-1s 一次

不是"选一个动作"——是"选一个节奏"。输出不是具体招式——是一个倾向：

| 倾向 | 行为 | 适用 |
|------|------|------|
| **前压** | 主动接近，攻击频率↑，防守靠本能层 | 压制对方，创造机会 |
| **稳守** | 保持距离，攻击频率↓，等对方先出手 | 信息不足，等待时机 |
| **周旋** | 横向移动，寻找破绽，不冒进 | 试探阶段，实力相当 |
| **猛攻** | 连续出招，不管防守，体力消耗快 | 抓住窗口，速战速决 |
| **脱战** | 拉开距离，优先逃跑路径 | 战略层决定撤退 |

节奏持续数秒——在这数秒内不重新决策。改变节奏触发条件：被重创（HP 骤降 >20%）、对方节奏明显改变、战略层新命令。

---

## 三、战略层——5-30s + 事件驱动

真正的"思考"。自察 → 总目标 → 阶段目标 → 限定条件。

战略层的决策 = 人格 × 处境 × 情绪 × 记忆 × 知识 × 外部输入 同时搅拌的产物。

日常周期 5-30s，但关键事件（阶段目标完成、被重创、队友濒死、敌方士气崩溃）→ 事件推送 → 立即重新评估。

---

## 四、层级关系——同时发生，彼此染色

不是"一层传给另一层"的数据流。三层同时运行，互相染色：

```
本能感到吃力 (雾中看不清 → 闪避成功率↓)
  → 不直接导致节奏改变
  → 但它"染了"节奏的底色: 一种身体层面的不安
  → 节奏没有"选"稳守——节奏只是"不稳"了

战略层同时感受到:
  - 身体的不安 (本能层染色)
  - 节奏的犹豫 (节奏层染色)
  - "他也在雾里" (战斗信息的推理)
  - "上次在雾里我差点死了" (记忆)
  - "但我答应过妻子不冒没必要的险" (限定条件)
```

---

## 五、信息不足 = 放大人格 × 知识

在信息充足时（晴天、开阔地）——不同人格的战术差异比较小，因为事实很清楚。
在信息不足时（雾、夜、烟幕）——不同人格的战术差异急剧放大，因为没有清楚的"正确选择"——每个人按自己的天性去填补空白。

**但知识可以覆盖人格的默认倾向**：一个急躁的老兵（高神经质），因为经历过十几次烟雾战——"急，但我知道雾里不能乱冲"。人格仍然急躁，知识压住了它。

---

## 六、人格驱动的冲突风格

人格不只在"参数"上影响战斗——它影响"冲突风格"：

- **高宜人性** → 保护者。保护队友和弱者的优先级 > 击杀敌人
- **高尽责性 + 低神经质** → 问题解决者。找到最高效的解决方案，目标精确，成本可控
- **高神经质 + 低宜人性** → 破坏者。最大伤害，可能过度杀戮
- **高开放性** → 实验者。不固定，看这场战斗有什么"有趣"的事情发生
