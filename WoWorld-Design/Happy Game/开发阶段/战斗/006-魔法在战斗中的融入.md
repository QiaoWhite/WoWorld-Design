# 006-魔法在战斗中的融入

## 一、魔法基础

- **十元素**：金/木/水/火/土/风/雷/电/血/灵——每元素有独立的战斗角色
- **七系魔法**：变幻/召唤/破坏/守护/惑心/疗愈/死灵——每系在战斗中有不同频率和层级
- **魔法方程式**：范围+对象+接触+元素+影响+时间+条件——决定了施法的代价和复杂度
- **五施法媒介**：语言(吟唱)/物质(触媒)/仪式(法阵)/意志/契约——不同媒介适合不同的战斗节奏
- **魔力体系**：灵元素+血元素→提炼魔力。储量有限，消耗=各要素乘积

---

## 二、施法媒介 × 战斗节奏

> 五种施法媒介（语言/物质/仪式/意志/契约）在战斗中表现为不同形式。下表前三行（完整吟唱/短吟唱/默发）均为**语言媒介**的熟练度进阶——不是独立媒介类型。

| 媒介 | 施法时间 | 所属层级 | 限制 |
|------|---------|---------|------|
| **完整吟唱** | 1-3s | 战略层 | 不可移动/减速——可被打断——需要队友保护施法窗口 |
| **短吟唱** | 0.3-0.8s | 节奏层 | 可移动但减速——战斗中常用的施法方式 |
| **默发(大师)** | 0.1s | 本能层 | 可与其他行动同步——500+次熟练度解锁 |
| **触媒施法** | 0.5s | 节奏层 | 需一只手空闲——法杖/符文/魔杖挥动——物理媒介有耐久 |
| **意志** | 取决于灵元素强度 | 各层 | 根本媒介——灵元素越强施法越快——不需要外部媒介 |
| **法阵** | 极长(数分钟-小时) | 战略层外 | 预设触发法阵→被动激活——战前准备——战斗中不可画出 |
| **契约** | 1-3s(召唤) | 战略层 | 召唤物可"自动护卫"——召唤后持续消耗魔力维持存在 |

## 三、七系魔法在战斗中的角色

| 系 | 战斗角色 | 典型频率 | 层级 | 说明 |
|----|---------|---------|------|------|
| **破坏系** | 直接伤害——最常用 | 节奏层高频 | 节奏/本能 | 元素伤害+附加效果——战士+法师都可使用 |
| **守护系** | 护盾/力场/魔法盔甲 | 本能/节奏层 | 本能/节奏 | 可本能化——"盾牌"不只是物理的——魔法护盾自动激活(熟练后) |
| **变幻系** | 改变物质/形态/环境 | 战略层 | 战略 | 改变地形/制造掩体/冰冻水面/石化敌人——场上环境操控 |
| **召唤系** | 传送/链接/召唤生物 | 战略层 | 战略 | 需要保护施法窗口——召唤后的生物自主战斗 |
| **惑心系** | 瘫痪敌方决策——恐惧/魅惑/混乱/催眠 | 战略层 | 战略 | 消耗极高——不直接伤害——攻击对方的心智层 |
| **疗愈系** | HP恢复/伤病治疗/状态解除 | 节奏层 | 节奏 | 和攻击交替——"打两下→退后→治疗→再冲回去" |
| **死灵系** | 扭曲灵-血关系/控制尸体/汲取生命 | 战略层 | 战略 | 社会接受度因文化而异——有些文化视其为禁忌——有些文化是国教 |

## 四、法术熟练度→自动化降级

```rust
fn spell_tier(cast_count: u32, affinity: f32) -> SpellTier {
    let effective = cast_count as f32 * (1.0 + affinity * 0.5);  // 高亲和加速
    match effective {
        c if c < 50   => SpellTier::Strategic,   // 完整吟唱——初学
        c if c < 200  => SpellTier::Rhythmic,     // 快速施放——熟练
        c if c < 500  => SpellTier::RhythmicFast, // 几乎瞬发——精通
        _             => SpellTier::Instinctive,  // 身体自动——炉火纯青
    }
}
```

| 施放次数 | 层级 | 说明 |
|----------|------|------|
| 1-49次 | 战略层 | 完整咏唱——初学 ~10-15场战斗。高失败率(20-50%)。需要专注 |
| 50-199次 | 节奏层 | 快速施放——熟练 ~15-60场。失败率5-15%。施法窗口缩短 |
| 200-499次 | 节奏层(本能化中) | 几乎瞬发——精通 ~60-150场。失败率<3% |
| 500+次 | 本能层 | 触发条件满足→身体自动施放——炉火纯青 ~100-150场。失败率<0.5% |

每场战斗约3-5次同法术使用。50次≈10-15场(入门到熟练)。500次≈100-150场(本能化)。实际速度受灵元素亲和+训练频率影响——高亲和力者加速1.5-2×。

## 五、元素交互 × 战术

- **亲和组合**(火+风→扩散/水+雷→导电)→降低消耗、增强效果
- **排斥组合**(水+火→蒸汽爆炸)→魔力消耗×3但效果独特——高手和普通法师的区别：知道什么时候值得付出额外代价
- **环境联动**：雨天→电系优势(水导电)/干燥→火势蔓延/水域→电导电范围扩大/暴风雪→水(冰)系增强/大风→火势扩散

## 六、魔力 vs 耐力——两种战斗资源

| | 魔力 | 耐力(体力) |
|---|---|---|
| **恢复速度** | 非战斗1%/s、战斗中0.2%/s、耗尽后0.05%/s | 非战斗5%/s、战斗中1%/s |
| **消耗方式** | 施法消耗——每法术固定消耗 | 移动+攻击+闪避持续消耗 |
| **消耗特点** | 成本高、恢复慢——需要精打细算 | 消耗快、恢复也快——短期节奏 |

物理型：消耗快恢复快——适合持久战。魔法型：成本高恢复慢——每次施法需要权衡。混合型：更灵活但每项不精——"什么都会一点，什么都不深"。

## 七、环境魔力浓度对战斗的影响

| 魔力浓度 | 效果 |
|----------|------|
| 高魔力区(魔法群系/魔力浓度>0.6) | 魔力恢复速度×2、法术效果×1.2、敌方也一样。高魔力区=施法者的天堂 |
| 正常区 | 正常 |
| 低魔力区(死魔法区/反魔法场) | 魔力恢复速度×0.2、法术效果×0.5、部分法术无法使用(消耗>当前灵元素) |
| 零魔力区 | 无法施法——只能物理攻击。对纯法师=战斗不能 |

## 八、魔法的社会评价——文化相对

魔法本身没有固定的"禁忌"标签。同一魔法在不同文化/不同人眼中反应完全不同——取决于文化规范 + 目击者人格 + 与该使用者的关系。死灵系在A文化是国教仪式、在B文化是最高的禁忌。惑心系在自由城市是"侵犯心智——比杀更恶劣"、在独裁政体是"有效的统治工具"。
