# 009-环境互动与掩体

## 一、环境评估

环境信息在战斗信息池中是预先加载的。自察不只评估"我"——也评估"我在哪"：地形开阔度、掩体密度、介质类型、高地差、可交互物位置。

三层更新模式：
- **初始快照**（战斗开始时一次）：进门时的第一眼
- **事件驱动推送**（环境剧变时）：森林起火、暴风雨突降——世界推给 NPC
- **惰性更新**（关注时）：看的时候才刷新，用的时候才确认。不看的就不关心——这就是信息不对称

## 二、掩体意识

### 2.1 本能层

被远程攻击时 → 自动微调位置进入掩体阴影。在掩体后 → 自动减少暴露面积。从掩体边缘探头→射箭→缩回（约 1.5-2s）。

### 2.2 节奏层

掩体的价值评估：掩体能挡当前威胁吗？继续待在掩体后 vs 离开掩体的机会窗口？

### 2.3 战略层

"先到那个断墙后面"→ 掩体是阶段目标的一部分。离开掩体的风险评估：掩体外的收益 / 掩体外的风险。人格影响冒险阈值。

### 2.4 何时"该冒一下险"

- 看到了关键射击窗口，机会稍纵即逝
- 队友正在被夹击，必须吸引火力
- 对方正在换位置（中间没有保护的最脆弱时刻）
- 对方吟唱关键法术，掩体后的角度射不到

## 三、地形伤害

敌人背靠墙壁 → 崩/撞攻击没有后退空间 → 冲击力×2.0(伤害加倍) + 眩晕 1.5s。
敌人在悬崖边缘 → 撩基元(上挑)→ 平衡检定(敏捷+重心 vs 冲击力)→ 失败→翻落(坠落伤害=高度×重力×落地介质)。坠落>10m→重伤; >30m→致命。
敌人在浅水中 → 雷系攻击→水域导电→范围×2.5, 伤害不衰减。

## 四、高度差

高度差不是"射程变长几米"——是改变战斗的整个物理规则。俯攻方：射程×1.3-1.5、视野开阔、掩体优势。仰攻方：移动速度×0.6-0.8、攻击力量×0.8、暴露面积增大。

## 五、掩体可被破坏

掩体生命周期：完整→裂缝（保护率-30%）→即将坍塌（-70%）→彻底损坏。NPC 可感知掩体状态。信息不对称：看不到的那面墙——可能不知道快塌了。
