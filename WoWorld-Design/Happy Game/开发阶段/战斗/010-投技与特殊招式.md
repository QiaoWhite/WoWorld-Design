# 010-投技与特殊招式

## 一、投技

### 1.1 四阶段

**抓取窗口**（极短）：距离 <1.5m + 对方硬直中 / 被精准弹反后失衡 / "化"基元成功卸力

**力量对抗检定**：投掷者(力量+熟练度) vs 被投者(力量+敏捷/体重)。悬殊→直接完成。均势→短暂僵持。对方更强→挣脱+可能反制

**投掷执行**：选择方向和效果——过肩摔(倒地)、侧向摔(撞向环境)、背摔(自身不动)、推向特定方向(悬崖/尖石/队友刀口)

**收招**：双方短暂后摇。投掷者收力 0.3-0.5s。被投者倒地硬直 0.5-1s

### 1.2 条件与限制

- 抓取窗口短，抓到→大回报，挥空→大空档
- 体型限制：不能过肩摔巨龙
- 霸体限制：霸体 1-2 级可抓但有加成，霸体 3 级不可抓
- 缠的反制技能→可能反投
- 一场 45s 1v1 近战：出现 1-4 次尝试，成功 1-2 次
- 投技不是主攻击手段——是关键时刻的破局招式。威胁大于实际使用频率

## 二、特殊招式

### 2.1 定义

不能从动作积木的常规组合中产生。有独特性、有代价、有叙事意义。

### 2.2 四种来源

| 来源 | 说明 |
|------|------|
| 训练/传承 | 传奇流派的核心招式，需拜师+长期练习 |
| 天赋觉醒 | 极度危机中极低概率开悟——高开放性+高技能 |
| 武器专属 | 特定材料解锁：龙鳞剑→"龙吟"，玄铁锤→"崩山" |
| 魔法/契约 | 和强大存在签订契约→专属特殊法术 |

### 2.3 战斗中使用

激活条件：消耗大于常规 2-5×。冷却长。效果独特。一场战斗 0-3 次。战略层决定时机，非节奏层随意选。

### 2.4 使用中成长

效果 = 招式基础值 × 武器适配度 × 使用者属性 × 熟练度。1-10 次：消耗×1.5、后摇×1.3。200+ 次：消耗×0.6、后摇×0.6。可能出现变体——使用者自然改进招式。
