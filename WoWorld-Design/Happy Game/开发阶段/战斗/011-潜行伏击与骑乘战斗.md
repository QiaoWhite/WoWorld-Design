# 011-潜行伏击与骑乘战斗

## 一、潜行与伏击

### 1.1 核心原则：持续性博弈

不是一次性检定——是每一步都在赌对方没注意到。

### 1.2 持续性的潜行博弈

每 ~0.2s 更新：暴露风险（距离+行动+环境+对方注意力）× 感知能力 × 注意力方向 × 直觉 → 觉察概率。随机数 < 觉察概率 → 目标产生模糊信息。模糊信息累积 → 信息可信度升级 → 目标主动检查 → 可能暴露。

### 1.3 人格在潜行中

**潜行者**：高尽责性→耐心等最佳时机。低尽责性→不耐烦冲→暴露风险×3 但可能因快成功。高开放性→利用环境制造假声响引开对方。

**目标**：高神经质→频繁回头。低神经质→松懈（但老兵的低神经质是有底气的判断）。高尽责性老兵→习惯性警觉（不紧张但稳定）。

### 1.4 先手优势

伏击成功 → 目标的信息池在攻击命中后才更新 → 攻击获得伤害加成 + 目标无法格挡/闪避（本能层没有准备）。

## 二、骑乘战斗

### 2.1 坐骑 = 独立 AI

马不是摩托车——它有独立的意志、恐惧和脾气。骑手和坐骑 = 两个有配合度的独立 AI。坐骑有自己的本能层+节奏层，部分有战略层（高级坐骑）。

### 2.2 骑乘三层模型

**本能层**：骑术高→人马合一，自动调整出刀时机。飞行坐骑：俯冲攻击重力帮→伤害+30%但闪避受限。

**节奏层**：马上弓手→需要"减速→射→加速"的节奏。飞行弓手→盘旋→射→俯冲→拉起→再盘旋。

**战略层**：利用机动性切位置、保护坐骑（受伤=失去机动性）、坐骑体力是独立资源。

### 2.3 骑手 × 坐骑配合度

配合度低 → 坐骑可能不服从指令、本能层的时机不协调。配合度高 → 人马合一、自动协同。
