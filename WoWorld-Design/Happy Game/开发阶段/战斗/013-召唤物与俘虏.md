# 013-召唤物与俘虏

## 一、召唤物 AI

### 1.1 三层模型（简化版）

**本能层**：自带物理感知+闪避。精度取决于主人灵元素强度。

**节奏层**：执行主人命令——召唤时设定节奏倾向，不改变。火元素→猛攻（无自我保护）。水元素→保护主人。石傀儡→固守（挡在主人前面）。

**战略层**：无。不自定目标。"攻击那个弓手"→一直执行直到目标死亡/新命令/召唤物被消灭。

### 1.2 契约物（更高级）

和普通召唤物不同——契约物有有限的自主性（有战略层）。和主人的关系 = 团队中的高度信任队友 + 契约约束。

### 1.3 性能

每个召唤物 = 极简 AI。成本 ≈ 1/10 个普通 NPC。1000 人战斗中出现 50 个召唤物 → 额外约 0.05ms。

## 二、俘虏与"被囚禁"模式

### 2.1 被囚禁 = 日常决策器的受限模式

同一套 AI（人格/情绪/记忆/决策器）——但行动选项受限。可用：等待、配合、求情、谈判、反抗（有限）、绝食、装病、试图逃跑。不可用：自由移动、自由攻击、自由使用物品。

### 2.2 俘虏的动机

人格：宜人性高→倾向于配合。宜人性低→恨他们，等机会。神经质高→恐惧主导，可能崩溃。开放性高→"也许可以和他们交易……"。

记忆：认识捕获者且之前有正面关系→"他不会杀我"。听说过捕快者的好名声→恐惧降低。听说过坏名声→恐惧极高。

### 2.3 捕获者的交互

审讯：问问题。回答质量 = f(人格, 恐惧, 对审讯者的评估)。释放→记忆+关系更新+声誉传播。赎金→放出消息等对方来赎。招降→"加入我们"→俘虏评估：出路？还是背叛？囚禁→长期关押。处决→违反"不杀"条件，道德后果。

### 2.4 俘虏的战后过渡

被俘本身就是创伤经历。情绪残留+记忆编码+人格消化→影响被释放后的长期行为。
