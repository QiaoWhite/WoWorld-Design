## 核心设计原则

1. **玩家 AI = NPC AI** — 同一套代码路径。玩家不操作时完全自主战斗。
2. **三层模型** — 本能(逐帧)→节奏(0.5-1s)→战略(5-30s)。同时运行，互相染色。
3. **信息不对称** — 没有上帝视角。感知驱动，非距离驱动。信息不足放大人格×知识。
4. **只创造规则，不创造故事** — 招式积木+武器模块化+人格→自然涌现千百种战斗风格。
5. **战斗 = NPC 活人感的高压延续** — 不是独立模块，是同一套人格/情绪/记忆在高压下的自然表达。
