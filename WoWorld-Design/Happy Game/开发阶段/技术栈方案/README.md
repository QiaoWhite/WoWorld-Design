## 关键决策（v3.0）

| 维度 | 决策 |
|------|------|
| **引擎** | Godot 4.6 LTS |
| **模拟** | Rust stable 1.80+ (GDExtension) |
| **世界** | 25 万 km² 初步（≈英国），Minecraft 级最终 |
| **海陆** | 7:3，Gerstner 波 |
| **地形** | 700m+ 山，~1500m 垂直范围，垂直稀疏 Chunk + Clipmap LOD |
| **战斗** | 半自动——玩家 AI = NPC AI = 同一套 Rust 代码 |
| **NPC** | GOAP + 概率行为树，L1/L2/L3/L4 分层模拟 |
| **天空** | 混合体积云（2D impostor + 3D 体积 + 噪声高云） |
| **载具** | 魔力驱动，移动参考系 + 局部 NavMesh |
| **建造** | 三途径蓝图（编辑器/导入/自然语言委托） |
| **硬件** | GTX 1660 SUPER 6GB VRAM 流畅运行 |
