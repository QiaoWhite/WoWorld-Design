## 关键决策（v4.0）

| 维度 | 决策 |
|------|------|
| **引擎** | Godot 4.7（从 4.6 升级——CHG-053） |
| **模拟** | Rust stable 1.80+ (GDExtension) |
| **世界** | 25 万 km² 初步（≈英国），Minecraft 级最终 |
| **海陆** | 7:3，Gerstner 波 |
| **地形** | 700m+ 山，~1500m 垂直范围，垂直稀疏 Chunk + Clipmap LOD |
| **战斗** | 半自动——玩家 AI = NPC AI = 同一套 Rust 代码 |
| **NPC** | GOAP (~9%) + 概率行为树 (~90%) + LLM增强(~1%)，L1/L2/L3/L4 分层模拟 |
| **天空** | 混合体积云（2D impostor + 3D 体积 + 噪声高云） |
| **载具** | 魔力驱动，移动参考系 + 局部 NavMesh |
| **建造** | 三途径蓝图（编辑器/导入/自然语言委托） |
| **硬件** | GTX 1660 SUPER 6GB VRAM 流畅运行 |

## 架构速览

```
Rust 模拟核心 (woworld_* crates)
├── 世界生成 15阶段管线 (P0-P13 + P2.25)
├── NPC GOAP+概率行为树 (L1/L2/L3/L4 分层)
├── 经济 (限价订单簿·Bootstrap千年快照)
├── 权力 (17原子涌现·Contract双边)
├── 战斗 (三层模型·玩家AI=NPC AI)
├── Transvoxel 体素 (垂直稀疏Chunk·Clipmap LOD)
├── 动画 (9层可叠加动画栈·Rust CPU → GPU skinning)
└── 存档 (LMDB单文件多named_db·SaveableModule v2.0)
        ↓ GDExtension (PackedByteArray)
Godot 4.7 客户端
├── Toon 2-tone cel渲染 (SSAO+Bloom+AgX+大气透视)
├── 天空 (混合体积云·Gerstner海洋)
├── 音频 (Rust引擎→Godot AudioServer)
└── 输入/UI/物理(仅玩家 CharacterBody3D)
```
