## 定位

本文件夹存放 WoWorld 文化系统的**正式开发规格**。文化是引用最多的缺失模块——53 个文件、327+ 次引用，跨越 14 个现有模块。

**核心原则**：
- **原子组装**：10 个 CultureCoreParams 是所有文化表达的不可再分原子——一切行为/审美/沟通风格从参数派生
- **涌现性**：文化从种子生成，不预设任何具体文化（无"诺斯文化""希腊文化"）
- **程序化生成**：全部文化特征（参数、命名、手势含义、技术等级）从种子确定性生成
- **玩家 AI = NPC AI**：同一套 `CultureQuery` trait 同时服务玩家和 NPC

## 三层参数架构速览

```
第一层：CultureCoreParams（10个 f32，0-1）
  individualism · power_distance · uncertainty_avoidance · competition_orientation
  · long_term_orientation · indulgence · openness_to_outsiders · religiosity
  · militarism · artistry

        ↓ 派生（确定性公式 × 环境上下文）

第二层：第一级派生 → CommunicationNorms · BuildingStylePreferences
  · CulturalBeautyStandard · HonorificSystem · FertilityNorms
  · DietaryBasePreferences · ClothingPreferences · honor_weight

        ↓ 派生（核心参数 × 人口 × 贸易 × 历史 × 其他模块数据）

第三层：第二级派生 → TechnologyProfile (8领域) · magic_type_affinity
  · mage_social_status · literacy_rate · cross_cultural_attitude
```

## 关键参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| **核心参数数** | 10 | 不可再分的文化原子 |
| **技术领域数** | 8 | 独立增长的 TechnologyProfile 领域 |
| **起源点数** | 3-8 | P2.5 从种子生成，数量 = f(大陆面积, 大陆数) |
| **群系修正上限** | ±0.05 | 环境对初始参数的温和偏移 |
| **漂移率** | σ=0.003/年 | 100年累积 ~0.03——玩家一生内文化稳定 |
| **分裂最小隔离年** | 200 | 亚文化分裂的隔离阈值 |
| **克里奥尔最小稳定年** | 300 | 边界带混合文化结晶的阈值 |
| **参数传染性** | 0.3-0.8 | 各维度通过贸易传播的不同速度 |
| **每文化存储** | ~380B | 100文化 ≈ 38KB |
| **每帧 CPU** | < 0.05ms | 不及 16.7ms 预算的 0.3% |
