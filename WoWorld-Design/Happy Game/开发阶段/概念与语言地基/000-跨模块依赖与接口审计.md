## 一、权威性声明

**本模块（概念与语言地基系统）的正式规格文档是本模块所有内容的唯一权威来源。** 其他模块文档中凡涉及 NPC 概念思维/语言 proficiency/文化概念空间/概念翻译/代际知识传递的任何描述，若与本模块正式规格冲突，以本模块为准。

本模块是**新增地基层**——它填补了语言表达模块（文本管道）和 NPC 认知模块（思维系统）之间的概念表示层空白。本模块不取代任何已有模块。

## 四、冲突与不一致（已识别）

| # | 冲突描述 | 解决方案 |
|---|---------|---------|
| 1 | 语言表达模块中 ExpressionRef 既用于书籍又用于瞬时话语 | UtteranceId 用于瞬时话语，ExpressionRef 保留用于持久可读物 |
| 2 | NPC ver2.0 中 SkillCategory 含 Social/Economic——但技能系统已删除这两类 | 以技能系统为权威 |
| 3 | 信息传播系统的 InformationPayload 使用 String summary | 新增 compact_concepts 字段，旧字段保留共存 |
| 4 | 现有 TextGenerator 只处理模板渲染——没有概念渲染入口 | 新增 render_utterance() / render_thought() |
| 5 | EventMemory.summary 是 String——没有概念维度 | 新增 compact_concepts 字段；summary 降级为惰性缓存 |
| 6 | AgentSnapshot 不含语言信息 | 新增 language_proficiencies / script_proficiencies 字段 |
| 7 | literacy 当前为 bool | script_proficiency: f32（连续值） |
| 8 | 语言学习无涌现模型 | 从原子日志 LISTEN/SPEAK/READ/WRITE/EXAMINE 涌现 |
