## 二、演绎推理——deductive_chain()

```rust
// woworld_core/src/reasoning.rs
// 纯函数——仅依赖 woworld_types

/// 将 MentalModel 集合链式应用到已知事实
pub fn deductive_chain(
    premises: &[ConceptFact],
    models: &[MentalModelRef],
    style: &CognitiveStyleParams,
    max_depth: u8,
) -> Vec<DeductiveConclusion> {
    let mut conclusions = Vec::new();
    let mut working_facts: Vec<ConceptFact> = premises.to_vec();
    let depth_limit = (max_depth.min(5) as f32 * style.analytic_intuitive) as u8 + 1;

    for depth in 0..depth_limit {
        let mut new_facts = Vec::new();
        for model in models.iter().filter(|m| m.confidence > 0.3) {
            if let Some(conclusion) = apply_model_to_facts(model, &working_facts) {
                let confidence = model.confidence
                    * conclusion.premise_match_quality
                    / (1.0 + depth as f32 * DEDUCTIVE_CONFIDENCE_DECAY);  // [TUNING] 0.15
                new_facts.push(ConceptFact {
                    content: conclusion.content,
                    confidence,
                    derivation: Derivation::Deduction {
                        from_model_id: model.id,
                        from_fact_ids: conclusion.matched_premise_ids,
                        depth,
                    },
                });
            }
        }
        if new_facts.is_empty() { break; }
        conclusions.extend(new_facts.clone());
        working_facts.extend(new_facts);
    }
    conclusions
}
```

### 2.2 浅层 vs 深层演绎

```
浅层演绎 (memory_encode 时, depth≤2):
  "儿子在铁匠铺" → "有工作" (depth 1)
  "有工作"      → "有收入" (depth 2)
  ✓ 到这儿就够了——"不用给钱了"是推理结论

深层演绎 (expand_inference 时, depth≤5, 被问到时才做):
  "有收入" → "可以买剑" → "铁匠铺需要更多铁" → "铁矿价格可能涨"
  → "我可以投资铁矿" → ...
  深层推理链在 NPC 被问到特定问题时才展开
```

### 2.4 memory_encode_with_deduction() —— NPC crate 包装器

```rust
// NPC crate —— XD3修正: 正式定义
// 记忆编码时自动执行浅层演绎

/// 将感知编码为 EventMemory，同时执行浅层演绎推理
/// 调用 woworld_core::deductive_chain() 做实际推理
/// ★ 方案D: Rust 实现中 cognitive_style 和 mental_models 从 CognitionColumn 读取——不经过 NpcData
pub fn memory_encode_with_deduction(
    percept: &PerceptEntry,
    encoding: &CompactConceptEncoding,
    npc: &NpcData,
    cognition_col: &CognitionColumn,  // ← 06-认知 own 的 SoA 列族
) -> EventMemory {
    // ① 将感知数据转化为概念事实
    let premises = percept_to_facts(percept, npc.effective_concept_space());

    // ② 浅层演绎 (depth ≤ 2)
    let conclusions = woworld_core::deductive_chain(
        &premises,
        &cognition_col.mental_models,         // ← SoA
        &cognition_col.cognitive_style.params(), // ← SoA
        2,  // max_depth=2: "儿子在铁匠铺"→"有工作"→"有收入"
    );

    // ③ 取最高置信度结论作为 inference_conclusion
    let best = conclusions.iter()
        .max_by(|a, b| a.confidence.partial_cmp(&b.confidence).unwrap());

    EventMemory {
        compact_concepts: *encoding,
        inference_conclusion: best.map(|c| InferenceConclusion {
            content: c.content.clone(),
            confidence: c.confidence,
            culture_id: npc.dominant_concept_culture(),
            concept_space_version: npc.concept_space_version,
            formed_at: GameInstant::now(),
            derivation_depth: c.derivation.depth(),
        }),
        // ... 其余 EventMemory 字段不变
    }
}
```

## 三、推理置信度传播

```rust
// 每层推理的置信度衰减:
// confidence(n+1) = confidence(n) × model.confidence × premise_match_quality / (1 + depth × 0.15)
//
// 示例:
// depth=0: model.confidence=0.9, match=1.0 → 0.90
// depth=1: model.confidence=0.85, match=0.9 → 0.85 × 0.9 / 1.15 = 0.67
// depth=2: model.confidence=0.8, match=0.8  → 0.80 × 0.8 / 1.30 = 0.49
// depth=3: 结论 confidence < 0.3 → 不再信任
```

## 四、理性对情绪的主动调控

```rust
// NPC crate —— deliberate_reasoning.rs

pub fn evaluate_deliberation(
    npc: &AgentSnapshot,
    emotion: &EmotionState,
    decision_importance: f32,
    cognitive_load: f32,
) -> DecisionMode {
    // 前提: 反思型 > 0.6
    if npc.cognitive_style.reflective_impulsive < 0.6 { return DecisionMode::Normal; }
    if cognitive_load > 0.8 { return DecisionMode::Normal; }

    let emotional_interference = emotion.arousal().abs() * 0.5
                               + (1.0 - emotion.pleasure()) * 0.3;
    let threshold = 0.5 - npc.cognitive_style.analytic_intuitive * 0.3;

    if emotional_interference > threshold || decision_importance > 0.7 {
        DecisionMode::Deliberate {
            reasoning_depth_boost: if emotional_interference > 0.6 { 2 } else { 1 },
            emotional_weight_multiplier: (1.0 - npc.cognitive_style.analytic_intuitive * 0.6).max(0.15),
            cognitive_load_cost: 0.15 + emotional_interference * 0.1,
        }
    } else {
        DecisionMode::Normal
    }
}
```

**涌现**: 高反思型 NPC 在情绪激动时会"冷静下来想一想"。高分析型 NPC 在重要决策时自动降低情绪影响。低反思的人冲动决定——这是真实的认知差异。

## 五、智慧——从已有维度派生

```rust
/// 智慧有效值——纯函数，不存储，每次调用时计算
pub fn wisdom_effective(
    npc: &AgentSnapshot,
    models: &[MentalModel],
    age_years: f32,
) -> f32 {
    // 自省: 反思 × (1 - 自我服务偏差)
    let self_awareness = npc.cognitive_style.reflective_impulsive
                       * (1.0 - npc.cognitive_biases.self_serving * 0.5);

    // 经验内化: 有多少模型是"自己的"
    let owned_count = models.iter()
        .filter(|m| m.source == ModelSource::SelfRefined || m.source == ModelSource::SelfDerived)
        .count();
    let owned_ratio = (owned_count as f32 / models.len().max(1) as f32).min(1.0);

    // 认知平衡: 不是极端分析也不是极端直觉
    let balance = 1.0 - (npc.cognitive_style.analytic_intuitive - 0.5).abs() * 2.0;

    // 年龄: 60 岁前线性增长
    let age_factor = (age_years.min(60.0) / 60.0) * 0.8
                   + (age_years.max(60.0) - 60.0).min(20.0) / 100.0;

    (self_awareness * 0.30 + owned_ratio * 0.25 + balance * 0.25
     + npc.cognitive_style.rigid_flexible * 0.10 + age_factor * 0.10).clamp(0.0, 1.0)
}
```

**智慧影响**: deliberate_reasoning 门槛降低、deductive_chain 的推理深度选择更恰当、creative_leap 中验证要求更高。

## 六、认知风格的生命历程演变

```rust
/// 重大生活事件对认知风格的方向性偏移
pub fn life_event_cognitive_shift(
    event: &EventMemory,
    style: &mut CognitiveStyle,
) -> Option<CognitiveShiftEvent> {
    let shift = match event.event_type {
        // 冲动导致失败 → 反思 +0.03-0.08
        EventType::MajorFailure if event.attributed_to_impulsivity() => {
            CognitiveShift { reflective_impulsive: event.impact_score * 0.08, .. }
        }
        // 深度教育 → 抽象 + 分析
        EventType::Education { years, quality, .. } => {
            CognitiveShift {
                abstract_concrete: years * quality * 0.012,
                analytic_intuitive: years * quality * 0.008,
            }
        }
        // 创伤 → 直觉偏移（生存模式）
        EventType::Trauma { severity, .. } if severity > 0.7 => {
            CognitiveShift { analytic_intuitive: -severity * 0.1, rigid_flexible: severity * 0.05 }
        }
        // 跨文化生活 → 认知灵活性
        EventType::CrossCulturalImmersion { years, cultural_distance, .. } => {
            CognitiveShift { rigid_flexible: years * cultural_distance * 0.01 }
        }
        _ => return None,
    };
    style.apply_shift(&shift);
    Some(CognitiveShiftEvent { shift, cause_event_id: event.id })
}
```
