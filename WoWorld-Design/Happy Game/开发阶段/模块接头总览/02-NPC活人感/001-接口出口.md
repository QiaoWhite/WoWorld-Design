## NpcData (根部 struct — 核心身份/状态数据)
- **概念**: NPC 的身份/状态核心。持久化字段：身份标识(Identity) + 人格(BigFive) + 生理(Physiology) + 情绪(EmotionState) + 记忆(MemoryStore, ~617KB) + 关系(BTreeMap) + 技能(BTreeMap) + 战斗属性 + 自我叙事(SelfNarrative) + 社会身份 + 共识缓存 + 行为统计 + 载具状态 + 天象感知 + 施工任务 + 感官状态机(SensoryState, 含瞬态缓存)。
  ★ v1.1 方案D: 以下持久化子域数据已从 NpcData 迁移至独立 SoA 列族(见下方 §SoA 列族出口):
  - `social_deficit` / `need_sensitivity` → NeedColumn (03-基本需求)
  - `esteem_deficit` / `competence_frustration` / `chronic_days` → GrowthColumn (04-进阶需求)
  - `aesthetic_taste` → TasteColumn (05-审美)
  - `cognitive_style` / `cognitive_tide` / `mental_models` / `mind_attributions` / `belief_history` / `cognitive_distress` / `mental_model_creation_count` / `chronic_intoxication_years` → CognitionColumn (06-认知)
  - `attraction_template` / `norm_internalizations` → AttractionColumn (02-性别)
  - `knowledge` / `aesthetic_frameworks` → PerceptionColumn (感官与知觉)
  - 过渡字段(mental/physical/appearance)暂留 NpcData——待 LifeEntity 融合
- **Owner**: [[../../../NPC活人感模块/NPC活人感开发文档ver2.0|NPC ver2.0 §Part 1]]
- **类型**: Rust struct (bincode → LMDB), woworld_npc
- **消费者**: [[../16-音频/002-接口入口|音频]] · [[../12-历史/002-接口入口|历史]] · [[../14-经济/002-接口入口|经济]] · [[../06-战斗/002-接口入口|战斗]]
