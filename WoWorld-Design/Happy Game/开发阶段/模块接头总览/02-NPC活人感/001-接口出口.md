- **Owner**: NPC 关系模块（新增，非本子模块直接拥有）
- **类型**: Rust struct, woworld_npc
- **消费者**: NPC 认知/情绪管线 · GOAP 伴侣寻找目标
- **契约**: CHG-041 (本模块)

## crowd_suggestibility() / assess_and_integrate_mental_model() ★ v1.1 (CHG-057)

- **概念**: crowd_suggestibility(npc) → f32 — 从 PerceptBatch.environment.crowd_emotional_field 和 cognitive_tide.mind_quietude 派生。调制 assess_and_integrate_mental_model() 内部 effective_trust() 计算——高 suggestibility → trust→0.7（Le Bon crowd suggestibility）。emotion传播快于信念传播——NPC 先感受后找理由（事后归因）。信息级联从已有机制交互中涌现——不需独立系统
- **Owner**: [[../../../NPC活人感模块/NPC活人感开发文档ver2.0|NPC ver2.0 §2.1.6 (群体可暗示性)]]
- **类型**: fn(npc: &NpcData) → f32, woworld_npc
- **消费者**: 信念评估管线（MentalModel 接受/拒绝） · 情绪传染系统 · 信息级联衰减
- **契约**: CHG-057

## receive_player_speech() ★ v1.2 (CHG-058)

- **概念**: 玩家→NPC 信念桥接——玩家对 NPC 说话时与 NPC↔NPC 走完全相同的认知管道。合成 incoming MentalModel (source = TaughtByNpc(PLAYER_ENTITY_ID)) → 路由通过 assess_and_integrate_mental_model()。玩家无需 MentalModel 存储——零新 trait · 零新跨模块依赖。玩家 reputation 影响 source_credibility——撒谎被抓→低 credibility→reject
- **Owner**: [[../../../NPC活人感模块/NPC活人感开发文档ver2.0|NPC ver2.0 §2.1 (玩家话语→NPC信念桥接)]]
- **类型**: fn(npc: &mut NpcData, utterance: &Utterance, player_reputation: &PlayerReputation), woworld_npc
- **消费者**: 语言表达系统（玩家 Utterance → NPC 认知管道） · UI 层（玩家对话输入）
- **契约**: CHG-058
