## 10 元素 (固定顺序)
- **概念**: 金/木/水/火/土/风/雷/电/血/灵 — 固定顺序不可变. 雷=声音/振动, 电=闪电/电荷 (CHG-013 关键区分). 血=肉身元素(金木水火土风电雷按比例组合). 灵=灵魂元素(寄宿于血, 施法核心桥梁). 两条公理: 二元构成原理+意志投射原理
- **Owner**: [[../../../魔法/01-基础层/002-十元素|魔法 §002]] + [[../../../魔法/01-基础层/001-魔法总纲|总纲 §零]]
- **类型**: Rust enum + const定义, woworld_magic (woworld_core定义)
- **消费者**: [[../06-战斗/002-接口入口|战斗]] 元素战斗互动 · [[../01-生命/002-接口入口|生命]] elemental_affinity[10]/element_surplus[8] · [[../08-技能/002-接口入口|技能]] 元素组边界 · [[../18-物品/002-接口入口|物品]] 附魔元素 · [[../12-历史/002-接口入口|历史]] 元素事件
- **契约依据**: CHG-013 (10元素固定顺序, 雷≠电)

## 魔法方程式 (7 要素)
- **概念**: 魔法 = 范围(点/线/面/体) + 对象(类型x数目x命中x指向) + 接触(过程/凭空) + 元素(单一/混合) + 影响(增减/吸收/释放/排斥 → 转化/转移/链接) + 时间(瞬时/持续/触发/永久) + 条件(自身/环境/物质/仪式/契约). 每个魔法都是这7个维度的参数组合—非预设法术列表
- **Owner**: [[../../../魔法/01-基础层/004-魔法方程式|魔法 §004]]
- **类型**: Rust struct MagicFormula, woworld_magic
- **消费者**: [[../06-战斗/002-接口入口|战斗]] 战斗法术生成 · [[../08-技能/002-接口入口|技能]] 法术学习

## 七系魔法 (MagicType)

- **Owner**: [[../../../魔法/02-机制层/006-魔法类型|魔法 §006]]
- **类型**: Rust enum MagicType, woworld_magic
- **消费者**: [[../06-战斗/002-接口入口|战斗]] 法术分类 · [[../08-技能/002-接口入口|技能]] 魔法技能树 · [[../10-信仰/002-接口入口|信仰]] 宗教魔法类型

## 魔力体系 (Mana u32刻)
- **概念**: max_mana_ke = (spirit_capacity x magic_power x race_base_mana) as u32. 运行时全部u32刻运算—避免浮点精度误差. 恢复: 活跃1-3%/h, 冥想5-10%/h. 环境魔力浓度→恢复速度翻倍. 学徒10-50刻/成熟法师100-500刻/传奇数千刻
- **Owner**: [[../../../魔法/02-机制层/005-魔力体系|魔法 §005]] → 来源: [[../../../生命/004-身体状态与生命过程|生命 §四]]
- **类型**: Rust u32 + struct, Life定义魔力容量 → Magic定义恢复规则
- **消费者**: [[../06-战斗/002-接口入口|战斗]] 施法消耗 · [[../01-生命/002-接口入口|生命]] MaxManaKe属性
- **契约依据**: CHG-013 (Mana运行时u32, 恢复速度以Magic为准)

## 过载施法 (Spirit过载)
- **概念**: Mana=0后强行施法→消耗raw spirit. 渐进症状: >50%轻微眩晕/30-50%视觉模糊+魔法失控+15%/10-30%意识模糊+元素反噬+禁止施法/0%不可逆死亡—灵魂消散. 过载后spirit恢复极慢(比Mana慢10倍以上). 累积过载→spirit_capacity永久下降
- **Owner**: [[../../../生命/004-身体状态与生命过程|生命 §四]] (Spirit消耗模型) + [[../../../魔法/01-基础层/001-魔法总纲|魔法总纲]] (施法端触发)
- **类型**: Rust 过载信号, Life存储spirit值 → Magic触发过载
- **消费者**: [[../06-战斗/002-接口入口|战斗]] 高风险战术 · [[../01-生命/002-接口入口|生命]] SpiritState更新
- **契约依据**: CHG-013 + Change/hand/005
