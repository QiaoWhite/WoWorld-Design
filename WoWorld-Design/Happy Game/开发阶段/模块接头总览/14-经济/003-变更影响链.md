```

│   ├── → Player: 玩家交易
│   └── → 物品: 库存变动
├── 五大货币稳定器参数变更
│   ├── → 世界生成: 初始货币
│   └── → 物品: 铸币

│   ├── → NPC: 身份系统
│   ├── → 世界生成: 初始分配

│   └── → 经济: 收入来源
├── 5 种交易模式变更
│   ├── → NPC
```

## 关键架构约束

| 约束 | 范围 | 违规后果 |
|------|------|---------|
| 禁止直写价格 | 全部模块 | 价格涌现机制破坏——经济系统失去可信度 |
| 玩家和 NPC 走同一交易路径 | 交易执行 | NPC 特权——不公平 |
| LLM 不参与经济计算 | LLM 增强层 | 价格失真——结构化计算优先 |
| Market ≠ 物理地点 | 空间系统 | 混淆市场聚合与物理商店 |
