# 载具系统 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 载具系统 v1.0 (CHG-026)
>
> 当载具系统发生变更时, 以下模块和概念会受影响.

## VehicleId/VehicleDef 变更
→ **物品系统**: 契书 ItemCategory 关联. VehicleDeed 复用 `ItemCategory::Financial`
→ **世界生成**: P8.5 初始载具分布
→ **NPC**: GOAP 目标池 (Crew GOAP)

## 动力类型 (5种) 变更
→ **生命**: MusclePowered → 乘员 stamina 消耗公式
→ **魔法**: MagicEngine → ManaConsumer trait 调用方式
→ **天气**: 灾害级硬约束应用范围

## 移动参考系 (T_V) 变更
→ **Godot渲染**: 共享内存直读变换矩阵格式
→ **NPC**: 载具内寻路/行为坐标转换
→ **CHG-033**: 物理已迁移至 Rust 空间查询, T_V 须与 EntityIndex 注册坐标一致

## 损伤模型 (3通道) 变更
→ **战斗**: collision_bodies/cover_surfaces 查询结果 → 伤害穿透计算
→ **物品**: 维修材料消耗

## 所有权/契书 (5路径) 变更
→ **权力**: Adjudicate 调用方式. `set_owner()` 仅 Adjudicate 结果调用
→ **经济**: 交易转让契书流程

## VehicleQuery trait 签名变更
→ **战斗**: damage_state/motion_vector/collision_bodies/cover_surfaces — 每帧热路径, 签名变更影响最大
→ **NPC GOAP**: vehicles_at_port/available_transport — 每决策周期
→ **经济**: physical_profile/cargo_space — 商业决策
→ **历史**: vehicle_state 快照 — 事件记录
→ **天气**: position — 灾害区域判定

## VehicleMut trait 签名变更
→ **世界生成**: register_vehicle — P8.5 初始分布 (一次性)
→ **天气**: update_navigation/apply_damage — 导航+灾害伤害
→ **战斗**: apply_damage — 战斗伤害
→ **权力**: set_owner — 裁决结果

## CrewSlotDef 变更
→ **技能**: required_skill → SkillId 引用
→ **文化**: label_key → 本地化键

## 命名系统变更
→ **文化**: 命名模板/参数
→ **语言表达**: 名字生成算法

## 涌现 (VehicleArchetype) 变更
→ **世界生成**: P8.5 初始分布 → P9 铁路涌现
→ **文化**: CulturalShipbuildingTradition::derive()

## 跨CHG影响
- **CHG-016 (天气)**: 天气004 §4.9 wind→航速 已删除. 载具系统为权威 — 冲突已解决
- **CHG-033 (空间查询)**: 物理方案改为 Rust 空间查询. 载具导航/碰撞依赖 TerrainQuery+EntityIndex
