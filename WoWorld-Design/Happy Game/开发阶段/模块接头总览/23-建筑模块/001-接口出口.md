# 23-建筑模块 — 接口出口

> **最后更新**: 2026-06-22 | **CHG-043 + CHG-057**

## 本模块提供给其他模块的概念

| # | 概念 | 类型 | 消费者 | 说明 |
|---|------|------|--------|------|
| 1 | **BuildingQuery trait** | Rust trait (Send+Sync) | Weather, Senses, NPC, Combat, Audio, Economy, Aesthetic, History, Culture | 建筑世界数据唯一只读接口。15+方法——空间查询/结构查询/表面查询/历史查询/估值因子。详见 [[建筑模块/003-建筑查询接口\|003]] |
| 2 | **Surface trait** | Rust trait | Items (PlacedItem) | 可放置物品的表面——Floor/Wall/Ceiling/Counter。建筑组件暴露表面，家具系统消费。详见 [[建筑模块/003-建筑查询接口#二、Surface trait\|003 §二]] |
| 3 | **ConstructionTask + MaterialRequirementList** | Rust struct | NPC 行动 (CONSTRUCT), NPC EconomicBehavior | 声明式施工需求——不指定材料获取方式。详见 [[建筑模块/005-施工调度系统\|005]] |
| 4 | **ConstructionModifier trait** | Rust trait（定义）| Magic（实现）| 魔法辅助建造预留接口。详见 [[建筑模块/005-施工调度系统#四、ConstructionModifier trait\|005 §四]] |
| 5 | **Blueprint TOML 格式** | 文件格式规范 | 玩家, Items (Blueprint item 0x52) | 玩家 DIY 建筑的设计文件格式。详见 [[建筑模块/006-Blueprint蓝图系统\|006]] |
| 6 | **BuildContext** | Rust struct（定义）| World Gen / 玩家编辑器 / NPC | 六维参数聚合——调用方填充后传入建筑模块。详见 [[建筑模块/008-跨模块接口与数据合同\|008]] |
| 7 | **BuildingGenerator trait** | Rust trait | World Gen P5/P6 | 多种建筑生成策略的抽象。详见 [[建筑模块/007-建筑生成器族谱\|007]] |
| 8 | **StructureValueFactors** | Rust struct | Economy（房产估值）| 估值因子——不含价格。详见 [[建筑模块/003-建筑查询接口#四、StructureValueFactors\|003 §四]] |
| 9 | **BuildingGeometry** | Rust struct | Weather（风场/雨遮蔽）, Audio（声音反射）| 建筑简化外壳。详见 [[建筑模块/003-建筑查询接口#三、BuildingGeometry\|003 §三]] |
| 10 | **BuildingMaterialProps** | Rust struct（定义）| 胶水代码 | 建筑物理属性——从 MaterialProperties 转换。详见 [[建筑模块/003-建筑查询接口#五、BuildingMaterialProps\|003 §五]] |
| 11 | **BuildingHistory** | Rust struct | History (AetherImprint), NPC 认知 (★ CHG-057 建筑考古) | 双层时间窗口存储——建筑考古。NPC 认知系统消费 room function tags 做建筑功能推断，MentalModel 存储建筑空间语义。详见 [[建筑模块/002-建筑数据模型\|002 §六]] |
| 12 | **Building / ComponentInstance** | Rust struct | World Gen, NPC, Combat, Senses | 权威建筑数据。详见 [[建筑模块/002-建筑数据模型\|002]] |
