## 影响方向：存档系统 → 其他模块

存档系统是被动消费方——它不主动影响其他模块。但 `SaveableModule` trait 的引入要求**所有有持久化需求的模块**实现此 trait。

| 影响模块 | 影响类型 | 说明 |
|---------|---------|------|
| 全部 14 模块 | 新增 impl | 实现 `SaveableModule` trait（2 个必须方法 + 按需覆盖其余 6 个） |
| 世界生成 | 删除操作 | P13 不再写 LMDB。LMDB 写入权移交存档系统 |
| 历史 | 架构变更 | 独立 LMDB 文件 → 存档内 named_db |
| 性能分析 | 描述修正 | "JSON Lines 格式" → bincode |

## 变更传播路径

```
存档系统 trait 定义变更
  → 影响所有 impl 模块（需同步更新）
  → 最小化：trait 方法大部分有默认实现

存档文件格式变更
  → 需更新 global_version
  → 各模块 module_versions 可能受影响
  → Header 字段增删 → 全局迁移
```
