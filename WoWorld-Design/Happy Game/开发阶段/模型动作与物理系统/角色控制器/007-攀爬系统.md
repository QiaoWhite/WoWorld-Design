## 〇、设计哲学

**自动挂墙——不需要"攀爬键"**。玩家推摇杆向墙 + 墙角度在攀爬能力内 → 自动过渡。这是 Zelda 式（碰墙自动爬），不是 Souls 式（按键爬梯子）。NPC 也走同一逻辑——GOAP 设 `desired_special = Climbing` → ClimbEntryDetectionSystem 自动检测。

---

## 一、Grip——独立握力资源

Grip 独立于 Stamina。两者的消耗模式完全不同：

| | Stamina | Grip |
|---|---|---|
| 含义 | 全身运动消耗 | 手/爪/触手对表面的附着力 |
| 消耗 | 冲刺/跳跃/闪避/挥剑 | 攀爬移动 |
| 受材质影响 | 否 | 极大——冰面 vs 粗糙岩石 |
| 再生 | 非运动时缓慢恢复 | 非攀爬时恢复 |

**涌现记忆点**：一个马拉松选手（高 Stamina）在冰墙上（低 Grip）爬不了几步。一个攀岩老手（高 Grip）冲刺一段就喘（低 Stamina）。

```rust
pub struct CGrip {
    pub current: f32, pub max: f32, pub regen_rate: f32,
}
impl CGrip {
    pub fn drain(&mut self, amount: f32) -> bool { self.current > 0.0 }
    pub fn regenerate(&mut self, dt: f32) { /* 非攀爬时恢复 */ }
    pub fn ratio(&self) -> f32 { self.current / self.max }
}
```

```rust
pub struct CClimbingAbility {

    pub base_grip: f32,       // 粗糙岩=1.0, 冰=0.15
    pub wet_penalty: f32,     // 湿后倍率损失
    pub ice_penalty: f32,
    pub friability: f32,      // 松散度——0=实心, 1=一碰就碎
```

## 三、攀爬生命周期

```
         推摇杆向墙(玩家) / GOAP设desired_special(NPC)
                        │
              ┌─────────▼─────────┐
              │ ClimbEntryDetection │  检测可攀爬面→发 MountClimb
              └─────────┬─────────┘
                        │ ActionController 接受 → Windup(150ms)
                        ▼
              ┌──────────────────┐
              │ MovementMode      │  special = Climbing
              │ ClimbingMovement  │  切平面投影 + 握力消耗
              └──────┬───────────┘
                     │
         ┌───────────┼───────────┐
         ▼           ▼           ▼
     平移移动    静止休息   Mantle上翻
    (消耗握力)  (回复握力)  (翻上平台→Grounded)
         │
         ├── 握力耗尽 → DetachClimb → PhysicsBody(坠落)
         ├── 按蹲键 → DetachClimb(松手)
         ├── 按跳键 → LeapFromWall(推离墙面+跳跃)
         └── 空中跳向墙 → JumpGrab(抓握)→Climbing
```

## 五、ClimbingMovementSystem

```rust
fn apply_climbing_movement(
    transform, climb_state, grip, ability, intent, terrain, dt
) -> ClimbResult {
    // 1. 切平面投影——移动方向投影到攀爬面
    let tangent_move = intent.direction - normal * intent.direction.dot(normal);

    // 2. 加速度——与地面移动同一套 accelerate()
    let target_speed = CLIMB_BASE_SPEED * ability.speed_multiplier;
    let displacement = tangent_move * new_speed * dt;

    // 3. 重新贴合表面——射线投射到新位置
    let probe_origin = new_pos + normal * CLIMB_PROBE_DIST;
    if let Some(hit) = terrain.terrain_raycast(probe_origin, -normal, CLIMB_PROBE_DIST * 2.0) {
        transform.position = hit.point - normal * CLIMB_BODY_OFFSET;
        climb_state.surface_normal = hit.normal;
    } else {
        return ClimbResult::SurfaceLost;  // 悬空/凹洞/凸角→Mantle或脱落
    }

    // 4. 握力消耗 = base × material_factor × weather_factor × fatigue_factor
    let total_drain = base_drain * material_factor * weather_factor * fatigue_factor * dt;
    if !grip.drain(total_drain) { return ClimbResult::GripExhausted; }
    ClimbResult::Ok
}
```

## 六、表面材质抓力表

```toml
# surface_grip.toml
[grip.rough_granite]    # base_grip=1.0, wet_penalty=0.2, ice_penalty=0.9
[grip.sandstone]        # base_grip=0.7, wet_penalty=0.5, ice_penalty=0.6, friability=0.3
[grip.ice_wall]         # base_grip=0.15——几乎不可攀爬（需要冰镐）
[grip.oak_bark]         # base_grip=0.85, wet_penalty=0.4
[grip.mossy_stone]      # base_grip=0.5, wet_penalty=0.7——湿后只剩 0.15
[grip.metal_smooth]     # base_grip=0.3——"这爬不了，找别的路"
```

**天气涌现**：`weather_factor = 1.0 + wet_penalty × rain_intensity`。雨天岩壁变滑——玩家能**感觉到**（grip_drain_rate 加倍）。

## 七、攀爬中战斗

| 条件 | 可用 |
|------|------|
| 单手武器 + can_fight | 攀爬攻击（简化的挥击） |

| 瞬发法术 | ✅ |
| 专注施法 | ❌——不能在墙上读条 |
