## 这是什么游戏？

**与其说这是一个游戏，不如说我们在以游戏的方式创建另一个"人世"。**

驱动这个项目的最初火花，是那些让我们心潮澎湃的幻想——像《龙之信条》那样组建一支冒险小队、像 D&D 那样探索充满未知的地下城、像 Kenshi 那样置身于一个不围着你转的残酷而自由的世界、像日式异世界幻想番剧那样"来到另一个世界开始新人生"。

但 WoWorld 不止于冒险。

你被丢进一个已经运转了千百万年的世界。这个世界以西方剑与魔法为基底，但全球各地的文化元素——东方的、中东的、非洲的、北欧的、美洲原住民的、南亚的……——都可以在这里找到自己的位置。更重要的：这个世界的各个文明可以自己发展、融合、演变出独一无二的文化特色，而非开发者预设的模板。每一棵树、每一条河、每一栋建筑都是由数学和规则生成。

这个世界有**历史**。书本、遗迹、墓碑、纪念碑——不同程度地保留了以往的先民们遗留写下来的活过的证据。不同的人对同一段历史有着不同的记忆和传说。你踏入这个世界，就像一个人第一次从母胎来到现实中那样——世界早就在这里了，你需要自己去理解它。

这个世界的 NPC 不是任务板。他们是被尽可能**完整**地塑造的人——有记忆、预测、情感、思想、人际关系、欲望、习惯、偏好。铁匠不只是"卖武器的NPC"。他可能是三个孩子的父亲，欠商人钱，暗恋隔壁的面包师，在昨天看到一队冒险者路过时想起了自己年轻时没能实现的冒险梦。他会饿、会累、会生气、会记仇、会恋爱、会死。

**你可以做任何事。** 组建冒险小队去无穷远方探索遗迹与奇迹？当然。做一个铁匠度过一生？可以。经商致富？从政掌权？当游侠？当情圣？当法外狂徒？都可以。冒险不是"主线"——在这个世界里，每一种生活方式都是一生的完整形态，拥有平等的地位。

但最重要的是——**这些故事没有一个是我们提前写好的。** 铁匠欠钱、暗恋面包师、冒险梦——这些不是脚本。是铁匠的人格参数、经济系统的供需波动、社会关系的概率互动，共同"长"出了这个故事。我们在做的不是一个装了100小时剧情内容的游戏，而是一个能够像现实世界那样永不停歇地产生新故事的生成器。

> 以往的游戏大多是与"物"打交道——采集、建造、制造。与"人"打交道才是人类的天性，是人类整日离不开的、又爱又恨的、充满张力和无限故事空间的事务。这就是 WoWorld 要做的。

## 核心竞争力

### 1. 完整的"人"（最重要）

每个NPC力求成为一个完整的人——大五人格、情绪引擎（30种复合情绪）、**2000条事件记忆**（上限，含遗忘曲线与记忆重构）、社会关系网、预测与意图识别、欲望与志向、习惯与偏好。他们的行为由概率驱动而非脚本——不可预测所以看起来聪明。与"人"打交道的深度和广度，决定了这款游戏的上限。

> 不是"NPC走到A点说B句话"。  
> 是"NPC因为讨厌邻居，决定搬家到隔壁镇子，在路上遇到土匪，被打伤后对高个子的人产生了持续的心理阴影——而三十年后，他的儿子在酒馆里听到了这件事的另一个版本"。

### 2. 每次新存档都是前所未有的世界

种子决定一切。山脉、河流、聚落、NPC、势力关系——全部由噪声函数和生成规则产生。但这不仅仅是"山的位置不同"——不同世界的文明形态、文化格局、势力版图可能天差地别。一个世界可能诞生出骑士王国与东方王朝并立的格局；另一个世界可能演化出城邦联盟与游牧帝国的对峙；第三个世界可能被某个宗教统一了大半疆域。就像《我的世界》那样，新建存档就是新世界——但这次，世界自己会写历史，每个文明也能自己发展出独一无二的特色文化。

### 3. 历史深度

玩家进入的世界已经运转了千百万年——也可以是零年，从刀耕火种的蛮荒时代起步。但重点是前者：一个有厚重历史的世界。人们对于过去有着相同的、不同的记忆和传说。遗迹和书本中保留着先民活过的证据。这个世界在你到来之前存在了很久，在你离开之后还会继续存在。

### 4. 故事生成器，不是故事容器

不写故事，只写规则。铁匠抢了面包师→面包师传播坏话→铁匠生意变差→铁匠更暴躁——这条因果链我们没有写，但它会发生。传统RPG（如D&D系列）用开发者手搓的剧情来驱动游戏，开发者写的故事结束的地方就是游戏寿命终止的地方。WoWorld 没有这个问题——因为根本没有预设的剧本。游戏寿命不由"写了多少剧情"决定，因为世界自己在源源不断地产生新故事。

### 5. AI增强但不依赖

接入你自己的LLM API Key → NPC说话更自然、游戏日志变成故事。不接？游戏完全正常运转。LLM是锦上花，不是地基。

## 技术栈

| 项 | 选型 | 说明 |
|----|------|------|
| **引擎** | Godot 4.6 LTS | 渲染/UI/音频/输入/物理 |
| **模拟核心** | Rust stable 1.80+ | NPC/世界生成/战斗AI/经济/魔法/载具——GDExtension 集成 |
| **画面** | 3D 低多边形 + 像素纹理 | 对标 Valheim 美学 |
| **地形** | Transvoxel（Rust 自建） | 垂直稀疏 Chunk + Clipmap LOD + 1500m 高程范围 |
| **海洋** | Gerstner 程序化波 | 海:陆 ≈ 7:3，`OceanProvider` trait 预留 FFT 升级 |
| **天空** | 混合体积云 | 2D impostor + 3D 体积密度场（可穿越）+ 噪声高云 |
| **世界尺度** | 25 万 km² 初步（≈英国） | Minecraft 级无限世界为最终目标 |
| **NPC 规模** | 100K+（L1-L4 分层模拟） | L4 为超远距统计层——远方城市人口基数 |
| **战斗** | 半自动（玩家 AI = NPC AI） | 三层模型：本能逐帧→节奏0.5-1s→战略5-30s |
| **平台** | Windows / Linux / macOS | PC 单机 |
| **美术** | AI 生成（Stable Diffusion）+ 手动调整（Blender） | — |
| **LLM** | 可选，玩家自备 Key | 锦上花，非地基 |

> 📘 完整技术规格见 `开发阶段/技术栈方案/001-WoWorld正式技术栈方案v3.md`

## 参考精神

- **龙之信条** — 与性格各异的同伴组成冒险小队的浪漫感
- **D&D（龙与地下城）** — 奇幻冒险的广度、组队探索未知领域的核心体验
- **日式异世界幻想番剧** — "来到另一个世界开始新人生"的沉浸感
- **Minecraft** — 每个存档都是独一无二的新世界；世界本身的多样性就是内容的无限性
- **Kenshi** — 世界不围着你转的自由感；残酷而真实的生存
- **RimWorld** — 需求驱动行为的系统设计；从蛮荒起步建立文明的体验
- **Elona/Elin** — "什么都能做"的广度；全球文化元素在西幻世界中的自然融入
- **Mount & Blade** — 势力政治的动态
- **Valheim** — 低多边形体素的美学可能
- **Dwarf Fortress** — 法则→涌现的设计哲学；世界历史的深度模拟
