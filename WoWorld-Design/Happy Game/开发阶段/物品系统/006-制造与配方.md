# 006 — 制造与配方

## 一、设计原则

**配方是数据，不是代码。** 所有制造配方通过 TOML 定义——物品系统只提供 CraftingRecipe 数据结构和执行框架。具体工艺（熔炼/锻造/酿造）的成败率和品质由技能系统和物品属性共同决定。

**配方发现途径**：学习（师父传授/读书）、实验（投入材料尝试——可能失败或发现新配方）、购买（商人出售配方书）、天启（罕见 event——"在梦中看见了新的锻造法"）。

## 二、CraftingRecipe 数据模型

```rust
pub struct CraftingRecipe {
    pub recipe_id: RecipeId,
    pub name: LocalizationKey,

    // === 输入 ===
    pub inputs: Vec<RecipeInput>,         // 所需材料
    pub required_tools: Vec<ToolRequirement>,  // 所需工具/工作站
    pub required_skill: SkillRequirement, // 所需技能

    // === 输出 ===
    pub output: RecipeOutput,             // 产物

    // === 过程 ===
    pub base_duration: Duration,          // 基础制造时间
    pub quality_formula: QualityFormula,  // 品质计算方式
    pub failure_consequence: FailureMode, // 失败后果

    // === 条件 ===
    pub required_location: Option<LocationRequirement>,  // 需要特定地点
    pub required_condition: Option<ConditionRequirement>, // 需要特定条件（月相/季节/天气）
}

pub struct RecipeInput {
    pub item_def: ItemDefId,              // 需要的物品（模板）
    pub quantity: u32,
    pub consumed_on_success: bool,        // 成功后是否消耗
    pub consumed_on_failure: bool,        // 失败后是否消耗
    pub acceptable_quality_range: Range<Quality>,  // 可接受的原料品质范围
}

pub struct ToolRequirement {
    pub tool_category: ToolType,           // 熔炉/铁砧/织机/炼金台/附魔台/...
    pub min_quality: Quality,              // 最低品质要求
    pub consumes_durability: f32,          // 每次使用损耗
}

pub struct SkillRequirement {
    pub skill: SkillId,
    pub min_level: f32,                    // 最低技能等级
    pub quality_influence: f32,            // 技能对品质的影响权重 (0-1)
}

pub struct RecipeOutput {
    pub item_def: ItemDefId,               // 产物（模板）
    pub quantity: u32,
    pub quality_inheritance: QualityInheritance,  // 产物品质继承规则
}

pub enum QualityInheritance {
    Fixed(Quality),                    // 固定品质
    FromSkill(f32),                    // 纯由技能决定
    FromSkillAndMaterials { skill_weight: f32, mat_weight: f32 },  // 技能+材料加权
}

pub enum FailureMode {
    Nothing,                           // 失败无事发生
    LoseMaterialsOnly,                 // 损失全部材料
    LoseMaterialsAndDamageTools,       // 材料+工具损耗
    CriticalFailure(f32),              // 爆炸/火灾/中毒概率
}

pub enum LocationRequirement {
    AnyForge,                          // 任何熔炉
    SpecificForge(ItemEntId),          // 特定熔炉（"只有王五的熔炉能做到"）
    ElementalSite(Element),            // 元素富集地（水元素位面→冰系炼金）
    RitualSite,                        // 仪式场所
}
```

## 三、主要制造类别

### 3.1 熔炼（矿石 → 金属锭）

```
配方示例: 铁锭
  输入: 铁矿石×3
  工具: 熔炉(ToolType::Furnace)
  技能: Artisan(Smelting), min=5
  时间: 15min 游戏时间
  产物: 铁锭×1
  品质: FromSkillAndMaterials { skill: 0.6, mat: 0.4 }
  失败: LoseMaterialsOnly
```

### 3.2 锻造（金属锭 + 组件 → 武器/防具/工具）

```
配方示例: 铁剑（模块化——生成的是一把参数化的剑）
  输入:
    - 铁锭×4 (刃部)
    - 铁锭×1 (护手)
    - 铁锭×1 (配重)
    - 木材×1 (柄)
  工具: 熔炉 + 铁砧(ToolType::Anvil) + 锤(ToolType::Hammer)
  技能: Artisan(Blacksmith), min=20
  时间: 2h 游戏时间
  产物: Weapon(StraightBlade, 铁, 品质=派生)
  品质: FromSkillAndMaterials { skill: 0.7, mat: 0.3 }
  失败: LoseMaterialsAndDamageTools
```

### 3.3 酿造（材料 → 药水）

```
配方示例: 生命药水
  输入: 治疗药草×2 + 清水×1 + 魔力晶粉×1
  工具: 炼金台(ToolType::AlchemyTable)
  技能: Artisan(Alchemy), min=15
  时间: 30min
  产物: Potion(Healing)×1
  品质: FromSkill { skill_weight: 1.0 }
  失败: CriticalFailure(0.05)  // 5% 概率爆炸
```

### 3.4 裁缝（纤维/皮革 → 衣物/常服/轻甲）

```
配方示例: 皮甲
  输入: 皮革×6 + 麻线×2
  工具: 缝纫台(ToolType::SewingTable)
  技能: Artisan(Tailoring), min=10
  时间: 3h
  产物: Armor(LeatherArmor, Chest)
  品质: FromSkillAndMaterials { skill: 0.5, mat: 0.5 }
```

### 3.5 木工（木材 → 家具/容器/弓/箭）

```
配方示例: 弓
  输入: 紫杉木×1(弓臂) + 橡木×1(握把) + 筋腱×1(弦)
  工具: 木工台(ToolType::WoodworkingBench)
  技能: Artisan(Carpentry), min=25
  时间: 4h
  产物: Weapon(Bow), 参数化——拉重/长度由材料和技能决定
  品质: FromSkillAndMaterials { skill: 0.6, mat: 0.4 }
```

```
配方示例: 烤肉
  输入: 生肉×1 + 盐×1 + 香料×1(可选)
  工具: 篝火(ToolType::Campfire) 或 炉灶(ToolType::Stove)
  技能: Survival(Cooking), min=0
```

## 五、批量制作

NPC 可以委托批量制造——不需要逐件盯着：

```rust
struct CraftingOrder {
    recipe: RecipeId,
    quantity: u32,
    crafter: NpcId,
    station: ItemEntId,  // 使用的工作站
    progress: f32,       // 0.0-1.0
    queue_position: u32, // 队列位置
}
```

NPC 铁匠在工作时间自动从队列中取订单 → 消耗材料 → 产出 → 放入完成箱。玩家可以去取（或等 NPC 通知）。

## 六、制造对经济系统的预留

- 原材料价格 × 技能效率 = 制造成本 → 市场定价基础
- 高品质产物稀缺 → 供不应求 → 高价
- NPC 铁匠技能 ≥80 → 垄断当地精良以上武器 → 地区经济支柱
- 详细留给经济系统设计
