## 一、Life 基类架构

```rust
/// 生命基类——所有生物的最小公共结构
struct LifeEntity {
    id: EntityId,
    species_id: SpeciesId,
    display_name: String,

    // === 生命阶段 ===
    life_stage: LifeStage,
    age_years: f32,
    max_lifespan: f32,
    growth_progress: f32,

    // === 核心数值（所有生命共享）===
    vitals: Vitals,
    physical: PhysicalAttributes,
    damage_pool: DamagePool,

    // === 灵元素/魔法（除亡灵外所有生命都有灵元素）===
    spirit: SpiritState,
    magic: MagicAttributes,

    // === 生态属性 ===
    ecology: EcologicalAttributes,

    // === 心智（按生命类型分层）===
    mind: MindLayer,

    // === 繁衍 ===
    reproduction: Reproduction,

    // === 产物与槽位 ===
    produce: Vec<CreatureProduce>,
    inherent_slots: Vec<CreatureSlot>,

    // === 标签 ===
    tags: HashSet<Tag>,
    status_effects: Vec<StatusEffectInstance>,

    // === 种族特质（查种族表，不存个体上）===
    race_traits: RaceTraitsId,
}
```

## 二、心智分层

| 心智层 | 拥有者 | 内容 |
|--------|--------|------|
| **无** | 植物 | 无任何心智——纯被动 |
| **InstinctiveMind** | 动物/灵兽/怪物 | 本能行为树 + 简单学习 |
| **SapientMind** | 智能种族（人类/精灵/矮人/…） | InstinctiveMind + GOAP + 记忆(2000条) + 情绪引擎 + 关系网络 |
| **DivineMind** | 神明(A模式)/龙 | 超越常规认知——全知或近似全知 |
| **PlayerMind** | 玩家 | 共享 BodyState/RaceTraits——心智由人类输入驱动 |

## 三、生命类型分类

| 类型 | 心智层 | 灵元素 | 可驯服 | 程序化生成 | 说明 |
|------|--------|--------|--------|-----------|------|
| **植物** | 无 | 低 | — | ✅ | 架构分离——无 BodyState/Mind |
| **动物** | InstinctiveMind | 低 | 部分 | ✅ | 含巨兽 |
| **灵兽** | InstinctiveMind+ | 中-高 | ✅ | ✅ | 5种驯服路径 |
| **怪物** | InstinctiveMind | 异常 | ❌ | ✅ | 7种来源+自然动物变体 |
| **亡灵** | MinimalMind | 异常残留 | ❌ | 部分 | 灵元素衰减 |
| **智能种族** | SapientMind | 中-高(可变) | — | ✅ | 人类/精灵/矮人/程序化种族 |
| **玩家** | PlayerMind | 种族决定 | — | — | 智能种族特例 |
| **神明(A)** | DivineMind | 极高 | — | — | 仅A模式存在实体 |
| **特殊造物** | 不定 | 不定 | 不定 | — | 开发者手搓（龙/小精灵） |

## 四、生成顺序

```
世界生成 P2: 生物群系完成
  → 每个群系参数筛选候选积木集
  → 12维积木采样（环境×历史）
  → 硬约束检查（物理不可能→拒绝）
  → 生态学过滤（活不下来→拒绝）
  → 模式模板匹配
  → 美学评分→稀有度分级
  → 物种池 + 个体投放密度
```

## 五、核心原则

- **所有伤害削减数值，所有数值可恢复**——无永久性伤害
- **特质从历史涌现，不预置**——种族参数 = f(环境, 历史, 种子)
- **灵元素普遍存在**——除亡灵外每个生命都有灵元素
- **标签可叠加、不互斥**——一个生命可有多个来源/行为/状态标签
- **群系驱动生物**——不是先生成生物再往群系里塞
