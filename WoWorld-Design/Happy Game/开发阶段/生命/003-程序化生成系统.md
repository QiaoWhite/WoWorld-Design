## 一、12 维度积木体系——完整展开

### 维度一：身体架构 (Body Plan)

决定生物的基本结构——"这是什么形状的生物"。

| 架构类型 | 说明 | 典型代表 | 约束 |
|----------|------|---------|------|
| **二足** (Bipedal) | 两条主肢直立/半直立 | 人类/鸟人/恐龙型 | 需要尾或前倾躯干保持平衡 |
| **四足** (Quadrupedal) | 四条腿——最常见的陆地动物 | 狼/牛/蜥蜴 | 前肢≈后肢长度（或前短后长跳跃型） |
| **六足** (Hexapod) | 六肢 | 巨型甲虫/飞龙(四腿+双翼) | 中间肢对功能分化多 |
| **八足+** (Octopod+) | 多肢 | 蜘蛛/蜈蚣型 | 协调成本——每增加一对肢，运动速度可能反而下降 |
| **蛇形** (Serpentine) | 无肢蜿蜒 | 蛇/鳗鱼/蠕虫 | 摩擦力驱动——冰面上极慢 |
| **半人马型** (Centauroid) | 下四足+上躯干 | 人马/蝎尾狮 | 上下两个功能分区——各自有器官系统 |
| **鸟类型** (Avian) | 前肢特化为翼，后肢行走 | 鹰/鹫 | 翼不能同时用于飞行和抓握（蝙蝠翼除外） |
| **水生型** (Aquatic) | 鳍/鳍肢为主，流线体 | 鱼/鲸/海爬 | 不能在陆地上长时间活动 |
| **软体/无定形** (Amorphous) | 无固定形态 | 史莱姆/软泥怪 | 没有传统意义上的"肢体"和"部位" |
| **多节肢** (Segmented) | 每节独立肢体——可自切可再生 | 蜈蚣/等足类 | 头部节和尾节有功能分化 |
| **悬浮型** (Hovering) | 无行走肢，浮力/魔法悬浮 | 眼球怪/浮游水母/元素精灵 | 需要持续能量（魔法/气体）维持悬浮 |
| **植根型** (Rooted) | 不可移动或极慢移动 | 食人花/孢子塔/珊瑚虫 | 植物与动物的边界——但有简单感官/攻击能力 |
| **集群型** (Swarm) | 多个小个体构成一个"生物" | 蜜蜂群/食人蚁群/魔法尘埃聚合体 | 单个个体HP极低——整体HP=个体HP×存活数 |

### 维度二：体型与比例 (Size & Proportions)

| 维度 | 范围 | 说明 |
|------|------|------|
| **体型等级** | 微型/极小/小型/中型/大型/巨型/超巨型/世界级 | 对标 D&D。每级体积约×8(长宽高各×2) |
| **头身比** | 头占身体 5%-60% | 影响"可爱/威猛"感知。幼体→头大、成体→头占比降 |
| **肢身比** | 腿长/身长 | 修长型(羚羊)→矮壮型(獾) |
| **尾身比** | 尾长/身长 | 0(无尾)→3.0(极长尾——如长尾鸡) |
| **颈长** | 无颈→长颈(长颈鹿/蛇颈龙型) | 影响取食高度/视野范围 |
| **体型轮廓** | 纤细/敦实/流线/扁平/球状/细长 | 不同生态位形状不同 |

### 维度三：外皮与覆盖 (Integument)

| 类型 | 子类 | 功能性 |
|------|------|--------|
| **毛皮** | 短毛/长毛/卷毛/刚毛/刺毛/绒羽 | 保暖(寒冷+30%耐受)、伪装(颜色匹配环境)、触觉(胡须) |
| **鳞片** | 小圆鳞/大板鳞/栉鳞/棱鳞/覆瓦状 | 防御(轻-中)、保水(沙漠+20%)、反光(闪光鳞片→威胁展示) |
| **羽毛** | 绒羽/飞羽/饰羽/防水羽 | 飞行(必须)、保暖、性选择展示 |
| **裸皮** | 光滑/粗糙/皱纹/透薄/厚革质 | 呼吸(两栖动物——通过皮肤)、感知、高灵活度 |
| **甲壳** | 完整龟壳/背甲/骨板/节段甲/钙化壳 | 极高防御——但重量-速度 |
| **外骨骼** | 几丁质/钙化/硅质 | 结构支撑+防御——需定期蜕皮(脆弱期) |
| **粘膜** | 湿润滑腻/干燥粘性 | 保持湿润(离水=持续掉血)、化学防御(分泌毒素) |
| **树皮/木纤维** | 粗糙树皮/光滑竹皮/苔藓覆盖 | 植物生物——可燃性弱点 |
| **结晶/矿物** | 水晶覆盖/岩壳/金属质鳞 | 极高防御+特定元素亲和、极重 |
| **元素外皮** | 火焰鬃毛/冰壳/阴影烟雾/电流纹 | 该元素的亲和≥0.8 才可能出现——同时有对应弱点(火怕水等) |

### 维度四：头部与感官 (Head & Senses)

| 维度 | 变体 |
|------|------|
| **头部数量** | 0(感官分散于身体)、1(绝大多数)、2-3(双头/三头蛇型)、多头(多头龙型——每个头有独立意志概率) |
| **眼睛数量** | 0(盲——洞穴/深海)、1(独眼)、2(标准)、3-4(多视角)、复眼(数千小眼——超强运动侦测/低分辨率) |
| **眼类型** | 简单眼(仅光感)、相机眼(成像——大多数脊椎动物)、复眼(运动侦测)、热感应窝(蛇类——黑暗中"看"热源)、盲(依赖其他感官) |
| **听觉** | 外耳(大/小/可旋转——定位精度高)、无外耳(鼓膜裸露)、震动感知(地下——通过骨骼传导)、回声定位(洞穴/蝙蝠型) |
| **嗅觉** | 鼻子/吻部(大多数)、触角化学感应(昆虫型——追踪信息素)、分叉舌(蛇/蜥蜴型——空中气味立体采样) |
| **口器** | 颌+牙(咬/咀嚼)、喙(啄/撕/嗑)、吸管(吸取液体——蝴蝶/蜂鸟型)、咀嚼口器(昆虫型)、滤食口(水生滤食)、针状口器(吸血/注毒)、无口(吸收——皮肤吸收养分/魔法) |
| **齿列** | 无齿(鸟/龟型)、切齿+臼齿(植食)、犬齿+裂齿(肉食)、毒牙(注毒——有管道)、研磨齿(硬壳食物)、终生替换齿(鲨/鳄型) |
| **头部装饰** | 角(多种形态——直刺/弯曲/螺旋/分叉)、冠/盔(骨盔/角质冠——求偶或防御)、肉垂/肉瘤(性选择信号)、触角(嗅觉+触觉——多种形态)、发光器(诱饵/通信/照明)、扇状饰(展开威胁/求偶) |

### 维度五：肢体与运动 (Limbs & Locomotion)

| 维度 | 变体 |
|------|------|
| **主肢数量** | 0(蛇形/软体)、2(二足)、4(四足)、6(六足——昆虫型/龙型)、8(蜘蛛型)、10+(多足)、可变(集群型——个体数不定) |
| **肢类** | 行走腿(承重/行走)、挖掘肢(强壮前肢+大爪)、攀援肢(长指/吸盘)、游泳肢(鳍/蹼)、抓握肢(手/爪——有拇指或对跖)、翼(飞行——消耗大量体力)、滑翔膜(前肢-后肢间皮膜——不提供动力仅滑翔) |
| **关节结构** | 跖行(掌着地——人类/熊——慢但稳)、趾行(趾着地——猫/狗——快)、蹄行(蹄着地——马/鹿——极速+地面敏感度牺牲)、反向关节(鸟类/蛙——后肢超长+弹跳) |
| **足部** | 爪(多形态——锐爪=攻击/钝爪=挖掘/长爪=攀爬)、蹄(单蹄=马/双蹄=牛/三趾=犀牛)、吸盘(攀爬光滑面)、蹼(水生——增加划水面积)、肉垫(静音潜行)、钩爪(悬挂——树懒/蝙蝠) |
| **翼类型** | 膜翼(蝙蝠/翼龙——翼由指骨撑开薄膜)、羽翼(鸟类——前肢骨+飞羽)、鞘翅+膜翅(甲虫——先张开硬鞘再展开膜翅)、鳍翼(飞鱼/鳐鱼——水中"飞行")、魔法翼(无实体翼——仅由魔力维持——高魔力消耗) |
| **尾类型** | 平衡尾(大多数——奔跑中急转)、抓握尾(猴/蜥蜴——额外的"手")、武器尾(尾锤/尾刺/尾刃——高肌肉密度)、储脂尾(蜥蜴/一些哺乳——食物充足时粗尾)、扇尾(展示——孔雀/雷龙型)、发光尾饵(深海/洞穴——诱捕猎物) |
| **运动方式** | 行走/奔跑、跳跃/弹跳(后肢极发达)、攀爬/摆荡、游泳、飞行(主动拍翼)、滑翔、挖掘/钻地、蛇行/蠕动、短距瞬移(魔法——高消耗——仅高魔力生物) |

### 维度六：攻击与防御 (Offense & Defense)

**攻击手段**：
- **角/獠牙**：直刺角(犀牛型——冲锋一击)、弯曲角(山羊型——锁角角力)、螺旋角(羚羊型)、上獠牙(海象/大象——展示+攻击)、下獠牙(野猪)、锯齿牙板(鲨型——切割)
- **爪/刺**：利爪(肉食——抓/撕)、钝爪(掘地)、毒刺(尾——蝎/蜜蜂型)、背刺阵列(刺猬/豪猪型——物理+视觉双重防御)、触手刺(水母型——远程注毒)
- **尾部武器**：尾锤(甲龙型——尾端骨质锤)、尾刃(剑龙型——尾侧骨质刃)、尾鞭(蜥蜴型——快速抽打产生音爆)、毒尾针(蝎型)
- **化学攻击**：毒液喷射(喷射毒液——有命中距离限制)、酸液(腐蚀——毁装备)、恶臭喷雾(驱赶——非致命)、麻痹毒素(让猎物不能动)、催泪气体(削弱视觉)
- **魔法攻击**：元素吐息(火/冰/雷/酸——特殊器官产生)、魔力冲击(直接用魔力冲击——无特定元素)、诅咒凝视(视觉范围内附加debuff)、催眠/魅惑(低意志 NPC 可能中招)、瞬移袭击(闪到身边攻击——高灵元素消耗)

**防御手段**：
- **物理装甲**：骨板(特定部位)、龟壳(全身可缩入)、鳞甲(柔性——不限制运动)、角质层(厚皮——简单)、矿石甲壳(极重但防御力最高)
- **伪装**：变色(主动改变——情绪也有关系——恐惧时变苍白)、拟态(形态/颜色模拟另一种生物或环境)、透明/半透明(部分不可见——魔法类更多)、静止伪装(完全不动=看起来像石头/树枝)
- **威胁展示**：展开伞状饰/颈膜(伞蜥/眼镜蛇型——让自己看起来很"大")、发光脉冲、声波冲击(震耳+震退)、膨胀体型(快速吸入气体/液体——立即变大一倍)、假眼斑(蝴蝶/蛾型——翅膀上有假眼——恐吓)
- **脱逃手段**：断尾(自动——尾脱落后还在蠕动吸引注意——可再生)、喷墨/烟幕(遮蔽视线后逃)、快速钻地(立即消失在地下)、弹射跳跃(极长腿型——一次跳数十米)、装死(停止一切动作+释放腐败气味)
- **再生**：断肢再生(非即时——需要数日-数周)、快速愈合(战斗中缓慢回血)、分裂(被切成两半=两只——只适用于软体/集群型)

### 维度七：生命周期与繁殖 (Life Cycle)

| 维度 | 变体 |
|------|------|
| **出生方式** | 卵生(硬壳=陆地/软壳=水中)、胎生(体内孕育→产出活体)、卵胎生(卵在体内孵化→产出活体——蛇/鲨)、出芽(母体长出新个体→脱离=独立)、孢子(释放大量微小孢子——低存活率×海量释放)、分裂(一分为二——仅软体/集群)、寄生繁殖(在其他生命体上/体内产卵——宿主不一定死亡) |
| **生长模式** | 变态发育(卵→幼虫→蛹→成虫——每个阶段形态完全不同)、渐进发育(幼体→亚成体→成体——形态相似仅大小变)、持续生长(终生长大——某些爬行类/鱼类/龙——个体年龄=体型)、阶段蜕变(关键事件触发——如灵元素积累到阈值→蜕变到更高级形态) |
| **寿命范围** | 数日(某些昆虫)、季节(一年生)、数年(小型哺乳/鸟类)、数十年(大型哺乳/人类)、上百年(精灵/龙/巨龟)、数千年(龙/巫妖/……)、不死(非亡灵类——神明/某些魔法造物) |
| **每胎/每窝数量** | 单胎(大型生物——大象/龙/人类)、少数 2-5(大多数中等体型)、多胎 6-20(小型哺乳/某些爬行)、海量 20+(昆虫/鱼类/孢子类——绝大多数在幼体阶段被吃掉) |
| **亲代抚育** | 无(生完即离——多数爬行/鱼/昆虫)、单亲抚育(多数哺乳——雌性)、双亲抚育(鸟类——双方轮孵+一起觅食/一些哺乳)、群体抚育(狮子/大象/真社会性——整个群体参与) |
| **衰老表现** | 身体缩小/变弱(体重↓、力量↓)、智慧增长(经验积累↑)、元素亲和力变化(年轻时活跃→老年时沉淀——或反向)、外观变化(毛色变灰白/脱毛/鳞片变钝/骨板增厚/结晶化)、生育能力衰退(达到一定年龄后不可繁殖) |

### 维度八：食性与生态位 (Diet & Ecology)

| 维度 | 变体 |
|------|------|
| **营养级** | 生产者(光合/化合——植物和某些微生物/魔法生物)、初级消费者(植食/滤食——以此为食的动物)、次级消费者(小型肉食/虫食——吃初级消费者)、顶级消费者(大型掠食——几乎无天敌)、分解者(真菌/蠕虫/食腐动物)、寄生者(寄生于其他生命——体型通常小于宿主)、共生者(两个不同物种互相帮助——如清洁鱼+大鱼) |
| **取食方式** | 啃食/割草(大量低营养植物)、浏览(挑选嫩叶/果实——需要较高智力)、追猎(速度+耐力——开阔地形)、伏击(伪装+爆发速度——森林/沼泽)、滤食(水生——通过鳃过滤微生物)、吸食(体液——蚊/吸血蝙蝠型——有麻醉物质防被发现、花蜜——蝴蝶/蜂鸟型)、腐食(捡尸体——嗅觉灵敏度高)、食矿/食土(咀嚼岩石/土壤获取矿物——特殊消化系统)、食魔(以环境魔力为食——不需要物理食物——极罕见——仅纯魔法生物) |
| **食性广度** | 特化(仅一种食物——如考拉仅吃桉叶——脆弱但高效)、窄食(几种——如狮子仅吃特定大猎物)、广食(杂食——什么都吃——适应力最强) |
| **取食周期** | 持续进食(小体型+高代谢——蜂鸟/鼩鼱——每天进食体重数倍)、间歇进食(大型掠食——一顿管数日)、长期休眠间进食(熊冬眠/沙漠蟾蜍——数月-数年旱季休眠+遇雨立即觅食繁殖) |

### 维度九：行为与社会性 (Behavior & Social)

| 维度 | 变体 |
|------|------|
| **社会结构** | 独居(仅繁殖期配对——虎/豹/大多数爬行)、配对(一雄一雌——长期或仅此季——狼/天鹅/很多鸟类)、小群 3-10(家族群体——狮子狮群)、大群/兽群(野牛/角马/沙丁鱼——数量安全)、等级群体(有首领/α——狼群/灵长类——首领有优先食/交配权)、真社会性(蚁/蜂——有工/兵/后——无法独立生存)、集群(无组织结构的大聚合——蝗虫群/某些鱼群) |
| **领域性** | 无领域、小巢域(仅守卫巢周边——鸟巢半径30m)、中等领域(主动巡逻——狼群领域数km²)、大迁徙范围(角马/驯鹿——数百-数千km年迁徙——无处是"自己的领地") |
| **活动节律** | 昼行、夜行、晨昏行(仅在日出/日落时活跃)、潮汐节律(海岸生物——按潮涨落)、无节律、仅特定季节活跃(冬眠/夏眠/雨季限定) |
| **性情** | 被动(你不惹它它不理你——牛/鹿/兔)、警觉(见人就跑——大多数小动物)、防御性(近巢才攻击——筑巢鸟/母熊+幼崽——"你靠近了她的孩子")、领地攻击(进入领地=攻击——河马/某些鱼/很多雄性有蹄类——"你闯入了他的地盘")、主动掠食(视一切合适的为食)、好奇(主动接近观察——猴/鸦科/某些高智力生物——"这什么？能吃吗？"——会试吃各种东西)、顽皮(高智力—非必需行为——海豚/鸦科/某些灵长类——会玩——智力指标) |
| **智力等级** | 本能反应(无学习——昆虫/大多数爬行/两栖)、简单学习(条件反射——多数哺乳/鸟类——可被训练——怕/不怕)、工具使用(少数——猩猩/海獭/某些鸦科——用石头/树枝获取食物)、语言理解(高智力——能理解简单符号/命令——灵兽/猿/海豚——人类语言的部分理解)、抽象思维(智能种族级别——概念/未来/过去/他人心智——人类/精灵/矮人等) |
| **睡眠模式** | 单相睡眠(一次长眠——人类/大多数昼行动物)、多相睡眠(多次短眠——猫/狗——"打盹")、半脑睡眠(水生哺乳——海豚/鲸——一半脑睡另一半醒着游+呼吸)、冬眠/夏眠/旱季蛰伏(代谢降到极低——环境极端时) |
| **特殊行为** | 囤积/筑巢(积累食物或建材——乌鸦收集闪光物/海狸建坝)、求偶仪式/展示(特定动作/装饰/声音——筑巢/献食/跳舞/比美/角力)、迁徙(季节固定路线——鸟/鱼/有蹄类)、利他行为(群体防御时牺牲自己/报警鸣叫暴露自己但提醒群体)、游戏/探索(不需要生存目的——就是玩——高智力标志) |

### 维度十：魔法与元素 (Magic & Elements)

| 维度 | 变体 |
|------|------|
| **灵元素含量** | 0(亡灵/无生命物体)、低 0.05-0.15(植物/低等动物)、中 0.15-0.39(高等动物)、高 0.4-0.7(智能种族/灵兽/灵兽精英)、极高 0.7-1.0(龙/神明/元素化身) |
| **十元素亲和** | 每元素独立的 [0.0, 1.0]——从环境+历史涌现。不是"天生火系"——是在火山环境生活数百代后火元素亲和逐渐上升。亲和 ≠ 能力——它决定了学该元素魔法的效率+该元素魔法的威力加成 |
| **魔力运用方式** | 无(被动——仅灵元素存在)、天生固定能力(喷火/放电/冰冻吐息——不是"施法"是"器官功能"——龙/某些灵兽)、可学习施法(与智能种族相同——需要智力+训练+文化传承)、魔力身体(身体由元素构成——火元素生物/冰元素生物——hp=魔力存量) |
| **魔力来源** | 自身灵元素转化(消耗 hunger——转化效率=灵元素容量×魔力控制)、环境魔力吸收(高魔力浓度区自然恢复——类似"呼吸魔力")、食物魔力摄取(吃魔法植物/魔法生物——灵元素含量越高食物恢复越多)、契约/神明授予(仅A模式——神明可向信徒授予额外魔力) |
| **魔法消耗** | 体力消耗(天生能力轻度使用——消耗 stamina)、魔力消耗(主动施法——消耗 spirit)、生命消耗(禁术——消耗 max_hp 上限——不可逆——除非极特殊治疗)、环境魔力借用(就地吸取——仅在魔力浓度≥0.7区域可行) |

### 维度十一：环境适应 (Environmental Adaptation)

| 维度 | 变体 |

| **温度适应** | 极寒适应(-60°C可存活——厚毛皮/厚脂层/短肢短耳/冬眠) ↔ 酷热适应(55°C可存活——大耳散热/少毛/夜行/储水) ——每个生物有一个可存活温度范围 |
| **湿度适应** | 极致干旱(沙漠——从食物中获取水分、极少排尿) ↔ 完全水生(鳃或皮肤呼吸、无法离开水超过数分钟) |

| **光照适应** | 全日照 ↔ 完全黑暗(退化眼/无眼+强化其他感官+可能自发荧光) |

| **特殊环境** | 火山/岩浆(耐高温+耐硫+踩岩浆不是即死——但也不是无限时间)、冰盖(耐寒+防滑+钻入冰层)、浮空岛(抗低压+高效呼吸+抗风——可能演化出翼/滑翔)、深渊裂隙(完全黑暗+高压+奇异的地质化学合成生态链→不需要阳光) |

## 二、组装规则

```rust
/// 程序化生物生成器——完整流程
struct CreatureAssembler {
    parts_catalog: PartsCatalog,         // 12维积木目录
    biome_constraints: BiomeConstraints, // 群系→允许的积木和形态空间
    evolutionary_pressure: EvolutionaryPressure, // 历史积累的适应压力
}

fn generate_creature_for_biome(
    biome: &BiomeType,
    history: &BiomeHistory,
    rng: &mut StdRng,
) -> Creature {
    // 1. 群系筛选合法积木候选集
    //    寒冷 → 毛皮/厚脂/短肢/小耳/白色
    //    洞穴 → 盲/回声定位/苍白/触觉强化
    //    沙漠 → 鳞片/裸皮/夜行/储水器官/大耳散热
    let candidates = biome.filter_legal_parts(&PARTS_CATALOG);

    // 2. 历史微调权重
    //    曾有大火 → 耐火外皮权重↑
    //    被大型掠食压力 → 伪装/速度/夜行权重↑
    //    长期无掠食 → 笨重体型/缓慢繁殖权重↑
    let weighted = history.adjust_weights(candidates);

    // 3. 选择身体模式模板(~15-20个预定义模式)
    //    模板约束比例范围——保证"看起来像生物"
    let template = select_body_template(biome, rng);

    // 4. 在模板约束范围内随机采样各部件的具体参数
    let parts = template.sample_parts(&weighted, rng);

    // 5. 硬约束——绝对矛盾=拒绝重采
    validate_hard_constraints(&parts)?;

    // 6. 生态学过滤——活不下来=拒绝重采
    if !ecology_filter(&parts, biome) {
        return generate_creature_for_biome(biome, history, rng);
    }

    // 7. 美学评分→稀有度
    let score = aesthetic_score(&parts);
    let rarity = score_to_rarity(score);

    // 8. 组装 + 命名（程序化命名——如"灰鬃狼""水晶角鹿"）
    Creature::assemble(parts, rarity, generate_name(&parts, rng))
}
```

## 三、四层质量防线——详解

### 第一层：硬约束规则

在生成前排除不合法组合。矛盾 = 拒绝 + 重新采样。

```
类型级矛盾:
  ✗ 膜翼 + 羽毛 → 两种翼type不能共存
  ✗ 甲壳(整体覆盖) + 毛皮 → 外皮 type 互斥
  ✗ 外骨骼 + 内骨骼 → 结构体系互斥
  ✗ 植根型 + 任何行走肢
  ✗ 软体 + 骨板(需要有骨骼才能支撑骨板)

力学级矛盾:
  ✗ 巨型体型 + 极细腿(腿直径/体长 < 0.05)
  ✗ 飞行 + 超巨型(翼展至少=体重×重力/升力系数——巨型生物需要超大的翼或魔法翼)
  ✗ 快速奔跑 + 极短腿(步幅/体长 < 0.15)
  ✗ 集群型 + 巨型(单个体不能同时是集群型又是巨型——一个集群由很多小个体组成)

功能级矛盾:
  ✗ 滤食口 + 顶级掠食生态位
  ✗ 盲 + 昼行 + 无替代感官
  ✗ 无肺/无鳃 + 无魔法呼吸（水生/陆生都需要气体交换器官或魔法替代）
  ✗ 食魔生物 + 低魔法环境 → 饿死——除非该群系有高魔力浓度

每块积木有一个 compatibility_mask 位域:
  if trait_a.mask & trait_b.forbidden != 0 → reject
```

淘汰率约 15-25%。

### 第二层：生态学过滤

检查生成的生物"能不能在这个群系中活下去"。

```rust
fn ecology_filter(candidate: &CreatureSpec, biome: &BiomeState) -> bool {
    // 1. 食物: 它的取食方式×取食对象在此群系中是否存在？
    if !biome.has_food_for(candidate.diet) { return false; }

    // 2. 防御: 此群系有掠食者吗？它能逃跑或防御吗？
    if biome.has_predators() && candidate.defense_score < MIN_DEFENSE { return false; }

    // 3. 繁殖: 寿命×繁殖率是否足以维持种群？
    if candidate.reproductive_viability() < VIABILITY_THRESHOLD { return false; }

    // 4. 气候: 温度/湿度/压力是否在其耐受范围内？
    if !candidate.can_survive_climate(biome.climate) { return false; }

    // 5. 生态位: 是否与已有物种完全重叠？
    if niche_overlap(candidate, biome.existing_species) > OVERLAP_LIMIT {
        // 不完全拒绝——降低密度（竞争）
        candidate.population_density *= 0.3;  // 竞争惩罚
    }

    true
}
```

淘汰率约 60-80%——这是最主要的一层。

### 第三层：模式模板

不是自由拼装——限定在自然界反复出现的身体模式中。模板定义"这个架构的合法比例范围"。

~15-20 个模板：
```
A.  四足兽模式(最通用陆地模式)     H.  蛇/蜿蜒模式
B.  二足直立模式(类人型)           I.  鱼/水生模式
C.  鸟/飞行模式(前肢=翼)           J.  昆虫/节肢模式
D.  半人马模式(四足+上躯干)        K.  软体/无定形模式
E.  多足模式(6+)                  L.  悬浮型模式
F.  半水生模式(四足+鳍状肢)        M.  集群型模式
G.  掘地模式(短肢+大爪+圆柱体)     N.  植根型模式
                                    O.  龙型模式(四足+翼——专属——极稀有)
```

模板不决定外观——它约束的是"这个架构的合理比例"。模板内具体尺寸/颜色/材质可自由变化。

### 第四层：美学评分与稀有度

```rust
fn aesthetic_score(creature: &CreatureSpec) -> f32 {
    let mut score = 0.0;

    // 1. 比例和谐度 (+0~0.20)
    //    各部位比例越接近数学上"悦目"的比例（如黄金分割）
    score += proportion_harmony(creature.proportions);

    // 2. 色彩一致性 (+0~0.20)
    //    外皮颜色是否构成互补/类似色协调
    score += color_harmony(creature.colors);

    // 3. 功能一致性 (+0~0.30)
    //    所有部件是否指向同一个生存策略
    //    利爪+锐齿+前向眼+短肠+敏捷→捕食者——一致——高分
    //    利爪+磨齿+侧向眼+长肠→攻+植食混合——矛盾——低分
    score += functional_coherence(creature);

    // 4. 轮廓可识别性 (+0~0.20)
    //    剪影是否容易识别？有独特轮廓高于混沌轮廓
    score += silhouette_distinctiveness(creature);

    // 5. 生态位契合 (+0~0.10)
    score += niche_fitness(creature, creature.biome);

    score  // 0.0(诡异) ~ 1.0(自然优美)
}
```

```
评分 → 稀有度 → 出现频率:

0.8-1.0 → 常见——随处可见(自然优美——大多数生物应该在这里)
0.5-0.8 → 不常见——经常能看到但不密集
0.3-0.5 → 稀有——偶尔遇到——"奇特但真实"(鸭嘴兽级)
0.1-0.3 → 极稀有——罕见——发现本身就是奖励
0.0-0.1 → 传说级——整个世界只有极少数——需要特殊条件才会出现
```

**关键——不是拒绝低分生物。自然界中鸭嘴兽也是真实的。低分=稀有=发现即奖励。**
