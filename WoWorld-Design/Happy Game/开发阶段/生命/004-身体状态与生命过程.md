# 004-身体状态与生命过程

> **状态**: 开发规格 v1.0
> **关联**: [[001-生命总纲]] · [[012-繁衍系统]]

---

## 一、核心生命值 (Vitals)

| 数值 | 范围 | 归零后果 |
|------|------|---------|
| `hp` | 0 ~ max_hp | 濒死→死亡 |
| `stamina` | 0 ~ max_stamina | 行动不能 |
| `hunger` | 0.0(饿死) ~ 1.0(饱) | 体力衰减→死亡 |
| `thirst` | 0.0(脱水) ~ 1.0(不渴) | 死亡(比饥饿快) |
| `fatigue` | 0.0(精力充沛) ~ 1.0(需睡眠) | >0.9→属性衰减→昏睡 |
| `body_temperature` | 种族正常值±耐受范围 | 过低→冻死 / 过高→热死 |
| `spirit` | 0 ~ spirit_capacity | 0→不可逆死亡(无法成亡灵) |

## 二、身体属性 (Physical)

`strength` / `agility` / `constitution` / `perception` / `dexterity` / `endurance` ——全部 0.0-1.0，由种族+体型+历史涌现。

## 三、心智属性 (Mental——智能种族/灵兽专有)

`intelligence` / `wisdom` / `willpower` / `charisma` / `memory_capacity`(0-2000)

## 四、魔法/灵元素

`spirit_capacity`(0-1) / `elemental_affinity[10]`(十元素各 0-1) / `magic_power` / `magic_control` / `magic_resistance` / `mana_regen_rate`

## 五、身体部位伤害

由形态模板决定部位集：`head_damage` / `torso_damage` / `left_arm_damage` / `right_arm_damage` / `left_leg_damage` / `right_leg_damage` / `tail_damage` / `wing_damage` / `[N]_limb_damage`(动态)

所有部位伤害削减数值、所有数值可治愈。

## 六、生命阶段

`Juvenile`(幼体 属性15-30%) → `Adolescent`(少年 50-70%) → `YoungAdult`(青年 90-100%) → `Adult`(成体 100%) → `MiddleAge`(中年 -0~10%) → `Elder`(老年 -10~50%)

## 七、生态属性

`size_class` / `diet_type` / `activity_rhythm` / `sociality`(0-1) / `territoriality`(0-1) / `aggression`(0-1) / `curiosity`(0-1) / `fearfulness`(0-1) / `trainability`(0-1) / `temperature_min/max` / `humidity_min/max` / `pressure_tolerance`(0-1) / `light_tolerance`(0-1) / `magic_tolerance`(0-1)

## 九、死亡原因

`VitalDamage`(HP归零) / `Starvation`(饥饿) / `Dehydration`(口渴) / `Hypothermia`(冻死) / `Hyperthermia`(热死) / `SpiritDepletion`(灵元素枯竭) / `OldAge`(寿终) / `Falling` / `Drowning` / `Magic`

## 十、生命产物与槽位

### 活体可采集资源
奶/毛/蛋/羽毛/丝/蜜/树脂/粪便/魔力结晶/元素精华等——由物种决定，定期可采集。

### 固有槽位（身上装载物品，不依赖装备）
`Mouth`(叼) / `Back`(背) / `Claws`(抓) / `Tail`(缠) / `Pouch`(袋) / `Gizzard`(吞储) / `Tentacle`(持) / `Magic_Anchor`(魔法悬浮)

### 死亡掉落
肉/皮/骨/角/鳞/翼膜/毒囊/甲壳/元素残片/鞍具(如曾驯服)。生前状态影响掉落的数量和质量（火烧=皮不可用但肉烤熟；诅咒=掉落物带诅咒属性）。
