## 一、分类

### 1.1 基础动物（保底——每个世界都有）

现实常见的——作为生物多样性的基准线。猫/狗/牛/羊/猪/狼/狮/虎/象/鹿/兔/狐/熊/鹰/蛇/蛙/鱼(多种)/鸡/鸭/鹅。

### 1.2 程序化动物（每个世界不同）

由 12 维积木拼装——群系适应+历史涌现。每个种子世界有自己独特的物种组合。程序化动物有程序化名称——"灰鬃狼""水晶角鹿""沙鳞蜥"等——由形态特征自动生成。

### 1.3 巨兽（可程序化生成）

大体型(大型以上)——类似怪物猎人中的狩猎对象。不是"怪物"——是自然生态中的顶级消费者。巨兽的覆盖密度远低于普通动物——每数十 km² 才有一头。

## 二、心智：InstinctiveMind

本能行为树 + 简单条件反射学习。无语言、无抽象思维、无"未来规划"。

典型行为树结构：
```
┌─ 生命维持（高优先级）
│   ├─ HP<30% → 逃跑/装死
│   ├─ hunger<0.3 → 觅食
│   ├─ thirst<0.3 → 找水
│   └─ fatigue>0.85 → 回巢/找安全处睡眠
│
├─ 安全（中优先级）
│   ├─ 感知威胁 → 警觉/逃跑/防御
│   ├─ 幼崽在场 → 防御性↑+不逃跑
│   └─ 领地内出现同类同性 → 攻击或退让（由体型差决定）
│
├─ 社会（低优先级——仅社会性>0.5）
│   ├─ 离群 → 回到群体
│   ├─ 首领竞争 → 角力（非致命——仪式化）
│   └─ 社交梳理 → 同种间互相清理
│
└─ 繁殖（季节性）
    ├─ 繁殖季节+无伴侣 → 寻找配偶
    ├─ 有伴侣 → 筑巢/交配
    └─ 有幼崽 → 抚育行为（喂食/保护/教捕猎）
```

## 三、种群动态

物种层面驱动。个体仅在 L1（玩家 50m 内）具体模拟。

```rust
fn tick_population(species: &Species, biome: &BiomeState, delta_days: f32) -> PopulationChange {
    let current = species.population_in(biome);
    let carrying = biome.carrying_capacity_for(species);
    if current < carrying * 0.7 { PopulationChange::Growth }
    else if current > carrying * 1.3 { PopulationChange::Decline }
    else { PopulationChange::Stable }
}
```

## 四、生态位

每种动物占据一个生态位——由食性+体型+行为+活动节律推导。同一群系内两个物种的生态位重叠 >70% → 竞争 → 一种逐渐淘汰另一种（或二者共存但密度都降低）。

生态位包括：取食高度(地面/灌木/树冠)、活动时间(昼/夜)、主要食物种类、体型等级。例如：同一森林中——白天吃灌木叶的小型鹿 vs 晚上吃落果的中型野猪——生态位不同，可以共存。

## 五、与灵兽的边界

灵兽 = 动物 + 灵元素 ≥ 0.4 + 可驯服标签 + 智能≥Trainable。

不是所有高智能动物都适合驯服——虎有高智能(可学习)但领地性极强且是大型掠食者→仅可通过挑战征服路径驯服。兔灵元素低(0.1)→不够格成为灵兽——它是普通动物。

详见 [[006-灵兽]]。

---

## 六、与怪物的边界

怪物 = 动物 + 异常来源。正常狼 vs 魔狼(同一基础物种+骨刺增生+夜光眼+元素泄漏)——同一个群系中两者并存。详见 [[007-怪物]]。
