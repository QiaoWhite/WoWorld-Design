## 一、分类

| 层级 | 说明 |
|------|------|
| **基础动物** | 现实中常见的——猫/狗/牛/羊/猪/狼/狮/虎/象（保底） |
| **程序化动物** | 由12维积木拼装——群系适应+历史涌现 |
| **巨兽** | 大体型(大型以上)——可程序化生成。类似怪物猎人中狩猎对象的生态地位 |

## 二、心智

`InstinctiveMind`——本能行为树 + 简单条件反射学习。无语言、无抽象思维。

## 三、种群动态

个体不逐帧模拟。物种层面用种群动态模型：

```rust
fn tick_population(species: &Species, biome: &BiomeState, delta_days: f32) -> PopulationChange {
    let current = species.population_in(biome);
    let carrying = biome.carrying_capacity_for(species);
    if current < carrying * 0.7 { PopulationChange::Growth }
    else if current > carrying * 1.3 { PopulationChange::Decline }
    else { PopulationChange::Stable }
}
```

仅在 L1（玩家附近）具体模拟个体行为——幼崽、巢穴、捕猎场景。

## 四、生态位

每种动物占据一个生态位——由 12 维度中的食性+行为+体型推导。同一群系内生态位过度重叠→竞争→一种逐渐淘汰另一种。

## 五、与灵兽的边界

灵兽 = 动物中灵元素含量 ≥ 0.4 + 可驯服 + 基础智能。不是所有高智能动物都是灵兽——虎高智能但不可驯服。详见 [[006-灵兽]]。
