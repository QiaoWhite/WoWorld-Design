## 〇、核心定位

动物是 WoWorld 中数量仅次于植物的生命类型。它们共享 Life 基类 + InstinctiveMind 心智层，通过 12 维积木体系程序化生成。动物跨越**陆地、水体、天空**三个域——这不是三个独立的分类，而是同一条 12 维管线选择不同身体模板和维度选项的结果。

本文档覆盖全部三个域——从共享基础开始，分别展开各域的物理约束、生态位分层、运动策略和 AI 行为。

## 一、共享基础——所有动物适用

### 1.1 分类

动物分为三类，全部由 12 维积木管线驱动：

| 类别 | 生成方式 | 实例数(25 万 km²) | 说明 |
|------|---------|-----------------|------|
| **基础动物** | 保底——每个世界都有 | ~10-50 万 | 现实常见的——猫/狗/牛/羊/猪/狼/狮/虎/象/鹿/兔/狐/熊/鹰/蛇/蛙/鱼(多种)/鸡/鸭/鹅。作为生物多样性的基准线 |
| **程序化动物** | 12 维积木拼装——群系适应+历史涌现 | ~50-200 万 | 每个种子世界有独特的物种组合。程序化名称——"灰鬃狼""水晶角鹿""沙鳞蜥"等——由形态特征自动生成 |
| **巨兽** | 可程序化生成——大体型 | ~数千 | 类似怪物猎人中的大型狩猎对象。不是"怪物"——是自然生态中的顶级消费者。密度极低——每数十 km² 一头 |

### 1.2 心智：InstinctiveMind——水陆空共享

所有动物（陆生、水生、飞行）共享同一套 InstinctiveMind 行为树。不因"在水中"或"在空中"而需要不同的 AI 框架：

```
典型行为树结构（水陆空通用——仅目标类型不同）:

┌─ 生命维持（高优先级）
│   ├─ HP<30% → 逃跑/装死（水生=深潜/飞行=飞远）
│   ├─ hunger<0.3 → 觅食（水=追踪鱼群/飞行=寻找猎物或栖息地）
│   ├─ thirst<0.3 → 找水（水生=不需要——从环境中摄取/从食物中获水）
│   └─ fatigue>0.85 → 回巢/找安全处睡眠（水=底栖休息处/飞行=栖息地）
│
├─ 安全（中优先级）
│   ├─ 感知威胁 → 警觉/逃跑/防御
│   ├─ 幼崽在场 → 防御性↑+不逃跑
│   └─ 领地内出现同类同性 → 攻击或退让（由体型差决定）
│
├─ 社会（低优先级——仅 sociality>0.5）
│   ├─ 离群 → 回到群体（水=鱼群/飞行=鸟群）
│   ├─ 首领竞争 → 角力（非致命——仪式化）
│   └─ 社交梳理 → 同种间互相清理
│
└─ 繁殖（季节性）
    ├─ 繁殖季节+无伴侣 → 寻找配偶
    ├─ 有伴侣 → 筑巢/交配
    └─ 有幼崽 → 抚育行为（喂食/保护/教生存技能）
```

### 1.3 种群动态——水陆空共享公式

```rust
fn tick_population(species: &Species, biome: &BiomeState, delta_days: f32) -> PopulationChange {
    let current = species.population_in(biome);
    let carrying = biome.carrying_capacity_for(species);
    // capacity 计算随地/水/空域不同——在各自部分展开
    if current < carrying * 0.7 { PopulationChange::Growth }
    else if current > carrying * 1.3 { PopulationChange::Decline }
    else { PopulationChange::Stable }
}
```

### 1.4 生态位——含深度层和高度层

每种动物占据一个生态位——由食性+体型+行为+活动节律推导。同一群系内两个物种的生态位重叠 >70% → 竞争 → 一种逐渐淘汰另一种（或二者共存但密度都降低）。

生态位维度扩展（加上水体和空域独有的深度/高度层）：

```
陆地生态位维度:
  取食高度: 地下/地表/灌木(<3m)/低树冠(3-10m)/高树冠(>10m)
  活动时间: 昼/夜/晨昏行
  主要食物种类
  体型等级

水体生态位维度(额外):
  深度层: 水面/浅水(<30m)/中层(30-200m)/深层(200-1000m)/深渊(1000m+)/底栖
  盐度: 淡水/河口/近海/远洋
  运动自主性: 主动游泳/被动漂流/固着(成年)

空中生态位维度(额外):
  飞行高度: 低空(<50m)/中空(50-500m)/高空(500m+)
  飞行方式: 拍翼/滑翔/悬停
  栖息地: 树冠/悬崖/地面/水面
```

### 1.5 主域 + 跨域能力——不是独立分类

每个动物有一个**主域**（PrimaryDomain），跨域能力通过**附加标签**表达：

```rust
enum PrimaryDomain {
    Terrestrial,   // 主要在地面——可能游泳或短距飞行
    Aquatic,       // 主要在水中——可能爬上岸（两栖/海爬）
    Aerial,        // 主要在空中——必须落地但觅食/交配在空中
}

struct CreatureTraits {
    // ... 已有字段
    primary_domain: PrimaryDomain,
    
    // 跨域能力——从主域溢出的能力组合
    can_swim: bool,
    max_swim_duration: f32,      // 水中最长停留(憋气时间——0=不能下水)
    can_fly: bool,
    flight_mode: Option<FlightMode>,
    under_water_breathing: bool, // 水下呼吸(鳃或魔法)
}
```

跨域不是身份标签——是**能力组合**。例子：
- **鸭子** = Terrestrial + can_swim=true + can_fly=true（水鸟——三个域都能活动但主域是陆地筑巢）
- **企鹅** = Terrestrial + can_swim=true + can_fly=false（用翼游泳——不会飞）
- **飞鱼** = Aquatic + can_fly=true + flight_mode=Glide（出水滑翔——数秒——极短飞行）
- **蛙** = Terrestrial + can_swim=true + under_water_breathing=true(蝌蚪阶段)
- **海豚** = Aquatic + can_swim=true + 哺乳返回海洋（完全水生但呼吸空气——憋气潜水）

### 1.6 NPC 如何获取动物产物——统一底层

```
获得动物产物的四种方式(水陆空共享底层):
  
  狩猎(陆地):   追踪/伏击/陷阱 → 杀死野生动物 → 肉+皮+骨+角
  钓鱼(水生):   抛竿/渔网/鱼笼 → 鱼+贝+偶尔稀有物
  捕鸟(空中):   网/陷阱/弹弓 → 小鸟+蛋 —— 小型+罕见(不如狩猎和钓鱼普遍)
  驯养屠宰:     养殖的家畜 → 定期宰杀 —— 和野生狩猎不同的稳定经济模式

共同底层:
  产物 = 动物物种 × 采集工具 × 技能加成
  写入记忆(位置+物种+产量)
  种群密度 → 采集压力 → 如果持续过度=种群下降
```

## 二、陆地动物

### 2.1 陆地运动

```
陆地运动约束——重力主导+地面摩擦:
  行走/奔跑: 能量消耗随速度线性增长
           耐力消耗 = f(体型, 腿长, 地面类型)
           硬地面(岩石/道路)=快，软地面(沙/雪/泥)=慢
           爬坡消耗↑——坡度>30°=显著减速
           下坡消耗↓——但坡度>45°=滑倒风险

  跳跃/弹跳: 爆发消耗——跳距= f(后肢力量, 体型)
           跳后需要几秒恢复不能再次跳
           
  挖掘/钻地: 极慢移动——仅适应性地面的生物
           挖掘时暴露(不能同时防御)

  攀爬: 树冠/悬崖——特殊适应(长指/吸盘/钩爪)
        树冠层=第二个地面——树上生物几乎从来不下地
```

### 2.2 地表生态位分层

```
地下(<0m):     穴居动物——鼹鼠/蚯蚓/蛇——依赖震动感知——几乎不出地面
地表(0-1m):    多数陆生动物——从昆虫到象——最密、竞争最激烈
低灌丛(1-3m):  小型鹿/兔/雉——在灌丛中隐藏——避掠食者
高灌丛(3-5m):  猴/浣熊/鸟类——半树上——敏捷
树冠层(>5m):   松鼠/树懒/树栖蟒/大量鸟类——树上=安全+食物
```

陆地部分相对简单——WOWorld 的技术栈和现有设计主要覆盖此域。下两个域是本次补充的重点。

## 三、水生生物

### 3.1 水的物理约束

```
水 vs 空气的关键差异:

  密度: 水比空气密 ~800 倍
        → 运动阻力远大于陆地
        → 流线型 = 最重要的形态约束(水阻力 ∝ v²)
        → 能耗随速度立方增长 —— 快速游泳极其昂贵

  浮力: 浮力抵消重力
        → 体型上限 > 陆地(蓝鲸 ~200 吨 >> 任何恐龙)
        → 不需要支撑自重的骨骼结构 = 更多能量用于生长
        → 但: 浮力需要主动管理(浮力策略——下详)

  水压: 每 10m 深度增加 ~1 个大气压
        → 浅水生物和深水生物完全不同的生理需求
        → 深水生物不能快速上升(减压=组织内气体膨胀=死亡)
        → 玩家同理——深潜=需要魔法辅助

  溶氧: 水中的氧气远少于空气
        → 水生呼吸(鳃=从水中提取氧)比肺呼吸效率低
        → 温水含氧 < 冷水(热带水域=氧气更少)
        → 死水区(无水流)=溶氧骤降 = 大多数鱼无法存活

  透光: 光在水中快速衰减
        → 清水: 藻/微粒少 → 可见 20-60m
        → 浊水: 沉积物 → 可见 <1m
        → 深层: 永久黑暗
        → 玩家视野 = 由浊度参数直接控制

  盐度: 淡水/海水 = 生理性硬边界
        → 绝大多数物种不可跨越
        → 仅少数广盐性物种适应河口(盐度波动)
```

### 3.2 浮力策略——水生运动的核心变量

浮力策略决定了水生动物的移动方式和能做什么。这不是 AI 选择——是物种在生成时确定的固有属性，由身体形态和生理推导。

```
负浮力(NegativeBuoyancy):
  不游就沉 —— 无鱼鳔或油肝不足以抵消体重
  代表: 鲨鱼/鳐/某些快速巡游鱼类
  行为: 必须持续游泳维持深度
        休息=沉底(在海底停下来)
        快速深度变化容易(不需要调整浮力——直接上游或潜)
        持续游=持续消耗 stamina —— 不休息=疲劳累积更快

中性浮力(NeutralBuoyancy):
  静止悬浮 —— 鱼鳔充气/放气调节
  代表: 硬骨鱼(石斑鱼/隆头鱼/大多数珊瑚鱼)
  行为: 维持当前位置零能耗
        但: 深度变化需要调整鱼鳔(慢——分钟级)
        快速上浮→鱼鳔膨胀→可能损伤或死亡
        可以"发呆"——悬停在水中不动

正浮力(PositiveBuoyancy):
  不游就浮 —— 厚脂肪/气囊/毛皮困空气
  代表: 海獭(皮毛困空气)/鲸(厚脂肪)/某些鱼(油肝)
  行为: 休息=浮在水面(可呼吸)
        下潜=主动游泳对抗浮力(持续消耗)
        上浮=自由(省力)

动态浮力(DynamicBuoyancy):
  靠游泳产生升力 —— 胸鳍像机翼——水流过=升力向上
  代表: 快速游动的鲨/金枪鱼
  行为: 低速时升力消失——必须加速或下沉
        高速时=几乎不费力维持深度
        但不能"静止"——永远在动
```

**浮力对游戏行为的影响**：玩家看到一条鲨鱼不断游、一条石斑鱼静静悬在礁石旁——**不是 AI 行为代码不同**。是浮力策略的固有属性导致。不需要写"鲨鱼永远在动"的 AI——只要标记鲨鱼浮力=负浮力，基础行为 "长时间不动=沉底" 自动产生永远在游的行为。

### 3.3 游泳推进——效率决定行为

```
推进方式          效率    极速      巡航速度   代表
────────────────────────────────────────────────
尾鳍侧摆(鲔鱼型)   最高    80km/h    40-50      金枪鱼/剑鱼/某些鲨
                  几乎不摆身体——仅尾柄+鳍——超高效巡游

尾鳍+身体全摆      中      30km/h    10-15      鲑鱼/鳟鱼/多数鱼
                  整条身体波动——加速快但低速效率差

胸鳍摆动(鳐型)     中低    15km/h    3-8        鳐/隆头鱼/鹦哥鱼
                  像"水下飞行"——极机动——适合珊瑚礁复杂地形

波动推进(蛇型)     低      10km/h    2-5        鳗鱼/海蛇/某些幼鱼
                  可后退+钻进狭缝——极高机动但不适合追猎

喷射推进           低(爆发) 40km/h    N/A        鱿鱼/章鱼/水母/鹦鹉螺
                  瞬间加速+滑行——不能巡航——伏击型

鳍肢划水(半水生)   极低    10km/h    2-4        海龟/企鹅/水獭/海蛇(部分)
                  需返回水面呼吸——不能在水下待太久
```

**推进方式 → 掠食策略的涌现**：
- 高速巡航型(金枪鱼) → 开阔水体追猎 → 不依赖地形
- 高机动型(鳐/鳗鱼) → 在珊瑚礁/岩缝中伏击 → 需要复杂地形
- 爆发喷射型(鱿鱼) → 躲藏+瞬间加速出击 → 也需要结构物藏身

AI 不需要被告知"去珊瑚礁附近"——它们的生存依赖于适合其推进方式的环境，生态过滤器在生成时已经确保它们只出现在合适的水体中。

### 3.4 水体生态位——深度分层

水体生态位是一个**连续参数梯度**——不是硬分类。但方便设计，按深度区间描述典型特征：

```
透光层(0-200m) —— 水下"地表"——生产力最高:
  光充足 → 浮游植物 → 动物食物链基础
  温度: 随表面季节变化(温带=夏季温暖/冬季寒冷, 热带=整年温暖)
  生物密度: 最高 —— 绝大多数鱼/海龟/海豚在此
  
  生物特征: 视觉导航为主、色彩丰富、日行性
  玩家体验: 浮潜=鲜艳珊瑚礁+密集鱼群。深潜水=蓝渐深

中层(200-1000m) —— 过渡带——微光→无光:
  光极弱(星光/月光无法到达)
  温跃层在此 —— 温度骤降 5-10°C
  垂直迁移生物白天在此(躲掠食者)、夜晚上升到表层觅食
  生物发光=主要光源——灯笼鱼/某些水母/发光鱿鱼
  
  生物特征: 大眼(捕捉微光)、发光器官(诱饵或通信)、低代谢率
  玩家体验: 暗蓝→深蓝→接近黑色。需要自带光源。发光生物=天然"星空"

深层(1000-4000m) —— 完全黑暗+高压:
  温度: 2-4°C 恒定
  食物来源: 有机碎屑雨(海雪——从表层沉下来的死浮游生物+粪便)
  生物密度: 极低——但多样性可能高(特殊适应=特殊形态)
  
  生物特征: 巨口+发光诱饵(鮟鱇)、退化眼(或无眼)、巨型化(深水巨人症——某些甲壳类和鱿鱼极大)
           极低代谢率——可能数月不进食
  玩家体验: 极难到达——需要魔法抗压+魔法照明

深渊(4000m+) —— 极端:
  温度: 恒定~2°C
  压力: 极高
  食物: 极稀少——化学合成(热液喷口)或海雪
  
  生物特征: 极端特化+极低代谢率
  玩家体验: 几乎不可能到达——除非魔法。热液喷口=唯一生命绿洲

底栖(不限深度——水底生活):
  扁平体+向上眼(鳐/鲽/某些鲨)
  固着(珊瑚/海绵/藤壶——成虫固着, 幼体漂流)
  穴居(某些鳗/蠕虫——在底泥中生活)
```

**和世界生成的接口**：深度生态带从 P1 海洋系统的"深度分级"直接映射——不需要新数据结构。世界生成已经定义了近海/远海/深海的深度分布。

### 3.5 淡水 vs 海水——生理性硬边界

```
淡水(河流/湖泊):
  盐度 ~0-0.5‰
  水化学: 高氧(河流——流动)或低氧(沼泽——停滞)
  温度: 随季节大波动——温带湖泊冬季结冰
  规模: 河流=线性、湖泊=孤立
  
  典型生物: 鲤鱼/鲈鱼/狗鱼/鳟鱼/鲑鱼(出生)/淡水贝/小龙虾
  独特生物: 肺鱼(旱季休眠)、电鳗(亚马逊型)、河豚(淡水豚)

河口(半咸水——淡水海水交汇):
  盐度波动大(潮汐每日变 0.5-30‰)
  极高生产力(河水带来营养盐+潮水带来海水浮游生物)
  幼鱼保育场——众多海鱼的幼体在此成长(浅+营养+躲避大型掠食者)
  
  生物: 广盐性物种——适应盐度波动的少数物种
       鲈鱼/某些鲨(牛鲨)/鲑鱼(洄游经过)

海水(海洋):
  盐度 ~30-40‰
  规模: 连续的、全球循环
  洋流系统: 温盐环流——营养盐分布+温度分布
  
  生物: 绝大多数鱼/鲸/海龟/海蛇/头足类
       分布由洋流+温度+营养盐分布决定——非随机
```

**生成时的硬约束**：淡水/海水边界是**不可跨越**的——大多数物种生成时仅在一侧。洄游物种（鲑鱼/鳗鱼）是唯一例外——它们的生命周期在生成时硬编码在迁徙路径上。

### 3.6 鱼群行为——1 AI = 上百条鱼

鱼群是水生生物性能的关键优化。不做个体 AI 模拟——一个群=1 个 AI 实例。

```
鱼群配置:
  1 个 AI 实例 = 决定群中心位置 + 整体方向 + 速度
  N 个渲染实例(10-200条) = GPU compute shader boids
  
  GPU boids 规则(每帧, compute shader):
    凝聚: 不要离群中心太远
    分离: 不要撞相邻个体(最小间距)
    对齐: 和邻居同方向同速度
    掠食者响应: 鲨/大鱼靠近 → 全群急速转向+闪避
    环境回避: 浅水/障碍物 → 转弯
    
  个体变异(避免"复制粘贴"):
    相位偏移(不同鱼不同频率的轻微摆动)
    色相微偏(同种不同个体的天然色差)
    大小微变(±5%)

L1 可见鱼群数: 5-10 群(珊瑚礁/浅海) 或 0-2 群(开阔远洋)
AI 成本: 5-10 × 0.05ms = 0.25-0.5ms (极少)
渲染: 5-10 MultiMesh instances = ~0.2ms
```

### 3.7 大型水生个体

```
大型主动游泳者(鲨/鲸/大型鱿/鱼龙型程序化生物/海豚):

  L1 可见范围: 0-5 头(开放海域密度极低)
  每头 = InstinctiveMind(多数) 或 SapientMind(鲸/海豚——高智力)
  AI: 5 × 0.05ms = 0.25ms

  3D 寻路:
    开阔海域 = 直线(不需要 NavMesh)
    靠近海岸/珊瑚礁 = 需要简单寻路
    和陆地寻路的不同: 中段有上升/下潜决策

  大型海洋哺乳的特殊行为(高智力——SapientMind):
    海豚: 好奇伴随船游 → 跃出水面 → 和同类社交
          = 高 playfulness + 高 sociality 参数 → 行为涌现
    鲸: 浮出换气 → 喷水 → 拱背 → 尾鳍上扬 → 深潜
        低频歌声(通信——可跨数百km)
        迁移(季节性——追踪浮游生物丰富区)
```

### 3.8 漂浮/漂流——零 AI 成本

```
浮游植物+浮游动物+鱼卵+幼体:

  不是个体 —— 是水体属性的一部分
  密度 = f(洋流, 温度, 营养盐, 季节, 深度)
  视觉: 水体 shader 中的微粒(浊度组件) —— 不在动物系统中
  
  藻华(浮游植物爆发):
    温暖+高营养 → 密度爆炸
    水面颜色变绿/褐 + 可见度降低
    死后沉到海底=碳输出(海雪的来源之一)

  垂直迁移(浮游动物——每日):
    白天在深层(躲避掠食者)
    黄昏→上升到表层(追逐浮游植物=食物源)
    黎明前→下降到深层
    L3 统计——深度分布图在昼夜周期中移动——零个体 AI
```

### 3.9 水下寻路——比陆地简单

```
水下 3D 寻路 vs 陆地 2D 寻路:

  陆地: 障碍物密集(每棵树/石头/建筑=passability检查)
        地形多变(坡度/地表类型影响速度)
        NavMesh 需重建(砍树/建造→局部 NavMesh 重建)

  水体: 障碍物极稀疏(仅珊瑚礁/海藻林/沉船/冰层)
        90%水体=开阔无障碍——直线游 = 零寻路成本
        仅复杂地形处需要寻路(珊瑚礁/沉船/洞穴)
        
  寻路成本: 几乎零 —— 仅大群体在复杂地形时需要
        
  3D NavMesh: 大部分水体不需要 NavMesh
              仅预计算珊瑚礁/海藻林区域的局部 3D NavMesh
              开阔水域 = 直线——不使用 NavMesh
```

### 3.10 水生性能总账单

```
水生动物开销(玩家在水中时):

  AI(L1):
    鱼群: 5-10 群 × 0.05ms = 0.25-0.5ms
    大型个体: 2-5 头 × 0.05ms = 0.1-0.25ms
    3D 寻路: ~0(开阔) 或 <0.1ms(复杂地形)
    总计 AI: ~0.4-0.75ms

  渲染:
    鱼群 MultiMesh: ~0.2ms
    大型个体: 2-5 draw calls——可忽略
    水下 fog/光柱: 已在水体 shader 中——不额外
    总计 GPU: ~0.3ms

  L2 (50-150m): 鱼群简化为统计——<0.01ms
  L3 (>150m): 种群动态——和陆地同系统——<0.01ms

  水生总计: ~0.75-1.1ms/帧

  和陆地动物的关系:
    玩家在水面上=陆地 AI 少(没有地面动物在水下)、水生 AI 多
    玩家在陆地上=水生 AI=0(仅 L3 统计)
    → 水生和陆地不叠加——共享总动物 AI 预算 ~2ms
```

## 四、飞行生物

### 4.1 不是物理模拟——是感觉自然

飞行生物的体验是"看起来像真的在飞"——不模拟空气动力学。翼载/升力/阻力/翼展公式 → 全部砍掉。飞行质量在动画，不在物理。

```
三个飞行模式——玩家肉眼可区分（每种飞行生物随情景自动切换）:

  拍翼: 能耗高、速度快、可爬升 —— 日常觅食、迁移
  滑翔: 能耗低、速度中、缓慢下降 —— 长途省力、空中搜寻
  悬停: 能耗极高、速度零 —— 仅小体型+蜂鸟型——在小空间内静止观察

飞行半径——从栖息地能飞多远:
  小型 (昆虫/蜂鸟):   数百m、极易受风影响
  中型 (鸟/蝙蝠):     数km、中等抗风
  大型 (鹰/巨鸟/龙):   数十km、强抗风、魔法辅助可选

魔法飞行——不卡物理约束:
  魔法翼或无翼悬浮 —— 体重/翼载/起飞着陆=全都不受限
  魔力替代体力 —— 消耗 spirit 不消耗 stamina
  龙能飞 = magic_capacity > 0.8 —— 不是物理, 是规则
  巨鸟能飞 = 物理为主(大翼展) + 微量魔法辅助(降低有效体重)
```

### 4.2 飞行 AI——spline 路径，不寻路

空中基本没有障碍物——飞行生物不需要传统的 2D NavMesh 寻路。

```
飞行 = 在节点之间走 spline 路径:

  AI 决定:
    去哪 —— 目的地(栖息地/食物源/迁徙目标区域)
    多高 —— 飞行高度(树冠上? 高空?)
    多快 —— 拍翼=快+耗能 / 滑翔=慢+省能
    停哪 —— 下一栖息点(飞行终需落地)

  路径生成:
    起点 → 终点 → bezier spline
    + perlin 噪声偏移控制点 → 自然弧度
    + 局部风偏移 → "受风影响"(纯视觉——不影响目的地)
    不需要避障 —— 空中无障碍物(除非峡谷/密林——极少)

  成本: 
    决定目的地: InstinctiveMind 行为树——"饿了→飞向食物源"
    决定路径: 不寻路——spline 插值——<0.001ms
    决定模式: 距离×紧迫度 → 拍翼(急) or 滑翔(不急)——简单规则
```

### 4.3 栖息地决定分布

任何飞行生物终须落地——繁殖/休息/喝水。栖息地决定了"在哪里能见到什么鸟"。

```
栖息地类型 → 典型飞行生物:

  树冠层 → 林鸟(小型)——在枝叶间穿梭——不飞到开阔上空
  悬崖/岩壁 → 海鸟群+鹰/鹫(大型)——利用崖面上升气流
  地面 → 草地鸟/走禽——短距飞或不飞
  水面 → 水鸟(鸭/鸬鹚/信天翁)——能游泳+飞行+潜水
  建筑(聚落) → 鸽/燕/麻雀——适应人类
  空中(极少落地) → 魔法飞行/龙——魔法维持长时飞行

  自然涌现:
    森林内部 = 只有小鸟(大树冠下能飞——不需要开阔空间)
              大型猛禽不在密林——起飞/着陆空间不够
    开阔平原 = 中大型鸟可见——猛禽在天上盘旋
    海岸悬崖 = 海鸟群+鹰——利用崖上风升起
    远洋 = 信天翁型/魔法飞行——极少落地(水面休息)
```

### 4.4 空域视觉——玩家从地面往上看

```
日常上空可见的飞行生物:

  小鸟(最常见——到处都有):
    树冠间穿梭、草地上跳、沙浴
    成群飞过天空(傍晚——鸽群/燕群)
    偶尔撞进窗户 → NPC 惊吓 → 笑/骂
    在 NPC 屋檐下筑巢(春)——巢=建筑装饰+小鸟叫声

  猛禽(稀有——需要抬头+仔细看):
    高空盘旋——翼尖微翘、几乎不拍翼——"空中霸主"
    俯冲——隼以极高速度俯冲地面猎物

  候鸟(季节性+群):
    V形编队横穿天空——缓慢移动
    "大雁南飞"——秋天标志
    NPC 反应: "燕子来了=春天到了"

  虫群(夏季黄昏):
    蜻蜓+蚊虫+萤火虫——不是个体 AI
    水面上的群飞——鱼在下方等
    黄昏+湿地 → 逆光虫群
```

### 4.5 玩家在空中——飞行坐骑/魔法飞行

```
飞行坐骑(灵兽驯服或魔法辅助):

  起飞:
    坐骑拍翼上升——地面越来越远——树木变小——视野急剧扩展
    游戏中具冲击力的时刻之一——和地面看到的完全不同

  空中体验:
    视野 = 数十km——远处山脉/海岸线/湖泊一览无余
    地面聚落像小棋子——烟囱烟柱是导航标记
    风 = 可见(坐骑毛发被吹动)+ 可听(呼啸)+ 可感(微震)
    
  导航:
    不需要 NavMesh——空中自由 3D——去哪都是直线
    唯一约束: 飞行耐力(拍翼=消耗 stamina, 滑翔=恢复)
    云层 = 可穿越的体积云——雾+湿+冷却——穿越=GPS丢失(迷航可能)
    
  降落:
    找开阔地——盘旋→降低→拍翼缓冲→接触→跑停
    森林中找不到降落点(树冠遮挡) → 只能降落在林窗/空地
    城市 → NPC 反应: 有飞行坐骑的人=稀有/高地位/"又是那个冒险者"
```

### 4.6 空中飞行灵兽——引用 006

飞行灵兽 = 灵兽 + 羽翼/膜翼/魔法翼。驯服走 006-灵兽已有的 5 路径——区别仅在于驯服对象是飞行生物。005 不重复——引 `[[006-灵兽]]` 一条。

### 4.7 飞行性能总账单

```
飞行动物开销(每帧):

  L1 (<100m 玩家距离):
    小型鸟: 0-5 只——极低 AI(简单采食+栖息)——<0.05ms
    中大型: 0-3 只——<0.03ms
    鸟群:   0-2 群(群 AI——1 AI/群)——<0.04ms
    AI 总计: <0.12ms

  鸟群个体渲染:
    1-2 MultiMesh(群内 spline 偏移)+20-200 个体
    GPU: <0.1ms

  个体鸟渲染:
    0-8 draw calls(少量个体——远)
    GPU: ~0.03ms

  迁徙(LOD3 高空鸟群):
    远 LOD——V形四边形+缓慢移动——几乎零成本
    <0.01ms

  空中飞行坐骑(玩家在空中时):
    坐骑渲染: 1 个个体+翅膀动画——在灵兽/载具预算中
    更多鸟更近: 和地面态共享——只是距离更近——不新增实例

  飞行总计: ~0.2-0.3ms/帧
  (密度远低于陆地——飞行能耗高限制空中生物量)
```

## 五、跨域物种——能力组合，不独立分类

不创建"跨域"分类。每一个物种有一个主域，跨域能力作为附加标签。以下为"典型的能力组合"——不是所有组合的穷举：

```
两栖型(蛙/蝾螈):
  Terrestrial + can_swim=true + under_water_breathing=true(幼体)
  生态约束: 需要淡水繁殖——不能远离水体
  生成: 仅在湖泊/河流/沼泽附近放置

水鸟型(鸭/鸬鹚/鹈鹕):
  Terrestrial + can_swim=true + can_fly=true
  生态约束: 需要水体+飞行空间+筑巢地(树/地面)
  生成: 在湖/河/海岸附近放置
  三域能力但每域效率中等——飞不如鹰、游不如鱼、走不如鸡

飞鱼型:
  Aquatic + can_fly=true + flight_mode=Glide(极短程——数秒)
  生态约束: 出水滑翔=掠食者逃逸(不是主动旅行方式)
  生成: 在开阔海域表层放置——被水下掠食者追逐时自动跃出水面

企鹅型:
  Terrestrial + can_swim=true + can_fly=false
  生态约束: 冷水域——翼特化为鳍肢——陆地笨拙
  生成: 仅在寒冷海岸/岛屿

两栖蛇/鳄型:
  Terrestrial/Aquatic(取决于物种) + can_swim=true + 憋气能力极强
  生态约束: 水边伏击——浅水+岸上都能活动
  生成: 河流/沼泽/浅海/红树林
```

## 六、总性能预算——全部动物

```
每帧动物总开销:

  陆地动物 AI:                ~0.8-1.2ms  (已含在 NPC 心智预算中)
  水生动物 AI:                ~0.4-0.75ms (玩家在水中时——替代陆地)
  飞行生物 AI:                ~0.1-0.2ms  (同步+附加——密度极低)
  动物骨骼 + 动画:            ~0.5ms
  动物 MultiMesh 渲染:        ~0.3ms (陆地) + ~0.2ms (鱼类) + ~0.1ms (鸟)
  动物寻路:                    ~0.3ms (陆地) + ~0 (水生) + ~0 (飞行)

  总计: 动物 AI ~1.5-2.0ms + 渲染 ~1.0ms + 寻路 ~0.3ms = ~2.8-3.3ms/帧
  (在整体 16.7ms 帧预算中约 17-20%——和 NPC 心智共享此预算段)
```
