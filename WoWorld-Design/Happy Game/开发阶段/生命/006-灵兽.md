# 006-灵兽

> **状态**: 开发规格 v1.0
> **定位**: 类似帕鲁/宝可梦——可捕获、可训练、可协作的伙伴生物
> **关联**: [[001-生命总纲]] · [[005-动物]] · [[003-程序化生成系统]]

---

## 一、定义边界

灵兽 = 动物 + 灵元素 ≥ 0.4 + 可驯服 + 基础智能(能理解并执行至少一个协作任务)。

| 维度 | 灵兽 | 普通动物 |
|------|------|---------|
| 灵元素含量 | ≥ 0.4 | < 0.4 |
| 智能 | 理解指令、执行协作任务 | 本能+简单学习 |
| 驯服 | 可被驯服(5种路径) | 不可或仅可驯养(如牛耕地) |
| 魔力运用 | 可能具备天生元素能力 | 仅被动灵元素 |
| 与人的关系 | 协作伙伴 | 资源来源 |

## 二、5 种驯服路径

由灵兽生成时的行为特质决定——同一种灵兽可有多条驯服途径：

| 路径 | 适用 | 方式 |
|------|------|------|
| **温和亲近** | 社会化+草食/杂食+小体型 | 喂食+互动→信任积累→跟随 |
| **挑战征服** | 领地性强+大体型+掠食者 | 战斗至残血→尊重强者 |
| **救助** | 所有类型都可能 | 受伤/被困时救助→感激 |
| **共鸣** | 高魔法亲和 | 灵元素共鸣→精神连接 |
| **交易** | 高智力(工具使用+) | "服务换报酬"——契约式协作 |

## 三、训练与忠诚

```rust
struct BondedCreature {
    creature: CreatureRef,
    bond_level: f32,           // 0-1 纽带强度
    loyalty: f32,              // 0-1 忠诚度(独立于bond——可以被伤害)
    trained_skills: Vec<TrainedSkill>,
    owner: OwnerRef,
}

enum TrainedSkill {
    CarryItems { weight_limit: f32 },
    FollowInCombat { combat_role: CombatRole },
    TrackResource { resource_type: ResourceType },
    DeliverMessage { range: f32 },
    ScoutAhead { detection_range: f32 },
    FarmAssist { efficiency_bonus: f32 },
    Mount { rider_count: u32, speed: f32 },
    PerformMagic { spell: SpellId },
}
```

忠诚度下降：喂食不足/战斗中让它濒死/长期不互动。低忠诚→逃跑或拒绝协作。

## 四、NPC 与灵兽

NPC 也可以拥有和训练灵兽——与玩家完全相同的驯服流程。聚落中可以看到：商人的驮兽灵兽、卫兵的护卫灵兽、牧羊人的牧羊灵兽。
