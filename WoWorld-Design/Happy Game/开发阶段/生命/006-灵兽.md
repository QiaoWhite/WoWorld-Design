# 006-灵兽

> **状态**: 开发规格 v1.0
> **定位**: 类似帕鲁/宝可梦——可捕获、可训练、可协作的伙伴生物
> **关联**: [[001-生命总纲]] · [[005-动物]] · [[003-程序化生成系统]]

---

## 一、定义边界

灵兽 = 动物 + `spirit_capacity ≥ 0.4` + `Tameable` 标签 + 智能 ≥ `Trainable`

| 维度 | 灵兽 | 普通动物 |
|------|------|---------|
| 灵元素 | ≥ 0.4(中-高) | < 0.4(低) |
| 智能 | 理解指令+执行协作任务 | 本能+简单学习 |
| 驯服 | 5种驯服路径——可主动驯服 | 不可或仅可驯养(牛耕地——不是驯服是驯化) |
| 魔力 | 可能天生元素能力 | 仅被动灵元素 |
| 与人关系 | 协作伙伴——战斗/运输/探索/生产 | 资源来源——肉/皮/骨/家畜 |

**够格成为灵兽的标准**：能理解并执行至少一个协作任务（携带物品/跟随战斗/探测矿脉/传递消息/驮运骑手/……）。

## 二、5种驯服路径

由灵兽生成时的生态属性决定——**同一种灵兽可有多条驯服途径，不同驯服者可以走不同路径**。

### 路径1：温和亲近 (Gentle)

```
适用: sociality > 0.5 + 非纯肉食 + 体型 ≤ Large + aggression < 0.5

方式:
  1. 发现灵兽→不要攻击
  2. 喂食喜欢的食物(由灵兽的食性决定)
  3. 每日互动(陪伴/游戏——灵兽的 curiosity 越高效果越好)
  4. 信任值逐渐积累→达到阈值→灵兽主动跟随

适用灵兽示例:
  - 小型草食灵兽(类似鹿/羊/兔型——但灵元素≥0.4的变体)
  - 社会化灵兽(类似犬/马型灵兽)
  - 幼年灵兽(幼体 trust 积累速度 ×2)

不可用路径的情况:
  - 独居大型掠食灵兽——不会因为喂食就信任你
  - 领地性>0.7的灵兽——你进入领地=视为威胁，喂食无效
```

### 路径2：挑战征服 (CombatRespect)

```
适用: territoriality > 0.4 或 size ≥ Large 或 diet=肉食

方式:
  1. 挑战灵兽→战斗
  2. 将其 HP 削弱至 <30%(但不能杀死——杀死=失去机会)
  3. 灵兽判定: "这个对手够强" → respect
  4. 停止攻击+表示不杀(收起武器/后退/喂食)→建立初步服从
  5. 后续仍需要训练建立忠诚

适用灵兽示例:
  - 大型掠食灵兽(类似虎/熊/龙型的灵兽)
  - 极强领地性的灵兽
  - 首领级(AIpha标签)灵兽——挑战并击败首领=整个群体尊重你

特别机制:
  - respect_from_damage: 如果你打出了极高的单次伤害(暴击)→尊重积累速度 ×1.5
  - flee_if_overwhelmed: 如果你碾压它(你的HP几乎无损)→它可能直接逃跑(尊重=0)
  - 战斗必须"公平"——你用远程打近战=不尊重
```

### 路径3：救助 (Rescue)

```
适用: 所有灵兽都可能(但独居型更倾向感激恩人)

方式:
  1. 发现受伤/陷入困境的灵兽
     - 受重伤(部位damage>50%)
     - 被捕兽夹/陷阱困住
     - 被其他掠食者攻击
  2. 救助——治疗/解除陷阱/驱赶掠食者
  3. 灵兽感激→自动获得高初始信任
  4. 如果救助后继续照顾至痊愈→信任达到满值→自动驯服

特殊:
  - 救助幼年灵兽→可能连带获得其母亲的感激(如果母亲在场)
  - 感激有持续时间→如果痊愈后你不继续互动，它可能回归野外
  - 有些灵兽会在被救后送回"谢礼"(它猎到的猎物/找到的稀有物品)
```

### 路径4：共鸣 (SpiritResonance)

```
适用: magic_affinity ≥ 0.6(任一元素) 的灵兽

方式:
  1. 驯服者需要有与灵兽匹配的元素亲和力
     (如火元素灵兽→驯服者火元素亲和≥0.5)
  2. 在灵兽附近释放该元素的魔力(温和的——不是攻击)
  3. 灵兽感知到"同频的灵元素"→好奇→接近
  4. 通过灵元素共鸣建立精神连接→不需要战斗或长期喂食
  5. 共鸣连接建立后→其他驯服路径的信任积累速度 ×2

适用灵兽示例:
  - 元素灵兽(火焰鬃毛/冰壳/电流纹——身体有元素表现的)
  - 魔力身体型灵兽

限制:
  - 共鸣需要时间和魔力消耗(每次共鸣消耗 spirit)
  - 如果驯服者元素亲和不够→无法触发
  - 异性元素(水vs火)共鸣→不可能(反而会引起灵兽的敌意)
```

### 路径5：交易 (Bargain)

```
适用: intelligence ≥ ToolUse(工具使用级) 的灵兽

方式:
  1. 与灵兽"谈判"——不是语言交流而是行为展示
  2. 驯服者展示自己能提供的: 食物/保护/住所/宝藏
  3. 灵兽展示它愿意提供的: 战斗力/运输/追踪/魔法
  4. 达成"默契"→灵兽主动跟随→定期需要"支付报酬"
  5. 如果驯服者持续不支付→灵兽会离开(不是逃跑——是"合同到期")

适用灵兽示例:
  - 高智力灵兽(接近智能种族的灵兽——会使用工具/解决复杂问题)
  - 好奇心极强的灵兽(curiosity>0.8)

特殊:
  - 报酬可以是: "每月10块肉"、"保护你不被那个掠食者吃"、"给你建个窝"
  - 可重新谈判——灵兽的要求可能随时间变化
  - 交易型驯服——灵兽不是"服从"你——它是"合作"——它随时可以走
```

## 三、驯服后的训练与忠诚

```rust
struct BondedCreature {
    creature: CreatureRef,
    bond_level: f32,           // 0.0-1.0 纽带深度——随时间和协作提升
    loyalty: f32,              // 0.0-1.0 忠诚——可以被伤害而下降(独立于bond)
    trained_skills: Vec<TrainedSkill>,
    owner: OwnerRef,           // 可以是玩家或NPC
    taming_path: TamingApproach, // 记录是通过哪条路径驯服的
}

enum TrainedSkill {
    CarryItems { weight_limit: f32 },
    FollowInCombat { combat_role: CombatRole },
    TrackResource { resource_type: ResourceType },
    DeliverMessage { range_km: f32 },
    ScoutAhead { detection_range_m: f32 },
    FarmAssist { efficiency_bonus: f32 },
    Mount { rider_count: u32, speed_kmh: f32 },
    PerformMagic { spell: SpellId },
    GuardLocation { alert_range_m: f32 },
    MineAssist { detect_mineral_range_m: f32 },
}

/// 忠诚度下降条件:
///   loyalty -= 0.1: 喂食不足超过3天
///   loyalty -= 0.2: 战斗中让它HP降到<10%
///   loyalty -= 0.3: 你攻击了它的同种(野生同类)
///   loyalty -= 0.05/天: 超过7天没有任何互动
///
/// 忠诚度后果:
///   loyalty < 0.3: 战斗中可能不听从指令
///   loyalty < 0.1: 可能逃跑或拒绝协作
///   loyalty = 0: 灵兽离开——回归野外
```

## 四、NPC与灵兽

NPC 也可以拥有和驯服灵兽——与玩家完全相同的流程。聚落中可以看到：
- 商人的驼兽灵兽——背着货物走在商路上
- 卫兵的护卫灵兽——在城门口或巡逻路线上
- 牧羊人的牧羊灵兽——管理羊群
- 猎人的追踪灵兽——鼻尖贴地追踪猎物
- 农民的农用灵兽——帮忙耕地/拉车
- NPC 对灵兽的感情——NPC 与灵兽有 bond_level 和关系值。NPC 可能"非常爱他的老伙计"——如果灵兽死了 NPC 会悲伤
