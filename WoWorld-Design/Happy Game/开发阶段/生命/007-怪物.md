## 一、定义

怪物不是自然生态的一部分——它们来自异常来源。与动物共享 12 维度积木，但额外叠加"异常特征层"。

## 二、7 种来源

| 来源 | 说明 |
|------|------|
| **魔法扭曲** (Corrupted) | 正常生物被高浓度魔法/元素失衡扭曲 |
| **亡灵** (Undead) | 灵元素异常残留——见 [[008-亡灵]] |
| **异界** (Extraplanar) | 从其他位面渗入（深渊/元素位面/灵界） |
| **人工创造** (Created) | 魔法实验/炼金合成/魔像 |
| **远古幸存** (Ancient) | 先于当前生态系统的远古物种 |
| **自然变异** | 正常动物个体因环境+魔法+基础异常积累产生新特征——见 三 |
| **混合来源** | 以上多种叠加 |

## 三、自然动物变体

部分怪物**从已生成的自然动物派生**——基础物种 + 异常特征叠加层：

```rust
enum MonsterSource {
    Independent(CreatureTemplate),     // 独立生成
    VariantOf {
        base_species: SpeciesId,       // 基础物种
        anomaly: AnomalyOverlay,       // 异常特征叠加
        // 例: 正常狼 + 骨刺增生+夜光眼+元素泄漏 → 魔狼
    },
}
```

这样同一群系中既有正常狼，又有其怪物变体——NPC 可以讲述"森林里有一头不一样的狼"的故事。

## 四、异常特征叠加层

| 特征 | 说明 |
|------|------|
| 骨刺增生 | 额外骨质突起——攻击力+、外貌恐怖 |
| 体型变异 | 巨大化/萎缩——超出物种正常范围 |
| 元素泄漏 | 身体散发元素能量（火焰鬃毛/冰冻气息/电光纹路） |
| 异常再生 | 超出正常再生能力——断肢快速重生 |
| 多生物融合 | 两个或多个生物的特征混合（合成兽） |
| 位面特征 | 带有所属位面的物理特性(暗影/光晕/熔岩皮肤) |
| 精神污染 | 周围生物受到恐惧/疯狂/魅惑光环 |
| 不可杀性 | 需要特定条件才能永久杀死(银器/圣水/特定魔法) |

## 五、生态学地位

怪物的生态学过滤用**不同标准**——不必通过正常食物链检查。怪物破坏正常生态，是食物链的"外来入侵物种"。

## 六、必涵盖的经典奇幻怪物

史莱姆 / 骷髅(→见亡灵) / 哥布林 / 幽灵(→见亡灵) ——作为保底内容。
