## 一、定义

亡灵 = 灵元素异常残留。身体已死，灵元素未散干净，或被外力拴在世界上。亡灵**不是"活着"的**——灵元素持续衰减，除非有外力补充。

```rust
struct Undead {
    former_life: LifeRecord,
    lingering_spirit: f32,       // 残留灵元素度(0→消散, 1→完整魂魄)
    binding_force: Vec<BindingForce>,  // 可多个绑定力叠加
}
```

## 二、7 种绑定来源

| 来源 | 说明 | 典型触发条件 |
|------|------|-------------|
| **怨恨** (Grudge) | 强烈未了情感将灵元素拴在世上 | 横死(谋杀/战争/冤案) |
| **诅咒** (Cursed) | 被主动施加——死者或生者被诅咒 | 死灵法师/复仇者/神明 |
| **死灵法术** (Necromantic) | 被魔法强制驱动 | 死灵法师施法 |
| **环境自然产生** (AmbientMagic) | 高魔力浓度区+尸体→自动"醒来" | 魔法群系/古战场 |
| **神圣意志** (Divine) | 神明让死者不得安息 | 仅 A 模式 |
| **自愿** (Voluntary) | 生前主动选择不死 | 巫妖化仪式 |
| **传染** (Infected) | 被已有亡灵杀死→成为新亡灵 | 亡灵密度高时 |

**几个来源可叠加**——三重来源的亡灵远比单来源的强。

## 三、来源权重由世界状态驱动

```
local_cursed_weight      = 被诅咒者密度
local_grudge_weight      = 横死率(战争/谋杀)
local_necromantic_weight = 死灵法师活动
local_ambient_weight     = 魔力浓度
local_divine_weight      = divine_mode == A
local_voluntary_weight   = 当地文化是否崇尚永生
local_infected_weight    = 已有亡灵密度

→ 古战场: 怨恨高+环境高+尸体集中→骷髅士兵+怨魂
→ 瘟疫废弃村: 传染高+怨恨中→僵尸群+病灵
→ 神殿废墟: 神圣(A)+自愿→守护灵+殉道者
→ 魔法实验室: 死灵法术+自愿→巫妖+合成亡灵
```

## 四、亡灵强度与衰减

```rust
fn undead_power(bindings: &[BindingForce], spirit: f32, decay_time_days: u32) -> f32 {
    let origin_mult = 1.0 + (bindings.len() as f32 - 1.0) * 0.5; // 多来源叠加
    let freshness = 1.0 / (1.0 + decay_time_days as f32 * 0.01);     // 越新鲜越强
    let feeding = if binds_include(Infect) { recent_kill_bonus } else { 1.0 };
    origin_mult * spirit * freshness * feeding
}
```

亡灵本能攻击活人——**汲取灵元素延缓自身消散**。

## 五、必备经典亡灵

骷髅 / 幽灵 / 僵尸 / 巫妖 / 缚灵 / 无头骑士——作为保底。
