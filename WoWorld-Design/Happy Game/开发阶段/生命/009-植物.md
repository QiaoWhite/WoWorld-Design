## 一、架构分离

植物**不继承 Life 完整基类**。与动物的关键差异：

| | 动物/怪物 | 植物 |
|---|---|---|
| 运动 | 能动——AI/寻路/行为树 | 不可动——极少数魔法植物有缓慢运动 |
| 智能 | 有本能→完全智能 | 无 |
| 行为AI | 需要 | 不需要——仅"生长+反应" |
| 身体状态 | 完整BodyState(饥饿/疲劳/体温/……) | **无**——仅"生长阶段"+"健康度" |
| 交互 | 战斗/驯服/捕猎/训练 | 采集/种植/收割/砍伐 |
| 性能 | L1-L4分层模拟 | MultiMesh 批量渲染——极低成本 |
| 灵元素 | 有(所有动物) | 有(所有植物)——但低(0.05-0.15) |

## 二、植物生成维度

```rust
struct PlantSpecies {
    morphology: PlantMorphology,
    biome_affinity: BiomeAffinity,
    growth_stages: Vec<GrowthStage>,
    harvestable: Vec<HarvestableProduct>,
    health: f32,             // 1.0(健康)~0.0(枯死)——简单的健康度——代替BodyState
}

struct PlantMorphology {
    size: PlantSize,             // 苔藓(贴地)/草本(<1m)/灌木(1-3m)/乔木(3-30m)/巨树(30m+)
    root_system: RootType,       // 浅根(广泛横向)/深主根(深扎)/块根(储水/营养)/气生根(从空气中吸水)/寄生根(刺入宿主)/共生根(与真菌互利)
    stem_type: StemType,         // 草本(一年生/多年生)/木本(乔木/灌木)/藤本(攀援)/多肉(储水)/空心(竹型)/匍匐(贴地蔓延)
    leaf_type: LeafType,         // 阔叶/针叶/鳞叶/多汁叶/无叶(以茎光合)/变色叶(季节)/发光叶(魔法)
    flower_type: Option<FlowerType>, // 花色/花形/花期/是否需要特定传粉者
    fruit_type: Option<FruitType>,   // 果型/可食用/有毒/魔法效果
    special_features: Vec<PlantFeature>,
}

enum PlantFeature {
    Luminescent,       // 发光——吸引传粉者/警告毒性/纯粹美化
    Thorny,            // 荆棘——物理防御——阻止动物啃食
    Carnivorous,       // 食肉——主动捕捉小动物/昆虫——补充氮素
    Parasitic,         // 寄生——吸取宿主养分——对宿主有害
    Symbiotic,         // 共生——与动物/真菌/其他植物互利
    Magical,           // 魔法——天然魔力储存/释放——稀有
    Animated,          // 能动——仅少量魔法植物有缓慢运动(如向光转/捕虫动作/收缩触须)
    ExplosiveSeeds,    // 爆炸播种——种子通过爆炸力散布
    WindDispersed,     // 风播——轻种子借助风长途传播
    WaterDispersed,    // 水播——种子可通过水流/洋流传播(跨洋!)
    AnimalDispersed,   // 动物传播——果实被动物吃→种子通过粪便远距离传播
    FireAdapted,       // 火适应——需要火才能释放种子(某些针叶树)
}
```

## 三、植物分类

| 类别 | 生成方式 | 实例数 | 说明 |
|------|---------|--------|------|
| **野生植物** | 按群系程序化生成形态+密度 | ~数百万 | 群系的"底色"——草/灌木/树木 |
| **农作物** | NPC种植决定种类+密度 | 聚落周边 | 与聚落农业系统关联 |
| **魔法植物** | 魔法群系特有——低概率稀有 | 稀有 | 有特殊魔法效果——可采集/炼金 |
| **巨树/世界树** | 极稀有——特殊生成 | 极少 | 单体极巨大——可能承载整个生态微系统 |

## 四、生长阶段

种子→发芽→幼苗(脆弱的——任何啃食可以摧毁)→生长→成熟(可采集)→开花→结果→种子散布→衰老→枯死(→变成腐殖质→新植物更易生长)。

植物的生命周期完全由时间+季节+环境条件驱动——不需要 AI 决策。简单健康度受：光照、水分、温度、被啃食量、疾病影响。

---

## 五、与动物/怪物的交互

- 植食动物啃食→植物密度下降(种群层面)
- 食肉植物吸引/捕捉小动物
- 某些植物与特定动物共生(如特定蜂类只采集特定花型)
- 魔法植物释放的魔力辐射→可能催生怪物(魔法扭曲型怪物)
- NPC砍伐/种植/采集→主动改变植物分布
