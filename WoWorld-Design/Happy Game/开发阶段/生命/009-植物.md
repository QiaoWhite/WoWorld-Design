## 一、架构分离

植物与动物/怪物**不共享 Life 完整基类**。植物无 Mind、无 BodyState(vitals)、无 AI 行为。

| | 动物/怪物 | 植物 |
|---|---|---|
| 运动 | 能动——AI/寻路 | 不可动(极少数魔法植物例外) |
| 智能 | 有本能→完全智能 | 无 |
| 行为 AI | 需要 | 不需要——仅生长+反应 |
| 交互 | 战斗/驯服/捕猎 | 采集/种植/收割 |
| 生成 | 形态+AI+生态 | 形态+生长+分布密度 |
| 性能 | L1-L4分层 | MultiMesh 批量渲染 |

## 二、植物生成

```rust
struct PlantSpecies {
    morphology: PlantMorphology,
    biome_affinity: BiomeAffinity,
    growth_stages: Vec<GrowthStage>,
    harvestable: Vec<HarvestableProduct>,
}

struct PlantMorphology {
    size: PlantSize,         // 苔藓/草本/灌木/乔木/巨树
    root_system: RootType,   // 浅根/深主根/块根/气生根/寄生根
    stem_type: StemType,     // 草本/木本/藤本/多肉/空心/匍匐
    leaf_type: LeafType,     // 阔叶/针叶/鳞叶/多汁叶/无叶(以茎光合)
    flower_type: Option<FlowerType>,
    fruit_type: Option<FruitType>,
    special_features: Vec<PlantFeature>,
}

enum PlantFeature {
    Luminescent,       // 发光
    Thorny,            // 荆棘
    Carnivorous,       // 食肉
    Parasitic,         // 寄生
    Symbiotic,         // 共生
    Magical,           // 魔法性(天然魔力储存)
    Animated,          // 能动(少量魔法植物)
}
```

## 三、植物分类

| 类别 | 说明 |
|------|------|
| **野生植物** | 按群系分布——程序化生成其形态和密度 |
| **农作物** | NPC 种植——由聚落农业决定种类 |
| **魔法植物** | 魔法群系特有——稀有+特殊效果 |
| **巨树/世界树** | 特殊——单体极巨大，可能承载生态系统(见自然景观) |

## 四、生长阶段

种子→发芽→幼苗→成熟→开花→结果→衰老→枯死。植物有独立的生长阶段系统，不需要 BodyState(饥饿/疲劳等)。

## 五、与动物/怪物的交互

- 植食动物啃食植物→植物密度变化
- 某些植物与动物/怪物共生（如食肉植物吸引猎物）
- 魔法植物的魔力辐射可能产生怪物
- NPC 种植/采集/砍伐植物
