## 一、动物繁衍——种群动态模型

个体不逐帧模拟繁衍。物种层面用种群动态：

```rust
fn tick_population(species: &Species, biome: &BiomeState, delta_days: f32) -> PopulationChange {
    let current = species.population_in(biome);
    let carrying = biome.carrying_capacity_for(species);
    if current < carrying * 0.7 { Growth }
    else if current > carrying * 1.3 { Decline }
    else { Stable }
}
```

个体模拟仅在 L1（玩家附近）：怀孕母兽、幼崽、巢穴。

## 二、智能种族繁衍——社会关系驱动

### 2.1 生理基础
与动物相同的 `Reproduction` 结构（出生方式/孕期/每胎数量/抚育方式——由种族决定）。

### 2.2 控制层——社会因素

```rust
struct FertilityDrivers {
    // 生理
    health_weight: f32,
    age_pressure_weight: f32,
    // 经济
    economic_security_weight: f32,   // "养得起吗？"
    // 伴侣
    partner_quality_weight: f32,     // 与伴侣的关系值
    // 已有子女
    existing_children_weight: f32,   // 越多→越不想生
    // 个人
    personal_desire_weight: f32,     // 由 NPC 人格决定
    // 稳定性
    stability_weight: f32,           // 战争/饥荒→降低
    // 三层社会约束（乘数修正）
    religious_modifier: f32,
    legal_modifier: f32,
    cultural_modifier: f32,
}
```

### 2.3 配偶/伴侣制度——由文化决定

`Monogamous`(一夫一妻) / `Polygamous`(多配偶) / `GroupMarriage`(群婚) / `FreeUnion`(自由) / `Arranged`(包办) / `Contractual`(契约制) / `Celibate`(独身)

### 2.4 生育决策

NPC 的 GOAP 中产生"想要孩子"目标——当经济稳定、关系良好、文化期望+个人意愿足够。宗教×法律×文化三层约束修正各权重。

### 2.5 子女继承

身体属性/心智属性/灵元素亲和 = 双亲混合 + 随机小变异。

跨种族：不可混血 / 母系 / 父系 / 混血 / 随机——由该世界的种族兼容性参数决定。

## 三、三层社会约束

### 3.1 宗教/信仰

| 教义 | 修正 |
|------|------|
| 多生多育 | 已有子女影响反转、文化期望×2 |
| 适度生育 | 标准 |
| 禁欲崇高 | 伴侣权重×0.3、个人意愿-0.4 |
| 神定时机 | 经济权重×0.5 |
| 神圣血脉 | 仅特定阶层可繁衍 |

### 3.2 法律/政策

| 政策 | 修正 |
|------|------|
| 鼓励生育 | 经济+0.2、反转已有子女 |
| 限制生育 | 经济-0.5、子女达上限→强衰减 |
| 战争恢复 | 强制配额、健康-0.3 |
| 继承法(长子) | 只在意第一个孩子 |
| 继承法(均分) | 多生=每人少分→-0.2 |

### 3.3 文化习俗

- 理想子女数——偏离理想→压力
- 性别偏好——"直到生出儿子"→持续生育
- 非婚生子女——无所谓/污名/禁制
- 收养接受度
- 氏族结构(父系/母系/双系)——影响命名和归属

## 四、种群调控——动物 vs 智能种族

| | 动物 | 智能种族 |
|---|---|---|
| 繁殖驱动 | 种群密度 vs 承载容量 | 社会因素(经济+伴侣+文化+个人) |
| 配偶 | 本能 | 文化制度 |
| 子女数量 | 物种固定范围 | 决策结果 |
| 抚育 | 本能行为 | 社会行为(父母+家族+社区) |
| 后代属性 | 随机采样+变异 | 双亲混合+教育环境影响 |
| 调控 | 食物链承载量 | 经济承载+战争+饥荒+瘟疫+文化政策 |
