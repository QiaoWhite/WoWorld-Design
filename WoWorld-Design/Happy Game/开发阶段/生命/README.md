## 定位

本文件夹存放 WoWorld 生命系统的**正式开发规格**。Life 是所有生物的通用基类——NPC、动物、灵兽、怪物、亡灵、神明均继承此基类。

**核心原则**：
- **Life 是基类，NPC 心智是上层**：BodyState/CreatureTraits 由 Life 定义，GOAP/记忆/情绪由 SapientMind 叠加
- **特质从历史涌现，不预置**：种族参数不是手写表，是世界生成+历史塑形的产物
- **12 维度积木拼装**：程序化生成适用于动物/怪物/植物/智能种族
- **四层质量防线**：硬约束→生态过滤→模式模板→美学评分
- **所有伤害是数值，所有数值可恢复**：无永久性伤害

## 架构速览

```
Life (生命基类 —— 所有生物共享)
├── BodyState        ← Vitals / 部位伤害 / Spirit
├── CreatureTraits   ← 种族参数化(从历史涌现)
├── Reproduction     ← 繁衍参数
├── Produce & Loot   ← 活体产物 + 死亡掉落
├── Tags             ← 200+ 可叠加标签
│
├── 植物（极简 LifeBase 继承——无 Mind/BodyState 完整继承——架构分离，详见 [[009-植物]] v2.0）
├── 动物（InstinctiveMind——本能行为树）
│   └── 灵兽（InstinctiveMind + 可驯服 + 5种驯服路径）
├── 怪物（InstinctiveMind + 异常特征叠加）
├── 亡灵（MinimalMind + 灵元素异常残留）
├── 智能种族（SapientMind——完整 GOAP/记忆/情绪/关系）
│   └── 玩家（心智由人类输入驱动）
└── 神明（DivineMind——仅 A 模式）

生成顺序: 群系 → 环境筛选 → 12维积木采样 → 硬约束 → 生态过滤 → 美学评分 → 物种池
```
