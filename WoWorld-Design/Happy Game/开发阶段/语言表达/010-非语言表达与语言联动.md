## 一、定位与已有基础

### 1.1 已有基础 (NPC 模块 §4.4)

NPC 模块已定义了从情绪到身体姿态的映射 `emotional_pose_offset()`：

```rust
pub fn emotional_pose_offset(emotion: &EmotionState) -> PoseOffset {
    PoseOffset {
        head_tilt: emotion.sadness * -5.0,       // 悲伤→低头
        shoulder_slump: emotion.fear * 3.0,       // 恐惧→蜷缩
        chest_expand: emotion.joy * 2.0,           // 喜悦→挺胸
        gaze_avert: emotion.disgust * 0.5 + emotion.fear * 0.3,
        fist_clench: emotion.anger * 4.0,          // 愤怒→握拳
        social_distance_bias: emotion.trust * -0.3 + emotion.fear * 0.5,
    }
}
```

这提供了**非语言表达的渲染基础**——但缺少与**语言表达的联动桥梁**。本文件填补这个缺口。

## 二、非语言表达数据模型

### 2.1 NonVerbalSignal

```rust
/// 非语言信号——伴随或独立于语言表达的肢体信息
pub struct NonVerbalSignal {
    /// 信号类型
    pub signal_type: NonVerbalType,
    /// 信号的强度 0-1
    pub intensity: f32,
    /// 信号的持续时间（秒）
    pub duration_seconds: f32,
    /// 信号的目标——对谁做的（None = 面对所有人）
    pub target: Option<EntityId>,
}

pub enum NonVerbalType {
    // ── 面部 ──
    FacialExpression(FacialExpression),
    // ── 手势 ──
    Gesture(Gesture),
    // ── 身体姿态 ──
    Posture(PostureShift),
    // ── 视线 ──
    Gaze(GazeAction),
    // ── 空间距离 ──
    Proxemics(ProxemicShift),
    // ── 触觉（稀有——文化限制）──
    Touch(TouchType),
}

pub enum FacialExpression {
    Smile { warmth: f32 },        // 0=礼貌假笑, 1=真诚大笑
    Frown { intensity: f32 },
    Surprise { intensity: f32 },
    Disgust { intensity: f32 },
    Fear { intensity: f32 },
    Anger { intensity: f32 },
    Neutral,
}

pub enum Gesture {
    /// 点头——同意/确认
    Nod { emphasis: f32 },
    /// 摇头——否定/拒绝
    HeadShake { intensity: f32 },
    /// 指向——"看那边"/"就是他"
    Point { direction: Vec3, target: Option<EntityId> },
    /// 耸肩——"不知道"/"无所谓"
    Shrug,
    /// 挥手——打招呼/告别
    Wave { style: WaveStyle },
    /// 双手摊开——"没有恶意"/"我也没办法"
    OpenPalms,
    /// 拇指/手指比划——讲价、计数、强调
    Counting { number: u8 },
    /// 握拳高举——胜利/愤怒
    FistRaise { emotion: Emotion },
    /// 拍桌——强调/愤怒/激动
    TableHit { intensity: f32 },
    /// 文化特定手势（由文化参数派生）
    CultureSpecific { gesture_id: u32 },
}

pub enum WaveStyle { Greeting, Farewell, Excited, Dismissive }
pub enum TouchType { Handshake, Hug, PatOnBack, ArmTouch, AggressiveGrab }
pub enum GazeAction { LookAt(EntityId), LookAway, ScanRoom, StareDown(EntityId), AvoidEyeContact }
pub enum PostureShift { LeanIn, LeanBack, StandTall, Slouch, CrossArms, HandsOnHips, TurnAway }
pub enum ProxemicShift { MoveCloser, StepBack, BlockPath, MakeSpace }
```

### 2.2 NonVerbalExpression——完整表达

```rust
/// 一轮对话中伴随语言表达的非语言信号集合
/// 对话中每个 TurnRecord 可以附带一个 NonVerbalExpression
pub struct NonVerbalExpression {
    /// 语言表达之前/同时/之后——不同阶段的非语言信号
    pub pre_utterance_signals: Vec<NonVerbalSignal>,
    pub concurrent_signals: Vec<NonVerbalSignal>,
    pub post_utterance_signals: Vec<NonVerbalSignal>,
    /// 整体的身体姿态基线——该轮对话期间持续
    pub posture_baseline: PoseOffset,
    /// 说话时的社交距离
    pub social_distance_m: f32,
    /// 这些信号的可观察性——距离越远部分信号不可见
    pub observability: SignalObservability,
}

pub struct SignalObservability {
    /// 面部表情需要近距离 (<3m) 才能分辨
    pub facial_visible: f32,     // distance threshold
    /// 手势可见距离更远 (<10m)
    pub gesture_visible: f32,
    /// 身体姿态可见距离最远 (<30m)
    pub posture_visible: f32,
    /// 视线方向——中等距离可见
    pub gaze_visible: f32,
    /// 触觉——仅极近距离
    pub touch_visible: f32,
}
```

## 三、非语言 ↔ 语言联动引擎

### 3.1 对话中非语言信号的生成

```rust
/// 从对话内容+说话者状态派生非语言表达
pub fn synthesize_nonverbal(
    utterance: &NpcResponse,
    speaker: &NpcData,
    listener: &NpcData,
    ctx: &DialogueContext,
) -> NonVerbalExpression {
    let mut expr = NonVerbalExpression::default();

    // ① 面部表情——来自 utterance.emotion + speaker 当前情绪
    expr.concurrent_signals.push(
        NonVerbalSignal {
            signal_type: NonVerbalType::FacialExpression(
                emotion_to_facial(utterance.emotion)
            ),
            intensity: utterance.emotion.arousal,
            duration_seconds: utterance.estimated_duration(),
            target: Some(listener.id),
        }
    );

    // ② 手势——来自 utterance 中的关键词/意图
    expr.concurrent_signals.extend(
        gesture_from_intent(&utterance.intent_type, speaker)
    );

    // ③ 身体姿态——来自 speaker 的 PoseOffset
    expr.posture_baseline = emotional_pose_offset(&speaker.emotion);

    // ④ 视线——来自 speaker 的个性 + 当前情绪
    let gaze = if speaker.personality.extraversion > 0.6 {
        GazeAction::LookAt(listener.id)  // 外向→直视对方
    } else if speaker.emotion.fear > 0.4 {
        GazeAction::AvoidEyeContact       // 恐惧→避免视线接触
    } else {
        GazeAction::LookAt(listener.id)   // 正常→间歇性视线接触
    };
    expr.concurrent_signals.push(NonVerbalSignal {
        signal_type: NonVerbalType::Gaze(gaze),
        intensity: 0.7,
        duration_seconds: 2.0,
        target: Some(listener.id),
    });

    // ⑤ 社交距离——来自情绪+个性+关系
    expr.social_distance_m = compute_social_distance(speaker, listener);

    expr
}

/// 从对话意图生成手势
fn gesture_from_intent(
    intent_type: &IntentType,
    speaker: &NpcData,
) -> Vec<NonVerbalSignal> {
    match intent_type {
        IntentType::Threatening { .. } => vec![
            NonVerbalSignal {
                signal_type: NonVerbalType::Gesture(Gesture::FistRaise { emotion: Emotion::ANGER }),
                intensity: 0.7,
                duration_seconds: 1.0,
                target: None,
            }
        ],
        IntentType::Trading { action: TradeAction::Bargain } => vec![
            NonVerbalSignal {
                signal_type: NonVerbalType::Gesture(Gesture::Counting { number: 3 }),
                intensity: 0.5,
                duration_seconds: 1.5,
                target: None,
            }
        ],
        IntentType::Greeting => {
            if speaker.personality.extraversion > 0.6 {
                vec![NonVerbalSignal {
                    signal_type: NonVerbalType::Gesture(Gesture::Wave { style: WaveStyle::Excited }),
                    intensity: 0.8,
                    duration_seconds: 1.5,
                    target: None,
                }]
            } else {
                vec![NonVerbalSignal {
                    signal_type: NonVerbalType::Gesture(Gesture::Nod { emphasis: 0.3 }),
                    intensity: 0.3,
                    duration_seconds: 0.5,
                    target: None,
                }]
            }
        }
        IntentType::Farewell => vec![
            NonVerbalSignal {
                signal_type: NonVerbalType::Gesture(Gesture::Wave { style: WaveStyle::Farewell }),
                intensity: 0.5,
                duration_seconds: 1.0,
                target: None,
            }
        ],
        _ => vec![],  // 中性意图——手势由个性+文化决定
    }
}

/// 社交距离计算
fn compute_social_distance(speaker: &NpcData, listener: &NpcData) -> f32 {
    let trust = speaker.trust_toward(listener);
    let intimacy = speaker.intimacy_with(listener);

    // 亲密距离: 0-0.45m (家人/恋人)
    // 个人距离: 0.45-1.2m (朋友)
    // 社交距离: 1.2-3.6m (陌生人/交易)
    // 公共距离: 3.6m+ (演讲/正式)

    let base = 2.5;  // 默认社交距离
    // trust 每增加 0.1，距离减少 0.2m
    // intimacy 每增加 0.1，距离减少 0.15m
    let adjusted = base - trust * 2.0 - intimacy * 1.5;
    adjusted.max(0.3)  // 不能小于 0.3m（个人空间底线）
}
```

### 3.2 NPC 读对方的非语言信号——作为对话输入

```rust
/// NPC 在对话中感知对方的非语言信号
/// 这些信号影响 DialogueIntent 的生成和 TopicSelector 的选择
pub struct NonVerbalPerception {
    /// 感知到的面部表情
    pub perceived_facial: Option<FacialExpression>,
    /// 感知到的手势
    pub perceived_gestures: Vec<Gesture>,
    /// 感知到的姿态变化
    pub perceived_posture: Option<PostureShift>,
    /// 感知到的视线行为
    pub perceived_gaze: Option<GazeAction>,
    /// 感知到的距离变化
    pub perceived_proxemics: Option<ProxemicShift>,
    /// 总体感知置信度（受距离/光照/注意力影响）
    pub overall_confidence: f32,
}

/// 非语言感知→对话意图修正
pub fn nonverbal_driven_intent_adjustment(
    perception: &NonVerbalPerception,
    observer: &NpcData,
    observed: &NpcData,
) -> Vec<DialogueIntent> {
    let mut intents = Vec::new();

    // 对方在愤怒 → 可能生成安抚/防御/离开的意图
    if matches!(perception.perceived_facial, Some(FacialExpression::Anger { intensity }) if intensity > 0.6)
    {
        if observer.personality.agreeableness > 0.6 {
            intents.push(DialogueIntent {
                intent_type: DialogueIntentType::Emotional {
                    emotion: Emotion::CONCERN,
                    intensity: 0.7,
                    cause_memory_id: None,
                },
                priority: 0.8,
                target: Some(observed.id),
                topic_params: HashMap::new(),
                expires_at: GameTimestamp::now() + 60,
            });
        }
    }

    // 对方回避视线 + 负面情绪 → 可能有什么事瞒着自己
    if matches!(perception.perceived_gaze, Some(GazeAction::AvoidEyeContact))
        && observer.emotion.trust < 0.4
    {
        intents.push(DialogueIntent {
            intent_type: DialogueIntentType::GoalDriven {
                goal_id: GoalId::new("confront_evasion"),
                action: "challenge_honesty".into(),
            },
            priority: 0.6,
            target: Some(observed.id),
            topic_params: HashMap::new(),
            expires_at: GameTimestamp::now() + 120,
        });
    }

    // 对方在微笑 + 高信任 → 关系维护意图
    // ...

    intents
}
```

## 四、文化差异

### 4.1 手势文化映射

```rust
/// 同一手势在不同文化中的含义不同
/// 例: 点头 = 同意 (大部分文化), 但某些文化中 = "我在听"（不一定同意）
///      直视 = 尊重 (西方), 但某些文化中 = 挑衅
pub struct GestureCultureMapping {
    /// 手势 ID → 文化区域 ID → 含义
    pub meanings: HashMap<u32, HashMap<CultureId, GestureMeaning>>,
}

pub struct GestureMeaning {
    /// 该手势在此文化中的含义
    pub interpretation: String,
    /// 情感极性（正面/中性/负面）
    pub valence: f32,
    /// 正式度层级——该手势在正式场合是否合适
    pub formality_appropriate: FormalityLevel,
}

impl GestureCultureMapping {
    /// 两个不同文化的 NPC 之间可能发生手势误解
    pub fn cross_cultural_misinterpretation(
        gesture: &Gesture,
        actor_culture: CultureId,
        observer_culture: CultureId,
    ) -> Option<GestureMeaning> {
        let actor_meaning = self.meanings.get(&gesture.id())?.get(&actor_culture)?;
        let observer_meaning = self.meanings.get(&gesture.id())?.get(&observer_culture)?;

        if actor_meaning.interpretation != observer_meaning.interpretation {
            // 误解！NPC A 做的手势在 NPC B 的文化中是另一个意思
            Some(observer_meaning.clone())
        } else {
            None  // 无误解
        }
    }
}
```

### 4.2 文化沟通规范

```rust
/// 每个文化区域定义了一套沟通规范
/// 影响对话的多个层面
pub struct CommunicationNorms {
    /// 是否允许打断
    pub interruption_tolerance: f32,    // 0=绝对不打断, 1=打断是热情的表示
    /// 视线接触规范
    pub eye_contact_norm: EyeContactNorm,
    /// 个人空间半径（米）
    pub personal_space_radius_m: f32,
    /// 直接 vs 间接——"你不合格" vs "也许还有改进空间"
    pub directness: f32,               // 0=极其委婉, 1=极其直接
    /// 沉默的容忍度——对话中的沉默是否让人不适
    pub silence_tolerance: f32,        // 0=沉默=尴尬, 1=沉默=深思熟虑
    /// 情感表达的公开程度
    pub emotional_expressiveness: f32, // 0=克制, 1=奔放
    /// 敬语系统
    pub honorifics: HonorificSystem,
    /// 身体接触规范
    pub touch_norms: TouchNorms,
}

pub enum EyeContactNorm {
    /// 直视=尊重
    Direct = 0,
    /// 低头=尊重（东亚式）
    Averted = 1,
    /// 取决于社会地位——低位者低头
    StatusBased = 2,
}

pub struct HonorificSystem {
    pub has_grammatical_honorifics: bool,
    pub age_based: bool,       // 长辈必须用敬语
    pub status_based: bool,    // 高地位必须用敬语
    pub intimacy_overrides: bool,  // 亲密关系可以省略敬语
}

pub struct TouchNorms {
    pub handshake_allowed: bool,
    pub hug_allowed: bool,
    pub opposite_gender_touch_allowed: bool,
    pub public_touch_allowed: bool,
    pub touch_as_dominance: bool,    // 拍肩=友好还是冒犯？
}
```

### 4.3 CommunicationNorms → 对话行为修正

```rust
impl CommunicationNorms {
    /// 文化规范修正发言权竞争 (FreeForm TurnMode)
    pub fn turn_taking_modifier(&self, speaker: &NpcData, listener: &NpcData) -> f32 {
        // 高 interruption_tolerance → 打断更容易 → interrupt_threshold 降低
        // 低 interruption_tolerance → 必须等对方说完 → interrupt_threshold 提高
        1.0 + (0.5 - self.interruption_tolerance)
    }

    /// 文化规范修正对话中的沉默处理
    pub fn silence_response(&self, duration_seconds: f32) -> Option<DialogueIntent> {
        if duration_seconds > 3.0 && self.silence_tolerance < 0.3 {
            // 该文化不能容忍沉默 → NPC 会说点什么填补
            Some(DialogueIntent {
                intent_type: DialogueIntentType::Emotional {
                    emotion: Emotion::ANXIETY,
                    intensity: 0.3,
                    cause_memory_id: None,
                },
                priority: 0.5,
                target: None,
                topic_params: HashMap::new(),
                expires_at: GameTimestamp::now() + 5,
            })
        } else {
            None  // 沉默很正常——不说话
        }
    }
}
```

## 五、Godot 渲染集成

### 5.1 非语言信号→动画

```
Rust NonVerbalExpression → GDExtension → Godot AnimationPlayer

面部表情:
  FacialExpression::Smile { warmth: 0.8 } → mix shape: "smile" = 0.8
  FacialExpression::Anger { intensity: 0.6 } → mix shape: "brow_furrow" = 0.6

手势:
  Gesture::Nod { emphasis: 0.5 } → blend animation: "head_nod", weight = 0.5
  Gesture::Point { direction, target } → IK → arm bone → point toward target

身体姿态:
  PoseOffset { chest_expand: 0.4 } → spine bone rotation +0.4 radians

视线:
  GazeAction::LookAt(target) → IK → head/neck bones → look_at(target.position)
  GazeAction::AvoidEyeContact → head bone rotation yaw +15° away from target

社交距离:
  social_distance_m: 1.5 → NPC movement → step back/forward to maintain distance
```

### 5.2 玩家感知非语言信号

非语言信号通过 Godot 的 NPC 渲染直接传达给玩家——不需要字幕：

- 玩家看到铁匠老王在说"那把剑——"时握紧了拳头 (anger)。玩家自然知道：老王对那笔交易不满意。
- 玩家看到学者 NPC 在说"这件事很复杂"时目光闪烁、看向别处 (fear/evasion)。玩家自然怀疑：学者在隐瞒什么。
- 玩家看到酒馆角落两个 NPC 低声说话、不时偷看周围 (Conspiracy mode visible behavior)。玩家自然会走过去听。

这是**非语言表达的魅力**——信息不需要通过文字传达，而是通过人类天然的社交直觉来感知。
