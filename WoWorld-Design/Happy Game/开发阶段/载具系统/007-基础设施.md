## 一、核心原则

> **港口和道路由世界生成。铁路由 NPC 建造（极低概率）。载具初始分布从经济参数派生。全部走已有系统。**

| 设施 | 来源 | 载具系统的角色 |
|------|------|-------------|
| **港口** | P4 port_quality + P6 WFC 港口建筑模板 | 定义 `ConstructionSite::Waterfront` 的条件——多深的水能建什么船 |
| **道路** | P7 MST | 不参与 |
| **海上航线** | P7 SeaRoute——但航速改为查询 VehicleDef.base_speed_ms | 提供航速参数 |
| **铁路** | P9 历史模拟——NPC 建造（非 P7 生成） | 定义 `requires_rail=true` 的载具约束 |
| **初始载具** | P8.5 载具初始化 | 定义分布逻辑的输入参数 |
| **后续新增** | P9 历史模拟 + 运行时建造 | 提供 VehicleConstructionSpec |

## 二、港口——扩展现有管线

港口不新建 Phase。它是已有多个 Phase 的**叠加效果**：

```
P1 海洋系统 → 海岸线形状 + 天然港湾质量 + 水深分布
P2 自然基底 → 海滩/悬崖/内河入海口类型
P4 聚落选址 → 沿海聚落 = trade_score × 港湾质量
               → 已有的 port_quality 字段定义港口潜力
P6 建筑生成 → WFC 模板新增港口类建筑:
                Dock(码头) / Slipway(造船滑道) / Shipyard(船厂)
                Lighthouse(灯塔) / Warehouse(仓库) / Breakwater(防波堤)
P7 交通网络 → SeaRoute 端点 = port_quality > 0 的聚落
```

### 2.1 载具系统的港口接口

```rust
/// 载具系统定义——供世界生成查询
/// "此地是否适合建港口？能建多大的船？"
struct PortSuitability {
    /// 港口的自然条件评分 0-1
    natural_harbor_quality: f32,
    /// 最大吃水深度（米）——该港口最深停泊位
    max_berth_draft_m: f32,
    /// 是否有造船滑道（Slipway）——决定了能建造什么载具
    has_slipway: bool,
    /// 是否有船坞（Drydock）——决定了能否进行港口维修
    has_drydock: bool,
    /// 是否有魔力充能法阵（仅少数大港口）
    has_mana_charging_array: bool,
}
```

载具系统不定义港口长什么样——那是建筑系统 WFC 模板的事。载具系统只定义"船需要什么条件才能建、才能修、才能停"。

## 三、铁路——不生成，由 NPC 建造

铁路**不在 P7 生成**。P7 的道路是"走出来的"（MST），P7 的海上航线是"算出来的"（洋流+港口间距）。铁路是**建出来的**（NPC 通过经济+权力+建造系统施工）。

### 3.1 铁路不生成的理由

| | 道路 | 铁路 |
|---|---|---|
| **形成方式** | 人走多了→路就出来了 | 必须主动铺设轨道——需要技术+资本+组织 |
| **维护** | 走的人多→路自然保持 | 必须持续维护——没人维护就废弃 |
| **所有权** | 公共（领主/Group） | 铁路公司（Group）私有或特许经营 |
| **可扩展性** | 踏出来的小路→自然形成网络 | 需要正式决策+融资+施工 |

### 3.2 铁路涌现条件——P9 历史模拟

```rust
/// [TUNING] 铁路涌现概率——每百年每个符合条件的城市对
const RAILWAY_EMERGENCE_BASE_CHANCE_PER_CENTURY: f32 = 0.03;

fn simulate_railway_emergence(world: &mut WorldState, era_span: GameDuration) {
    // 查技术前提
    let rail_capable_cultures = world.cultures.iter()
        .filter(|c| c.technology.rail_engineering > 0.2);

    for culture in rail_capable_cultures {
        // 候选城市对——tier≥3 且属于同一文化区域
        let city_pairs = find_high_trade_city_pairs(world, culture);

        for (city_a, city_b) in city_pairs {
            // 经济动机: 铁路比马车便宜 40%+ 才值得建
            let wagon_cost = estimate_transport_cost(Wagon, city_a, city_b, world);
            let rail_cost = estimate_transport_cost(Rail, city_a, city_b, world);
            if rail_cost > wagon_cost * 0.6 { continue; }

            // 整体概率 × 经济动机强度 = 涌现概率
            let emerge_chance = RAILWAY_EMERGENCE_BASE_CHANCE_PER_CENTURY
                * (1.0 - rail_cost / wagon_cost)  // 越便宜越可能
                * culture.technology.rail_engineering;  // 技术越高越可能

            if seeded_rng(world.seed, city_a, city_b).gen_bool(emerge_chance) {
                trigger_railway_construction(city_a, city_b, world);
            }
        }
    }
}
```

### 3.3 轨距涌现

```
第一家铁路公司铺设第一条铁路时 → 选定一个轨距
之后同一文化区域的后续铁路 → 大概率兼容同一轨距
跨文化区域的铁路相遇 → 轨距可能不同：
  → 转运站涌现（货物从A轨距列车卸下→装到B轨距列车）
  → 可适配的双轨距列车（技术更高才能制造）
  → 这个转运站可能发展成繁荣的贸易城镇（涌现！）
```

## 四、初始载具分布——P8.5

在 P8 NPC 初始化之后、P9 历史模拟之前。从经济参数派生——数量由当地经济和人口决定，船长和船员从 NPC 池中分配。

```rust
fn place_initial_vehicles(
    seed: u64,
    settlements: &[Settlement],
    cultures: &CultureRegistry,
    npcs: &NpcRegistry,
) -> Vec<(VehicleDefId, WorldPos, InitialCrew)> {

    let mut vehicles = Vec::new();

    for settlement in settlements {
        let rng = seeded_rng(seed, settlement.id);

        // ── 港口载具 ──
        if settlement.port_quality > 0.0 {
            let fleet_strength = settlement.port.trade_volume
                * settlement.economic_prosperity
                * settlement.port_quality;

            // 渔船: 数量 ∝ 沿海人口 × 鱼类资源
            let fishing_boats = poisson_sample(
                settlement.population * 0.03 * settlement.port.fishing_potential
            );
            for _ in 0..fishing_boats {
                vehicles.push(create_initial_vehicle(
                    "小型渔船", settlement, rng, npcs, cultures
                ));
            }

            // 商船: 数量 ∝ 贸易路线数 × 经济繁荣度
            let merchant_ships = poisson_sample(
                settlement.port.sea_routes.len() as f32 * 0.15
                * settlement.economic_prosperity
            );
            for _ in 0..merchant_ships {
                vehicles.push(create_initial_vehicle(
                    "中型商船", settlement, rng, npcs, cultures
                ));
            }

            // 偶尔有大船（旗舰/探险船/大型商船）
            if rng.gen_bool(fleet_strength * 0.02) {
                vehicles.push(create_initial_vehicle(
                    "大型远洋商船", settlement, rng, npcs, cultures
                ));
            }
        }

        // ── 陆地载具 ──
        // 马车: 按人口比例 + 商队路线经过频率
        let carts = (settlement.population * 0.02) as u32;
        for _ in 0..carts {
            vehicles.push(create_initial_vehicle(
                "小型马车", settlement, rng, npcs, cultures
            ));
        }
    }

    vehicles
}
```
