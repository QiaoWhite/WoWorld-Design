# 005 — 语音管道 v1.0

> **核心设计**：话语的内容和声音是两层——内容归语言表达模块（ExpressionRef + 文本 + resolve_partial），声音归音频模块（传播 + 时间维度 + 清晰度 + 打断）。语言表达不知道音频在轮询 `CurrentSpeech` 字段，音频不知道文本内容——只提供 `fraction_heard + clarity` 让听者自己去取。

> **所有权变更**：VoiceProfile（声学参数）、VoiceEmotionModulation（情绪→声学调制）、VoiceManager（播放队列+打断）、audible_radius（传播距离）——**所有权从语言表达模块迁至音频模块**。TtsEngine trait + TtsConfig 保留在语言表达（渲染层）。

## §一 内容×声音分离

| 层 | 归属模块 | 内容 | 对标 |
|----|---------|------|------|
| 内容（说什么） | 语言表达 | ExpressionRef + 完整文本 + resolve_partial() | 书的内容 |

**零直接依赖**：语言表达设 `NpcData.current_speech`（不知道音频在读）；音频每帧轮询（不知道文本内容）。

## §三 音频模块检测与 OngoingSpeech 追踪

### 3.1 每帧轮询

```rust
// 音频模块内部——ingestion/speech_detect.rs
fn detect_speech_changes(
    speeches: &[(EntityId, Option<&CurrentSpeech>)],
    ongoing: &mut HashMap<EntityId, OngoingSpeech>,
) -> Vec<SpeechEvent> {
    for (entity, speech) in speeches {
        match (speech, ongoing.get(entity)) {
            (Some(current), None) => {
                // 开始说话
                ongoing.insert(*entity, OngoingSpeech::new(*entity, current));
                push(SpeechEvent::Started { entity: *entity });
            }
            (Some(current), Some(prev)) => {
                // 检查是否换了新句子（started_at 不同）
                if prev.started_at != current.started_at {
                    *ongoing.get_mut(entity).unwrap() = OngoingSpeech::new(*entity, current);
                    push(SpeechEvent::NewSentence { entity: *entity });
                }
            }
            (None, Some(prev)) => {
                // 停止了
                let reason = detect_cancel_reason(*entity);
                push(SpeechEvent::Stopped { entity: *entity, reason, elapsed: prev.elapsed(), total: prev.total_duration });
                ongoing.remove(entity);
            }
            (None, None) => {}
        }
    }
}
```

### 3.2 OngoingSpeech

```rust
struct OngoingSpeech {
    speaker: EntityId,
    started_at: GameInstant,
    total_duration: Duration,
    intensity_db: f32,
    frequency_band: FrequencyBand,  // 人声 = Mid
    words_per_second: f32,
    delivery: SpeechDelivery,
}

impl OngoingSpeech {
    fn fraction_at(&self, now: GameInstant, propagation_delay: f32) -> f32 {
        let effective = self.elapsed_at(now) - propagation_delay;
        if effective <= 0.0 { 0.0 }
        else { (effective / self.total_duration.as_secs_f32()).min(1.0) }
    }
}
```

### 4.2 听者整合——获取部分内容

```rust
fn digest_speech(audio: &dyn AudioQuery, lang: &dyn ExpressionQuery,
                 speaker: EntityId, listener: EntityId, now: GameInstant) -> Option<PerceivedUtterance> {
    let perception = audio.perceived_speech(listener_pos, speaker, now, hearing)?;
    if perception.fraction_heard <= 0.0 { return None; }
    let level = speech_clarity_level(&perception.clarity);
    let partial = lang.resolve_partial(speech_ref, perception.fraction_heard, level);
    Some(PerceivedUtterance {
        speaker,
        text_snippet: partial,
        is_complete: perception.fraction_heard >= 1.0 && !perception.is_ongoing,
        was_interrupted: perception.cancel_reason.is_some(),
        clarity_level: level,
    })
}
```

### 5.1 CancelReason

```rust
pub enum CancelReason {
    SourceDied,              // 源死亡
    SourceUnconscious,       // 源昏迷
    SourceTeleported,
    SourceSubmerged,         // 入水
    CombatImpact,            // 外部物理打断
    Preempted(SoundEventId), // 更高优先级抢占
    MagicallySilenced,
    SourceVoluntaryStop,     // 正常说完（非打断）
}
```

### 5.2 打断检测

```rust
fn detect_cancel_reason(speaker: EntityId) -> CancelReason {
    if is_dead(speaker) { return CancelReason::SourceDied; }
    if is_unconscious(speaker) { return CancelReason::SourceUnconscious; }
    if is_submerged(speaker) { return CancelReason::SourceSubmerged; }
    if is_magically_silenced(speaker) { return CancelReason::MagicallySilenced; }
    CancelReason::SourceVoluntaryStop
}
```

### 5.3 五种打断来源

| 来源 | 触发 | 仲裁 |
|------|------|------|
| 源实体状态变化 | 死亡/昏迷/传送/入水 | 音频每帧检查 |
| 更高优先级事件 | Critical 级语音 | VoicePriority 五级队列 |
| 距离变化 | 源超出听觉范围 | 传播模型 |
| 环境覆盖 | 雷声/爆炸掩盖语音 | 掩蔽引擎 |
| 听者状态变化 | 听者昏迷/传送 | 听者侧过滤 |

pub struct VoiceProfile {
    pub profile_id: VoiceProfileId,

    // 声学参数
    pub base_pitch_hz: f32,        // 基频
    pub pitch_variance: f32,       // 自然波动 0-1
    pub timbre: TimbrePreset,      // 音色
    pub base_speed_wpm: f32,       // 词/分钟
    pub expressiveness: f32,       // 0=单音调，1=丰富抑扬
    pub breathiness: f32,          // 0=清爽，1=气声
    pub roughness: f32,            // 0=平滑，1=粗粝

    // 声源信息
    pub source: VoiceSource,
}

pub enum TimbrePreset {
    Deep, Bright, Husky, Crisp, Warm, Nasal, Breathy, Rough, Smooth, Thin,
}

/// 生成公式：种族 × 性别 × 年龄 × 大五人格
impl VoiceProfile {
    fn generate(race: &RaceTraits, gender: Gender, age: f32, personality: &BigFive) -> Self {
        VoiceProfile {
            base_pitch_hz: race.voice_pitch_base(gender) + age_pitch_shift(age),
            pitch_variance: personality.neuroticism * 0.3 + personality.extraversion * 0.2,
            timbre: derive_timbre(race, gender, age),
            base_speed_wpm: race.base_speech_rate
                * (1.0 - age * 0.002)
                * (1.0 + personality.extraversion * 0.2),
            expressiveness: personality.extraversion * 0.6 + personality.openness * 0.4,
            breathiness: (1.0 - personality.conscientiousness) * 0.3,
            roughness: (age.max(20.0) - 20.0) * 0.004,
            source: VoiceSource { race_id: race.id, gender, age },
        }
    }
}

## §七 VoiceEmotionModulation（情绪→声学调制）

```rust
pub struct VoiceEmotionModulation {
    pub pitch_shift_st: f32,        // 半音偏移
    pub speed_multiplier: f32,      // 速度倍率
    pub volume_multiplier: f32,     // 音量倍率
    pub tremor: f32,                // 颤抖 0-1
    pub breathiness_add: f32,       // 额外气声
    pub roughness_add: f32,         // 额外粗粝
}

fn emotion_to_modulation(emotion: &EmotionState, baseline: &VoiceProfile) -> VoiceEmotionModulation {
    VoiceEmotionModulation {
        pitch_shift_st: emotion.joy * 2.0 - emotion.sadness * 3.0 + emotion.anger * 1.0,
        speed_multiplier: 1.0 + emotion.anger * 0.3 + emotion.fear * 0.5 - emotion.sadness * 0.3,
        volume_multiplier: 1.0 + emotion.anger * 0.4 - emotion.fear * 0.4,
        tremor: emotion.fear * 0.6 + emotion.sadness * 0.3,
        breathiness_add: (1.0 - baseline.breathiness) * emotion.fatigue() * 0.5,
        roughness_add: emotion.anger * 0.3,
    }
}
```

## §八 VoiceManager（播放队列+打断）

```rust
struct VoiceManager {
    queue: VecDeque<VoicePacket>,
    current: Option<VoicePacket>,
    engine: Box<dyn TtsEngine>,    // TtsEngine 在语言表达模块
    config: TtsConfig,
}

impl VoiceManager {
    fn enqueue(&mut self, packet: VoicePacket) {
        match packet.priority {
            VoicePriority::Critical => {
                self.engine.stop();  // 立即打断
                self.current = Some(packet);
            }
            VoicePriority::High | VoicePriority::Normal => {
                self.queue.push_back(packet);
            }
            VoicePriority::Ambient | VoicePriority::Background => {
                self.queue.push_back(packet);
                self.queue.make_contiguous().sort_by_key(|p| p.priority as u8);
            }
        }
    }
}

pub enum VoicePriority {
    Critical = 0,   // 战斗警报/呼救——打断一切
    High = 1,       // 回应玩家对话
    Normal = 2,     // 一般对话
    Ambient = 3,    // NPC 间闲聊
    Background = 4, // 旁白/环境文字
}
```
