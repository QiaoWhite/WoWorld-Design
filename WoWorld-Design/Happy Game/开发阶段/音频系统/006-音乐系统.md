# 006 — 音乐系统 v1.0

> **核心设计**：WoWorld 的音乐有三层——系统驱动的动态 BGM（情境×区域×分层强度）、世内 NPC 演奏（走进酒馆→诗人的演奏就是背景音乐）、玩家自定义音乐（mp3/flac 播放列表）。三层可以在交叉淡入淡出中自然过渡——对标天气系统的 `transition_blend`。音乐完全在渲染层——NPC 听不到（模拟层不可感知）。

## §一 三层模型

```
Layer 1: 系统驱动动态音乐
  ├─ 情境→预设映射（探索/战斗/城镇/室内/地下/高空/水下/节日/叙事）
  ├─ 每首曲目 3-5 层（基础/节奏/和声/旋律）根据威胁/氛围动态混合
  └─ 区域音乐（群系×文化区域→音乐风格）

Layer 2: 世内 NPC 演奏
  ├─ FineArts.music_performance 技能驱动
  ├─ 街头艺人/酒馆乐队/宫廷乐团/宗教合唱
  └─ 进入可闻范围→覆盖 BGM

Layer 3: 玩家自定义音乐
  ├─ 文件夹/播放列表（mp3/flac/ogg/wav）
  └─ 按情境覆盖规则
```

三者通过交叉淡入淡出自然过渡——播放器一次只有一个活跃源，但过渡期间可以混合。

### 2.2 音乐分层模型

```rust
pub struct MusicLayerState {
    /// 每层的当前混合比例 [0.0, 1.0]
    /// [0]=基础层(持续), [1]=节奏层, [2]=和声层, [3]=旋律层
    pub layers: [f32; 4],

    /// 目标混合比例（tween 目标）
    pub target: [f32; 4],

    /// 过渡速度 (0-1)
    pub transition_speed: f32,
}
```

**层如何随游戏情境变化**：

| 情境 | 基础 | 节奏 | 和声 | 旋律 |
|------|------|------|------|------|
| 探索·和平 | 1.0 | 0.4 | 0.7 | 0.5 |
| 探索·黄昏 | 1.0 | 0.3 | 0.9 | 0.6 |
| 战斗·小规模 | 1.0 | 0.8 | 0.6 | 0.7 |
| 战斗·Boss | 1.0 | 1.0 | 1.0 | 1.0 |
| 城镇·白天 | 0.8 | 0.3 | 0.6 | 0.4 |
| 酒馆·室内 | 0.5 | 0.2 | 0.3 | — |
| 水下 | 0.6 | 0.2 | 0.5 | 0.3 |

### 2.3 过渡规则

```rust
fn music_transition(from: &MusicState, to: &MusicContext, config: &MusicConfig) -> MusicTransition {
    let target_preset = config.preset_for(to);
    let crossfade_duration = match (from.context, to) {
        // 和平→战斗：快速过渡
        (MusicContext::Exploration {..}, MusicContext::Combat {..}) => 1.5,
        // 战斗→和平：慢速回落
        (MusicContext::Combat {..}, MusicContext::Exploration {..}) => 3.0,
        // 城镇→室内：中速
        (MusicContext::Settlement {..}, MusicContext::Interior {..}) => 2.0,
        // 默认
        _ => config.crossfade_default_s,
    };
    MusicTransition { target_preset, duration_s: crossfade_duration }
}
```

### 3.2 世内演奏→BGM 覆盖

```rust
fn compute_active_music_source(
    pos: Vec3,
    context: MusicContext,
    performers: &[InWorldPerformer],
    player_config: &PlayerMusicConfig,
) -> Vec<WeightedMusicSource> {
    // 优先级：玩家音乐 > 世内演奏 > 系统 BGM（可配置）

    // 1. 检查玩家自定义音乐
    if player_config.enabled && player_config.should_override(&context) {
        return vec![WeightedMusicSource { source: MusicSource::PlayerMusic, weight: 1.0 }];
    }

    // 2. 检查世内演奏者（在可闻范围内）
    let nearby: Vec<_> = performers.iter()
        .filter(|p| distance(p.pos, pos) < 50.0)
        .collect();
    if !nearby.is_empty() {
        let best = nearby.iter().max_by_key(|p| (p.performance_quality * 100.0) as u32).unwrap();
        return vec![WeightedMusicSource { source: MusicSource::InWorldPerformer { performer: best.performer, performance: best.performance_ref }, weight: 1.0 }];
    }

    // 3. 系统 BGM
    let preset = resolve_preset(&context);
    vec![WeightedMusicSource { source: MusicSource::System { preset }, weight: 1.0 }]
}
```

### 3.3 世内演奏的审美价值

NPC 演奏 = `AestheticSignal` 的来源——技能系统已有 `FineArts.music_performance`。听众通过审美模块的 `judge()` 产生审美判断。

## §四 Layer 3：玩家自定义音乐

```rust
pub struct PlayerMusicConfig {
    pub enabled: bool,
    pub playlist_path: Option<String>,     // 文件夹路径或 .m3u
    pub override_rules: PlayerMusicOverride,
    pub volume_ratio: f32,                 // 玩家音乐 vs 游戏音乐（0.0-1.0）
    pub supported_formats: Vec<String>,    // ["mp3","flac","ogg","wav"]
}

pub enum PlayerMusicOverride {
    Never,               // 永远游戏音乐
    ExplorationOnly,     // 仅探索时（默认）
    ExceptCombat,        // 战斗以外
    Always,              // 总是玩家音乐
}
```

**纯 Godot 实现**——Rust 不处理音频文件 I/O。

## §五 过渡模型

### 5.1 交叉淡入淡出

对标天气系统的 `transition_blend` 模式：

```rust
pub struct MusicBlend {
    pub active_sources: SmallVec<[WeightedMusicSource; 3]>,  // 最多 3 个混合
    pub crossfade_progress: f32,  // 0.0-1.0
    pub crossfade_duration_s: f32,
}

pub struct WeightedMusicSource {
    pub source: MusicSource,
    pub weight: f32,  // 0.0-1.0（所有活跃源权重和为 1.0）
}

pub enum MusicSource {
    System { preset: MusicPreset },
    InWorldPerformer { performer: EntityId, performance: PerformanceRef },
    InWorldDevice { device_id: EntityId },
    PlayerMusic { track: String },
}
```

### 5.2 过渡实例

```
玩家从野外走进有吟游诗人的酒馆：
  t=0.0s  System: Exploration(forest)     1.0
  t=1.0s  进入酒馆 → 检测到吟游诗人
  t=1.0-4.0s  交叉淡入淡出:
    System: Exploration(forest)    1.0 → 0.0
    InWorldPerformer: bard         0.0 → 1.0
  t=4.0s+ InWorldPerformer: bard         1.0

吟游诗人弹完曲子：
  t=0.0s  诗人停止
  t=0.0-2.5s  交叉淡入淡出:
    InWorldPerformer: bard         1.0 → 0.0
    System: Interior(tavern)       0.0 → 0.3（酒馆低音量氛围）
```

```toml
[music]
crossfade_default_s = 2.5
combat_transition_speed = 0.8       # 快速进入战斗音乐

[music.player]
default_override = "exploration_only"
max_playlist_size = 1000

forest_day = "music/exploration/forest_day"
forest_night = "music/exploration/forest_night"

[music.context.combat]

[music.context.interior]
```
