# 001-魔法总纲

魔法是通过特定仪式、物质、语言、意志来运用元素以影响世界的一种力量。魔法的基本元素是金、木、水、火、土、风、雷、电、血、灵。世界中每一个物质都包含这些元素中的一种或几种。其中"灵"是生物固有的元素，也是施法者与其他元素联系的桥梁。

---

## 〇、两条公理

整个魔法体系建立在这两条不可动摇的公理之上：

**1. 二元构成原理**：万物由十元素构成。**血**是金木水火土风电雷按特定比例组合而成的"肉身元素"；**灵**是寄宿于血、连接精神与物质的"灵魂元素"，是施法的核心桥梁。关于灵的本质是否为血的某种形式，学术界仍有争论。

**2. 意志投射原理**：施法本质是施法者以自身**灵**元素为桥梁，将意志投射于目标，支配其内部元素，引发**增加、减少、吸收、释放、排斥**五种基本变化，以及由基本变化组合而成的**转化、转移、链接**等复合影响。普通人的意志只能驱动自身**血**元素（肉体），而施法者的意志可直接与外界元素交互。

---

## 一、魔法在WoWorld中的定位

魔法不是"战斗技能"——它是**世界的基本物理法则**。每一个生物（除亡灵外）都有灵元素→都有魔法潜能。农民用灵元素驱动自己的血元素种地；铁匠用灵元素感知铁的纯度；施法者用灵元素直接支配外界元素。

魔法与战斗、经济、文化、政治、技术的关系是全方位的——不仅是"学技能"。详见各子文档。

## 三、魔法方程式速览

> 完整详解见 [[004-魔法方程式]]。

**魔法 = 范围 + 对象 + 接触 + 元素 + 影响 + 时间 + 条件**

| 要素 | 核心问题 | 选项 |
|------|----------|------|
| **范围** | 作用于多大空间？ | 点 / 线 / 面 / 体 |
| **对象** | 作用于谁？ | 类型 × 数目 × 命中方式 × 指向 |
| **接触** | 怎样到达目标？ | 过程接触(沿路径) / 凭空接触(直接投射) |
| **元素** | 涉及哪种元素？ | 单一元素 / 多元混合 |
| **影响** | 对目标做什么？ | 基本操作(增减吸收释放排斥) / 复合影响(转化转移链接) |
| **时间** | 持续多久？ | 瞬时 / 持续 / 触发式 / 永久 |
| **条件** | 需要满足什么？ | 自身 / 环境 / 物质 / 仪式 / 契约 |

## 四、魔法类型速览

> 完整详解见 [[006-魔法类型]]。

| 系 | 核心影响 | 简述 | 战斗层级 |
|----|---------|------|---------|
| **变幻系** | 转化、转移 | 改变物质元素构成与形态 | 战略层 |
| **召唤系** | 转移、链接 | 与远方事物建立链接并传送 | 战略层 |
| **破坏系** | 释放、减少 | 瞬间释放大量元素能量——最常用 | 节奏/本能层 |
| **守护系** | 吸收、排斥 | 制造元素力场/护盾 | 本能/节奏层 |
| **惑心系** | 影响(心理) | 以灵元素直接影响意志 | 战略层 |
| **疗愈系** | 增加、转化 | 精细补充或转化血元素 | 节奏层 |
| **死灵系** | 链接、扭曲 | 扭曲灵-血关系，驱使亡骸 | 战略层 |

每一系都包含两种接触方式的具体魔法——接触方式不是某一系的专属。

## 六、元素→魔法类型适配参考

| 元素 | 天然适合的系 | 天然困难的系 |
|------|------------|------------|
| 火 | 破坏系(扩散性天然适合释放) | 守护系(火不擅长静态力场) |
| 水 | 变幻系(相态变化) + 疗愈系 | 破坏系(纯水破坏力有限) |
| 土 | 守护系(惰性天然适合静态防御) | 惑心系(土不涉及意志) |
| 风 | 召唤系(渗透性适合传送) | — |
| 雷 | 破坏系(振动冲击) + 惑心系(声音影响意志) | — |
| 电 | 破坏系(闪电) + 召唤系(信号/通信) | 守护系(电难以形成静态屏障) |
| 金 | 变幻系(塑形) + 守护系(物理屏障) | 惑心系 |
| 木 | 疗愈系(生命生长) + 变幻系 | 破坏系(木不倾向主动毁灭) |
| 血 | 疗愈系(重组) | 破坏系(破坏血=伤害生命) |
| 灵 | 所有系——灵是桥梁 | — |

## 七、与生命系统的接口

- **灵元素**：所有生命（除亡灵）都有灵元素。灵元素含量=法术潜能。见 [[004-身体状态与生命过程|生命系统-身体状态]]
- **魔力天赋**：生命基类的 `spirit_capacity` + `magic_power` 决定施法者的强度上限
- **法术熟练度**：生命/战斗系统共享——使用次数→自动化降级（战略→节奏→本能）
- **亡灵**：灵元素异常残留——死灵系魔法可直接控制亡灵
