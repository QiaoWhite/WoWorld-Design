## 亲和关系

某些元素在自然状态下倾向于协同作用。在施法方程式中混合亲和元素，通常可以降低魔力消耗或增强效果。

| 主导元素 | 亲和元素 | 协同效果举例 |
|----------|----------|-------------|
| 火 | 风 | 风助火势——火焰范围扩大，持续时间延长 |
| 火 | 木 | 木生火——可作为持续燃料，延长燃烧 |
| 水 | 土 | 水土混合——形成泥沼、流沙等复合地形效果 |
| 水 | 雷 | 水传声——雷元素（声音）在水中传播更远 |
| 风 | 电 | 风带电——电荷随气流扩散 |
| 风 | 木 | 风携种子/花粉——木元素在风中的扩散增强 |
| 金 | 火 | 火炼金——高温可改变金属形态 |
| 金 | 电 | 导电——电元素在金介质中精确传导 |
| 土 | 木 | 土养木——木元素在土中扎根生长 |

> 亲和关系不强制二元绑定。一个魔法可以同时利用多组亲和关系，但方程式的复杂度会相应提升。详见 [[04-魔法方程式]]。

## 排斥关系

某些元素之间存在天然的对抗性。在同一个魔法方程式中强行混合排斥元素，将大幅增加魔力消耗或导致施法不稳定。

| 元素A | 元素B | 排斥表现 |
|-------|-------|----------|
| 水 | 火 | 水灭火——两者同时存在时，强势一方压制弱势一方 |
| 土 | 风 | 土阻风——风难以穿透致密的土元素 |
| 金 | 木 | 金克木——金属切割/压制植物生长 |
| 火 | 金 | 火熔金——高温削弱金属的结构完整性 |
| 电 | 土 | 土绝缘——土元素阻碍电的传导 |
| 木 | 土 | 木破土——根系可以破坏土元素的结构 |

> 排斥关系不是绝对的禁止——一个技艺高超的法师可以在同一个魔法中同时运用水火元素（如制造高温蒸汽），但这需要在方程式中引入额外的调和参数，消耗远高于使用亲和元素组合。

## 组合规则

### 元素数量的限制

一个魔法方程式中可以指定的元素数量，受施法者的 [[06-魔力体系|魔力]] 和意志强度制约：

- **单一元素**（1种）：最基础、最稳定。魔力消耗基准值 ×1.0
- **二元组合**（2种）：常见。魔力消耗基准值 ×1.5（亲和）/ ×3.0（排斥）
- **三元混合**（3种）：高阶。魔力消耗基准值 ×3.0（全亲和）/ ×8.0（含排斥）
- **四元及以上**：传奇级。每增加一种元素，魔力消耗呈指数增长

混合排斥元素不是不可能，但需要极其精密的方程式设计和强大的灵元素控制力。许多传奇魔法正是通过对排斥元素的精准调和而实现的。

### 混合后不产生新元素

金、木、水、火、土、风、雷、电八种元素按不同比例混合，可以模拟自然界的各种物质形态——但这种混合不会创造出"第十一种元素"。元素的种类是宇宙的常量。

**血**元素本身就是八元素的特定组合（参见 [[02-元素]]），因此在常规魔法方程式中，指定"血"元素等同于指定一组预调和的八元素比例。这是疗愈系魔法的理论基础。

**灵**元素独立于上述八元素体系，不参与常规的亲和/排斥计算。详见下文"灵与血的特殊地位"。

## 环境元素浓度

世界不同区域的元素浓度并非均匀分布。环境浓度影响施法难度和魔力消耗——在高浓度区域施展对应元素的魔法更加容易，反之则需要施法者从自身魔力中补足差额。

| 环境 | 高浓度元素 | 低浓度元素 |
|------|-----------|-----------|
| 火山 / 熔岩区 | 火、土 | 水 |
| 海洋 / 大型水体 | 水 | 火 |
| 密林 | 木、土 | 风（受树冠阻挡） |
| 高山 / 开阔平原 | 风 | — |
| 矿脉 / 地下 | 金、土 | 风、火 |
| 雷暴区 | 雷、电、风 | — |
| 战场 / 墓地 | 血 | — |
| 人群密集区（城市） | 灵（多人意志聚集） | — |
| 地脉节点 | 全部元素浓度提升 | — |

环境浓度会在游戏中动态变化——一场森林大火会暂时降低木元素浓度并升高火元素浓度，暴雨会提升水元素浓度。详见 [[想法/世界生成/自然景观|自然景观生成]]。

## 元素极化：粗元素 vs 精炼元素

自然环境中的元素大多是"粗元素"——浓度低、杂质多、不够纯粹。通过特定技术（如法阵过滤、触媒催化、长时冥想），可以将粗元素精炼为高纯度的"精炼元素"。

- **粗元素**：随处可见，响应慢，魔力消耗高，效果上限低
- **精炼元素**：需主动提炼，响应快，魔力消耗低，效果上限高
- **极化成本**：精炼过程本身消耗魔力和时间——是否值得取决于法术的需求

部分高级法术（特别是需要精确控制的疗愈系和惑心系）要求使用精炼元素才能安全施展。

## 灵与血的特殊地位

### 灵元素：不参与亲和/排斥

**灵**元素在元素交互体系中享有特殊地位——它不与金、木、水、火、土、风、雷、电中的任何一种产生亲和或排斥关系。灵是**中立桥梁**，它的作用是将施法者的意志传递到其他元素，而非直接参与元素间的化学式交互。

这意味着：
- 涉及"灵"元素的魔法（如惑心系）的魔力消耗计算不适用亲和/排斥规则
- 灵元素的"浓度"取决于施法者自身的精神力强度，而非环境
- 灵元素不能被"精炼"——它始终以最纯粹的形式存在于每个生物体内

### 血元素：八元素的预调和

**血**元素是八元素的特定比例组合，它在大多数元素交互场景下表现为"中立"——一个以血为目标的魔法（如疗愈系），其成败不取决于血元素与其他元素的亲和/排斥，而取决于施法者对八元素各自比例的精确理解和操控。

唯一的例外是死灵系魔法——它通过扭曲灵-血的正常关系来运作，这是其被列为禁忌的根本原因。详见 [[17-魔法禁忌与风险]]。

## 元素交互与魔法设计

在 [[12-魔法设计]] 的理论推导阶段，设计者必须根据目标效果选择合适的元素组合，并评估亲和/排斥关系对魔力消耗的影响。在参数建模阶段，元素亲和度是魔力消耗公式的重要系数。

> 一个好的魔法设计不仅是"把几个要素拼在一起"——它在元素层面已经考虑到了自然界本身的倾向。利用亲和、规避排斥、善用环境浓度——这正是区分熟练设计者与新手的关键。
