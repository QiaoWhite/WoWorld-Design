## 魔力的本质

魔力（Mana）是施法者以自身**灵**元素为媒介，从体内**血**元素中提炼并储存的一种可消耗能量。它是施法者将意志转化为元素变化的"燃料"。

在魔法学的视角下，魔力的产生过程可以描述为：

```
灵元素（意志驱动） → 作用于血元素（肉身能量） → 提炼出魔力 → 储存于灵-血界面
```

一个不会魔法的人同样拥有灵元素和血元素，但其灵元素缺乏"提炼魔力"这一训练出来的能力。这正是魔法资质的核心含义——详见 [[16-魔法学习与传承]]。

魔力与灵元素强度直接相关，但两者并非同一个概念：灵元素强度是先天+后天培养的**潜力上限**，魔力是当前可用的**能量储备**。

## 魔力储量

### 量化方式

魔力的基本单位为"刻"（游戏内概念，取"时刻"之意——一刻魔力约等于维持一个基础法术一瞬所需的能量）。在游戏实现中，"刻"可以映射为 mana point（MP）数值。

### 储量决定因素

| 因素 | 影响方式 | 可变性 |
|------|----------|--------|
| 先天灵元素强度 | 决定储量上限的基数 | 不可变 |
| 长期训练 | 逐步提升储量上限 | 可成长 |
| 血元素健康状态 | 影响当前可用魔力的百分比 | 随身体状况波动 |
| 年龄 | 幼年储量低 → 成年达到峰值 → 老年缓慢衰减 | 自然变化 |
| 特殊事件/契约 | 可能大幅改变上限（正向或负向） | 罕见 |

### 储量的个体差异

普通人的魔力储量为零（未开发），经过基础训练的学徒约为 10-50 刻，成熟法师约为 100-500 刻，传奇级施法者可达数千刻。但这些数字仅用于相对比较——实际游戏中以 NPC 的对应属性值呈现。

## 魔力消耗

### 消耗公式

一个魔法的魔力消耗由 [[04-魔法方程式]] 中的七大要素共同决定：

```
魔力消耗 = 基准消耗 × 范围系数 × 对象系数 × 接触系数 × 元素系数 × 影响系数 × 时间系数
```

其中：

| 要素 | 消耗权重 | 说明 |
|------|---------|------|
| 范围 | 高（指数增长） | 半径翻倍 → 魔力消耗约×4 |
| 对象 | 高（线性+额外系数） | 从单个到群体，每个额外目标增加链接消耗 |
| 接触 | 中 | 凭空接触比过程接触消耗更高（约×1.5-2） |
| 元素 | 中 | 受 [[03-元素交互]] 亲和/排斥影响；混合多元素显著增加消耗 |
| 影响 | 中-高 | 复合影响（转化、转移、链接）消耗高于基本操作 |
| 时间 | 高 | 持续时间的延长呈指数增长 |
| 条件 | 可变 | 触媒和法阵可以**降低**魔力消耗（替代施法者提供部分魔力） |

### 消耗与储量

如果魔法所需的魔力消耗超出施法者当前储量，有以下选择：
- 使用 [[09-触媒与材料|触媒]] 补足差额
- 通过 [[08-法阵系统|法阵]] 从环境中汲取元素能量
- 进行 [[10-仪式魔法|仪式]]，聚合多人魔力
- 透支施法（见下文"魔力过载"）

## 魔力回复

### 自然回复

魔力通过灵元素从血元素中持续提炼而自然回复。回复速率取决于：

- **基础速率**：约 5-10% 储量上限 / 小时（冥想状态下）
- **活跃状态**：非冥想状态下，回复速率降至 1-3% / 小时
- **睡眠**：高质量睡眠可回复 50-80% 储量
- **血元素状态**：饥饿、疾病、受伤会降低回复速率

### 环境加成

处于高浓度灵元素或对应元素环境中，回复速率可提升 1.5-3 倍。地脉节点是最理想的魔力回复场所。

### 辅助回复

- **魔力药水**：即时回复一定量魔力（属于 [[15-魔法物品与法器]]）
- **触媒吸收**：直接从高纯度触媒中汲取魔力
- **仪式充能**：通过 [[10-仪式魔法|仪式]] 从其他施法者处获得魔力传输

## 魔力过载

当施法者尝试施展消耗超出当前储量的魔法时，可以主动选择"过载"：

### 过载方式

| 方式 | 操作 | 代价 |
|------|------|------|
| 透支 | 从血元素中强行榨取魔力 | 身体虚弱、昏迷、永久降低血元素健康度 |
| 借力 | 从环境中强行汲取未经精炼的粗元素 | 魔力不稳定、法术效果打折、可能失控 |
| 燃灵 | 燃烧自身灵元素换取瞬间大量魔力 | 灵元素永久损伤、魔法资质下降、极端情况下死亡 |

过载不是常态施法手段——它是绝境中的最后选择。每次过载都会在施法者的灵-血界面上留下痕迹，累积到一定程度后会导致不可逆的后果。详见 [[17-魔法禁忌与风险]]。

## 魔力感知

施法者可以通过灵元素感知魔力：

- **自我感知**：随时知道自己当前的魔力储量（精确到"刻"）
- **他人感知**：可以大致感知附近施法者的魔力强度（模糊的强/弱判断；高精度感知需要专门的惑心系魔法）
- **环境感知**：可以感知一个区域的元素浓度分布（经验越丰富，感知越精确）
- **物品感知**：可以判断一件物品是否被附魔、大致魔力强度（详见 [[15-魔法物品与法器]]）

普通人对魔力没有感知能力——他们只能看到魔法的外在效果（火焰、冰霜等），而感觉不到魔力本身的存在。

## 魔力与游戏系统的接口

在 NPC 系统中，魔力储量是 NPCData 的属性之一，对应 `开发阶段/NPC活人感模块/` 中的人属性定义。NPC 的魔力消耗和回复由模拟管线中的对应模块处理。

> 魔力体系的具体数值（基准消耗、回复速率、过载阈值等）将在实际开发阶段进行参数化设计。本文档定义的是体系框架和逻辑关系。
