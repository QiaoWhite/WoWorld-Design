NPC设计概述

| 层级          | 说明                                                                                     |
| ----------- | -------------------------------------------------------------------------------------- |
| **NPC规模**   | 世界常驻目标 **100,000 NPC**（含约1,000个L1全模拟），按一定规则一部分一部分地生成，后续通过自然繁衍、战争、灾害等方式演化和消亡。随着技术迭代，模拟规模将持续扩大。                             |
| **NPC决策模型** | **接近现实的决策逻辑**：基于性格、利益、需求、人际关系等驱动行为，而非脚本。                                               |
| **心理动机系统**  | 需一个**灵敏的描摹系统**来生成NPC的"想法"与目标（可类比RimWorld的心情/需求系统与《艾瑞亚之歌》的动机网络）。                        |
| **自发社会组织**  | NPC会因共同利益/性格等原因结成团体，范围涵盖**家庭、小队、党派、国家**等级别。                                            |
| **LLM增强选项** | - 可选接入LLM（玩家自备Key）提升交互生动性：NPC对话更自然、生成个人历程日志、辅助人际关系/国际关系演算。  <br>- 无LLM时游戏仍可**完整正常运行**。 |
| **涌现式交互**   | 世界内所有单位（NPC、动物、环境事件）的行为构成动态网络，彼此影响并改变世界状态。                                             |
