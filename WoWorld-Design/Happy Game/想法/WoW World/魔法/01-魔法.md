魔法是通过特定仪式、物质、语言、意志来运用元素以影响世界的一种力量。
魔法的基本元素是金、木、水、火、土、风、雷、电、血、灵。世界中每一个物质都包含这些元素中的一种或几种。其中"灵"指的是心灵，是人以及其他生物的固有元素，也是生物施放魔法时与其他元素联系的桥梁。

### 核心原理：两条公理

整个魔法体系建立在这两条不可动摇的公理之上：

1. **二元构成原理**：万物由十元素构成。其中，**血**是金、木、水、火、土、风、雷、电按特定比例组合成的"肉身元素"；在实用魔法学的框架下，**灵**被视为寄宿于血、连接精神与物质的"灵魂元素"，是施法的核心桥梁。关于灵的本质是否为血的某种形式，学术界仍有争论，详见 [[02-元素]]。
    
2. **意志投射原理**：施法本质是施法者以自身**灵**元素为桥梁，将意志投射于目标，支配其内部元素，引发**增加、减少、吸收、释放、排斥**五种基本变化，以及由基本变化组合而成的**转化、转移、链接**等复合影响。普通人的意志只能驱动自身**血**元素（即肉体），而施法者的意志可直接与外界的元素交互。

## 魔法方程式速览

> 完整详解见 [[04-魔法方程式]]。

**魔法 = 范围 + 对象 + 接触 + 元素 + 影响 + 时间 + 条件**

| 要素 | 核心问题 | 详情 |
|------|----------|------|
| 范围 | 魔法作用于多大空间？ | 点 / 线 / 面 / 体 |
| 对象 | 魔法作用于谁？ | 类型 × 数目 × 命中 × 指向 |
| 接触 | 魔法怎样到达目标？ | 过程接触 / 凭空接触 |
| 元素 | 魔法涉及哪种元素？ | 单一或多元混合 |
| 影响 | 魔法对目标做什么？ | 基本操作 / 复合影响 |
| 时间 | 魔法持续多久？ | 瞬时 / 持续 / 触发 / 永久 |
| 条件 | 施展需要满足什么？ | 自身 / 环境 / 物质 / 仪式 / 契约 |

## 魔法类型速览

> 完整详解见 [[05-魔法类型]]。

| 类型 | 核心影响 | 简述 |
|------|----------|------|
| **变幻系** | 转化、转移 | 改变物质元素构成与形态 |
| **召唤系** | 转移 | 与远方事物建立链接并传送 |
| **破坏系** | 释放、减少 | 瞬间释放大量元素能量 |
| **守护系** | 吸收、排斥 | 制造元素力场 |
| **惑心系** | 影响(指向心理) | 以**灵**元素直接影响意志 |
| **疗愈系** | 增加、转化 | 精细补充或转化**血**元素 |
| **死灵系（禁忌）** | 链接 | 扭曲**灵-血**关系，驱使亡骸 |

每一系魔法都包含两种接触方式（过程/凭空）的具体魔法——接触方式不是某一系魔法的专属。
