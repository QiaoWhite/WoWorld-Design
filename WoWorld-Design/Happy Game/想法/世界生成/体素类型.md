## 体素 = 大地素材

体素仅用于地形。建筑、家具、角色、物品均为网格模型。

每个体素存储两个值：`密度(0=空气~1=实心)` + `材质ID`。

## 明确不作为体素的事物

| 事物            | 实现方式                |
| ------------- | ------------------- |
| 建筑(墙/屋顶/地板)   | 模块化网格               |
| 家具            | 网格道具                |
| 角色(NPC/玩家/动物) | 骨骼网格                |
| 植被            | MultiMeshInstance3D |
| 物品(武器/工具/食物)  | 网格道具                |
| 水面效果          | 着色器平面               |
| 粒子效果          | GPU粒子               |
