## 这是什么游戏？

你被丢进一个完全由种子生成的奇幻中世纪世界。这个世界在你到来之前已经运行了很久——有城市、村庄、家族、势力、文化。每一棵树、每一条河、每一栋建筑都是由数学和规则生成，不是美术手工摆放。

这个世界的 NPC 不是任务板。他们是活的。

铁匠不只是"卖武器的NPC"。他可能是三个孩子的父亲，欠商人钱，暗恋隔壁的面包师，在昨天看到一队冒险者路过时想起了自己年轻时没能实现的冒险梦。他会饿、会累、会生气、会记仇、会恋爱、会死。

**你可以做任何事。** 成为佣兵队长？商人？农夫？土匪？国王？都可以。游戏不告诉你该做什么。你觉得有趣的事，就是正确的事。

## 核心竞争力

### 1. 活着的NPC（最重要）

每个NPC有独立的大五人格、情绪引擎（30种复合情绪）、200条事件记忆、社会关系网。他们的行为由概率驱动而非脚本——不可预测所以看起来聪明。

> 不是"NPC走到A点说B句话"。  
> 是"NPC因为讨厌邻居，决定搬家到隔壁镇子，在路上遇到土匪，被打伤后对高个子的人产生了持续的心理阴影"。

### 2. 每次新游戏都是新世界

种子决定一切。山脉、河流、聚落、NPC、势力关系——全部由噪声函数和生成规则产生。没有两个世界相同。

### 3. 涌现叙事

不写故事，只写规则。铁匠抢了面包师→面包师传播坏话→铁匠生意变差→铁匠更暴躁——这条因果链我们没有写，但它会发生。

### 4. AI增强但不依赖

接入你自己的LLM API Key → NPC说话更自然、游戏日志变成故事。不接？游戏完全正常运转。LLM是锦上花，不是地基。
