## 一、魔法在战斗三层模型中的融入

### 1.1 施法媒介 vs 战斗节奏

```
战斗中的五种施法媒介:

语言 (吟唱):
  完整吟唱 → 1-3s 战略层行为, 不可移动 (或减速), 可被打断
  短吟唱 → 0.3-0.8s 节奏层行为, 可移动但减速
  默发 (大师) → 0.1s 本能层行为, 和其他行动同步执行

物质 (触媒):
  消耗触媒施法 → 0.5s 节奏层行为, 需要一只手空闲取出触媒
  稀有触媒 → 更强效果但使用次数有限 (资源稀缺影响决策)

意志:
  最根本的媒介。纯意志施法 → 灵元素强度决定速度
  战士的低阶意志: "点燃剑刃" → 节奏层
  法师的高阶意志: "远处的石头飞起来" → 战略层 (需要完整的七要素方程式)

仪式 (法阵):
  战斗中几乎不可用 (需要长时间+静止)
  但有例外: 预制的"触发法阵" (陷阱式) → 战斗前预设, 战斗中敌人触发 → 被动激活
  契约法阵: 特定的召唤/转移 → 战略层行为, 需要队友保护施法窗口

契约:
  呼唤契约召唤物 → 1-3s 战略层行为, 和完整吟唱同级
  契约生物的"自动护卫" → 本能层 (和"法术本能化"同理)
```

### 1.2 元素交互 → 战斗中的战术选择

```
亲和关系产生的战术:

火+风: "燃烧+扩散"
  → 火球命中后, 火势随风扩散到附近敌人 (风是扩散介质)
  → 战术: 先放风, 再放火。风把火推向敌人密集区。

水+雷: "导电+传播"
  → 雷击落点附近如果有水 → 导电范围扩大
  → 战术: 先用水魔法制造湿地, 再用雷
  → 雨天 → 自然湿地 → 雷系法师优势

金+电: "精确传导"
  → 电击可以沿金属精准传导 → 对穿金属甲的敌人伤害加深
  → 战术: 先用金系魔法标记敌人 (金属化表面) → 再放电

排斥关系产生的代价和高手玩法:

水+火: 通常排斥。但高手可以制造"高温蒸汽"。
  → 需要额外调和参数 → 魔力消耗 ×3
  → 效果: 范围蒸汽爆炸——更痛但成本更高
  → 高手和普通法师的区别: 高手知道什么时候"值得付出额外代价"

实际战斗中的元素选择逻辑 (战略层):
  1. 我的灵元素亲和哪个元素? (先天+训练)
  2. 敌人的元素弱点是什么? (战斗信息→记忆中获取)
  3. 当前环境对我有利吗? (雨天→雷, 干燥→火)
  4. 队友在用什么元素? (避免互相抵消)
  5. 我的魔力还剩多少? (资源稀缺染色)
```

### 1.3 七系魔法在战斗中的角色

```
破坏系: 直接伤害。战斗中使用最频繁。
  火球/冰箭/雷击/风刃/土刺/金剑/木蔓/电击/血蚀/灵震
  → 可能是节奏层最常选的法术

守护系: 防御/力场。
  元素护盾/排斥力场/吸收场
  → 本能层可触发 (护盾自动激活——如果已习得+魔力足够)
  → 和本能层的闪避/格挡并列——"自动防御"的一个新分支

变幻系: 改变物质/形态。
  石化敌人/变形自身/化石为泥
  → 战略层行为。需要评估"改变什么目标 + 代价是否值得"
  → 对巨物: 石化它的脚 (部分破坏)

召唤系: 传送/链接/召唤。
  召唤灵体/远距离传送物体/和自己远处的物品建立链接
  → 战略层行为。战斗中召唤→需要保护施法窗口

惑心系: 直接影响意志。
  恐惧/混乱/暗示/沉睡
  → 战略层行为。不以"伤害"为目标——以"瘫痪敌方决策能力"为目标
  → 对低灵元素亲和的目标 (如野兽) 效果更好
  → 不适合高频使用——消耗极高 + 可能触发道德/法律问题

疗愈系: 恢复/修复。
  治疗伤口/止血/解毒
  → 节奏层行为。和攻击交替: "砍一刀→后退→给自己止血→再上"
  → 团队中: 队友受伤 → 节奏层插入"治疗队友"行动

死灵系 (禁忌):
  扭曲灵血关系→控制尸体/吸取生命
  → 战略层行为。但使用它在大多数文化中触发道德/法律后果
  → NPC 的记忆会标记: "他用死灵魔法"——影响声望
  → 一个绝望的被包围的 NPC 可能"不得已"使用——
    这就是道德/处境/人格在战斗中碰撞的瞬间
```

## 二、武器模块化与战斗

### 2.1 武器组成

```
武器 = 柄 + 刃/攻击部 + 卡槽(0-3个) + 饰品(可选)

柄 (决定持握方式、长度、重量):
  材质: 木/铁/钢/秘银/龙骨/...
  长度: 滑块式, 0.3m-2.5m
    短柄 (0.3-0.6m): 仅单持。快。匕首/短剑/单手斧/单手锤。
    中等柄 (0.6-1.2m): 可单可双。长剑/武士刀/单手矛
    长柄 (1.2-2.5m): 仅双持。长矛/大剑/戟/双手斧
  粗细: 影响握感和速度。细→快轻, 粗→慢重
  重量 = f(材质, 长度, 粗细)

刃/攻击部 (决定伤害类型和可用的动作积木):
  直刃 (剑形): 刺√ 斩√ 劈√ 撩√ 点√ 崩× 缠×
  弯刃 (刀形): 刺× 斩√ 劈× 撩√ 点× 崩× 缠√
  锤头: 刺× 斩× 劈√ 撩× 点× 崩√ 缠×
  矛头: 刺√ 斩× 劈× 撩× 点√ 崩× 缠×
  斧刃: 刺× 斩× 劈√ 撩√ 点× 崩√ 缠×
  手(体术): 刺√ 斩× 劈× 撩× 点√ 崩√ 缠√
  法杖 (不用于物理攻击, 用于施法): 增加魔力稳定性和施法速度

  材质: 铁/钢/银/秘银/玄铁/黑曜石/龙鳞/...
  长度/宽度/厚度: 滑块式, 决定伤害量和攻击速度

卡槽 (附魔符文):
  火焰符文: 攻击附带燃烧
  冰霜符文: 击中减速
  雷电符文: 概率硬直
  吸血符文: 击中微量回血
  破甲符文: 忽略部分护甲
  ... (约 15-20 种符文)
  直接附魔 (不可拆换): 更强但永久绑定
  卡槽附魔 (可随时换): 灵活

饰品 (装饰/少量属性):
  配重球: 增加攻击速度, 降低伤害
  护手: 增加格挡稳定性
  流苏/丝带: 少量干扰敌人视觉
```

### 2.2 武器使用门槛

```
武器的物理属性 → 使用者的最低要求:

重量 → 需要力量 ≥ 门槛值
  (玄铁重剑, 一个矮小瘦弱的人根本拿不起来)
长度 → 需要力量+敏捷的综合门槛
  (长矛需要足够的上肢力量 + 足够的敏捷来控制尖端)

材质硬度 → 需要武器技能 ≥ 门槛值
  (一把秘银剑比铁剑更难驾驭——需要更精确的角度控制)

使用者的属性超过门槛 → 使用没有惩罚
使用者的属性不足 → effectiveness × 0.5-0.8 (勉强使用)
使用者的属性远远不足 → 根本拿不起来 (effectiveness = 0)
```

### 2.3 武器 × 流派 × 招式积木

```
一把具体的武器 = 一组可用的动作积木的"许可列表"

例: 单手铁柄 + 钢直剑刃 + 无卡槽
  可用积木: 刺 斩 劈 撩 点
  不可用积木: 崩 缠
  速度修饰: 快·中 (单手轻剑)
  攻击范围: 中 (柄长 0.8m + 刃长 0.9m = 总长 1.7m)

例: 双手玄铁柄 + 玄铁锤头 + 火焰符文
  可用积木: 劈 崩 (锤)
  不可用积木: 刺 斩 撩 点 缠
  速度修饰: 慢·重 (双手重锤)
  攻击范围: 中等 (柄长 1.6m)
  额外效果: 击中附燃烧 (符文)
  门槛: 力量 ≥ 30。低于 30 → 拿不动。

一个使用这把锤子的 NPC:
  他能用的积木只有劈和崩。
  他的招式库 = 从劈/崩出发的所有合理派生链。
  他的流派不是"玄铁锤法"——是"用劈和崩的组合来战斗的方式"。

  如果他的力量刚好够门槛 (30):
    → 攻击速度慢, 后摇长。一波猛击后需要喘息。
  如果他的力量远超门槛 (50):
    → 攻击速度快, 后摇短。这把锤在他手里像个玩具。
    → 他会发展出和别人不同的"用锤方式"——连崩三次再劈。
      不是因为流派不同——是因为他的身体能承受更快的节奏。
```

## 三、综合自查——战斗系统全面审视

### 3.1 魔法 vs 物理的平衡

**魔法消耗魔力, 物理消耗耐力 → 双重资源管理**

```
物理攻击者: 耐力是消耗品。
  猛攻 → 耐力消耗快 → 需要"稳住"
  耐力耗尽 → 攻击速度和力量均下降 → 必须后退喘息

魔法攻击者: 魔力是消耗品。
  强力法术 → 魔力消耗大 → 用完就没有
  魔力耗尽 → 只能靠意志施法 (极慢) 或切换物理攻击

混合型 (魔法剑士/战斗法师):
  耐力+魔力 双重管理
  可以: 用剑消耗对方耐力 (格挡消耗耐力) → 对方耐力低时切换魔法突袭
  战斗弹性更大——但每项都不如"专精者"精
```

**缺失检查**：耐力的恢复速度 vs 魔力的恢复速度是否平衡？魔力需要"提炼"（灵+血转换）——比耐力的"呼吸即可恢复"更慢。这是对的——魔法更强大但补给更困难。

### 3.2 战斗信息在魔法中的特殊应用

```
魔法的"可读性":

物理攻击:
  前摇 → 攻击 → 后摇 (动作可见, 时间可预测)
  剑举起来 → 要砍了 → 本能层自动反应

魔法攻击 (对战物理攻击者):
  施法动作 → 咏唱声音 → 元素粒子聚集 → 法术释放
  物理攻击者需要识别这些"信号"——因为魔法没有物理武器的"前摇"那么直观
  → 有经验的战士能"读"出法师的施法阶段
  → 新手可能"看到他在念咒"但不知道是什么法术/还有多久放出来

魔法对魔法:
  火球 vs 冰箭。元素颜色让它们容易区分。
  同类法术的对抗: 两个火球相撞 → 强者吞噬弱者
  异类法术的对抗: 火球 vs 水球 → 排斥关系生效 → 谁元素浓度高谁赢
  → 法师对战法师 → 从"谁能先打中"变成了"谁的元素压得过谁"

信息战——误导:
  法师可以"假装吟唱一个破坏系法术" (夸张的火焰粒子)
  → 对方格挡/闪避预备
  → 但实际释放的是一个惑心系法术
  → 对方白白浪费了宝贵的闪避窗口
  → (需要高技能+高开放性才能策划这种策略)
```

### 3.3 我们的设计仍然是"围绕核心目的分解为小规划"吗？

**不是。** 我们已经从"战斗v0.1 的总目标→阶段目标→限定条件"的 GOAP 模型——演化为**三层模型 + 目标被多重因素实时搅拌**。

```
进化的关键:

v0.1 (原版):
  总目标 → 阶段目标 → 行动
  线性。可预测。GOAP 风格。

v3 (当前):
  战略层: 总目标 + 阶段目标 (5-30s 更新, 但关键事件立即触发重评估)
  节奏层: 倾向 (前压/稳守/周旋/猛攻/脱战), 0.5-1s 更新
  本能层: 自动闪避/格挡/反击/本能化法术, 逐帧执行
  
  三层同时运行, 互相染色。
  目标不是固定的——随时被 处境 + 人格 + 情绪 + 记忆 + 知识 + 外部输入 搅拌。
  
  是。核心目的仍然在。但"分解成子规划"的方式不是"专家系统的规划树"——
  是"三层模型 + 多因素实时搅拌 + 事件驱动重评估"。
  规划更松散、更灵活、更像人。
```

### 3.4 战斗者能围绕目的做出有逻辑、有拉扯、有人格特色的决策吗？

```
战斗者视角: 一个战术型剑客的完整决策过程

战略层 (每 ~8s 更新):
  当前总目标: 拖住对方剑客, 等我的弓手队友解决掉对方法师然后来支援。
  阶段目标: 不主动进攻, 保持距离, 消耗时间。
  限定条件: 不能受重伤 (否则没法继续后面的任务)。
  输出→ 节奏层: 倾向 = 稳守+周旋

节奏层 (每 ~0.8s):
  倾向: 稳守+周旋。
  当前行动: 持盾, 慢慢后退。
  敌人压上来 → 格挡 (本能层自动处理)。
  敌人露出破绽 → 我看到了, 但阶段目标说"不主动进攻"。
     → 不进攻。即使有破绽也不打。这是人格+目标的体现。
     → 一个狂暴型的剑客在同样处境下可能: "管不了那么多, 有破绽就打!"

本能层 (逐帧):
  格挡。闪避。反击 (仅当敌人攻击被完美格挡后的自然反弹)。

拉扯感从何而来:
  我不是被动地"反应"——我是主动地在"控制节奏"。
  我的后退不是"怕他"——是"在等队友"。
  敌人前压, 我后退。敌人犹豫, 我也稳住。他试探我, 我不上钩。
  → 这就是拉扯。不是"AI 随机选动作"——是有明确意图的节奏控制。

人格特色从何而来:
  同样处境, 不同人格的战术型剑客:
    高尽责性+低神经质: "稳住。按计划来。"
    高神经质+低尽责性: "我等不了那么久。我现在就上了！"
    高宜人性: "我的队友会不会有危险? 是不是应该过去帮他?"
    低宜人性: "我的队友能不能快点——我已经在消耗体力了, 这都是为了配合他。"
  → 同一个总目标。同样在拉扯。但拉扯的"心态"和"节奏"不同。
  → 赢了之后, 他们回想这场战斗的叙述也不同。
```

## 四、完整战斗循环模拟

### 模拟场景

三个角色：玩家（剑客，战术型）、NPC_22（弓手，高尽责性，玩家的同伴）、NPC_45（法师，高神经质，敌对）。

地形：河流旁边的森林废墟。有掩体（断墙、巨石）。下午。天色渐暗。

---

### 阶段 0: 战前。日常→冲突过渡。

```
14:30。玩家和 NPC_22 在森林中追踪 NPC_45（目标是抓捕，不是击杀）。

日常决策器在进行:
  玩家: 行走。观察周边。
  NPC_22: 行走。偶尔停下确认方向。高尽责性 → 他会定期检查路上标记。
  NPC_45: 在前方约 60m。不知道被跟踪。正在采集魔法材料。

→ NPC_22 的听觉感知检测到了 NPC_45 的脚步声 (15m 外有枯枝断裂)。
→ NPC_22 对玩家做了一个手势 (非语言交流: "前面有人")
→ 玩家也听到了。两人进入警戒状态。但还没有进入战斗——只是日常决策器的权重变了:
  探索/行走权重↓, 警觉权重↑。
```

### 阶段 1: 战斗发起。冲突初接触。

```
→ NPC_45 回头看到了两人。
→ 他的遭遇响应触发: 威胁感知 → 评估。

NPC_45 的评估 (人格: 高神经质 + 高开放性):
  处境: 2对1。我方劣势。
  人格: 高神经质 → 我现在的感觉是——危险。很危险。
  知识: 附近有一个废墟。有掩体。可以进入。不利于远程弓箭手。
  目标: 不是我方先动手。我可以先威慑 → "别过来！靠近一步你就得死！"
  → 他选择了: 虚张声势 + 往废墟后退。

玩家和 NPC_22 的评估:
  玩家战略层: "我们要抓活的。不能冲动——他是法师, 近身战对我们有利但废墟给他提供了掩体。"
  玩家给 NPC_22 的命令: "我去抄他后面。你在这吸引他的注意力。"
  NPC_22: "明白。" (层级团队的快速指令——他接受了玩家的命令)
```

### 阶段 2: 战斗正式开始。三层模型全开。

```
战略层:

玩家: 总目标 = 抓捕 (致残, 非击杀)。限定条件 = 不能放跑。
  → 阶段目标 1: 绕到废墟后面
  → 阶段目标 2: 和 NPC_22 前后夹击

节奏层:
```

## 六、掩体意识——远程战斗的空间维度

### 6.1 掩体不是"墙壁"——是"战术资源"

掩体意识不是"有墙就躲"——是在三层模型各层中自然存在的空间评估。

```
本能层:
  当被远程攻击时 (箭矢/法术飞来):
    → 自动评估: 我身边 2m 内有可以挡的东西吗?
    → 有: 本能性地微调位置 (侧移一步进入掩体阴影)
    → 这个"微调"不是"决定"——是身体自动做的。和闪避一样。
    → 精度取决于: 技能熟练度 + 对周围环境的预先感知

  当正在掩体后时:
    → 自动减少身体的暴露面积。只露出必要的部分。
    → 弓手本能: 从掩体边缘探头→射出一箭→缩回掩体 (整个动作约 1.5-2s)
    → 法师本能: 躲在掩体后吟唱 (掩体阻挡物理攻击, 但不能阻挡范围魔法)

节奏层:
  掩体后的节奏倾向 = f(当前目标, 信息充足度, 风险评估):

  情形 A: 我有掩体, 敌人在开阔地。
    → 节奏倾向: 压制 (我不需要冒险出去。等他犯错。或等他必须移动。)

  情形 B: 我躲在掩体后, 但我的队友正在开阔地被围殴。
    → 战略层: "我必须帮他" → 节奏倾向: 准备离开掩体
    → 节奏层评估: 离开掩体的"机会窗口"——什么时候出去风险最小?
    → 等待对方射完一箭 (后摇) 或对方注意力转移到队友身上时 → 果断跳出掩体

  情形 C: 我躲在掩体后, 但对方正在接近 (准备近战)。
    → 掩体继续提供保护——但掩体也被他接近了。
    → 掩体对我的价值在下降——因为一旦他到了掩体旁, 就变成了近战
    → 节奏层: 评估是否切换位置 (离开这个掩体→去下一个掩体)

战略层:
  掩体是阶段目标的一部分:

  "先到那个断墙后面。从那可以看到整个战场。"
  "这个掩体太矮——我需要找一个能挡住火球的。"
  "放弃掩体。直接冲过去。赌他射不中我。"

  战略层的掩体选择 = f(当前总目标, 敌人位置, 敌人武器, 自身资源)
```

### 6.2 掩体评估——不是所有掩体都"能用"

每个可用的掩体位置被战斗信息池记录为一个"掩体条目":

```
掩体条目:
  位置: (x, y, z)
  类型: 高墙 / 矮墙 / 巨石 / 树木 / 废墟 / 盾牌(临时)
  高度: 1.2m (矮墙——蹲下可全身掩蔽, 站立仅掩下半身)
  宽度: 2.5m (可掩护 1-2 人)
  材质: 石 (可挡箭矢+火球, 不能挡雷击)
  状态: 完整 (尚未被破坏)
  可达性: 我离掩体 8m, 途中有一段暴露路程 (需 ~1.5s)

掩体对当前威胁的有效性:
  威胁 = { 类型: 弓手, 位置: 我的 NE 30m, 射界: 120° }
  掩体对我的保护: 80% (石墙可完全挡箭→但敌人可能在侧面射击)
  掩体对我的行动限制: 我可以探头射箭 (低局限性), 但我不能后退 (掩体后面是悬崖)
```

### 6.3 什么时候"该冒一下险"？

离开掩体的风险评估——不是"掩体 vs 无掩体"的二元选择。是"掩体提供的保护 vs 此刻出去做成某件事的收益"。

```
评估 = 掩体外的收益 / 掩体外的风险

收益:
  1. 我看到了一个关键的射击窗口。机会稍纵即逝。
  2. 我的队友正在被夹击，必须我吸引火力。
  3. 对方在换位置 (从一个掩体跑向另一个——中间没有保护)。
     这是他最脆弱的时刻。如果我留在掩体后 → 错失窗口。
  4. 对方正在吟唱一个关键法术。我需要打断它。掩体后的角度射不到他。

风险:
  1. 对方有弓手在注意我。一旦我探头→箭就在路上。
  2. 对方有雷系法师。掩体对雷没有保护 (雷穿透掩体)。
  3. 我 HP 很低。一次命中可能致命。

人格染色:
  高神经质 → 风险评估中的"风险"权重大 (倾向于躲在掩体后多等一会儿)
  低神经质 → "我想出去就出去。" (可能冒更大的险)
  高尽责性 → 更精确的时间把握 (他会等对方刚射完后摇→才探头。时间掐得准)
  低尽责性 → 可能没看时机就出去了 (然后被射中——"下次我会注意" 或 "活该")

知识染色:
  老练的弓手: 他知道对方法师的施法时间是多少秒 → 
    精确地掐在吟唱的最后 0.5s 时探头→放箭→缩回。在对方释放之前打断了他。
  新手的弓手: 早探头了 1.5s → 暴露在对方精准的射界中太久 → 中箭。

关键时刻:
  如果收益 / 风险 > 人格_冒险阈值 → 离开掩体
  不是"一直躲到对方走"——是"在需要行动的时候行动"
```

### 6.4 掩体也是"被破坏的"

掩体不是永久的。火球可以打碎石墙。持续攻击可以摧毁树木。

```
掩体的生命周期:

完整 → 经受攻击 → 出现裂缝 (保护率 -30%)
→ 继续受击 → 即将坍塌 (保护率 -70%)
→ 彻底损坏 → 掩体消失 (保护率 = 0)

NPC 可以感知掩体的状态变化:
  → "这道墙扛不住下一发火球了——我得换位置。"
  → 如果 NPC 没注意到 (掩体在他身后, 他看不到裂缝) → 
    掩体倒塌时 → 事件推送: "掩体已失效!"
  → 信息不对称: 敌人可能不知道掩体快塌了 (如果他看不到那面墙)
     → 他可以继续躲在后面——直到下一次火球穿透墙壁
```

## 八、潜行与伏击——持续性的信息博弈

### 8.1 潜行不是"隐身"——是每时每刻都在"赌"

潜行者靠近目标的过程 = 每一步都在和目标的感知能力对抗。

```
潜行博弈 (每 ~0.2s 更新):

潜行者的"暴露风险" 随距离和行动而变:
  距离 50m, 夜色, 不动:     暴露风险 ~0.01  (几乎不可能被发现)
  距离 30m, 夜色, 缓慢移动:  暴露风险 ~0.05
  距离 15m, 夜色, 踩到枯枝:  暴露风险 ~0.25  (突然上升!)
  距离 10m, 夜色, 对方转身:  暴露风险 ~0.40  (他正朝向我!)
  距离 5m, 任何环境:         暴露风险 ~0.70  (太近了!)

目标的"觉察概率" = 暴露风险 × 感知能力 × 注意力方向 × 直觉

  感知能力: 视觉+听觉+魔法感知+经验
  注意力方向: 正面 > 侧面 > 背后
  直觉: 灵元素亲和 + 人格(神经质→更警觉)

每次更新:
  生成 [0,1) 随机数。
  如果 < 觉察概率 → 目标产生一条模糊信息。
  如果 ≥ → 没有发现。潜行者可以继续接近。

  模糊信息的累积:
    第一次: "那边好像有动静。" (可信度: 低, 信息不完整)
    第二次: "又听到了。不是风吹的。" (可信度: 中, 目标方向模糊)
    第三次: "有人在靠近!" (可信度: 高, 目标大致位置确定)
    → 目标转身 → 视觉确认 → 信息升级为确定。
    → 伏击失败。潜行者暴露。

  如果潜行者一直成功 (随机数一直 ≥ 觉察概率):
    → 潜行者到达攻击距离 (如 3m 的匕首背刺)
    → 发起伏击。先手优势: 目标本能层没有预先加载这个方向。
```

### 8.2 目标在做什么也重要

```
目标的状态影响暴露风险:

目标专注在做某件事 → 注意力窄 → 觉察概率 ×0.5
  例: 在吟唱法术、在瞄准远处、在吃东西、在睡觉

目标在警戒状态 → 注意力宽 → 觉察概率 ×1.5
  例: 刚刚听到了一个模糊信息、队友警告了"小心周围"、
       知道附近有敌人但不知道在哪

目标在战斗中 → 注意力高度分散 → 觉察概率 ×0.3
  例: 正在和前方的剑客缠斗 → 对身后几乎没有感知
       → 这就是为什么战斗中"背刺"如此致命
       → 这也是信息战中"让队友吸引注意"的核心价值

目标在主动检查 → 觉察概率 ×2.0
  例: 老兵的习惯——每走几步回头看一眼
       高神经质 → 频繁检查 (即使没有理由)
       低神经质 → "这森林很安静。没必要回头。"
```

### 8.3 人格在潜行博弈中

```
潜行者的人格:

高尽责性: 耐心等待最佳时机。
  "不急。等他注意力转到那边的时候我移动。"
  行动: 只在对方注意力方向改变时移动 → 暴露风险低 → 成功率更高

低尽责性: 不耐烦。
  "太慢了, 我直接冲过去。"
  行动: 加速移动 → 暴露风险 ×3 → 成功率低 (但可能因为快而成功——
       对方还没来得及回头看)

高开放性: 利用环境。
  "那边的石头可以扔——制造假声响, 把他引到另一边去。"
  行动: 自己不是被动等待——主动创造有利于自己的条件

高神经质: 每一步都紧张。
  这种紧张可能让他更小心 → 暴露风险低
  但也可能让他反应过度 → 听到对方脚步声就僵住了 → 错过窗口

────────────────────────────────────────

目标的人格:

高神经质: 过度警觉。
  "我总觉得有什么不对劲……" → 频繁回头 → 觉察概率高
  但他也可能"疑神疑鬼" → 把风声当成敌人 → 虚报 → 浪费注意力

低神经质: 松懈。
  "好安静。是个睡觉的好夜晚。" → 觉察概率低
  老练的低神经质: 他的"松懈"是有底气的——
    他有足够的经验知道"现在这个环境没有危险"。
    他的安静不是愚蠢——是判断。

高尽责性 (老兵): 习惯性警觉。
  不需要"紧张"——他的警觉是训练出来的自动行为。
  → 觉察概率高, 但因为不是"紧张"驱动的——
     他不会虚报。他就是有规律地、系统地检查盲区。
```

### 8.4 群体潜行——信息在不同目标之间传播

```
潜行者靠近一个三人小队:

NPC_A (面向潜行者): 觉察概率最高
NPC_B (侧面): 觉察概率中等
NPC_C (背对): 觉察概率最低

如果 NPC_C 被攻击:
  → NPC_C 的信息池立刻更新:"我被袭击!"
  → NPC_A 和 NPC_B 的信息池:
      听到声音 + 听到 NPC_C 的喊叫 →
      "那边有情况!" (可信度: 中, 距离: 近)
      → 立即转身 → 可能看到潜行者 → 伏击→正面战斗

如果 NPC_A 产生了模糊信息:
  → NPC_A 对 NPC_B 说:"你有听到什么吗?"
  → NPC_B 的注意力被引导 → 觉察概率临时 ×1.5
  → 团队中的"注意力传递"——一个人的警觉可以唤醒整队
```

## 十、地形伤害——环境作为武器

### 10.1 地形伤害的触发方式

```
地形伤害不是"额外系统"——是"攻击 + 环境条件 = 附加伤害"。

触发方式:

1. 位移冲击 (将敌人打飞到危险地形):
   位移技能/攻击中包含的"击退/击飞"向量 → 敌人撞向危险地形
   
   例: 重锤的"崩"基元 → 击飞敌人 → 敌人飞出 5m → 
        撞到尖石 → 尖石碰撞伤害 + 短暂硬直
        掉下悬崖 → 坠落伤害 (高度 × 重力 × 落地介质)
        撞入火焰 → 着火

2. 地形利用 (利用地形特征放大攻击效果):
   敌人背靠墙壁 → 我的崩/撞攻击 → 他没有后退空间 → 
   → 承受的不是"击退"而是"全部冲击力" → 伤害加倍 + 眩晕
   (这就是为什么"不要背靠墙"是近战的基本原则)

   敌人在浅水中 → 我的雷系攻击 → 水域导电 → 伤害范围扩大
   敌人在悬崖边缘 → 我用"撩"基元 (上挑) → 他可能失去平衡 →
   → 平衡检定失败 → 从边缘翻落

3. 地形破坏 (攻击导致地形崩塌):
   敌人在断桥上 → 我的重击连续砸桥面 → 桥面断裂 →
   → 两人都可能坠落 (取决于反应速度) → 杀敌一千, 自损八百
```

### 10.2 地形伤害在信息池中

```
战斗信息池中的环境危险标记:

每个 NPC 的战斗信息池记录了:
  自身周围 ~10m 内的危险地形:
    { 位置: (x, y, z), 类型: 悬崖, 危险: 坠落伤害, 距离: 2m (在我身后!) }
    { 位置: (x, y, z), 类型: 尖石, 危险: 碰撞伤害, 距离: 5m (在我左边) }
    { 位置: (x, y, z), 类型: 浅水, 危险: 导电, 距离: 3m (在我前面) }

  敌人的信息:
    敌人离悬崖还有 3m → "我可以把他逼下去!" (战略层: 调整攻击方向)
    我自己离悬崖 2m → "我不能背对悬崖!" (节奏层: 调整站位)
    敌人在尖石旁边 → "把他撞到石头上" (节奏层: 调整攻击角度)
```

### 10.3 人格 + 地形利用

```
高开放性 → "看那个悬崖! 我能把他推下去!"
  开放性高的人更善于在战场上发现和利用地形机会。

高尽责性 → "我不能背对那个悬崖。先换位置。"
  尽责性高的人更谨慎地管理自己的站位。

高神经质 → 靠近悬崖时本能层紧张 → 微动作更僵硬。
  他"怕"掉下去——这个恐惧本身可能让他在悬崖边更容易出错。
```

## 十一、投技——抓取与摔投

### 11.1 投技的积木定义

```
投技 = 特殊的"缠"类动作基元 + 力量对抗检定

投技的四个阶段:
  1. 抓取窗口 (极短的瞬间):
     需要满足一组严格的条件:
       - 距离 < 1.5m (近身)
       - 对方处于硬直中 (后摇/收招/受击中)
       - OR 对方的攻击被精准招架/弹反后 (对方瞬间失衡)
       - OR 自己使用了"化"基元成功卸掉了对方的力 (借力打力)

  2. 力量对抗检定:
     抓取成功后 → 双方进行力量对抗:
       投掷者的 (力量 + 投技熟练度) vs 被投者的 (力量 + 敏捷/体重)
       对抗结果决定: 是否能完成投掷? 还是对方挣脱?
       
       如果力量悬殊: 投掷者直接完成。
       如果势均力敌: 短暂僵持 (双方都在用力)。
       如果对方更强: 挣脱 + 可能反制 (他反过来投我!)

  3. 投掷执行:
     检定成功 → 投掷者选择投掷方向和效果:
       过肩摔: 对方砸在前方地面 → 地面碰撞伤害 + 倒地
       侧向投: 对方被横向摔出 → 位移 + 撞向环境 (地形伤害的可能)
       背摔: 对方被摔向自己身后 → 自己的位置不变, 对方位移
       推向特定方向: 推向悬崖/尖石/队友的刀口

  4. 收招:
     投技完成后 → 双方都有一个短暂的后摇/起身时间
     投掷者: 短暂的"收力" (0.3-0.5s)
     被投者: 倒地硬直 (0.5-1s) → 起身 → 恢复

### 11.2 投技的条件和限制

```
投技不能随意用:
  1. 抓取窗口短。错过 → 挥空。反被对方利用后摇。
  2. 抓取失败 → 自己暴露一个很大的空档。
  3. 对方如果有"缠"的反制技能 → 可能反投。
  4. 体型的限制:
     人可以投人。
     人可以投和自身体型相近的生物 (狼/哥布林)。
     人不能投比他大好几倍的东西 (你不能过肩摔巨龙)。
  5. 投技对抗霸体:
     霸体 1 级的敌人: 轻击不中断, 但不能免疫投技 (投技是"缠", 不是"击")
     霸体 2 级: 投技的抓取窗口仍然有效——但力量对抗时对方有加成
     霸体 3 级: 投技不可用——对方的身体太庞大/太重

### 11.3 人格 + 投技

```
高尽责性:
  把投技当精确工具。他不是"找机会就抓"——
  他在等那个"完美的抓取窗口": 对方后摇最大的那一帧。
  抓取成功率 ×1.3 (技术精准)

低尽责性:
  投技当蛮力用。"管你什么时候——我试试!"
  抓取成功率低——但一旦成功——他不在乎精确的方向——
  随手就把敌人往地上砸。

高开放性:
  利用环境最多的投技使用者。
  他不仅仅在"想投对方"——他在想"把对方投到那个火堆里"。
  他的投掷方向总是朝向最近的危险地形或队友的武器。

投技+信息战:
  对方刚被投过一次 (从他的信息池中: "我被这个人的投技摔倒了")
  → 战略层更新:"下次他靠近时, 我得留一个'缠'来防他的投技。"
  → 投技之间的心理博弈——投技成为一种威慑。
```

### 11.4 投技的性能

```
投技在每场近战中的使用频率:
  不是每个回合都有——抓取窗口窄。
  一场 45s 的 1v1 近战: 可能出现 1-4 次投技尝试。
  成功: ~1-2 次。
  
  投技不是"主攻击手段"——是"关键时刻的破局招式"。
  它的威胁大于它的实际使用频率——因为对手知道: 
  "如果我在他面前使了一个大后摇的动作, 他会抓我。"
  这个"威胁"本身改变了双方的战斗节奏。
```

## 十二、战斗 HUD——玩家作为"战略干预者"

### 12.1 核心原则

```
1. 玩家不操作 = 角色完全自主战斗。
   玩家角色和 NPC 共用同一套三层模型 + 决策引擎。
   唯一的差异: 玩家可以在任何时候覆盖 AI 的决策。

2. 不弹窗。不暂停。不减速。不子弹时间。
   战斗始终全速运行。玩家在实时战斗中做高层决策——和现实中指挥作战一样。

3. 看到什么取决于你知道什么。
   不对玩家"上帝视角"。队友的状态不自动显示——
   玩家需要看向队友才能了解他们的状态。
   这和战斗信息不对称的原则一致——玩家也是"战斗者"，不是"全知者"。

4. 随时干预。不干预就自动。
   如果玩家 30 秒没有任何输入——AI 继续自主战斗。
   这不是"挂机"——是"信任你的角色的战斗能力"。
```

### 12.2 玩家角色人格

玩家角色人格 = 创建角色时设定（和 NPC 一样的大五人格滑动条）。
玩家 AI = NPC AI。同一套三层模型。不操作 = 角色按自己的人格自动战斗。

```
一个"暴躁型"玩家 (高神经质 + 低宜人性):
  默认: 猛攻、不防守、危急时暴走。
  玩家可手动让他冷静——但每次覆盖都是在和"他的天性"对抗。
  
一个"冷静型"玩家 (低神经质 + 高尽责性):
  默认: 精确、耐心。但可能因过于保守而错失机会——
  玩家看到 AI 发现窗口→手动按下进攻→打破了自己的谨慎。
```

## 十三、系统缺口解决方案

### 13.1 魔力 vs 耐力恢复

```
耐力 (体力):
  来源: 呼吸 + 营养 → 自动恢复
  恢复速度: 基准 5%/秒 (非战斗时)
           战斗中 1%/秒 (剧烈运动抑制恢复)
           疲劳度高时 0.5%/秒 (越累恢复越慢)
  满耐力: ~20s (非战斗) / ~100s (战斗中)
  
  人格修正:
    高尽责性: 战斗中耐力恢复 ×1.2 (懂得节奏, 会"省力")
    低尽责性: 战斗中耐力恢复 ×0.8 (乱用力, 消耗快)
    年轻 (18-30): 恢复 ×1.2
    年老 (50+): 恢复 ×0.7

魔力 (法力):
  来源: 灵元素从血元素中提炼 → 主动恢复 (需要时间+安静)
  恢复速度: 基准 1%/秒 (非战斗, 静坐/冥想)
           战斗中 0.2%/秒 (分心, 提炼效率低)
           战斗中完全耗尽后 0.05%/秒 (魔力池枯竭, 极慢)
  满魔力: ~100s (静坐) / ~500s (战斗中) / 不建议在战斗中耗到空
  
  人格修正:
    高灵元素亲和: 恢复 ×1.5
    高尽责性: 冥想效率 ×1.3 (更专注)
    血元素健康差 (受伤/饥饿): 恢复 ×0.5 (身体是魔力的"工厂")

平衡:
  物理: 消耗快, 恢复快。节奏: 猛攻→喘气→再攻。
  魔法: 消耗慢(每发成本高), 恢复慢。节奏: 谨慎使用→关键时刻释放。
  混合 (魔法剑士): 双重管理。更灵活但每项都不如专精。
```

### 13.2 召唤物/契约物 AI

```
召唤物 = 简化 NPC。三层模型中:

本能层: 自带的物理感知+闪避。和普通 NPC 相同。
        精度取决于主人的灵元素强度 (不是召唤物自己的属性)。
  
节奏层: 执行主人的命令。在召唤时设定节奏倾向, 之后不改变。
        火元素 → 猛攻 (无自我保护)。水元素 → 保护主人。
        石傀儡 → 固守 (挡在主人前面)。

战略层: 无。召唤物不自定目标。不评估处境。
        "攻击那个弓手" → 节奏层一直执行, 直到:
        a) 弓手死了
        b) 主人新命令覆盖
        c) 召唤物被消灭/召唤时间到

契约物 (更高级):
  和普通召唤物不同——契约物有有限的自主性。
  契约生物 (如: 和火灵签订了长期契约):
    → 有战略层 (有限)。可以自定: "主人现在危险, 我要保护他"
    → 但契约条款限制了它的自主范围。
    → 和主人的关系 = 团队中的"高度信任队友" + "契约约束"
```

### 13.3 魔法的社会评价——文化相对 + 个人差异

魔法本身没有固定的"禁忌"标签。一个魔法是否被接受——
取决于：谁在看、在哪个文化里、被用在什么情境下。

```
同一魔法, 不同文化:

死灵魔法:
  文化 A (祖先崇拜): "他在和先祖的灵魂沟通。这是神圣的。"
    → 目击者好感度 +0.2, 敬畏
  文化 B (排斥亡者): "他在亵渎死者! 抓住他!"
    → 目击者好感度 -0.5, 可能触发法律追责
  文化 C (学术自由): "这是前沿魔法研究。我们不赞成, 但不禁绝。"
    → 目击者可能回避, 但不会举报

惑心魔法 (控制他人意志):
  某文化: "侵犯自由意志是最恶劣的罪行。" → 极度排斥
  另一文化: "只要能达成目的, 手段是次要的。" → 默许 (但私下可能警惕)

同一魔法, 同一文化, 不同的人:

目击者的个人反应 = f(人格, 知识, 与该使用者的关系):

  高宜人性 + 高神经质:
    → 看到死灵魔法 → 恐惧 + 厌恶 → 可能去举报
  低宜人性 + 高开放性:
    → 看到死灵魔法 → "有意思。他怎么做到的?" → 可能去搭话
  高尽责性 + 法律的维护者 (卫兵):
    → 看到任何"本地法律禁止"的魔法 → 执法 (职业职责)
  追求力量的人 (任何人格, 但有"追求力量"的自我标签):
    → 看到任何强大魔法 → "这种力量……我想要。"

与该使用者的关系:
  如果是朋友: "我不在乎他用了什么——他救了我们。"
  如果是敌人: "看! 他用禁忌魔法! 抓住他!" (利用文化规范当武器)
  如果是陌生人: 取决于人格 + 该文化对该魔法的默认态度

长远后果——不由"系统"决定, 由"社会"决定:

目击者记忆 → 日常社交中传播 → 传播中信息衰减/扭曲 (和战斗声誉相同)
  → 传到执法者耳中 → 触发追责 (如果该法律存在)
  → 传到对该魔法有兴趣的人耳中 → 可能来拜师/挑战/偷学
  → 不传到任何人耳中 (目击者没传播) → 没有社会后果

但有一个"自然后果"是不依赖社会判断的:
  长期大量使用某种元素 → 施法者本身被该元素"染色":
    大量使用火 → 脾气可能变得更暴躁 (火元素影响灵元素)
    大量使用死灵 → 身体可能变得苍白、容易生病
    大量使用惑心 → 自己对幻象的抵抗力可能反而下降
  → 这是魔法本身的物理反馈——不是社会惩罚
```

### 13.4 武器切换冷却

```
切换时间 (从当前武器→目标武器):

收当前武器:
  单手武器 (剑/斧/锤): 0.3s
  双手武器 (大剑/长矛): 0.6s
  弓 (卸下): 0.4s
  法杖 (放下): 0.5s

取新武器:
  单手武器 (从腰间): 0.4s
  双手武器 (从背上): 0.8s
  弓 (从背上取下+搭箭): 0.7s
  法杖 (从背上/手边): 0.6s

切换总时间 = 收 + 取。例:
  剑→弓: 0.3s(收剑) + 0.7s(取弓) = 1.0s
  弓→剑: 0.4s(卸弓) + 0.4s(取剑) = 0.8s  (弓手被近身→拔剑比对方收剑取弓快)
  法杖→单手剑: 0.5s + 0.4s = 0.9s

切换期间: 不能攻击/格挡。本能层只能闪避。
切换速度受敏捷属性影响: 高敏捷 → ×0.7-0.8。低敏捷 → ×1.2-1.5。

人格影响:
  高尽责性: 切换时机精准——选在对方后摇时切换 (对方无法打断)
  低尽责性: 可能在不恰当的时机切换→被对方抓住空档
```

### 13.5 俘虏后交互

```
俘虏 = NPC 进入"被囚禁"情境模式。

同一套 NPC AI (人格/情绪/记忆/决策器)——但行动选项受限:
  可用行动: 等待、配合、求情、谈判、反抗(有限)、绝食、装病、试图逃跑
  不可用: 自由移动、自由攻击、自由使用物品

俘虏对捕获者的评估 (持续性的"社交感知"):
  人格: 宜人性高 → 倾向于配合。"他们是讲道理的人。"
        宜人性低 → "我恨他们。我在等机会。"
        神经质高 → 恐惧主导。可能崩溃招供、可能求饶、可能装死。
        开放性高 → "也许我可以和他们交易……"
  
  记忆: 如果认识捕获者 (之前有正面关系) → 俘虏可能觉得"他不会杀我"
        如果听说过捕获者的声誉 ("他从不杀俘虏") → 恐惧降低
        如果听说过捕快者的坏名声 ("他杀过投降的人") → 恐惧极高

捕获者的交互选项:
  审讯: 问问题。俘虏的回答质量和态度 = f(人格, 恐惧, 对审讯者的评估)
        可能说谎、可能招供、可能沉默。
  释放: "你走吧。" → 俘虏的记忆 + 关系更新 → 声誉传播
  赎金: 放出消息→等对方团体/家人来赎
  招降: "加入我们。" → 俘虏评估: 这是出路? 还是背叛?
  囚禁: 长期关押。被囚禁模式持续。
  处决: 违反"不杀"条件。道德后果 + 声誉。

俘虏后的"战后过渡" (和战斗结束的 PostCombatTransition 相同逻辑):
  被俘本身就是一次"创伤经历"。
  情绪残留 + 记忆编码 + 人格消化 → 影响俘虏被释放后的长期行为。
```
