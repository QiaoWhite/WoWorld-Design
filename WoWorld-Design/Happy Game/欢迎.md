# WoWorld — Wonder World

> 与其说这是一个游戏，不如说我们在创建另一个"人世"。以西方剑与魔法为基底，全球多元文化自由融入。独立游戏开发中。

---

## 这是什么？

**WoWorld** 是一款程序化生成的体素开放世界RPG。世界中目标常驻100,000个活着的NPC（包括约1,000个L1全模拟NPC）——他们力求成为完整的人：有记忆、预测、情感、思想、人际关系、欲望、习惯。你在这个世界做什么都行：组建冒险小队去无穷远方探索失落的文明与奇迹，或者做一个铁匠、商人、政客、法外狂徒度过一生——每条路都是"主线"。世界会记住你的一切。随着开发深入与技术革新，模拟规模将不断提高。

### 这游戏跟别的RPG有什么不同？

**没有剧本。** 传统RPG——比如D&D系列——用开发者手搓的剧情和代码来呈现一个庞大的故事。设计师的故事写到哪里，游戏的寿命就基本上终止在哪里。

WoWorld 不是这样。它意图创建一个能够像现实中的人世间那样，自由、自然地生成各种各样事件的世界。**开发者做的不是一个故事，而是一个故事生成器。** 我们只定义规则——物理规则、心理规则、社会规则——具体的故事由世界自己长出来。

**每个存档都是一个全新的世界。** 就像《我的世界》那样，创建一个存档就创建一个新世界。同一个世界内的内容眼花缭乱、多姿多彩；不同世界之间更是可以大相径庭、异彩纷呈。一个世界可能诞生出骑士王国与东方王朝并立的格局，另一个世界则可能演化出完全不同的文明形态——各个文明还可以自己发展出独一无二的文化特色。

**这个世界有历史。** 玩家进入时，世界已经运转了千百万年。书本、遗迹、传说中保留着先民活过的证据。玩家像一个新生儿第一次睁开眼睛——世界早就在这里了，你需要自己去理解它。

> **核心理念：只创造规则，不创造故事。游戏的寿命不由"开发者写了多少剧情"决定——因为根本没写剧情。世界自己会写。**

## 技术栈

- **引擎**: Godot 4.6.2
- **画面**: 3D 低多边形 + 像素纹理
- **地形**: 连续体素（Marching Cubes/Transvoxel）+ 网格物体混合
- **平台**: Windows / Linux / macOS
- **美术**: AI生成 + 手动调整
- **LLM**: 可选，玩家自备Key
