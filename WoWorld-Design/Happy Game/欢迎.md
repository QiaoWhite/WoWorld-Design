# WoWorld — Wonder World

> 一个真正活着的剑与魔法世界。独立游戏开发中。

---

## 这是什么？

**WoWorld** 是一款程序化生成的体素开放世界RPG。世界中有10,000+个活着的NPC——他们有记忆、情绪、欲望和自己的生活。你在这个世界做什么都行：冒险、经商、种田、从政、战争。世界会记住你的一切。

## 技术栈

- **引擎**: Godot 4.6.2
- **画面**: 3D 低多边形 + 像素纹理
- **地形**: 连续体素（Marching Cubes/Transvoxel）+ 网格物体混合
- **平台**: Windows / Linux / macOS
- **美术**: AI生成 + 手动调整
- **LLM**: 可选，玩家自备Key
