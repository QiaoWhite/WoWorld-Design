# 017-开发阶段测试记录

> **日期**：2026-06-10  
> **定位**：逐 Phase 的"双视角沉浸测试"——验证设计是否被兑现、涌现是否被感知  
> **测试次数**：每 Phase 10 次独立会话，不同种子，共 70 份报告  
> **参照技术方案**：`016-正式技术栈方案v2-20260610/`

---

## 一、测试方法论

### 1.1 核心思路

传统开发测试问的是"系统有没有 bug？"。WoWorld 多了一层更根本的问题：**"系统的设计意图，玩家有没有感知到？"**

这套测试记录专门回答这个问题。每个 Phase 完成后，一名开发者测试员以双视角进行 10 次独立测试会话：

```
测试员先以"无知的普通玩家"身份进入世界
  → 记录所见、所闻、所感
  → 然后切换到"开发者调试面板"
  → 检查系统实际数据
  → 对比差距：玩家感受到的 vs 系统实际发生的
```

### 1.2 双视角混合

| 视角 | 身份 | 记录内容 | 价值 |
|------|------|---------|------|
| A 面：沉浸玩家 | 不知道系统怎么运作的普通人 | 看到了什么、听到了什么、感受到了什么——用自然语言描述 | 回答"涌现是否被感知" |
| B 面：开发者 | 能打开调试面板、看 NPC 数据、追踪决策树 | 系统实际状态：NPC 需求值、情绪值、记忆条目、决策路径 | 回答"设计是否被兑现" |
| 差距分析 | — | A 面感知 vs B 面实际的差异 | 回答"哪些设计做了但玩家没发现、哪些是玩家脑补的" |

### 1.3 测试参数

| 参数 | 设定 |
|------|------|
| 初始条件 | 每次测试用不同种子（seed），产生不同的地形/NPC/初始状态 |
| 测试时长 | Phase 动态设定（见下表） |
| 行动方式 | 测试者自由行动，不预设路线——模拟真实玩家行为 |
| 记录格式 | 统一模板（见 §三） |

### 1.4 专项测试维度：远景可见性

WoWorld 的核心体验之一——"站在高处看到壮阔的延伸至地平线的远景"——不是可有可无的锦上添花，而是玩家对"这是一个真实世界"的第一印象来源。远景缺失 = 世界感缺失。从 Phase 1 开始，每一次测试都**必须**包含远景评估。

**远景体验的三层要求**：

| 层级 | 要求 | 对应技术 | 首次纳入测试 |
|------|------|---------|-------------|
| V1：能看到 | 站在平地能看多远？站在高处能看到地平线吗？ | LOD 0-2 基础渲染距离 | Phase 1 |
| V2：能看清 | 远处的地形轮廓清晰吗？LOD 切换有可见的跳变吗？ | Transvoxel LOD 3-5 | Phase 4 |
| V3：有壮阔感 | 远景是否产生"哇"的感觉？天际线是否自然？奇幻特征（浮空岛、巨菇林）在远景中是否可见？ | 8 级 LOD + Billboard + 大气散射 | Phase 6 |

**远景测试的关键动作**（每 Phase 每次测试至少执行）：

| 动作 | 测试什么 | 适用 Phase |
|------|---------|-----------|
| **平地望远** | 站在平地，视线水平方向——能看到多远？远处是"一片模糊"还是"能看到轮廓"？ | 全部 |
| **登高望远** | 爬到本场景最高点——视野范围是否明显扩大？远景是否带来了"壮阔感"？ | 全部 |
| **高速移动远望** | 奔跑/骑马/飞行时看向地平线——远景是否抖动/闪烁/消失？LOD 切换是否平滑？ | Phase 4 起 |
| **不同时间/天气远望** | 正午 vs 黄昏 vs 暴风雨——远景在不同光照/大气条件下的表现是否自然？ | Phase 5 起 |
| **奇幻特征远望** | 浮空岛/巨菇林/水晶簇——在 500m/1km/3km 外是否可见？ | Phase 5-6 |

**远景的性能约束**：

| Phase | 目标视距 | 帧预算上限 | 远景渲染占帧预算 |
|-------|---------|-----------|-----------------|
| 1 | 500m（单层地形，无 LOD） | 16.7ms 总 | ≤2ms |
| 4 | 1.6km（Transvoxel LOD 0-5） | 16.7ms 总 | ≤2.5ms |
| 6 | 6.4km+（8 级 LOD + Billboard） | 16.7ms 总 | ≤3ms |

> **核心原则**：远景体验不能以帧率崩塌为代价。每次测试必须同时记录"看到了什么"（A面）和"帧时间/内存/VRAM"（B面）。如果远景视觉达标但帧率 <30fps，该项测试标记为**不合格**。

### 1.5 专项测试维度：聚落与建筑

聚落不只是"一堆房子的集合"——它是 NPC 生活的舞台，是玩家对一个地区"文明程度"的第一印象来源。四级聚落体系（城市→城堡→小镇→村庄）+ 独立建筑 + 遗迹，从 Phase 4 开始陆续上线。每个阶段的聚落测试必须回答"这个地方像有人住过吗？"

**聚落体验的四层递进**：

| 层级 | 要求 | 首次纳入测试 |
|------|------|-------------|
| S1：有村 | 聚落存在且可见——从远处能看到聚落轮廓（炊烟/建筑群/城墙），走近后有可识别的建筑 | Phase 4 |
| S2：有差异 | 不同种子/文化的聚落视觉风格不同——不是"同一套模型换了个位置" | Phase 5 |
| S3：有生活 | 建筑内部有人、有物、有使用痕迹——NPC 在建筑内外的活动让聚落"活"起来 | Phase 5-6 |
| S4：有历史 | 聚落之间有道路连接，有遗迹散落，有新旧建筑的年代层次——聚落本身在"讲故事" | Phase 6 |

**聚落测试的关键动作**（适用 Phase 的每次测试至少执行）：

| 动作 | 测试什么 | 适用 Phase |
|------|---------|-----------|
| **远眺聚落** | 从 500m/1km/3km 外望聚落——它是否构成天际线上的一个"锚点"？ | Phase 4 起 |
| **走进聚落** | 从野外走进聚落的过渡感——有没有"进入了一个人类聚居地"的感觉？ | Phase 4 起 |
| **街道漫步** | 在聚落内部行走——街道宽度、建筑间距、空间节奏是否自然？有没有"死胡同"或太空旷的广场？ | Phase 4 起 |
| **建筑进入** | 推门进入一个建筑——内部空间是否与外部体量匹配？内饰是否与建筑类型一致？ | Phase 5 起 |
| **聚落功能验证** | 铁匠铺在打铁吗？市场在交易吗？教堂有人祈祷吗？——建筑的功能是否被 NPC 行为兑现？ | Phase 5 起 |
| **文化风格辨识** | 不看标签，能通过建筑风格识别出这是哪个文化区域的聚落吗？ | Phase 5-6 |
| **道路漫游** | 沿着道路走——它最终通向另一个聚落吗？道路的铺设质量/宽度是否随聚落等级变化？ | Phase 5 起 |
| **遗迹探索** | 找到一个遗迹——它看起来像"曾经有人住过的废弃建筑"还是"随机摆放的废墟块"？ | Phase 6 |
| **聚落发现感** | 10 次测试每次用不同种子——每次的聚落布局、位置、组合是否产生了"探索的新鲜感"？ | 全部 |

**聚落测试的差距关注**：
- A面：玩家觉得"这是一个真实的小镇"——B面：建筑是按 WFC 生成的模板排列，但玩家脑补出了"这个镇子建在河边是因为贸易"
- A面：玩家没注意到建筑有"文化风格"——B面：文化参数确实影响了建筑生成规则，但视觉差异不够大到被感知
- 核心问题：**建筑的生成规则（数学）和建筑的体验（感性）之间的翻译是否成功？**

### 1.6 专项测试维度：战斗

战斗不是"砍怪升级"——WoWorld 的战斗是 NPC 心理模块的自然延伸。NPC 在战斗中的行为由人格驱动（狂暴/战术/支援/施法），战斗结果写入记忆，影响关系网络。战斗测试不只是测手感，更要测"战斗是不是在讲故事"。

**战斗系统的测试递进**（对应总设计文档的三步渐进）：

| 阶段 | 内容 | 首次纳入测试 |
|------|------|-------------|
| F1：基础碰撞战斗 | 碰撞盒+轻重攻击+格挡+血条——1v1 和 1vN 的基本"打得起来" | Phase 3 |
| F2：元素魔法+战斗风格 | 火冰雷元素+人格驱动的战斗风格（狂暴/战术/支援/施法） | Phase 5 |
| F3：完整战斗 | 10元素+信息战+空战+环境交互+战斗士气+社会后果 | Phase 6 |

**战斗测试的关键动作**（每次测试至少执行适用的动作）：

| 动作 | 测试什么 | 适用 Phase |
|------|---------|-----------|
| **1v1 基础** | 与一个 NPC 战斗——命中判定有感觉吗？格挡时机合理吗？血条变化清晰吗？ | Phase 3 起 |
| **NPC 战斗风格识别** | 不看面板，通过战斗行为判断——这个 NPC 是狂暴型/战术型/支援型/施法者？人格驱动在战斗中是否可见？ | Phase 5 起 |
| **魔法战斗** | 施放火球/冰箭/雷击——元素相克有可感知的差异吗？魔法与物理的轻重感差异明显吗？ | Phase 5 起 |
| **战斗的社会后果** | 战斗中/战斗后——围观 NPC 反应？战斗是否被"著名度系统"记录？被你打伤的 NPC 后来对你的态度变了？ | Phase 5 起 |
| **团队战** | 组队（哪怕临时同盟）2v2 或 3v3——NPC 队友的协作行为看起来智能吗？ | Phase 5-6 |
| **环境互动** | 在建筑内/森林中/悬崖边战斗——环境是否影响战斗策略？建筑有破坏痕迹吗？ | Phase 6 |
| **信息战** | 高伤害后"强装镇定"能否影响对方士气？对方的实力认知是否来自"过往信息+当前观察"而非全知？ | Phase 6 |
| **空战** | 飞行坐骑上对空/对地攻击——空中战斗的手感和规则与地面有合理差异吗？ | Phase 6 |
| **战斗涌现故事** | 这场战斗有没有产生一个"可以讲给别人听的故事"？还是只是"我砍了他，他死了"？ | Phase 5 起 |

**战斗测试的差距关注**：
- A面：玩家觉得"这场战斗很刺激"——B面：GOAP 安全网驱动 NPC 做出了合理但"可预测"的战斗决策，玩家感受到的刺激来自碰撞物理而非 AI
- A面：玩家完全没注意到 NPC 的"战斗风格"——B面：人格参数确实影响了攻击频率/防御倾向，但动画和动作没有足够传达差异
- 核心问题：**战斗是否像 NPC 人格的延伸——还是像一个独立的"战斗小游戏"？**

### 1.7 各 Phase 测试时长

| Phase | 内容 | 建议单次时长 | 理由 |
|-------|------|-------------|------|
| 1 | 地形框架+玩家存在 | 10-15 分钟 | 内容薄——走、看、感受空间和光 |
| 2 | 第一个活人 | 10-15 分钟 | 1-5 NPC，观察一个 NPC 的日常行为足够 |
| 3 | 性格与情绪 | 15-20 分钟 | 需要观察到 NPC 之间的行为差异才能得出结论 |
| 4 | 记忆与关系+世界 | 20-30 分钟 | NPC 社交网络和记忆效应需要更长观察窗口 |
| 5 | 社会与文化 | 30-45 分钟 | 社会事件（节日/冲突/团体）出现需要时间 |
| 6 | 完整人世 | 45-60 分钟 | 100K NPC + 完整系统，玩法多样需要时间探索 |
| 7 | 打磨与深度 | 60 分钟 | 持续迭代中的回归测试 + 新内容探索 |

## 二、各 Phase 测试重点

### Phase 1：地形框架 + 玩家存在
> **产出**：一个平坦大地 + 简单噪声起伏 + 昼夜循环 + 玩家移动/跳跃/镜头  
> **远景目标**：V1——"能看到"，平地视距 ≥500m，不崩帧

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-2 | 地形感知 | 地面是"一大片平地"还是有微妙的起伏？视觉上有"这里和那里不一样"的感觉吗？ |
| 3-4 | **远景：平地** | 站在平地上水平方向能看多远？远处是"一片模糊/雾"还是"能看到地形轮廓延伸到天际"？ |
| 5-6 | 昼夜循环 | 昼夜的过渡自然吗？黄昏/黎明时远景可见度变化是否自然？ |
| 7-8 | **远景：登高** | 爬到场景最高点（哪怕是人工土坡）——视野范围有没有明显扩大？有没有"世界变大了"的感觉？ |
| 9 | 空间感 | 走 10 分钟后回望出发点——能看到出发区域吗？地平线始终在吗？ |
| 10 | 综合 | 大平地+移动+昼夜+远景——作为一个"世界的基底"，它有没有"辽阔感"？有没有一刻让你想停下来看远方？ |

**关键差距关注**：
- 地形噪声参数是否产生了玩家可感知的视觉多样性？还是玩家只觉得"到处都一样"？
- **远景关键差距**：500m 视距下，玩家 A 面感觉"能看到很远"——但 B 面数据显示实际渲染只有 200m（被雾隐藏了）。两者是否一致？

### Phase 2：第一个活人
> **产出**：1-5 NPC，需求槽驱动行为（找食物、找水、睡觉）

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-3 | NPC 存在感 | 看到一个 NPC 时——"他在干什么？"他的行为看起来有目的吗？ |
| 4-6 | NPC 自主性 | 跟着一个 NPC 看 5 分钟——他的行为连贯吗？中断后换目标了吗？ |
| 7-9 | 需求可观测性 | 不看面板，能猜到 NPC 饿了/渴了/累了吗？他的动画/动作传达了状态吗？ |
| 10 | 综合 | 5 个 NPC 同时在做什么？他们之间有互动（哪怕只是擦肩而过）吗？ |

**关键差距关注**：NPC 行为在数据上是连贯的（需求→决策→行动），但玩家感知到的是"有目的的行动"还是"随机游荡"？

### Phase 3：性格与情绪
> **产出**：10-30 NPC，大五人格+8 情绪+短记忆（20-50条），基础碰撞战斗（F1）  
> **聚落目标**：无聚落——但关注 NPC 聚集行为（是否自发形成"人群"？）  
> **战斗目标**：F1——"能打起来"，碰撞盒+轻重攻击+格挡+血条

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-2 | 人格可观测性 | 不看面板——能不能通过观察行为区分"暴躁的 NPC"和"友善的 NPC"？ |
| 3-4 | 情绪影响 + **战斗触发** | NPC 情绪变化后行为是否可见改变？情绪冲突升级为肢体冲突时——"打得起来"吗？ |
| 5-6 | **基础战斗** | 与一个 NPC 战斗——命中判定有感觉吗？格挡时机合理吗？血条变化清晰吗？碰撞反馈有"打击感"吗？ |
| 7-8 | 短期记忆 + **战斗记忆** | 你惹了一个 NPC → 走开 → 回来——他记得你吗？你打了一个 NPC → 第二天他见你的反应？ |
| 9-10 | 涌现多样性 | 10 次测试每次种子不同——30 个 NPC 的"群体氛围"每次都不一样吗？有因性格摩擦自然产生的冲突事件吗？ |

**关键差距关注**：
- 人格参数在数据上差异很大，但行为表现差异是否足以让玩家感知到？
- **战斗差距**：战斗在数据上"发生了"（碰撞检测→伤害→血条减少）——但玩家 A 面感受到的是"打架"还是"两个胶囊体在互相穿过对方"？

### Phase 4：记忆与关系 + 世界扩展
> **产出**：50-100 NPC + 聚落，完整记忆（200条/人）+ 关系网络 + GOAP + 分层密度场 + 生物群系  
> **远景目标**：V2——"能看清"，视距 ≥1.6km，LOD 切换不可感知  
> **聚落目标**：S1——"有村"，村庄级聚落出现，从远处可识别，内部可穿行

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-2 | 世界多样性 + **远景多样性** | 10 次测试产生了多少种不同的地貌组合？不同生物群系的远景是否有差异？ |
| 3-4 | **聚落首印象（S1）** | 从 500m 外能看到村子吗？走近的过程有没有"从野外进入人类聚居地"的过渡感？村子布局是否自然——建筑间距、街道宽度、中心/边缘的关系？ |
| 5-6 | 关系可见性 + **聚落 NPC 分布** | NPC 见面时的态度能反映关系吗？村子里的 NPC 活动是否和建筑功能匹配（铁匠在铁匠铺附近、农民在田地附近）？ |
| 7-8 | GOAP 安全网 + **远景 LOD** | 把 NPC 的食物偷走——他的反应合理吗？远处的建筑多边形有可见跳变吗？ |
| 9 | **战斗与关系** | 在村子里战斗——围观 NPC 什么反应？战斗后你的名声变了吗？受伤 NPC 后续对你的态度改变了吗？ |
| 10 | 综合 | 登高望远——有山有水有村有人——聚落是视觉锚点还是融入了地形？有"这个地方有人在过日子"的感觉吗？ |

**关键差距关注**：
- 关系网络和记忆系统跑起来了，但 NPC 行为的变化精细到足以让玩家"脑补出故事"吗？
- **聚落差距**：建筑是 WFC 生成的，但玩家感觉"这个村子是自然长出来的"还是"随机刷了一堆房子"？
- **远景差距**：LOD 3→4 切换时帧尖峰是否超过预算？

### Phase 5：社会与文化
> **产出**：500-1000 NPC，文化三层+团体+事件+经济+法律，建筑内部+道路+奇幻特征  
> **远景目标**：V2→V3 过渡——奇幻特征在远景中开始可见，不同时间/天气的远景有差异  
> **聚落目标**：S2-S3——"有差异、有生活"，四级聚落体系完整，建筑可进入，文化风格可辨识  
> **战斗目标**：F2——"元素魔法+战斗风格"，火冰雷+人格驱动战斗风格可见

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-2 | **聚落文化差异（S2）** | 不同文明种子的聚落——建筑风格、街道布局、装饰纹样有可感知的差异吗？不看标签能判断"这是东方式/西方式/中东式"聚落吗？ |
| 3-4 | **建筑内部（S3）** | 推门进入铁匠铺/教堂/民居——内部空间与外部体量匹配吗？内饰是否与建筑类型+文化风格+NPC 职业一致？家具摆放是否自然？ |
| 5-6 | **战斗风格识别（F2）** | 分别与高神经质 NPC 和高尽责性 NPC 战斗——他们的战斗风格差异（狂暴 vs 战术）在动作/节奏/决策上是否可感知？元素魔法（火球/冰箭/雷击）的手感和视觉反馈有区分度吗？ |
| 7-8 | **道路+聚落连接** | 沿着道路走——道路是否自然连接聚落？道路铺设质量是否随聚落等级变化？有没有"野路变成石板路"的过渡？ |
| 9 | **战斗社会后果** | 在聚落中战斗——"战斗显眼度"决定多少 NPC 围观？战斗结果是否写入社会记忆？有没有 NPC 因为"目睹了你打人"而改变对你的行为？ |
| 10 | 综合 | 这个世界有城市有村庄有道路有建筑内部有战斗——聚落是否从"建筑集合"变成了"人类社会生活的容器"？ |

**关键差距关注**：
- **聚落差距**：WFC 生成了建筑内部，但玩家走进一个房间时——感觉"这是一个铁匠的家"还是"一个贴了铁匠标签的通用房间"？
- **战斗差距**：人格驱动战斗风格在数据上差异明显（攻击频率/防御倾向不同）——但视觉传达是否足够让玩家在战斗中"认出"对手的战斗风格？还是所有 NPC 打起来都一样？
- **远景差距**：奇幻特征从 1km 外是否可识别为"浮空岛"而非模糊的点？

### Phase 6：完整人世
> **产出**：100K NPC（L1/L2/L3），完整情绪+记忆+全球化+天气+季节+全部职业+存档+Mod  
> **远景目标**：V3——"有壮阔感"，8级LOD完整上线，6.4km+视距，奇幻地标成为视觉锚点，帧率稳定  
> **聚落目标**：S4——"有历史"，遗迹+年代层次+聚落间道路网络完整，聚落本身在"讲故事"  
> **战斗目标**：F3——"完整战斗"，10元素+信息战+空战+环境交互+战斗士气+社会后果全链路

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-2 | 规模感 + **超远景 + 聚落天际线** | 站在高山顶——6.4km 视野中能看到多少个聚落？城市/城堡/小镇/村庄的天际线有层级差异吗？聚落在远景中是否形成了"文明分布的视觉地图"？ |
| 3-4 | **完整战斗体验（F3）** | 组队 3v3 战斗——队友协作、元素相克、环境利用（掩体/高地/狭窄空间）是否自然发生？空战（飞行坐骑）的手感与地面战斗有合理差异吗？ |
| 5-6 | **遗迹探索（S4）** | 找到一个遗迹——它看起来像"曾经有人住过的废弃建筑"吗？遗迹与周围聚落有历史关系吗（比如"这个废弃哨站曾经保护过下面那个村庄"）？遗迹中有"先民活过的证据"吗？ |
| 7-8 | **信息战 + 战斗涌现故事** | 高伤害后"强装镇定"能否影响对方士气？NPC 对敌方实力的认知来自"过往信息+当前观察"吗？这场战斗结束后——你的脑中有一个"可以讲给别人听的故事"吗？还是只是"我赢了/输了"？ |
| 9 | **不同人生 + 聚落功能** | 开一个铁匠档——铁匠铺真的在运作吗？有 NPC 来找你打铁吗？聚落中的建筑功能是否被 NPC 行为"兑现"了（市场有交易、教堂有祈祷、酒馆有闲聊）？ |
| 10 | 终极综合 + **聚落叙事** | 漫步一个城市——建筑的新旧程度有差异吗？有没有"老城区"和"新城区"的感觉？聚落的物理布局本身是否在讲这个文明的故事？**远景+聚落+战斗+NPC——这些系统叠加后，是否产生了一个"活的人世"？** |

**关键差距关注**：
- 100K 规模是否带来质的飞跃？
- **聚落差距**：遗迹和年代层次被系统生成了——但玩家走在城市中，感受到的是"有历史的城市"还是"建筑随机排列"？
- **战斗差距**：信息战和战斗士气在数据上运作——但玩家在战斗中是否感受到了"心理博弈"的层面？还是仍然只是砍血条？
- **远景差距**：8级LOD全开 + 聚落天际线 + 战斗+特效 + 天气——帧率是否稳定？

### Phase 7：打磨与深度
> **产出**：性能稳定、Bug 修复、LLM 增强、内容丰富、本地化  
> **远景目标**：V3 巩固——远景体验不因优化而退化，新内容在远景中持续可见  
> **聚落目标**：S4 巩固——新建筑/新文化风格在聚落中的整合质量  
> **战斗目标**：F3 巩固——战斗手感打磨，边缘情况处理

| # | 测试焦点 | 关键问题 |
|----|---------|---------|
| 1-3 | **回归验证（远景+聚落+战斗）** | Phase 1-6 的核心体验还在吗？优化有没有"优化掉"涌现？远景有没有因性能优化而退化？聚落生成质量有没有因新内容加入而"稀释"？战斗手感有没有因某次优化而变差？ |
| 4-6 | LLM 增强 + **LLM 对话与战斗叙事** | LLM 对话比模板好多少？LLM 生成的战斗描述（战斗日志/战后回顾）是否让战斗"更有故事性"？ |
| 7-8 | 新内容 + **新聚落风格/新战斗方式** | 新加的文化种子——其建筑风格在聚落中是否自然融合（而非"贴标签"）？新武器/新魔法类型——战斗体验是否因此产生了新的战术维度？ |
| 9 | **本地化 + 聚落文化文本** | 不同语言的体验一致吗？聚落名称、建筑标识、文化相关文本翻译后还有味道吗？ |
| 10 | **最终综合：新人盲测** | 一个完全新鲜的测试员进入游戏——不需要任何解释——他能否在 30 分钟内：①发现地平线的壮阔 ②找到一个聚落并走进去 ③与 NPC 发生一场有意义的互动（对话或战斗）④产生"我想继续探索"的冲动？ |

## 五、使用方法

1. **Phase 完成后**：在对应子文件夹中创建 10 份测试报告
2. **每次测试**：新种子 → 玩家视角体验 → 开发者面板检查 → 填写模板
3. **10 次完成后**：汇总该 Phase 的"差距分析"→ 发现的问题分三类处理
   - **P0 立即修复**：影响核心体验的 bug 或设计缺陷
   - **P1 Phase 内调整**：需要在本 Phase 范围内调优的参数和实现
   - **P2 后续 Phase 留意**：当前不做改动，但需在后续设计/开发中关注
4. **跨 Phase 对比**：随着 Phase 推进，对比同一维度的感知变化——比如"NPC 行为自然度"从 Phase 2 到 Phase 6 的提升曲线
