| # | 维度 | v2.0 | v3.0 |

| 7 | 海洋 | 无 | **Gerstner 波 + OceanProvider trait** |
| 8 | 战斗 | 动作游戏（轻攻/重攻/格挡按键） | **半自动 (玩家 AI = NPC AI + 高层指令)** |
