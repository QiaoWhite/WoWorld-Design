# 010-内部架构与子插件设计（大纲）

> **开发代号**: WoWorld (Wonder World)
> **状态**: 设计草稿（大纲·要点）

---

## 一、模块内部原则

- 11 个子插件，每个定义 `pub(crate)` trait
- 换一个组件 = 重新 impl 对应 trait，不动其他 10 个
- 每个公式支持多个可选算法，TOML 选择

## 二、11 子插件

| # | 子插件 | 职责 | pub(crate) trait |
|---|--------|------|-----------------|
| ① | ingestion | 摄入：轮询实体+发射器+对话+环境+玩家 | IngestionEngine |
| ② | propagation | 传播：衰减+吸收+风+温度+遮挡+多普勒+介质 | PropagationEngine |
| ③ | mixing | 混合：频段分割+掩蔽+声场+主导+环境坍缩 | MixingEngine |
| ④ | perception | 感知：perceived_sounds/speech/noise/acoustic_space | PerceptionEngine |
| ⑤ | lifecycle | 生命周期：进入缓冲+老化+打断+源监测+渐弱 | LifecycleManager |
| ⑥ | music | 音乐：情境映射+分层+过渡+世内+玩家音乐 | MusicEngine |
| ⑦ | notification | 提示音：事件订阅+规则表+冷却+优先级 | NotificationEngine |
| ⑧ | voice | 语音：VoiceProfile+情绪调制+队列+TTS路由 | VoicePipeline |
| ⑨ | accessibility | 无障碍：字幕+方向指示+频谱可视化 | AccessibilityEngine |
| ⑩ | assets | 资产：AudioAssetId+材质映射+规则表+校验 | AssetRegistry |
| ⑪ | render | 渲染输出：AudioRenderPacket 序列化→GDExtension | RenderAssembler |

## 三、完全 TOML 化

- audio_config.toml：全局参数（缓冲区/掩蔽阈值/传播模型选择/性能/无障碍/默认音量）
- material_sound_map.toml：材质→声音映射+湿润修正
- notification_sounds.toml：事件→提示音映射
- music_config.toml：情境→音乐预设+过渡规则

Mod 替换 = 替换 TOML + 替换音频文件。零代码。

## 四、数学工具注入点

- 衰减模型：InverseSquare / InverseSquareWithAbsorption / Custom(a,b,c)
- 掩蔽模型：SimpleThreshold / FrequencyBandComparison / ZwickerPartialLoudness
- 混响模型：PresetBased / Statistical(rt60) / SimpleGeometry
- TOML 的 [propagation.model] 字段切换

## 五、音频文件组织

```
woworld/assets/audio/
├── sfx/  (materials/creatures/combat/magic/vehicles/environment/objects/society/ui/)
├── music/ (exploration/combat/settlement/interior/festival/narrative/player/)
└── ambient/ (biome/cave/ocean/sky/)
```

引用：AudioAssetId(u32) + audio_asset_map.toml 映射表

## 六、修改时的目标精准度

| 想改什么 | 去哪里 | 影响范围 |
|---------|--------|---------|
| 某个材质声音 | material_sound_map.toml | 零代码 |
| 传播模型 | propagation/ 重新 impl PropagationEngine | 不碰其他 10 组件 |
| BGM 规则 | music_config.toml 或 music/transition.rs | 不碰音乐分层 |
| 加新提示音 | notification_sounds.toml | 零代码 |
| 换整个音乐系统 | 重新 impl MusicEngine trait | AudioQuery 30 方法不变 |
| Mod 替换全部声音 | 替换文件 + TOML | 零代码 |
