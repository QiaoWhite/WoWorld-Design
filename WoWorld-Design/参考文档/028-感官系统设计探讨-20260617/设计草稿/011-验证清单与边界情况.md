## 10个涌现场景

| # | 场景 | 预期行为 | 关键机制 |
|---|------|---------|---------|
| 1 | 玩家从NPC背后悄悄靠近 | NPC不转身——直到声音或外周运动触发 | 视觉=定向的，声音=事件推送 |
| 2 | 浓雾中两NPC擦肩而过 | 都感知到模糊运动但不辨识 | `WeatherSample.visibility_m`→`effective_range` |
| 3 | 铁匠专注打铁，玩家在面前晃 | 初期不察觉，动作间隙抬头 | 沉浸度0.7→阈值升高→决策间隔长 |
| 4 | 守卫值夜，远处狼嚎 | 守卫警觉↑，平民不受影响 | 职业调制注意阈值 |
| 5 | 诗人看夕阳，战士无感 | 情绪差异从框架涌现 | AestheticFramework差异×人格 |
| 6 | 文化A红色=喜庆，文化B红色=血腥 | 同一夕阳，相反感受 | CulturalNaturalAesthetic从核心参数涌现 |
| 7 | 老人年轻时学过南方审美 | 深度衰减但残留痕迹(depth≥0.01) | 衰减仅受memory影响，永不完全归零 |
| 8 | 醉酒NPC判断同伴伤势 | 视觉模糊+认知清晰度↓→误判 | PerceptualModifiers |
| 9 | NPC在地下洞穴待三天 | 时间感知不确定 | solar_elevation=NAN(sky_exposure<0.1) |
| 10 | 同一战斗三个目击者 | 三个不同版本EventMemory | 各自独立的PerceptualSeed+噪声 |

## 边界情况

| 边界 | 处理方式 |
|------|---------|
| NPC死亡 | SpatialEntity不注销——velocity→0。VisionQuery仍看到尸体。prepare_facts()从"不动了+流血+姿态异常"推断死亡 |
| 实体从世界消失（被拿走/被毁） | 期望验证失败——"本应看到但没看到"→Fact::Vanished |
| L1↔L2过渡（升级） | 空缓存→新奇驱动探索→10周期后稳定 |
| L1↔L2过渡（降级） | PerceptualCache丢弃——下次升级时重新探索 |
| 读档 | PerceptualCache+DarkAdaptation不持久化——读档=重新进入世界→自然环顾行为 |
| 极高密度（200人广场） | 绝大多数标记为Background→跳过→射线预算不随密度线性增长 |
| 极低密度（荒野独行） | 射线预算降到最低——仅检测远处运动 |
| 透明/半透明实体 | `SpatialEntity.is_transparent=true`→不阻断视线 |
| 睡眠/昏迷 | 视觉关闭+听觉阈值升高+20dB——仅强烈刺激能唤醒 |
| 地下（无天空） | sky_exposure=0→solar_elevation=NAN→时间感知完全依赖内感受 |
| 存档噪声确定性 | PerceptualSeed = hash(world_seed, npc_id, time_quantum, modality) → Pcg64 |
| 记忆权重翻转文化基线 | 强烈个人经历(emotional_impact>0.8)可以单次翻转框架值——1次改变人生的事件就够了 |
