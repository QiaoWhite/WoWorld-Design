> **紧迫度**: 🔴 高 — P3.5 生命世界初始化阶段必须消费
> **日期**: 2026-06-20

## 〇、总因：为什么生命系统需要补充

> **核心矛盾**: 当前世界生成仅通过 002-自然景观 §十 的 `HabitatProvider` trait 预留了生物栖息地接口——"世界生成不决定具体生物种类"。但现在玩家进入的世界需要一个**有动植物存在、怪物潜伏、亡灵在古战场徘徊**的初始状态。如果没有"生命世界初始化"阶段，世界是空的——只有地形，没有生物。
>
> **缺失的能力**:
> 1. **初始种群分布** — 没有从群系参数场推导植物/动物分布的算法和接口
> 2. **怪物放置逻辑** — 怪物从哪里来、放哪里、放多少——全部未定义
> 3. **亡灵来源** — 需要历史事件数据才能推导，但历史事件在 P9 才生成
> 4. **动物行为模型** — 动物不能使用 NPC 的 GOAP 规划器——需要独立的简化行为系统

## 一、新增接口：initialize_life_world()

> **★ 为什么需要**
>
> 生命系统当前没有暴露"批量初始化整个世界的生命分布"的接口。世界生成 P3.5 需要传入群系参数场 + 历史事件，获取植物种群 + 动物种群 + 怪物位置 + 亡灵分布的统一返回值。
>
> 内部必须按营养层顺序生成（植物→草食→肉食→顶级捕食者），每层校验食物链承载力——否则会出现"100 只狼在 1 只兔子的 Chunk 里"的生物荒谬。

```rust
pub fn initialize_life_world(
    land_mask: &LandSeaMask,
    biome_params: &BiomeParamField,
    habitat: &dyn HabitatProvider,
    history_events: &EventLog,       // 亡灵需要历史事件中的大规模死亡地点
    fantasy_terrains: &FantasyTerrainMap,
    rng: &mut DeterministicRng,
) -> LifeWorldState;
```

## 二、种群模板格式

> **★ 为什么需要种群模板（而非预生成全部个体）**
>
> 25 万 km² 的陆地上，如果每 Chunk（32m×32m）都预生成每棵植物和每只动物：
> - 植物个体: ~10M-100M → 不可能全量存储
> - 动物个体: ~1M-10M → 不可能全量模拟
>
> 种群模板 = 每 Chunk 存储"这里有什么物种、多少、几岁了"的紧凑描述（~100B/Chunk）。运行时，玩家进入 Chunk 时从模板 + 种子**确定性实例化**个体。离开后种群退回统计描述，由 L3 统计层追踪。
>
> 这个策略和 LOD 架构（L1 全精度→L4 统计）完全一致——只是应用到生命系统。

```rust
pub struct PlantPopulation {     // ~32B
    species_id: PlantSpeciesId,
    coverage: f32,
    average_growth_stage: f32,
    variant_seed: u32,
    health: f32,
    is_crop: bool,
}

pub struct AnimalPopulation {    // ~64B
    species_id: AnimalSpeciesId,
    estimated_count: u16,
    age_distribution: AgeDistribution,
    territory_center: Vec2,
    territory_radius: f32,
    is_migratory: bool,
    migration_route: Option<Vec<Vec2>>,
    seasonal_modifier: Option<SeasonalDensityMod>,
}
```

## 三、动物行为澄清：非 GOAP

> **★ 为什么动物不能用 GOAP**
>
> 1. 动物没有 BigFive 人格、没有 MemoryStore、没有 SelfNarrative——GOAP 的前提条件不成立
> 2. 即时是简化版 GOAP，每只动物每决策周期运行规划器 = 数百万次 GOAP/帧——超出 CPU 预算 7ms
> 3. 动物行为可以用**确定性行为树**高效模拟：HerdGrazing 动物在食物充裕时原地觅食，受惊时沿逃离向量移动，不需要"评估多个目标的效用"

```rust
pub enum AnimalBehaviorModel {
    HerdGrazing, SolitaryPredator, PackHunter,
    Migratory, Territorial, Aerial, Aquatic,
}
```
