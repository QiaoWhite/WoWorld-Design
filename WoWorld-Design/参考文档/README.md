# 参考文档

本文件夹包含 WoWorld 各核心系统的参考性设计文档。每次生成参考文档时，放入一个独立编号的子文件夹中。

## 文件夹命名规范

```
NNN-简短描述-YYYYMMDD
```

| 部分 | 说明 |
|------|------|
| `NNN` | 三位数字序号，从 001 开始递增 |
| `简短描述` | 中文简要说明本批次参考文档的主题 |
| `YYYYMMDD` | 创建日期 |
