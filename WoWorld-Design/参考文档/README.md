# 参考文档

本文件夹包含 WoWorld 各核心系统的参考性设计文档。每次生成参考文档时，放入一个独立编号的子文件夹中。

## 文件夹命名规范

```
NNN-简短描述-YYYYMMDD
```

| 部分 | 说明 |
|------|------|
| `NNN` | 三位数字序号，从 001 开始递增 |
| `简短描述` | 中文简要说明本批次参考文档的主题 |
| `YYYYMMDD` | 创建日期 |

## 文件夹索引

| 编号 | 文件夹 | 日期 | 内容 |
|------|--------|------|------|
| 001 | [核心系统设计参考](001-核心系统设计参考-20260609/) | 2026-06-09 | 涌现叙事、NPC心智、世界生成、全球文化、GOAP决策、体素地形 |

## 子文件夹内的文档

每个子文件夹内的文档按顺序编号（001、002...），独立于文件夹编号。具体索引见各子文件夹的 README。
