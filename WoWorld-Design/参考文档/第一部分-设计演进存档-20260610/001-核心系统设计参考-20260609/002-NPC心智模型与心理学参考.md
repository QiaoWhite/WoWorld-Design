# 002-NPC心智模型与心理学参考

> 对应设计：WoWorld NPC活人感系统——Big Five人格、Plutchik情绪、记忆系统、GOAP决策、社会认知

---

## 一、人格模型参考

### 1.1 Big Five（大五人格）— OCEAN

WoWorld 选择 Big Five 作为 NPC 人格的基础框架。以下是设计维度的详细参考。

| 维度                        | 低分表现 (0.0-0.3)   | 高分表现 (0.7-1.0)    |
| ------------------------- | ---------------- | ----------------- |
| **O**penness 开放性          | 保守、喜欢熟悉的事物、抗拒变化  | 好奇、爱尝试新事物、欣赏艺术和美  |
| **C**onscientiousness 尽责性 | 随性、即兴、可能拖延、不按计划  | 自律、有条理、提前规划、追求成就  |
| **E**xtraversion 外向性      | 喜欢独处、少言寡语、社交消耗能量 | 主动社交、在人群中感到愉悦、健谈  |
| **A**greeableness 宜人性     | 竞争性、怀疑他人动机、不易妥协  | 信任他人、合作优先、富有同情心   |
| **N**euroticism 神经质       | 情绪稳定、快速恢复、不易焦虑   | 敏感、容易焦虑、情绪波动大、恢复慢 |

**设计注记**：
- Big Five 是连续值而非二分法——每个维度 0.0-1.0 的连续值提供了丰富的人格空间
- 人格应影响几乎所有决策：社交风格、职业偏好、战斗风格、消费习惯、政治倾向
- 人格应具有稳定性但也有漂移——重大事件（创伤、成功、失去）可以改变人格参数

**扩展阅读**：
- McCrae & Costa (1999) "A Five-Factor Theory of Personality"
- 游戏中的人格建模：The Sims 3/4 的 Trait 系统、Crusader Kings 的特质系统

### 1.2 互补模型：可考虑融入的特质维度

Big Five 不能完全覆盖所有的行为倾向。以下维度可作为"附加特质"补充：

| 特质 | 说明 | 对 NPC 行为的影响 |
|------|------|-----------------|
| **勇气/胆怯** | 面对危险时的倾向 | 战斗风格、冒险意愿、紧急决策 |
| **诚实/狡诈** | 是否倾向于欺骗 | 交易行为、政治手腕、社交策略 |
| **勤勉/懒惰** | 工作投入度 | 职业发展速度、日常行为节奏 |
| **慷慨/贪婪** | 对财富和资源的态度 | 交易定价、慈善行为、资源分享 |
| **忠贞/多情** | 对关系的专一度 | 恋爱行为、婚姻稳定性 |
| **虔诚/世俗** | 对宗教/超自然的态度 | 仪式参与、道德决策、世界观 |

## 二、情绪系统参考

### 2.1 Plutchik 情绪轮

WoWorld 使用 Plutchik 的情绪八基色模型。以下是设计参考。

```
         optimism
        /        \
   serenity      interest
      |            |
    joy ------- anticipation
      |            |
   acceptance    vigilance
      |            |
   trust ------- fear
      |            |
  submission     terror
      |            |
  apprehension   amazement
      |            |
  sadness ------ surprise
      |            |
   pensiveness  distraction
      |            |
    grief ------ awe
        \        /
        disapproval
```

**八个基本情绪**：Joy（愉悦）、Trust（信任）、Fear（恐惧）、Surprise（惊讶）、Sadness（悲伤）、Disgust（厌恶）、Anger（愤怒）、Anticipation（期待）

**二元对立**：Joy ↔ Sadness / Trust ↔ Disgust / Fear ↔ Anger / Surprise ↔ Anticipation

**设计注记**：
- 基本情绪两两混合产生复合情绪：Joy + Trust = Love / Fear + Surprise = Awe / Sadness + Disgust = Remorse
- NPC 不应只有一个"当前情绪"标签——同时运行多种情绪，每种有独立强度
- 情绪的持续时间应遵循心理学规律：惊吓数秒、愤怒数分钟到数小时、悲伤数天、喜悦因人因事而异

### 2.2 三维情绪轴 (Pleasure-Arousal-Control)

WoWorld 使用 PAD 三维模型作为情绪的底层表示：

| 轴 | 范围 | 含义 |
|----|------|------|
| **P**leasure 愉悦 | -1 ~ +1 | 当前体验是愉快的还是不愉快的 |
| **A**rousal 觉醒 | 0 ~ 1 | 情绪的激活程度（平静→激动） |
| **D**ominance/**C**ontrol 掌控 | -1 ~ +1 | 对当前情境的控制感（无力→掌控） |

**设计建议**：
- PAD 是慢变量——不像具体情绪那样快速波动
- 生理状态持续拉扯 PAD（饥饿→P↓、疲劳→A↓、疾病→P↓D↓）
- PAD 的长期均值可以作为 NPC "生活满意度"的隐性指标

### 2.3 情绪感染

情绪在NPC之间传播——这是"人群氛围"的基础。

**传播规则**：
- 传播范围：视野内的NPC + 同一物理空间
- 传播强度：与关系密切度正相关，与距离负相关
- 衰减因子：每经过一个传播节点，强度衰减 20-40%
- 抵抗因子：高神经质的人更容易被感染（无论是正面还是负面情绪）

**涌现效果**：恐慌在人群中蔓延、欢庆气氛传遍整个市集、一个愤怒的顾客影响整条街的氛围

## 三、记忆系统参考

### 3.1 人类记忆的心理学基础

WoWorld 的记忆系统（2000条上限，含遗忘与重构）的设计灵感来自真实的记忆心理学：

| 现象 | 心理学描述 | 在 WoWorld 中的实现 |
|------|-----------|-------------------|
| **遗忘曲线**（Ebbinghaus）| 记忆随时间指数衰减 | 低冲击力记忆随时间自动压缩为摘要 |
| **记忆重构**（Bartlett）| 每次回忆都在重塑记忆 | 被重新调用的压缩记忆在解压时可能发生偏差 |
| **闪光灯记忆** | 高情绪冲击力的事件记得特别清楚 | 冲击力评分高的事件衰减极慢 |
| **自我辩护偏差** | 人会不自觉地将自己的行为合理化 | 记忆重构时向有利于自己的方向偏移 |
| **情绪一致性** | 更容易记起与当前情绪一致的事 | 记忆检索时情绪匹配加权 |
| **罗生门效应** | 不同人对同一事件有不同记忆 | 每个NPC保留独立的主观记忆版本 |

**推荐阅读**：
- Daniel Schacter《The Seven Sins of Memory》— 记忆的七种"错误"（稍纵即逝、心不在焉、阻塞、错误归因、受暗示性、偏差、持久性）
- Elizabeth Loftus 关于虚假记忆的研究

### 3.2 记忆的分层架构

```
┌─────────────────────────────────────────┐
│  L0: 客观事件事实锚点 (Event Fact Anchor) │
│  存在于全局记忆数据库中                    │
│  去重：同一事件仅存一份                    │
├─────────────────────────────────────────┤
│  L1: 个人主观记忆 (Per-NPC Subjective)    │
│  每个NPC对事件的独立版本                   │
│  包含：情绪编码、归因、视角偏差、压缩状态    │
├─────────────────────────────────────────┤
│  L2: 社会传播记忆 (Socially Transmitted)  │
│  从他人那里听来的记忆                      │
│  包含：来源标记、传谣路线、置信度衰减       │
├─────────────────────────────────────────┤
│  L3: 文化记忆 (Cultural Memory)           │
│  进入传说、歌谣、书本、纪念碑的事件         │
│  极度简化、高度符号化、可能严重失真         │
└─────────────────────────────────────────┘
```

## 四、决策系统参考

### 4.1 日常行为：概率行为树 vs 脚本

人类90%以上的日常行为是**习惯性的、概率性的**——不是每次开门都在做理性规划。

- 概率决策器根据时间、地点、人格、当前需求，从行为库中加权采样
- 行为一旦选定持续执行数分钟——不需要每帧都在"决定做什么"
- 行为中断机制：遇到更高优先级事件时终止当前行为

### 4.2 目标驱动：GOAP 安全网

当日常行为无法满足需求时（饥饿找不到食物、目标受阻），GOAP规划器介入。

**参考实现**：
- F.E.A.R. (2005) — 首个商业GOAP实现，证明了GOAP在实时游戏中的可行性
- Jeff Orkin 的 GOAP 论文 "Applying Goal-Oriented Action Planning to Games"

### 4.3 社会决策：意图识别与预测

社会决策需要"读人"的能力：
- 推测他人的意图（他为什么走向面包店？）
- 预测他人的行为（他接下来可能做什么？）
- 识别欺骗和伪装

**参考**：
- Theory of Mind（心智理论）研究——人类如何理解和推测他人的心理状态
- 博弈论中的"层次思维"（k-level thinking）：我以为他会以为我以为……

## 五、分层模拟的认知基础

WoWorld 的三层 LOD（L1/L2/L3）不仅是性能策略，也是认知合理性的体现：

| 距离 | 人类感知 | 模拟等级 |
|------|---------|---------|
| 近处 (<50m) | 可见细节、可互动 | L1：完整模拟 |
| 中距 (50-150m) | 可见轮廓、模糊感知 | L2：轻量模拟 |
| 远处 (>150m) | 不可见、仅存在于"概念"中 | L3：统计存在 |

**认知哲学基础**：你从来看不到你家三条街外那个陌生人的微表情。但你"知道"他在那里，过着他的生活。当你走近他——他的世界才从概念变为细节。这就是分层模拟的本质。

---

> **对 WoWorld 而言**：NPC的心智模型不是越复杂越好——而是在每一层模拟等级下，其行为都足够"看起来像人"。L3的NPC不需要完整的情绪引擎，但L1的NPC应该让人觉得"他不仅仅是AI，他是一个人"。
