# 004-全球多元文化体系设计参考

> 对应设计：WoWorld "全球多元文化自由融入"与"各文明自行演化独特文化特色"

---

## 一、文化设计的核心挑战

WoWorld 的文化体系面临一个独特的挑战：**既要以现实世界的人类文明为灵感来源，又要让每个世界中的文化是"自己长出来的"而非"开发者放置的"。**

这不是一个"在游戏里塞几个东方建筑、几个中东市集"的问题。这是在设计一个**文化能够自然演化的生态系统**。

---

## 二、文化生成的参数化

### 2.1 文化种子（Culture Seed）

每个文化需要一个"种子"——一组初始参数，决定这个文化的原始倾向。文化种子在聚落选址时被赋予该区域。

```
CultureSeed {
    // 核心参数
    social_structure:   collectivist ←→ individualist   (0-1)
    power_distance:     egalitarian  ←→ hierarchical     (0-1)
    time_orientation:   traditional ←→ progressive       (0-1)
    conflict_style:     diplomatic  ←→ warlike           (0-1)
    spiritual_style:    secular     ←→ devout            (0-1)
    gender_norms:       strict      ←→ fluid             (0-1)
    wealth_attitude:    communal    ←→ accumulative       (0-1)

    // 美学参数（决定建筑、服饰、艺术风格）
    ornament_density:   minimal     ←→ ornate            (0-1)
    color_palette:      earth_tones ←→ vibrant           (0-1)
    form_language:      angular     ←→ curved            (0-1)
    symmetry_preference: asymmetric  ←→ symmetric        (0-1)
    scale_preference:   intimate    ←→ monumental        (0-1)

    // 物质文化
    primary_material:   enum { wood, stone, clay, textile, metal }
    textile_tradition:  enum { woven, knitted, felted, dyed, embroidered }
    cuisine_profile:    enum { grain_based, pastoral, maritime, mixed }
}
```

### 2.2 文化参数与现实文明的映射（仅作灵感参考）

以下映射不是"模板"——不应该有一个"日本文化"预设。这些是灵感：

| 灵感源 | 可能的参数倾向 |
|--------|-------------|
| 东亚（中/日/韩） | 高集体主义、高权力距离、高传统取向、木质建筑传统、对称偏好高、曲线形式语言 |
| 中东/波斯 | 高装饰密度、鲜艳色调、几何花纹、纺织传统强、巴扎（市集）文化 |
| 北欧/凯尔特 | 中等集体主义、平等主义倾向、石材+木材混合、符文/结绳装饰、海/河文化 |
| 非洲（撒哈拉以南） | 多样化的社会结构、部落-王国谱系、口头传统强、鼓乐/面具/图腾、大地色调 |
| 南亚/东南亚 | 高装饰密度、鲜艳色彩、香料烹饪、浮屠/寺庙建筑、复杂的社会等级 |
| 美洲原住民 | 自然崇拜、部落联盟、图腾/羽饰、皮革/羽毛工艺、大地/天空二元哲学 |

**关键原则**：这些参数映射仅作为设计者的灵感参考——在 WoWorld 中，没有预设的"中国文化"或"北欧文化"。一个世界的文化参数是由种子和地理条件共同决定的，其结果可能与现实中的任何文化都不同——这正是我们想要的。

## 三、文化的演化机制

文化不是静态的。它随时间漂移、分化、融合。

### 3.1 文化漂移（Cultural Drift）

- 代际漂移：每一代人对上一代的文化参数有少量随机偏移（±0.02-0.05）
- 环境适应：地理环境缓慢塑造文化参数（如：资源匮乏→集体主义↑、安全环境→个人主义↑）
- 重大事件冲击：战争、瘟疫、移民潮可以在短时间内大幅改变文化参数

### 3.2 文化分化（Cultural Divergence）

当一个文化群体地理隔绝时——比如一部分人翻过山脉去了另一侧——两边的文化参数开始独立漂移。几百年后，同一个祖先文化分化成了两个不同的文化。

### 3.3 文化融合（Cultural Convergence）

当两个文化群体长期接触时（贸易、移民、征服）：
- 高频接触的文化特征更快传播（服装风格、建筑装饰）
- 深层文化特征传播较慢（价值观、宗教）
- 融合结果不是"A+B的简单平均"——可能产生全新的文化形态

**涌现示例**：一个骑士王国征服了一个东方风格的城邦 → 200年后，该地区出现了一种"穿着丝绸、使用长剑、崇拜祖先英灵"的混合文化。这不是开发者设计的——是系统演化出来的。

## 四、文化在游戏世界中的可见表达

### 4.1 建筑与城市

| 文化参数 | 对建筑的影响 |
|---------|------------|
| material=stone + monumental | 高大的石质建筑、拱门、穹顶 |
| material=wood + intimate | 低矮木屋、庭院、回廊 |
| ornament=ornate + symmetric | 严格对称的繁复装饰 |
| ornament=ornate + asymmetric | 巴洛克式的流动装饰 |
| color=earth_tones | 建筑使用当地天然材料的原色 |
| color=vibrant | 彩绘外墙、彩色琉璃瓦 |

### 4.2 服饰与外观

- 服装剪影：宽松 vs 紧身、层叠 vs 单层
- 装饰风格：刺绣 vs 印花 vs 珠宝 vs 素面
- 头饰与发型：冠/帽/头巾/散发/编发/髻
- 颜色偏好：从文化调色板中采样

### 4.3 食物与饮食

由 cuisine_profile 和当地可获取的食材共同决定：
- grain_based → 面包、面条、粥、酿酒
- pastoral → 奶制品、烤肉、发酵乳
- maritime → 鱼类、海藻、盐腌保存

### 4.4 宗教与仪式

- 神祇体系：单神/多神/泛灵/萨满/无神论/祖先崇拜
- 仪式频率与形式：日常祈祷/周期性节日/人生里程碑仪式（出生/成年/婚/死）
- 圣地类型：教堂/寺庙/巨石阵/圣林/圣泉/祖先祠堂

### 4.5 语言（远期）

- 文字系统：字母/音节/象形/表意
- 命名模式：名字的长度、音节结构、常见的名字后缀/前缀
- 语言亲属关系：分化自同一祖先语言的文化之间有相似但不相同的语言

## 五、避免"文化拼贴感"

最大的设计陷阱是："这里是中国区，这里是以色列区，这里是维京区"——这会让世界像主题公园。

**避免方法**：
- 不预设文化的地理分布——让文化种子+地理+历史共同决定
- 文化的边界应该是模糊的、重叠的、渐变的——就像现实中，没有一条线说"过了这条河文化就变了"
- 混合文化（如丝绸之路沿线的文化融合区）应该比纯文化更常见

---

> **对 WoWorld 而言**：全球多元文化不是"内容多样化"的一次性工作——它是一个系统。我们设计的是"文化如何诞生、演化、融合、消亡"的规则，而不是"这个世界有哪些文化"。文化应该是从规则中生长出来的结果，而不是输入。
