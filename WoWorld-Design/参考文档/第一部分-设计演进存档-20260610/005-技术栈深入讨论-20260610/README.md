## 核心立场

经过深入分析，**方案 A（Rust 模拟核心 + Godot 游戏客户端）是当前最务实的选择**。

- NPC 模拟的性能瓶颈由 Rust 解决（这是唯一不可协商的需求）
- Godot 保留它最擅长的东西：编辑器、UI、动画、音频、物理
- Bevy 是更好的长期选择——但 2026 年的它还不够成熟
- 架构设计使 Rust 模拟核心"不依赖 Godot"，未来可以在 Bevy 足够成熟时迁移
