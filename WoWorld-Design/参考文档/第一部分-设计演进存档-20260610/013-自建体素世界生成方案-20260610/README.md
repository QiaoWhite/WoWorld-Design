## 核心创新

**分层密度场**——基础地形层 + 文化覆盖层 + NPC修改层 + 玩家修改层。存档只存修改层，从种子可重现基础层。这是 WoWorld 独有的需求，现成引擎无法提供。

**Desire Path 道路形成**——NPC 频繁行走自动踩出土径。Trail SDF 操作累积深度，100次通过=可见凹陷，1000次=清晰道路。
