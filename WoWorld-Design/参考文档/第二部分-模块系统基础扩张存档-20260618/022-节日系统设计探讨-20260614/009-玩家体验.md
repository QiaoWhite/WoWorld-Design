## 一、玩家如何发现节日

### 1.1 发现层级——从远到近

| 距离 | 线索 | 玩家感知 |
|------|------|---------|
| **远距离**（进入区域前） | 旅行者 NPC 在对话中提及——"你是去参加河湾镇的收获节吗？" | 玩家得知远处有节日 |
| **中距离**（步行 5-10 分钟） | 聚落方向有烟雾/灯火（如果夜间）——L3 视觉提示 | 玩家看到远处的节日活动 |
| **近距离**（进入聚落） | 街道装饰（灯笼/旗帜）、NPC 穿 Ceremonial 服装、人群聚集、音乐/钟声（音频） | 玩家明确感知——这里有节日 |
| **对话**（与 NPC 交谈） | NPC 提及节日——"来得正好！今天是一年一度的龙王祭！" | 玩家了解节日详情 |

### 1.2 UI 辅助

| UI 元素 | 内容 |
|---------|------|
| **布告板** | 聚落布告板贴有节日通告——玩家阅读后日志记录 |
| **日志** | 如果玩家与聚落有关联 → 日志自动标记"X镇将在5天后举行收获节" |
| **地图** | 玩家探索过的区域 → 节日标记（可选——取决于玩家是否购买了"历法"知识） |
| **HUD** | 进入聚落时短暂浮动提示——"河湾镇——收获节第 2 天" |

### 1.3 无意错过

重要设计决策：**玩家可以完全错过节日**。如果玩家在冬天穿越一个夏天庆祝节日的地区——他们不会看到节日。这增加了世界的真实感——世界不围绕玩家转动。但如果玩家反复经过该地区 → NPC 可能在对话中回忆去年的节日（"你去年的收获节真应该来看看！"）。

## 二、玩家在节日中的角色

### 2.1 六个参与层次

| 层次 | 玩家行为 | 后果 |
|------|---------|------|
| **1. 未察觉** | 玩家根本不知道有节日 | 无——世界继续运转 |
| **2. 旁观** | 玩家看到装饰/人群但选择不参与 | 无直接后果。NPC 可能评论玩家的冷漠 |
| **3. 外围参与** | 参加宴饮、逛集市、买节日物品 | 经济消费 + 温和的 NPC 关系提升 |
| **4. 主动参与** | 参加舞蹈、赠礼、与 NPC 社交 | 关系显著提升。玩家被视为"参与了我们的节日" |
| **5. 竞技** | 报名参加竞技——与 NPC 比赛 | 获胜 → 名声事件（荣誉/勇武提升）。失败 → 无负面（"光荣的失败"） |
| **6. 干预** | 影响节日的进程——做演讲、捣乱、拯救被事故打断的节日 | 重大后果——名声显著改变、文化偏移催化剂 |

### 2.2 玩家能做什么（具体操作列表）

```
旁观类：
  ✓ 观看仪式/竞技/表演
  ✓ 与参加节日的 NPC 交谈（专用对话选项）
  ✓ 阅读布告板上的节日信息

参与类：
  ✓ 跳舞（QTE 小游戏——跟着节奏按键）
  ✓ 宴饮（购买/食用节日食物——获得临时 buff）
  ✓ 赠礼（向 NPC 赠送节日礼物——关系提升）
  ✓ 祈福（在宗教节日中——如果玩家与该信仰有联系）

竞技类：
  ✓ 报名竞技（如果节日包含 Tournament/Race/Contest）
  ✓ 与 NPC 比赛
  ✓ 下赌注（如果该文化允许博彩）

经济类：
  ✓ 在节日集市买卖（利用价格波动赚钱）
  ✓ 接受节日相关的小任务（"帮我送这份礼物给XXX"）

干预类：
  ✓ 发表演讲（需要足够的声望/关系）
  ✓ 救场（如：竞技中有人受伤→玩家施救→名声事件）
  ✓ 捣乱（偷窃节日物品/破坏仪式/打架）→法律后果
```

## 三、节日对玩家的约束

### 3.1 "节日规则"

节日期间当地法律和规范可能改变——但文化对玩家的约束不是硬性的（除非有权力强制）：

| 约束 | 是否有硬性后果 | 说明 |
|------|-------------|------|
| 强制参加 | 否（对玩家） | 玩家是外来者——不受本地义务约束。但拒绝可能被视为"不尊重" |
| 节日期间禁止战斗 | 视文化而定 | `militarism < 0.3` + `uncertainty_avoidance > 0.6` → 违反是严重罪行 |
| 节日期间禁止工作 | 否（对玩家） | 大多数商店关闭——玩家无法买卖 |
| 饮食限制（斋戒） | 否（对玩家） | 但违反可能冒犯本地 NPC |
| 服装要求 | 否（对玩家） | 但穿日常衣服去正式节日 → NPC 反应（轻蔑/好奇） |

### 3.2 玩家的外来者优势

作为外来者，玩家享有一些灵活性：
- 不必遵守所有本地义务
- 可以提出"愚蠢的问题"（"这节日是干什么的？"）而不被视为无礼
- 可以打破禁忌——但后果自负

## 四、玩家创立节日

### 4.1 条件

玩家不能直接"创立节日"——你不是神，不能告诉一群人"从今年起你们每年这个日子必须庆祝"。但你可以通过以下途径间接触发节日的创建：

| 途径 | 触发条件 | 等待时间 |
|------|---------|---------|
| **完成史诗任务** | 拯救了城市/杀死了龙/发现了重要古代遗迹 → 当地文化可能建立纪念性节日 | 需 3 年自发纪念行为 → 系统检测并正式化 |
| **长期赞助** | 连续 5 年资助某聚落的特定活动 → 可能在玩家离开后持续 → 变为年度传统 | 5+ 年 |
| **成为统治者** | 如果玩家成为聚落/政体的统治者 → 可以使用权力系统创立新的国家节日 | 立即（但需合法性支持） |

### 4.2 不给玩家"创立节日"按钮

玩家不应该有一个 UI 按钮来"创建节日"。节日必须从**有意义的行为**中自然涌现——玩家杀了龙 → 当地居民自发纪念 → 3 年后系统检测到重复模式 → 正式化。玩家可能永远不会知道自己的行为创造了一个节日——直到某年回来时发现 NPC 在庆祝"屠龙日"。

## 五、跨文化节日体验

### 5.1 玩家参加异文化节日

当玩家的文化与举办节日的文化不同时：

```
文化距离小（similarity > 0.6）：
  → 节日感觉"不同但可以理解"——类似地球上一个欧洲人参加另一个欧洲国家的节日
  → NPC 对玩家态度：好奇/友好
  
文化距离中（similarity 0.3-0.6）：
  → 节日感觉"新奇"——有些元素可以理解，有些完全陌生
  → NPC 对玩家态度：好奇/谨慎
  
文化距离大（similarity < 0.3）：
  → 节日感觉"完全陌生甚至怪异"——仪式/食物/行为完全出乎预料
  → NPC 对玩家态度：怀疑/好奇/冷漠
  → 可能触发文化误解事件（玩家做了什么在本地文化中被视为冒犯的事）
```

### 5.2 节日的"翻译"

当玩家从 NPC A 的文化视角听说文化 B 的节日时——NPC A 会使用 A 文化对 B 节日的**他称（exonym）**：

```
文化 A（航海文化）的 NPC 说：
  "下周是山民的'龙王祭'——他们相信这天龙会从山顶飞下来。"
  ↑ "龙王祭"是 A 文化给 B 文化节日取的 exonym
  
文化 B（山地文化）的 NPC 说：
  "明天是'苍翼之日'——最重要的节日。"
  ↑ "苍翼之日"是 B 文化自己的 endonym
```

这与地名系统的 endonym/exonym 模型一致——节日的命名差异是地名系统在文化领域中的自然延伸。
