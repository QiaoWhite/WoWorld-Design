# 006-音乐三层模型（大纲）

> **开发代号**: WoWorld (Wonder World)
> **状态**: 设计草稿（大纲·要点）

---

## 一、Layer 1: 系统驱动动态音乐

- MusicContext 枚举：Exploration/Combat/Settlement/Interior/Underground/Aerial/Underwater/Festival/Narrative/Menu
- 4 层分层（基础/节奏/和声/旋律），每层独立音量，根据威胁/氛围动态混合
- crossfade_transition 对标天气 transition_blend
- 音乐状态机：情境切换→交叉淡入淡出(2-3s)→目标状态

## 二、Layer 2: 世内 NPC 演奏

- NPC FineArts.music_performance 技能驱动
- 酒馆乐队/街头艺人/宫廷乐团/宗教合唱
- 进入可闻范围→世内演奏覆盖 BGM（交叉淡入淡出）
- 演奏结束→BGM 逐渐淡回
- NPC 演奏 = 其他 NPC 的 AestheticSignal（通过审美模块）

## 三、Layer 3: 玩家自定义音乐

- PlayerMusicConfig：文件夹/播放列表 m3u，支持 mp3/flac/ogg/wav
- 覆盖规则：Never / ExplorationOnly(default) / ExceptCombat / Always
- 音量混合比：玩家音乐 vs 游戏音乐

## 四、音乐源枚举

- System { preset } / InWorldPerformer { performer, performance } / InWorldDevice { device } / PlayerMusic { track }
- 活跃音乐源最多 3 个混合

## 五、配置

- music_config.toml：情境→预设映射、过渡时长、覆盖规则
