## 核心讨论：NPC 如何"感知"他人的内心状态？

### 已有通道

| 通道 | 机制 | 距离 |
|------|------|------|
| 面部表情 | `NonVerbalPerception`→FacialExpression 7种 | <3m |
| 手势 | `NonVerbalPerception`→Gesture 10种 | <10m |
| 姿态 | `NonVerbalPerception`→PostureShift 7种 | <30m |
| 视线方向 | `NonVerbalPerception`→GazeAction 5种 | 中距 |
| 空间行为 | `NonVerbalPerception`→ProxemicShift 4种 | 近距离 |
| 触觉 | `NonVerbalPerception`→Touch 5种 | 极近 |
| 情绪传染 | 情绪传染引擎 8因子 | <15m |
| 语言内容 | `CurrentSpeech`→`AudioQuery::perceived_speech()`→`resolve_partial()` | 距离+清晰度 |

### 关键设计：情绪→身体姿态的信号泄露

```rust
fn emotional_pose_offset(emotion: &EmotionState) -> PoseOffset {
    // 情绪→可观测姿态的映射
    // 这些信号"泄露"出NPC的内心状态→被其他NPC的NonVerbalPerception捕获
    head_tilt:   sadness * -5.0,        // 悲伤→低头
    shoulder:    fear * 3.0,             // 恐惧→蜷缩
    chest:       joy * 2.0,              // 喜悦→挺胸
    gaze:        disgust*0.5 + fear*0.3, // 厌恶/恐惧→避开视线
    fist:        anger * 4.0,            // 愤怒→握拳
    distance:    trust*-0.3 + fear*0.5,  // 不信任/恐惧→保持距离
}
```

**文化面具**：`CommunicationNorms.emotional_expressiveness` 调制泄露程度。压抑文化→愤怒不握拳→信号衰减 50%。观察者的解读也会被共情能力+关系调制。

### 关键设计：社会性共同注意——涌现，不特殊编码

NPC A 抬头看天。B 感知到 A 的视线向上（NonVerbalPerception）→好奇心→射线朝上→发现天上的龙。没有"共同注意系统"——从已有组件涌现。

### 关键设计：流言感知——定居点信息池

不从数据库直接查询声望。L2 NPC 在定居点内时→`SettlementInformationPool` 懒采样（每游戏日更新，对标 `SettlementFaithSnapshot` 模式）。听到流言=概率事件×openness。听到的内容可能有版本冲突、夸张化、过时信息。
