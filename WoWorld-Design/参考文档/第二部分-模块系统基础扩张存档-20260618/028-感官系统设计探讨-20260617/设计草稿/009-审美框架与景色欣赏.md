## 核心讨论：谁决定"景色美不美"？

### 问题

NPC 能否欣赏壮丽景色（日落、山脉、大海）？如果"美"由文化定义——但每个文化的定义不同——这怎么表达？

### 关键决策：感官不定义美，文化从核心参数涌现审美基线

**否定的方案**：枚举所有自然景观类型（`twilight_valence`、`mountain_valence`、`ocean_valence`...）。否决理由——枚举陷阱，明天就有人要加 `fog_valence`。

**采纳的方案**：不新增任何字段。文化对自然景观的审美全部从已有 `CultureCoreParams`（10个参数）+ `CultureHistory`（历史事件数据）涌现。

**派生公式**：
```
cultural_twilight_response =
    + history.event_valence_at_time_of_day(Dusk) × 0.3  // 历史上黄昏发生过坏事？
    - uncertainty_avoidance × 0.15                         // 不确定→害怕变化
    + (long_term_orientation - 0.5) × 0.2                 // 长期导向:结束=新开始
    + artistry × 0.3                                       // 艺术文化→欣赏色彩
    + indulgence × 0.2                                     // 享乐→享受感官体验
    - militancy × 0.25                                     // 军事→红色=血腥
```

### 关键决策：个体通过框架竞争超越文化基线

**否定的方案**：文化基线永远在底部，个人经历只能在上面加偏置（±0.5 上限）。否决理由——限制了个体的自主性。

**采纳的方案**：AestheticFrameworks = native（文化初始）+ personal（经历修改）+ adopted（跨文化采纳）。四独立过程：

1. **获取**：intelligence + openness 门槛。低门槛→容易学。高门槛→学不到。但已学到的框架不受影响。
2. **深化**：当前 openness 调制。开放性降低→深化变慢，但不停止。
3. **衰减**：仅受 memory 影响。深度永不完全归零——残留痕迹（depth_min=0.01）。
4. **激活**：当前 cognitive_clarity + 社会语境。醉酒→退回到 native。社会压力→文化框架激活。

**无硬上限**：能学几个学几个——数量 = learn_rate/decay_rate 的自然均衡。

### 关键决策：同一时刻多框架加权混合——不是单选

**否定的方案**：只选一个框架（当前最相关/权重最高的）。

**采纳的方案**：所有相关框架加权混合。在北境的精灵聚集区看夕阳：personal(-0.18×0.50) + native(-0.65×0.10) + 南方(+0.70×0.25) + 精灵(+0.40×0.15) = +0.08。混合结果→微弱的正面。文化在拉扯，但个人体验和南方影响略胜。→ 情绪: 复杂——"挺美的，但总觉得不太对劲"。

### 关键决策：天际线采样——奇观涌现，不预标记

**否定的方案**：WorldGen 预标记"这里有奇观"→感官查询→NPC看到奇观。否决理由——违反程序化生成原则。

**采纳的方案**：天际线剖面查询 `SpatialQuery::sample_horizon()`。从高度图中涌现：天际线突然在某方向显著抬高→"那边有座山"。同一座山，从平原看→仰角大→奇观✓。从山腰看→仰角小→不触发✓。世界生成不需要标记任何东西。

### 关键决策：空间氛围——感官直通情绪，不经审美

空间属性（封闭/开阔/拥挤/寂静）直接调制情绪——不经审美判断，不经信念推导。这是生理性的。对标已有的"血腥气味→恐惧"的嗅觉→情绪硬编码映射：

```
封闭+暗 → 压抑/恐惧（幽闭恐惧的生理基础）
开阔+高 → 壮阔/眩晕（恐高/敬畏）
拥挤+高噪音 → 焦虑（感官过载）
安静+孤立 → 不安 或 平和（取决于人格）
```
