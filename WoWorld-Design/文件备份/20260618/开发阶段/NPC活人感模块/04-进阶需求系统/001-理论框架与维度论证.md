# 001 — 理论框架与维度论证

## 一、现有系统回顾：我们已经有什么？

### 1.1 基本需求系统（7维，deficit-driven）

| 维度 | 类型 | 存储 | 进GOAP？ |
|------|------|------|----------|
| hunger | 生理/消耗型 | Life.Vitals | ✅ |
| thirst | 生理/消耗型 | Life.Vitals | ✅ |
| fatigue | 生理/累积型 | Life.Vitals | ✅ |
| health | 生理/阈值型 | Life.Vitals | ✅ |
| element_balance | 生理/累积型 | Life.Vitals | ❌ |
| libido | 生理/周期型 | Life.Vitals | ❌ |
| social_deficit | **心理**/缺席型 | NPC.NpcData | ❌ |

关键观察：social_deficit 已经在7维框架内，但它是**唯一的心理维度**。它代表"社交连接的量"，但不代表"社交连接带来的认可"或"社交地位的质"。

### 1.2 SelfNarrative 引擎（已有，aspiration-driven）

- `IntrinsicGoal` 6 种：ExploreUnknown / MasterSkill / ActOnValue / CreateSomething / DeepenConnection / LeaveLegacy
- `SelfNarrative`：SelfConcept / LifeChapters / CoreValues / Aspirations / StagnationSense / LifeTheme
- 反思周期：7 游戏天
- `intrinsic_motivation_weight`：存在于概率权重链中，但**公式未详述**

关键观察：SelfNarrative 已覆盖 SDT 的三大核心需求（自主/胜任/归属），但它们以"内在目标"形式存在，而非"需求维度"。这意味着：
- 它们没有 urgency 信号
- 它们不与基本需求在同一个公式框架中竞争
- 它们与基本需求之间没有抑制/促进的正式关系

### 1.3 其他相关系统

| 系统 | 已有与"进阶需求"相关的部分 |
|------|--------------------------|
| 技能系统 | 技能等级、aspiration.target_level、innate_aptitude |
| 权力系统 | PowerTopology、Legitimacy、地位梯度 |
| 文化系统 | honor_weight、CommunicationNorms |
| 经济系统 | 财富展示、奢侈品消费 |
| 信仰系统 | 宗教认可（Divine authorization） |
| 语言表达 | 对话中的赞美/批评/社交信号 |
| 战斗系统 | 战斗荣誉、胜利认可 |

## 二、理论参照系

### 2.1 马斯洛需求层次

```
自我实现 (Self-Actualization)
    ↑
尊重需求 (Esteem) — 内在自尊 + 外在认可
    ↑
归属与爱 (Belonging/Love) — 现有 social_deficit 覆盖
    ↑
安全需求 (Safety) — 现有 health/combat_threat 覆盖
    ↑
生理需求 (Physiological) — 现有 hunger/thirst/fatigue 覆盖
```

WoWorld 现状：
- 生理 ✅（4维）
- 安全 ⚠️（health + combat_threat，但缺少"财产安全""职业安全"等——这些由其他系统覆盖，不需要做需求维度）
- 归属 ⚠️（social_deficit 覆盖了"量"，缺少"质"和"认可"）
- 尊重 ❌（完全缺失！esteem 不在现有任何需求维度中）
- 自我实现 ⚠️（SelfNarrative/IntrinsicGoal 已覆盖，但不是 deficit 模型）

**核心洞见**：尊重/认可是马斯洛金字塔中**唯一完全缺失**的层次。它处于归属之上、自我实现之下——是连接"社会需求"和"成长需求"的桥梁。

### 2.2 ERG 理论

三种核心需求：
- **Existence**（生存）→ 已有 hunger/thirst/fatigue/health/element_balance
- **Relatedness**（关系）→ 已有 social_deficit，**缺 esteem/recognition**
- **Growth**（成长）→ 已有 IntrinsicGoal/SelfNarrative

ERG 的两个关键洞见：
1. **挫折-回归**：成长需求受挫 → 向下回归到关系需求（更强烈地寻求 social/esteem）
2. **满足-前进**：关系需求满足 → 向上追求成长需求

这两个机制在 WoWorld 中尚未被建模。

### 2.3 自我决定理论 (SDT)

三种普适基本心理需求：
- **自主** (Autonomy) → 已有 IntrinsicGoal::ActOnValue
- **胜任** (Competence) → 已有 IntrinsicGoal::MasterSkill + 技能系统
- **归属** (Relatedness) → 已有 social_deficit + IntrinsicGoal::DeepenConnection

SDT 的核心主张：这三种需求是**平行且同等基础**的——不是层次化的。一个人可以在饥饿时仍然追求自主。这支持我们的"软抑制而非硬层次"设计。

但 SDT 也指出：当胜任需求长期受挫时，内在动机会崩溃。这正是我们要用 `competence_frustration` 建模的。

### 2.4 赫兹伯格双因素理论

- **保健因素**：不满足 → 不满；满足 → 不不满（但不激励）
- **激励因素**：不满足 → 不不满；满足 → 主动激励

对应到 WoWorld：
- 保健因素 ≈ 基本需求（满足后消除不适，但不驱动"追求意义"）
- 激励因素 ≈ 成长驱力（IntrinsicGoal——驱动"做得更好"而非"消除不适"）

这支持我们区分 deficit 维度和 aspiration 维度——它们是不同的数学对象。

## 三、候选维度评估

### 3.1 评估矩阵

对每个候选维度，用以下标准评估：

| 标准 | 权重 | 说明 |
|------|------|------|
| **Deficit 适配** | 必过 | 是否适合"消耗-恢复"模型？不适配的维度应走其他路径 |
| **现有覆盖度** | — | 已有系统是否已处理？避免重复 |
| **行为多样性** | 关键 | 加了这个维度后，不同 NPC 的行为差异有多大？ |
| **跨系统交互** | 关键 | 是否桥接多个已有系统？ |
| **玩家可感知** | 重要 | 玩家能否从 NPC 行为中感受到这个维度？ |
| **性能成本** | — | 内存 + CPU 增量 |

### 3.2 逐项评估

#### 尊重/认可 (Esteem/Recognition)

- **Deficit 适配**：✅ 认可会"代谢"——昨天被夸≠今天被认可。需要持续的认可供给
- **现有覆盖度**：❌ 未覆盖。social_deficit 是"有人陪伴"，不是"被尊重"
- **行为多样性**：⭐⭐⭐⭐⭐ 极高——寻求认可的路径极其多样（战斗/创作/社交/财富/权力/信仰），且人格差异巨大
- **跨系统交互**：⭐⭐⭐⭐⭐ 桥接经济/权力/文化/战斗/信仰/语言表达 6+ 系统
- **玩家可感知**：⭐⭐⭐⭐⭐ 玩家能直观看到"骄傲的矮人""虚荣的贵族""淡泊的隐士"
- **性能成本**：1 f32/NPC ≈ 4 bytes
- **结论**：✅ **强烈推荐添加**

#### 胜任挫折 (Competence Frustration)

- **Deficit 适配**：⚠️ 非纯粹 deficit——设定点是动态的（aspiration 变化时移动）。属于 **allostatic** 模型
- **现有覆盖度**：❌ 未覆盖。技能系统有 gap 数据但无人消费。stagnation_sense 7天才更新一次，太慢
- **行为多样性**：⭐⭐⭐⭐ 高——挫败的学徒可能加倍努力/换方向/放弃/酗酒/嫉妒他人
- **跨系统交互**：⭐⭐⭐ 桥接技能/情绪/SelfNarrative/社交（嫉妒）
- **玩家可感知**：⭐⭐⭐⭐ 玩家能看到"遇到瓶颈的铁匠""放弃学魔法的学徒"
- **性能成本**：1 f32/NPC + 每天1次重新计算
- **结论**：✅ **推荐添加**，使用 allostatic 模型

#### 审美需求 (Aesthetic)

- **Deficit 适配**：❌ 审美不是消耗型资源——不是"看了美的东西就用完了"
- **现有覆盖度**：⚠️ CulturalBeautyStandard 已存在，BuildingStylePreferences 已处理建筑审美
- **行为多样性**：⭐⭐ 低——在中世纪设定中，只有极小部分 NPC（艺术家/贵族）会表现出审美驱动的行为
- **跨系统交互**：⭐ 弱
- **结论**：❌ 不添加。更适合作为文化参数，而非个人需求维度

#### 认知/好奇心 (Cognitive/Curiosity)

- **Deficit 适配**：❌ 好奇心是 trait-like 的，不是消耗型资源
- **现有覆盖度**：✅ IntrinsicGoal::ExploreUnknown 已处理
- **结论**：❌ 不添加。已有 IntrinsicGoal 机制

#### 自主 (Autonomy)

- **Deficit 适配**：❌ 自主不是"消耗-恢复"循环——做出自主选择后不会"用掉"自主感
- **现有覆盖度**：✅ IntrinsicGoal::ActOnValue + CoreValues 约束
- **结论**：❌ 不添加。已有 IntrinsicGoal + SelfConcept 机制

#### 超越/传承 (Transcendence/Legacy)

- **Deficit 适配**：❌ 传承是 LifeTheme 驱动的方向性驱力，不是匮乏信号
- **现有覆盖度**：✅ IntrinsicGoal::LeaveLegacy + LifeTheme::Legacy
- **结论**：❌ 不添加。已有 SelfNarrative 机制

## 四、最终维度选择：2+2 模型

### 4.1 为什么是"两个新维度"？

1. **尊重/认可**：马斯洛金字塔中唯一完全缺失的层次。Estem 的 deficit 动态天然适合 urgency 公式
2. **胜任挫折**：技能系统与心理状态的缺失桥梁。Allostatic 模型优雅处理动态设定点

### 4.2 为什么不更多？

- 审美/认知/自主/超越 → 已被 IntrinsicGoal 或文化系统覆盖
- 每加一个维度必须有**不可替代的行为多样性价值**
- YAGNI：在 WoWorld 当前阶段，2 个精心设计的维度比 5 个粗糙维度更好

### 4.3 为什么是"两个跨层机制"？

- **生存抑制**：防止"濒死 NPC 还在写诗"的荒谬涌现
- **挫折回归**：ERG 理论的核心机制，创造心理深度（中年危机、转行、酗酒）

### 4.4 与 IntrinsicGoal 的关系

- IntrinsicGoal **不纳入** deficit 公式——保持其"志向"本质
- 但**形式化**其 consumption pathway（commitment × relevance → 偏好偏置）
- 使 IntrinsicGoal 和 9 维需求在同一个概率权重链中**平等竞争**
