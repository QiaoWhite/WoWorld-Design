## 模块概述

信仰系统管理 WoWorld 中一切与"超自然信念与崇拜实践"相关的数据与逻辑——从全球宗教的种子生成到个体 NPC 的祈祷行为，从教会层级到圣物祝福。

**核心设计**：实践优先模型。NPC 没有"神学立场"——他们有参与实践（去不去庙、拜不拜神）。神学标签从实践派生，不驱动实践。

**原则**：
1. **有人的地方才有信仰**——信仰在 NPC 心中，不从空间地图独立存在
2. **信仰系统 = 条件与修饰**——所有执行走现有系统（权力/Group/CulturalNorm/战斗/经济）
3. **热路径零跨模块调用**——NPC 信仰数据本地存储（~44B/NPC）
4. **DivineMode 透明**——A/B/C 三种神明存在模式下，NPC 信仰行为完全相同
