## 〇、总览

`VisionParams` 定义每个 NPC 的视觉感知参数。对标音频系统的 `HearingModel`——由感官 crate 定义 struct，Life crate 负责从种族+属性+年龄+人格初始化派生，NPC crate 存储于 `SensoryState.vision`。

---

## 一、VisionParams 结构

```rust
pub struct VisionParams {
    pub fov_horizontal_deg: f32,
    pub fov_vertical_deg: f32,
    pub max_range_m: f32,
    pub acuity_arcmin: f32,
    pub dark_adaptation_speed: f32,
    pub color_vision: ColorVision,
    pub motion_sensitivity: f32,
    pub vision_type: VisionType,
}

pub enum ColorVision { Monochrome, Dichrome, Trichrome, Tetrachrome }
pub enum VisionType { Standard, LowLight, Thermal, BlindSight, TremorSense }
```

| 字段 | 人类默认 | 说明 |
|------|---------|------|
| `fov_horizontal_deg` | 180 | 水平视场角。猎物动物340° |
| `fov_vertical_deg` | 120 | 垂直视场角 |
| `max_range_m` | 500 | 晴朗大气最远可见距离 |
| `acuity_arcmin` | 1.0 | 角分辨率。1.0=20/20。鹰0.3 |
| `dark_adaptation_speed` | 1.0 | 基准速率乘数。精灵1.5 |
| `color_vision` | Trichrome | RGB三色。狗Dichrome(蓝黄) |
| `motion_sensitivity` | 0.5 | 对外周运动的敏感度 |
| `vision_type` | Standard | 低光视觉(LowLight)穿透黑暗；热视觉(Thermal)看热源；震颤感知(TremorSense)感知地面震动 |

## 二、派生公式

```rust
impl VisionParams {
    pub fn from_race(
        race: &RaceTraits,
        perception: f32,      // PhysicalAttributes.perception (0-1)
        age_years: f32,
        personality: &BigFive,
    ) -> Self {
        let base = race.base_vision();  // 种族基线——从 RaceTraits 获取

        // perception 调制
        let range_mod = 0.5 + perception * 0.5;  // 0.5 ~ 1.0
        let acuity_mod = 2.0 - perception * 1.0;  // 2.0 ~ 1.0 (arcmin越小越好)
        let motion_mod = 0.3 + perception * 0.7;  // 0.3 ~ 1.0

        // 年龄衰减（40岁后开始）
        let age_factor = if age_years > 40.0 {
            1.0 - (age_years - 40.0) * 0.005  // 每年衰减0.5%
        } else { 1.0 };

        // 开放性调制外周敏感度
        let openness_bonus = (personality.openness - 0.5) * 0.2;  // -0.1 ~ +0.1

        Self {
            fov_horizontal_deg: base.fov_horizontal_deg,
            fov_vertical_deg: base.fov_vertical_deg,
            max_range_m: base.max_range_m * range_mod * age_factor.clamp(0.3, 1.0),
            acuity_arcmin: base.acuity_arcmin * acuity_mod * (2.0 - age_factor).clamp(1.0, 2.5),
            dark_adaptation_speed: base.dark_adaptation_speed,
            color_vision: base.color_vision,
            motion_sensitivity: (base.motion_sensitivity * motion_mod + openness_bonus).clamp(0.1, 1.0),
            vision_type: base.vision_type,
        }
    }
}
```

## 三、种族视觉差异示例

| 种族 | FOV H | Max Range | Acuity | Color | VisionType | 说明 |
|------|-------|-----------|--------|-------|------------|------|
| 人类 | 180° | 500m | 1.0 | Trichrome | Standard | 基准 |
| 精灵 | 190° | 650m | 0.7 | Trichrome | LowLight | 耳尖视觉+暗视 |
| 矮人 | 170° | 400m | 1.2 | Trichrome | LowLight | 地下适应 |
| 鹰身人 | 220° | 2000m | 0.3 | Tetrachrome | Standard | 猛禽视觉 |
| 狼人 | 200° | 350m | 1.5 | Dichrome | LowLight | 犬类视觉 |
| 龙 | 300° | 5000m | 0.2 | Tetrachrome | Thermal | 顶级掠食者 |
| 史莱姆 | 360° | 20m | 5.0 | Monochrome | TremorSense | 无眼——振动感知 |
| 蝙蝠族 | 180° | 100m | 3.0 | Monochrome | BlindSight | 回声定位 |

## 四、状态效果对视力的调制

通过 `PerceptualModifiers`：

| 状态 | vision_range_mult | noise_mult | 其他 |
|------|------------------|------------|------|
| 醉酒 | 0.7 | 1.5 | motion_sensitivity -0.2 |
| 致盲 | 0.0 | — | 全黑 |
| 夜视药水 | 1.0 | — | vision_type 临时→LowLight |
| 恐吓 | 0.85 | 1.2 | threat_bias +0.3（威胁误判） |
| 极度疲劳 | 0.8 | 1.3 | motion_sensitivity -0.15 |
| 兴奋剂 | 1.2 | 1.0 | motion_sensitivity +0.2 |
