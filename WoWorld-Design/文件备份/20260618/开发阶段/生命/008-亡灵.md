# 008-亡灵

## 一、定义

亡灵 = 灵元素异常残留。身体已死，灵元素未散干净——或被外力拴在世界上。亡灵**不是"活着"的**——灵元素在持续衰减，除非有外力补充（攻击活人汲取灵元素）。

```rust
struct Undead {
    /// 生前记录——它曾是哪种生命
    former_life: LifeRecord,
    /// 残留灵元素——持续衰减中
    lingering_spirit: f32,       // 1.0(完整魂魄)→0.0(彻底消散=真正的死亡)
    /// 绑定力——什么力量把它拴在世界上（可有多个叠加）
    binding_forces: Vec<BindingForce>,
    /// 衰减速率(每天失去的spirit量)
    decay_rate: f32,
}
```

## 二、7种绑定来源——可叠加

| 来源 | 说明 | 产生条件 | 亡灵类型 |
|------|------|---------|---------|
| **怨恨** (Grudge) | 强烈的未了情感将灵元素拴住 | 横死(谋杀/冤案/战争/饥荒) | 怨魂、复仇亡灵、无头骑士 |
| **诅咒** (Cursed) | 被主动施加——死者或生者被诅咒不得安息 | 死灵法师/复仇者施加诅咒 | 缚灵、僵尸(被诅咒的尸体) |
| **死灵法术** (Necromantic) | 被魔法强制驱动——不是主动的"意愿" | 死灵法师施法 | 骷髅士兵/奴隶、亡灵傀儡 |
| **环境自然产生** (AmbientMagic) | 高魔力浓度区+尸体→自动"醒来" | 魔力浓度≥0.5+区域内存在尸体 | 僵尸(自然)、动物亡灵 |
| **神圣意志** (Divine) | 神明让死者不得安息——作为惩罚或守护 | 仅A模式——神明的主动行为 | 守护灵、殉道者、受罚者 |
| **自愿** (Voluntary) | 生前主动选择不死——通过特定仪式保留灵元素 | 巫妖化仪式/亡灵化魔法——需高魔力+知识 | 巫妖(极高智能)、亡灵领主 |
| **传染** (Infected) | 被已有亡灵杀死→灵元素被强行拴住→成为新亡灵 | 被亡灵杀死 | 僵尸(传染型)、幽灵(传染型) |

**叠加规则**——同一个亡灵可有多个绑定力。三重来源=远比单来源强。

## 三、来源权重由世界状态驱动

亡灵的产生不是"随机决定有几个骷髅"——而是由**当地世界状态**动态决定：

```
local_grudge_weight      = 横死率——战争/谋杀/瘟疫区域→怨魂多
local_cursed_weight      = 被诅咒者密度——死灵法师活动/复仇文化
local_necromantic_weight = 死灵法师数量——有亡灵法师的聚落附近→骷髅多
local_ambient_weight     = 魔力浓度——魔法群系/古战场→自然亡灵多
local_divine_weight      = divine_mode == A ——仅A模式生效——神明诅咒/惩罚
local_voluntary_weight   = 当地文化是否崇尚永生——某些文化的精英选择巫妖化
local_infected_weight    = 已有亡灵密度——传播链——亡灵越多的地区越容易产生更多亡灵

→ 古战场: 怨恨(横死率极高)+环境(尸体集中+魔力浓度可能高)→骷髅士兵+怨魂
→ 瘟疫废弃村: 传染(传播链)+怨恨(大规模横死)→僵尸群+病灵(感染型)
→ 神殿废墟: 神圣(A模式——神惩罚/守护)+自愿(殉道者)→守护灵+殉道者
→ 地下魔法实验室: 死灵法术+自愿→巫妖+合成亡灵+被奴役的骷髅
→ 刑场/绞架: 怨恨(横死)+低概率诅咒→复仇亡灵、无头骑士(极稀有)
```

## 四、亡灵强度与衰减

```rust
fn undead_power_score(
    bindings: &[BindingForce],
    lingering_spirit: f32,
    decay_time_days: u32,
    recent_kills: u32,
) -> f32 {
    // 多来源叠加——每多一个来源→强度×1.5
    let origin_mult = 1.0 + (bindings.len() as f32 - 1.0) * 0.5;

    // 刚死不久→强(残留灵元素多)
    let freshness = 1.0 / (1.0 + decay_time_days as f32 * 0.01);

    // 最近杀过活人→汲取了新鲜灵元素→临时强化
    let feeding_bonus = 1.0 + (recent_kills as f32 * 0.1);

    origin_mult * lingering_spirit * freshness * feeding_bonus
}
```

**亡灵的"生命"目标**——本能地攻击活人，从活人身上汲取灵元素延缓自身消散。这不是"邪恶"——这是**生存本能**。

## 五、亡灵与灵元素的最终命运

- 灵元素持续衰减→lingering_spirit→0→亡灵"死亡"——真正的死亡——彻底消散——灵魂不再
- 亡灵被外力永久杀死(圣水/神圣魔法/火化)→强制消散→但绑定力被打破——反而可能让灵魂终于安息
- 高力量亡灵(巫妖)可能有办法补充灵元素(定期猎杀活人/从魔力环境吸取/献祭)——从而长期"存活"

---

## 六、必备经典亡灵

骷髅(死灵法术型+环境型——最常见)、幽灵(怨恨型+传染型——不可物理攻击——需魔法/银器)、僵尸(传染型+环境型——缓慢但数量多)、巫妖(自愿型——高智力——往往藏在深处)、缚灵(诅咒型——被绑在特定地点——不能离开)、无头骑士(怨恨型——极稀有——横死的将领/贵族)。
