## 一、总览

自然景观是世界生成的"画布"。在 P0（全球大陆架构）和 P1（海洋系统）确定了海陆边界之后，P2 在陆地内部展开完整的自然景观体系。

**v3 核心变化**：
- **海洋景观独立成体系**——70% 海洋不是"空地"
- **海拔体系完全重算**——对齐 v3.0 的 ~1500m 高程范围
- **幻想地形按特性深入分类**——而非简单类型列表
- **气候带与纬度绑定**——25 万 km² 足以跨越多个气候带
- **生物栖息地接口**——为未来生物系统预留

## 二、海拔与高程体系（v3 重算）

v3.0 世界高程范围约 1500m（从海平面到山巅）。

| 区域 | 高程范围 | 特征 |
|------|---------|------|
| **深海** | < -200m | 大陆架以外 |
| **浅海** | -200m ~ 0m | 大陆架，光线可及 |
| **海岸低地** | 0 ~ 50m | 冲积平原、沙滩、滩涂、三角洲 |
| **低地** | 50 ~ 200m | 平原、草原、稀树草原、农田 |
| **丘陵** | 200 ~ 500m | 起伏和缓，连绵不断 |
| **低山** | 500 ~ 900m | 明显山形，山谷分明 |
| **中山** | 900 ~ 1300m | 陡峭山脊，岩石裸露增多 |
| **高山** | 1300 ~ 1700m | 山巅区域，植被稀疏至消失 |
| **极高** | > 1700m | 仅魔法/奇幻手段存在的区域 |

> 注：1500m 总范围中，700m+ 山是 v3.0 的硬性要求（`max_elevation(10km_radius) ≥ 700m`）。

## 四、河流系统

### 4.1 核心原则

水往低处流。河流从高海拔发源，沿坡度下流，支流汇入干流，干流入海或入内陆湖。

### 4.2 Strahler 分级（简化版）

| 级别 | 描述 | 宽度 | 深度 | 流速 | 可航行 |
|------|------|------|------|------|--------|
| **1级** | 山涧溪流，无支流 | 0.5-2m | 0.1-0.5m | 快 | ❌ |
| **2级** | 小型河流，1-2条支流 | 2-8m | 0.5-1.5m | 中-快 | 小型船 |
| **3级** | 中型河流，3-5条支流 | 8-30m | 1-3m | 中 | ✅ 中型船 |
| **4级** | 大型干流 | 30-100m | 2-8m | 缓 | ✅ 大型船 |
| **5级** | 巨川（入海口） | 100-500m+ | 5-20m | 极缓 | ✅ 海船可入 |

### 4.3 河流与游戏性

- **天然边界**：大型河流常成为政治实体之间的国境线
- **交通要道**：3级及以上河流可航行→贸易路线
- **渡口/桥梁**：道路与河流交叉处→聚落形成点
- **水力资源**：水磨坊；落差大的河段→水力驱动工业
- **特殊水体**：瀑布、峡谷河、地下河、温泉河、间歇河

### 5.4 洋流系统

简化的全球洋流模型：

```rust
struct OceanCurrents {
    // 洋流由全球温度梯度+大陆形状+科里奥利力决定
    gyres: Vec<OceanGyre>,       // 环流（如北大西洋环流）
    warm_currents: Vec<Current>, // 暖流——影响沿岸气候
    cold_currents: Vec<Current>, // 寒流——影响沿岸气候
}

struct OceanGyre {
    center: Vec2,
    radius: f64,        // 环流半径
    direction: f64,     // 旋转方向（±1）
    speed: f32,         // 流速
    seasonality: f32,   // 季节性变化幅度
}
```

洋流同时影响：
- **气候**：暖流→沿岸温暖湿润；寒流→沿岸干旱或寒冷
- **航行**：顺洋流航行加速，逆洋流减速
- **海洋生物分布**：冷暖流交汇处→渔场

## 六、气候带系统

### 6.1 纬度-温度映射

世界 Y 坐标映射纬度（种子决定赤道位置和纬度范围）：

| 气候带 | 纬度范围 | 年平均温度 | 特征 |
|--------|---------|-----------|------|
| **热带** | 0-23° | 20-35°C | 无明显季节，对流强 |
| **亚热带** | 23-35° | 15-25°C | 温和冬季、炎热夏季 |
| **温带** | 35-55° | 5-18°C | 四季分明 |
| **亚寒带** | 55-66° | -5~10°C | 长冬短夏 |
| **寒带** | 66-90° | <0°C | 永久冻土 |

### 6.2 温度修正因子

```
实际温度 = 纬度基准温度
         - 高程衰减（每上升100m ≈ -0.6°C）
         + 暖流接近度 × 暖流升温
         - 寒流接近度 × 寒流降温
         + 地形修正（背风坡/向阳坡）
```

### 6.3 湿度

```
湿度 = f(距海距离, 盛行风向, 雨影效应, 洋流温度)
```

## 七、群系参数场（Biome Parameter Field）

P2 输出的不是离散群系标签，而是**连续参数场**——一组覆盖全域的 2D 纹理/数组，每个像素存储该位置的群系相关参数：

| 参数 | 范围 | 含义 |
|------|------|------|
| `temperature_annual` | -30 ~ 45 (°C) | 年均温 |
| `temperature_seasonal_range` | 0 ~ 40 (°C) | 季节温差幅度 |
| `humidity` | 0.0 ~ 1.0 | 年均湿度 |
| `soil_fertility` | 0.0 ~ 1.0 | 土壤肥力 |
| `soil_type` | enum | 沙/壤/黏/砾/泥炭/火山灰 |
| `elevation_zone` | enum | 低地/丘陵/山地/高山/极高 |
| `water_table_depth` | 0.0 ~ 1.0 | 地下水位深度（归一化） |

离散群系标签（如"温带阔叶林""热带雨林"）由这些参数场通过分类函数产生——但这仅作为**人类可读的辅助标签**，不参与下游生成逻辑。下游模块（资源 P3、聚落 P4、生命系统的植物/动物）直接读取参数场而非标签：

```
参数场 (连续值, 生成管线消费) → 离散标签 (人类可读, 仅用于日志/显示)
```

**与 Life 模块的接口**：`[[009-植物|生命/009-植物]]` 和 `[[005-动物|生命/005-动物]]` 的生成以参数场为输入。Life 模块在生成完成后可反推离散群系标签作为显示用途。

## 八、幻想地形系统（深入分类）

幻想地形按**形成机制和特性**分类，而非简单类型列表。世界生成时，种子决定每种类型是否出现及数量。

### 8.1 浮空型——重力异常区

| 类型 | 特征 | 生成条件 | 世界观解释 |
|------|------|---------|-----------|
| **空岛/浮空岛** | 悬浮大陆碎片，大小不等（数m→数km） | <2%概率，群聚出现 | 古代魔法战争遗迹，元素失衡 |
| **浮空岛链** | 沿特定路径排列的浮空岛群 | 与空岛关联 | 断裂的古代浮空大陆碎片 |
| **翻转山** | 山体倒悬于空中 | 极低概率 | 重力魔法失控的永久后果 |
| **云中城迹** | 极高空的极小浮空遗迹 | 仅在高空可见 | 已灭绝的飞行文明的最后痕迹 |

### 8.2 能量浸染型——魔法能量改变地质

| 类型 | 特征 | 生成条件 | 魔法类型 |
|------|------|---------|---------|
| **魔化地** | 地面被魔法能量浸染，异常颜色/形态 | 魔法森林深处/古战场遗址 | 任何类型——颜色=浸染元素 |
| **永燃之地** | 持续燃烧不熄的区域（火元素） | 火元素裂隙附近 | 火元素 |
| **永冻区** | 冻结万年不化，冰并非水冰（冰元素） | 冰元素裂隙附近 | 冰元素 |
| **雷暴平原** | 持续雷暴的天空和导电的地面 | 雷元素裂隙附近 | 雷元素 |
| **元素倾泻区** | 单一元素极度富集 | 元素位面与主世界裂隙处 | 该元素的极端表现 |

### 8.3 结晶/矿化型——魔法与地质的化学反应

| 类型 | 特征 | 生成条件 |
|------|------|---------|
| **水晶地貌** | 地表生长巨大天然水晶柱/簇 | 地下魔法水晶矿脉+特定湿度 |
| **石化森林** | 树木全部石化的森林遗迹 | 远古森林被瞬间掩埋后矿化 |
| **盐晶荒漠** | 巨型盐晶形成的奇幻地貌 | 远古内海蒸发+魔法能量 |
| **金属森林** | 树木组织被金属元素替代 | 高浓度金属矿脉+魔法渗透 |
| **菌菇森林** | 巨型真菌构成的"森林"，昏暗奇幻 | 洞穴深处或特定魔法群系 |

### 8.4 巨型生物遗迹型——远古巨型存在的遗骸

| 类型 | 特征 | 生成条件 | 世界观解释 |
|------|------|---------|-----------|
| **龙陨之地** | 远古巨龙死亡之处 | <1%概率 | 龙血改变土壤，形成特殊生态微环境 |
| **巨人遗迹** | 超大规模石柱/石环/雕刻地貌 | 低概率 | 先于当前文明的未知种族遗留 |
| **世界树/巨树** | 单体极巨大的古树，可居住 | 极低概率 | 世界之树的末裔或最初森林的中心 |
| **巨兽遗骨** | 巨大生物的骨骼化石暴露于地表 | 低概率 | 远古巨型生物灭绝后的化石 |
| **神陨之地** | 神明死亡或受创之处 | 极低概率（0.1%） | 神血/神力残余改变了物理法则 |

### 8.5 维度异常型——物理法则失稳

| 类型 | 特征 | 生成条件 |
|------|------|---------|
| **深渊裂隙** | 深不见底的地面裂缝，宽度可达数十米 | 地震带/板块交界 |
| **时间异常区** | 区域内时间流速异常（快/慢/逆） | 极低概率 |
| **维度重叠区** | 主世界与其他位面在此处重叠 | 极低概率 |
| **失重区** | 重力轻微降低或消失的区域 | 浮空岛群附近 |
| **反色调区** | 颜色感官异常的扭曲区域 | 极低概率 |

### 8.6 星空/气象型——天空与地面的交界面异常

| 类型 | 特征 |
|------|------|
| **彩虹谷** | 持续或频繁出现彩虹的特殊山谷 |
| **极光原野** | 高纬度地区+魔法能量→极光全年可见且异常明亮 |
| **星降荒野** | 陨石频繁降落的区域，地面布满陨石坑 |
| **永夜谷/永昼峰** | 因地形+魔法→某些区域日照时间极端异常 |
| **镜湖群** | 水面异常平静的湖泊群，完美反射天空——但深度未知 |

## 九、地形过渡带

不同地貌之间不是突变——而是渐进过渡：

| 过渡类型 | 宽度 | 特征 |
|----------|------|------|
| 森林→草原 | 100-500m | 树木逐渐稀疏，林间空地增多 |
| 草原→沙漠 | 200-800m | 草地→半干旱灌木地→荒漠 |
| 山地→平原 | 200-1000m | 山麓冲积扇→缓坡→平地 |
| 河流沿岸 | 10-200m | 河岸植被带→周边地貌 |
| 海岸→内陆 | 100-500m | 耐盐植被→内陆植被 |
| 魔法区→正常区 | 50-500m | 魔法特征逐渐淡化 |

## 十、生物栖息地接口（预留）

世界生成不决定具体生物种类——只产出栖息地标记，供未来独立的生物系统读取。

```rust
/// 生物系统读取的栖息地接口
trait HabitatProvider {
    /// 查询指定位置的栖息地参数
    fn habitat_at(&self, pos: Vec2) -> HabitatData;
}

struct HabitatData {
    /// 生物群系类型
    biome: BiomeType,
    /// 初级生产力估算 (kcal/m²/年)
    primary_productivity: f32,
    /// 水体类型（淡水/咸水/流动/静止/无）
    water_type: WaterType,
    /// 特殊标记
    flags: HabitatFlags,
}

bitflags! {
    struct HabitatFlags: u32 {
        const MAGIC_ZONE    = 0b00000001;  // 魔法影响区
        const POLLUTED      = 0b00000010;  // 污染区
        const SACRED        = 0b00000100;  // 圣地
        const CURSED        = 0b00001000;  // 诅咒之地
        const CAVE_ENTRANCE = 0b00010000;  // 洞穴入口附近
        const MIGRATION_CORRIDOR = 0b00100000; // 动物迁徙通道
    }
}

/// 未来生物系统使用：
/// let spawn_table = creature_spawn_table.for_biome(habitat.biome);
/// let density_modifier = habitat.primary_productivity / global_average;
```

**目前不实现** `CreatureSpawnTable`、`MonsterSpawnCondition` 等——这些属于独立的生物系统模块。

> **v2.0 更新**: 此接口现由 [[010-生命世界初始化|010 P3.5 生命世界初始化]] 消费。`HabitatData` 的 `primary_productivity` 用于食物链承载力计算，`biome` + `flags` 用于物种筛选。
