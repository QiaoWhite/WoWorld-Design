## 三、EventMemory 增量

| 字段 | 大小 |
|------|------|
| compact_concepts | 64B |
| inference_conclusion | ~40B (Option) |
| summary_cache (惰性) | ~30B (引用) |
| summary_lang_id | 4B |
| encoding_version | 1B |
| **每条 memory 增量** | **~72B**（有效） |
| 旧字段保留 (summary, tags) | 0B 新增 (已存在) |

**1000 L1 NPC × 2000 条 × ~300B（含旧字段）= ~600MB**
新增部分: 2000条 × 72B = 144KB/NPC → 1000 NPC = **~144MB 新增**

## 五、CPU 预算

### 5.1 每决策周期 (0.3s, ~100 L1 NPC)

| 操作 | 频率 | 单次 | 帧均 |
|------|------|------|------|
| AgentSnapshot 派生 | 每 NPC | <2µs | <0.2ms |
| classify_pattern (桶索引) | 每社会事件×每感知 NPC | <1µs | ~3事件×30 NPC×1µs = <0.1ms |
| deductive_chain (浅层) | 每社会事件×每感知 NPC | <10µs | ~3×10µs = <0.03ms |
| memory_encode | 同上 | <5µs | <0.02ms |
| evaluate_deliberation | 每 NPC | <1µs | <0.1ms |
| **帧均总量** | | | **<0.5ms** |

### 5.2 每游戏日 (分摊到帧)
