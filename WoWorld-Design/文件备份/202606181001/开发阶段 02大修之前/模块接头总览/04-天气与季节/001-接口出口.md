# 天气与季节模块 — 接口出口

> **最后验证**: 2026-06-18 | **源文档版本**: v1.0 | **CHG-016 权威**

## WeatherQuery trait (唯一轮询接口 — 零事件总线)
- **概念**: `sample(pos, time) → WeatherSample` + `extreme_events_since(from, to) → Vec<ExtremeWeatherRecord>` + `current_season() → SeasonState` + `day_length_hours() → f32` + `wind_at(pos, time) → WindSample` (⚠️ CHG-033: wind_at 从旧 SpatialQuery 迁移至此)
- **Owner**: [[../../../天气与季节系统/001-天气系统总纲|天气 §二]] + [[../../../天气与季节系统/004-跨模块接口与数据合同|§004]]
- **类型**: Rust trait (woworld_core 定义, woworld_weather 实现)
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] SkyPerception · [[../01-生命/002-接口入口|生命]] 植物生长/动物迁徙 · [[../06-战斗/002-接口入口|战斗]] 环境判定 · [[../07-魔法/002-接口入口|魔法]] 元素浓度因子 · [[../05-感官与知觉/002-接口入口|感官]] perceive() 输入 · [[../16-音频/002-接口入口|音频]] 环境音 · [[../15-载具/002-接口入口|载具]] 航行条件 · [[../14-经济/002-接口入口|经济]] 农业产出 · [[../12-历史/002-接口入口|历史]] 气象异常
- **契约依据**: CHG-016 (零事件总线, 纯函数 (pos,time) → WeatherSample)

## WeatherSample (~120B Copy 类型)
- **概念**: 某时某地的完整天气样本 — 栈上 Copy. 字段: regional_temperature/ground_temperature (双层温度) + regional_humidity/ground_humidity (双层湿度) + precipitation_type/intensity/trend + regional_wind_speed/ground_wind_speed/wind_direction/gust_factor + cloud_coverage/cloud_type/visibility + surface_state + lightning_risk/lightning_strikes + is_extreme_weather + storm_front_distance + weather_stability + rainbow_visible + sun_altitude
- **Owner**: [[../../../天气与季节系统/001-天气系统总纲|天气 §四]]
- **类型**: Rust struct (Copy), woworld_weather
- **消费者**: 全部模块通过 WeatherQuery::sample() 消费
- **契约依据**: CHG-016 (双层温度: NPC 根据高程上下文自选基准温度)

## SeasonClock (季节时钟 — 纯时间函数)
- **概念**: 120天/年, 60分钟/天(默认·玩家可调15-120), 均匀四季(各30天) [TUNING]. 春分开局. 虚数年号(种子决定). 日长 = f(纬度, 季节). 天象生成 (彩虹归天气/极光日食归SeasonClock)
- **Owner**: [[../../../天气与季节系统/003-季节循环与时间|天气 §003]]
- **类型**: Rust struct, woworld_weather
- **消费者**: [[../01-生命/002-接口入口|生命]] 植物光周期/动物节律 · [[../02-NPC活人感/002-接口入口|NPC]] 日常节律
- **契约依据**: CHG-016 (纯时间函数 — 输入 GameTime, 输出 SeasonState)

## ClimateRegion (~50km grid)
- **概念**: ~100个气候区域. 从世界生成气候参数场降采样. 每游戏日更新. 预计算查找表 (~6.4KB)
- **Owner**: [[../../../天气与季节系统/001-天气系统总纲|天气 §三]] + [[../../../天气与季节系统/002-天气生成算法|§002]]
- **类型**: Rust 预计算表, woworld_weather (内部)

## Markov 6 状态机 + 雾独立维度
- **概念**: Clear → PartlyCloudy → Overcast → LightPrecip → ModeratePrecip → HeavyStorm. 雾独立布尔维度 (可与任意状态共存). 转移矩阵参数化生成 (非硬编码)
- **Owner**: [[../../../天气与季节系统/002-天气生成算法|天气 §002]]
- **类型**: Rust, woworld_weather (内部)
- **契约依据**: CHG-016 (6状态Markov + 雾独立)

## ExtremeWeather (极端天气 — 不命名枚举)
- **概念**: 参数组合自然区分: 暴风雨+风速>30m/s+海岸=台风效果, 雪+强度>0.7+风速>15m/s=暴风雪. 极端天气冷却管理
- **Owner**: [[../../../天气与季节系统/002-天气生成算法|天气 §002]]
- **类型**: Rust 阈值判定, woworld_weather
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 三层响应(L1温和偏移→L2警告强制→L3灾难硬约束) · [[../06-战斗/002-接口入口|战斗]] 环境剧变 · [[../15-载具/002-接口入口|载具]] 危险预警 · [[../12-历史/002-接口入口|历史]] 灾难事件
- **契约依据**: CHG-016

## WeatherVisualPacket (~200 bytes/帧)
- **概念**: sky_color_grade / cloud_config / precipitation_config / fog_params / wind_visual_params / lightning_flash / surface_wetness / snow_accumulation_rate / transition_blend. transition_blend 仅存在于 WeatherVisualPacket (非 WeatherSample)
- **Owner**: [[../../../天气与季节系统/001-天气系统总纲|天气 §二]]
- **类型**: Rust struct → Godot 消费
- **消费者**: [[../17-模型动作与物理/002-接口入口|Godot渲染]] 天空/云/海洋shader · [[../16-音频/002-接口入口|音频]] Godot音频层
- **契约依据**: CHG-016 (视觉~200B/帧, 降水粒子<=0.4ms GPU, VRAM增量<=2.5MB)

## HistoricalWeatherAnomaly (历史气象异常)
- **概念**: ~5,000-20,000条纯气象事实. 种子驱动极值采样 + 时间聚集 + 灾害集群. 供历史系统P9转化为HistoricalEvent
- **Owner**: [[../../../天气与季节系统/002-天气生成算法|天气 §002]]
- **类型**: Rust Vec<HistoricalWeatherAnomaly>, woworld_weather
- **消费者**: [[../12-历史/002-接口入口|历史]] P9消费并转化为HistoricalEvent
- **契约依据**: CHG-016 (历史系统必须查询气象异常数据, 不能独立随机)

## TerrainModifier / BiomeMicroclimateModifier
- **概念**: 地形修正(迎风坡/雨影/山谷风) + 群系微气候(冠层遮阳-2~6C/雪面反照/沙地辐射+8~20C/水体缓冲+-2~5C/城市热岛+1~5C/洞穴恒温)
- **Owner**: [[../../../天气与季节系统/002-天气生成算法|天气 §002]]
- **类型**: Rust, woworld_weather (内部 — 消费世界生成注入的 Query trait)

## PrecipitationType / SurfaceState / CloudType 枚举
- **概念**: PrecipitationType { None/Rain/FreezingRain/Snow/Hail/Dust } · SurfaceState { Dry/Wet/Ice/Snow/Mud } · CloudType { Cumulus/Stratus/Cumulonimbus/Cirrus }
- **Owner**: [[../../../天气与季节系统/001-天气系统总纲|天气 §二]]
- **类型**: Rust enum, woworld_weather
