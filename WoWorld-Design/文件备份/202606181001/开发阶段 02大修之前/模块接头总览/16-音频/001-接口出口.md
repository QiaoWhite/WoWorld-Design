- **概念**: pub trait(Send+Sync) — 对标 WeatherQuery/CultureQuery. 零分配
- **方法分组**: perceived_sounds() / perceived_speech() / ambient_noise_level() / acoustic_space() / speed_of_sound() / audible_radius() 等30个只读方法
- **Owner**: [[../../../音频系统/008-跨模块接口与数据合同|音频008]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] (感知) · [[../06-战斗/002-接口入口|战斗]] (声音情报) · [[../05-感官与知觉/002-接口入口|感官]] (听觉通道) · [[../13-语言表达/002-接口入口|语言表达]] (TTS渲染)

## 双层模型 (Rust模拟 + Godot渲染)
- **概念**: Rust侧物理声学模拟 → AudioRenderPacket → Godot渲染层播放
- **Owner**: [[../../../音频系统/001-音频系统总纲|音频001]]
- **消费者**: Godot渲染层
- **AudioRenderPacket**: 每帧 ~3KB — SoundRenderCommand[] + VoicePacket + MusicState + Notification + AmbientMix. 对标 WeatherVisualPacket

- **概念**: 描述物理现实 (表面接触+运动+发声+物品交互+沉默) — 不描述游戏语义
- **Owner**: [[../../../音频系统/002-声音足迹与发射模型|音频002]] | **类型**: Rust struct, woworld_types
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] ActionRingBuffer · [[../06-战斗/002-接口入口|战斗]] · [[../18-物品/002-接口入口|物品]]

## 传播引擎
- **概念**: 衰减(3种模型可选·TOML) / 吸收(频段) / 风矢量调制 / 温度梯度 / 遮挡(射线) / 多普勒频移 / 介质边界(水下)
- **Owner**: [[../../../音频系统/003-传播引擎|音频003]]

## 话语清晰度5档

## VoiceProfile (★从语言表达迁入 CHG-030)

- **Owner**: [[../../../音频系统/005-语音管道|音频005]] | **⚠️ CHG-030**: 所有权从语言表达012迁入音频模块
