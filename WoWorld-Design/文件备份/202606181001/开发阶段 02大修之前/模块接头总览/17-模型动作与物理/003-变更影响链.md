# 模型动作与物理 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 模型动作与物理系统 v1.0 (CHG-033)
>
> 当模型/动画/物理系统发生变更时, 以下模块和概念会受影响.

## SkeletonDef / BodyPlan 变更 ⚠️ CRITICAL
→ **NPC模块**: 骨架绑定/动画数据/GOAP姿态. 33→N骨或分区变化直接影响所有NPC动画
→ **战斗模块**: 骨骼蒙皮/受击判定. 骨骼ID变更→战斗伤害定位失效
→ **物品模块**: 装备槽位从BodyPlan派生. 骨架变更→装备挂点全部需更新
→ **生命模块**: BodyPlan引用(已提升至woworld_core). 生物生成参数
→ **载具模块**: 骑乘骨骼约束(CrewSlotDef → 骨骼锚点)

## 空间查询四trait 签名变更 ⚠️ CRITICAL (CHG-033)
→ **感官系统 (05)**: VisionQuery/ScentQuery/SpatialQuery — 感知管线核心依赖
→ **音频系统 (16)**: 遮挡/传播/混响查询
→ **战斗系统 (06)**: 战斗环境/视线/掩体查询
→ **载具系统 (15)**: 导航地面/碰撞检测
→ **NPC模块 (02)**: 寻路/感知/GOAP空间查询
→ **世界生成 (03)**: (trait实现方) TerrainQuery 实现
→ **大日志**: 玩家视线查询

## 9层动画栈 变更
→ **NPC模块**: GOAP→动画触发映射. 层级优先级/混合权重
→ **战斗模块**: 战斗姿态→动画栈插入点
→ **载具模块**: 骑乘/驾驶→动画层栈模式切换

## 38模块姿态 + 15基元轨迹 变更
→ **NPC模块**: 姿态数据库 → GOAP/情绪→姿态选择
→ **战斗模块**: 攻击轨迹适配器 → 武器基元轨迹
→ **TOML数据文件**: 姿态/轨迹数据格式变更

## GaitStyle (9参数) 变更
→ **NPC模块**: BigFive×Emotion→步态派生公式
→ **生命模块**: 疲劳度→步态调制

## 面部系统变更
→ **NPC模块**: EmotionState→面部映射. 6字节INSTANCE_CUSTOM 格式
→ **GPU shader**: 512²图集布局(16嘴×16眉×8眼). UV偏移公式
→ **文化模块**: 文化默认面部+个人偏离

## COM抛物体飞行 变更
→ **战斗模块**: 投掷武器/击飞轨迹
→ **环境/物品**: 掉落物品物理

## 骨架松弛死亡 变更
→ **战斗模块**: 死亡动画→骨骼松弛替代布娃娃
→ **生命模块**: 死亡事件→骨架松弛触发
→ **环境**: 尸体物理(不保留PhysicsServer3D)

## 双人交互/骑乘 TOML 变更
→ **NPC社交**: 握手/拥抱/推搡动画
→ **战斗**: 格斗投技/擒拿
→ **载具**: 骑乘约束

## 渲染管线变更
→ **Godot shader**: TABS cel 渲染(单diffuse, 无PBR)
→ **MultiMesh**: 双骨蒙皮 INSTANCE_CUSTOM 16B 格式
→ **VRAM预算**: 所有角色纹理 <5MB
