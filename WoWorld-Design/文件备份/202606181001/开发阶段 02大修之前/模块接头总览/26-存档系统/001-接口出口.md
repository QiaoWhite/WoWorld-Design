## 提供给其他模块的接口

### SaveableModule trait

存档系统定义 `SaveableModule` trait——所有需要持久化的模块实现此 trait。存档系统不提供数据给其他模块——它只消费模块的 trait 实现。

```rust
trait SaveableModule: Send + Sync {
    fn module_name(&self) -> &'static str;
    fn current_version(&self) -> u32;
    fn is_critical(&self) -> bool;
    fn has_dirty(&self) -> bool;
    fn snapshot_dirty(&self) -> Result<DirtySnapshot>;
    fn confirm_snapshot_written(&mut self, snapshot: &DirtySnapshot);
    fn load(&mut self, txn: &RoTransaction) -> Result<()>;
    fn migrate(&self, from_version: u32, txn: &RwTransaction) -> Result<()>;
}
```

### SaveSystem 运行时 LMDB 查询

```rust
impl SaveSystem {
    fn create_read_txn(&self) -> Result<RoTransaction>;
}
```

模块通过此方法获得只读 LMDB 事务，用于运行时渐进加载数据（如物品模块按 Chunk 加载物品）。
