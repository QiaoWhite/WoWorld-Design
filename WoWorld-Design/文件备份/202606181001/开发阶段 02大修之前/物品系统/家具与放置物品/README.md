## 核心问题

本模块回答：**一把椅子被摆在地上之后，它是什么？NPC 怎么知道可以坐它？桌上的盘子在地板被摧毁后会怎样？穷村和富城的家具为什么不同？魔法火球砸到餐桌——餐桌有木元素吗？**

---

## 关键设计决策

- **统一放置实体连续谱**——不区分"建筑组件"和"家具实体"。一切通过 `EntityIndex` 统一查询。
- **一切皆物品，一切皆可交互**——所有被摆放的实体都有 `ItemEntId`，物理原子层的 35 个原子统一操作。
- **参数化家族**——9 族 × ~27 mesh 模板 × 连续参数 × 文化扭曲 = ~12,000+ 视觉变体。
- **双 Pass 种子确定生成**——Pass A (附着家具) + Pass B (自由家具) + Pass C (野外)，全量预生成写入 LMDB。
