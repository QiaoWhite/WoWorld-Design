# 003 — 胜任挫折维度详细设计

## 一、为什么不是"胜任需求"而是"胜任挫折"？

### 1.1 命名意图

"胜任需求" (Need for Competence) 暗示：
- 有一个"理想胜任水平"需要达到
- 不足时产生驱力去弥补
- 达到后驱力消失

但这不符合 SDT 对胜任需求的描述。SDT 的胜任不是"我需要变强"，而是**"我需要在与环境的互动中感到有效能"**。它不是被消耗的资源，而是一种**持续的心理状态**。

"胜任挫折" (Competence Frustration) 更精确：
- 它不是"我不够强"，而是**"我的进步速度跟不上我的期望"**
- 它不因技能等级高而自动消失——大师也可以感到挫败（如果他的 aspiration 更高）
- 它来自**aspiration-reality gap**，而非来自绝对的技能水平

### 1.2 与相关系统的区别

| 概念 | 系统 | 本质 |
|------|------|------|
| 技能等级 | 技能系统 | 客观能力数值 |
| aspiration.progress | SelfNarrative | 志向达成的客观进展 |
| **competence_frustration** | 进阶需求系统 | 对 gap 的**主观心理反应** |
| stagnation_sense | SelfNarrative | 对生活**整体意义匮乏**的感知 |

**关键区分**：
- 一个技能 Lv.80 但 aspiration 是 Lv.85 的铁匠 → high frustration（差 5 级却进步缓慢）
- 一个技能 Lv.5 但 aspiration 是 Lv.6 的学徒 → low frustration（在预期轨道上）
- 一个没有任何 skill aspiration 的 NPC → frustration = 0（不关心能力进步）

## 二、Allostatic 模型

### 2.1 什么是 Allostatic？

**Homeostatic**（如 hunger）：设定点固定——吃饱=0 deviation，饿=1 deviation
**Allostatic**（如 competence）：设定点动态——取决于 aspiration 的当前水平

```rust
// 不是：competence_frustration = fixed_set_point - current_level
// 而是：competence_frustration = f(aspiration.target, current_level, recent_progress)
```

### 2.2 每日更新公式

```
competence_frustration =
    weighted_gap × chronic_factor × personality_modulation
    
其中:
    weighted_gap = Σ(gap_i × aspiration_priority_i) / Σ(priority_i)
    gap_i = max(0, (target_level_i - current_level_i) / target_level_i)
    
    chronic_factor = 1.0 + min(days_above_0_3 / 30.0, 1.0) × 0.5
    // 持续30天 gap>0.3 → ×1.5 慢性放大
    
    personality_modulation = neuroticism_amp × conscientiousness_buffer
    neuroticism_amp = 0.7 + neuroticism × 0.6    // 0.7~1.3
    conscientiousness_buffer = 1.0 - conscientiousness × 0.3  // 0.7~1.0
```

### 2.3 分步推演

**Step 1: 收集活跃的 SkillMastery aspirations**

```rust
let skill_aspirations: Vec<(SkillId, u32, f32)> = aspirations.iter()
    .filter_map(|a| match &a.category {
        AspirationCategory::SkillMastery { skill_id, target_level } => {
            Some((*skill_id, *target_level, a.priority))
        }
        _ => None,
    })
    .collect();
```

**Step 2: 计算加权 gap**

```rust
if skill_aspirations.is_empty() {
    return (0.0, 0.0); // (frustration_value, chronic_days_reset_flag)
}

let mut total_weighted_gap = 0.0;
let mut total_priority = 0.0;

for (skill_id, target, priority) in &skill_aspirations {
    let current = skill_query.get_level(npc_id, *skill_id).unwrap_or(0) as f32;
    let gap = ((*target as f32 - current) / *target as f32).max(0.0);
    total_weighted_gap += gap * priority;
    total_priority += priority;
}

let avg_gap = total_weighted_gap / total_priority;
```

**Step 3: 慢性累积**

```rust
let chronic_days = if avg_gap > 0.3 {
    npc.competence_frustration_chronic_days.saturating_add(1)
} else {
    // gap 缩小到阈值以下 → 慢性快速消退
    npc.competence_frustration_chronic_days.saturating_sub(2)
};

let chronic_factor = 1.0 + (chronic_days as f32 / 30.0).min(1.0) * 0.5;
// Day 0: 1.0
// Day 15: 1.25
// Day 30+: 1.5 (饱和)
```

**Step 4: 人格调制**

```rust
let neuroticism = npc.big_five.neuroticism;
let conscientiousness = npc.big_five.conscientiousness;

let neuroticism_amp = 0.7 + neuroticism * 0.6; // N=0→0.7, N=1→1.3
// 高神经质放大挫败感 86% (0.7 vs 1.3)

let conscientiousness_buffer = 1.0 - conscientiousness * 0.3; // C=0→1.0, C=1→0.7
// 高尽责性缓冲挫败感 30%——他们相信坚持能突破

let frustration = (avg_gap * chronic_factor * neuroticism_amp * conscientiousness_buffer)
    .clamp(0.0, 1.0);
```

### 2.4 人格调制的涌现效果

| 人格组合 | N | C | 效果 | 行为表现 |
|----------|---|---|------|---------|
| 淡定大师 | 0.2 | 0.8 | 0.7×0.76=0.53 | 遇到瓶颈也很冷静，坚持练习 |
| 焦虑学徒 | 0.9 | 0.2 | 1.24×0.94=1.17 | 容易挫败，可能频繁换方向 |
| 完美主义者 | 0.9 | 0.9 | 1.24×0.73=0.90 | 高标准但高韧性——痛苦地坚持 |
| 佛系NPC | 0.1 | 0.1 | 0.73×0.97=0.71 | 对 gap 不太在意 |

## 三、与其他心理信号的交互

### 3.1 对情绪引擎的牵引

```
competence_frustration > 0.5 →
    pleasure -= frustration × 0.03 / frame   // 挫败→不愉快
    arousal  += frustration × 0.02 / frame   // 挫败→焦虑/烦躁
    control  -= frustration × 0.04 / frame   // 挫败→失控感
```

这个牵引比基本需求（hunger→pleasure 约 0.005/frame）更强——**心理挫折对情绪的直接影响比生理不适更尖锐**。

### 3.2 对 SelfNarrative 的影响

```
// SelfNarrative.reflect() 中
if competence_frustration > 0.7 && chronic_days > 60 {
    // 慢性严重挫折 → 可能触发 LifeTheme 转换
    // Building/Mastery → Crisis (中年危机路径)
    // 或 aspiration 放弃/重设
    let give_up_probability = 
        competence_frustration * neuroticism * (1.0 - conscientiousness) * 0.3;
    // 高N+低C → 最高 ~30% 概率放弃当前 aspiration
}
```

### 3.3 与 stagnation_sense 的区分

| 维度 | competence_frustration | stagnation_sense |
|------|----------------------|-------------------|
| **粒度** | 技能特定 | 人生整体 |
| **更新频率** | 每日 | 每7天 |
| **来源** | aspiration gap | 有意义的记忆数量 |
| **方向** | "我在这个技能上卡住了" | "我的生活缺乏意义" |
| **解决** | 提升技能或降低志向 | 改变生活轨迹 |

两者可以同时高：
- 一个铁匠在技能上卡住（高 frustration）+ 每天重复单调生活（高 stagnation）= **中年危机**

两者也可以分离：
- 一个探索者在不同技能间切换（低 frustration）+ 见识了许多新事物（低 stagnation）= 充实的人生

## 四、边界情况

### 4.1 无技能志向的 NPC

大量 NPC 没有 SkillMastery aspiration——他们只是"过日子的人"：
- competence_frustration = 0.0
- 这些 NPC 的行为完全由基本需求 + habit + culture 驱动
- 他们是世界的"背景"，但在统计上构成了社会的稳定性

这是**正确且期望的行为**——不是每个 NPC 都需要有"成长焦虑"。

### 4.2 已经达成 aspiration 的 NPC

NPC 的 aspiration.target_level 被达到：
- gap → 0
- competence_frustration → 0（快速衰减）
- SelfNarrative 反思时可能生成新的 aspiration（更高目标）→ gap 重新出现
- 或者 NPC "退休"——不再有 skill aspiration → frustration 归零

### 4.3 技能等级碾压 aspiration

NPC 已经远超 aspiration 目标（current >> target）：
- gap = 0（不可能是负数——我们没有"过度胜任焦虑"）
- 但这表明 aspiration 设置过低
- SelfNarrative 反思时会检测到并更新 aspiration

### 4.4 儿童与老年 NPC

- 儿童（< Adolescent）：无 SkillMastery aspiration → frustration = 0
- 老年（LifeTheme::Decline/Legacy）：aspiration 可能从 SkillMastery 转向 LeaveLegacy → frustration 下降，被 transcendence 驱力取代

## 五、恢复路径

competence_frustration 的"恢复"不同于 esteem_deficit 的"恢复"：

| | esteem_deficit | competence_frustration |
|---|---|---|
| **恢复方式** | 社交钦佩事件 | gap 缩小（技能提升或志向降低） |
| **主动恢复** | 寻求社交认可 | 训练技能 / 更换志向 |
| **被动恢复** | 无（只能通过新认可） | gap 缩小后快速消退 |
| **"假恢复"** | 奉承（低效） | 降低志向（放弃） |

### 5.1 积极恢复：技能突破

```
NPC 的 skill level 提升（level up 事件）：
  → gap 缩小
  → chronic_days 重置
  → competence_frustration 重新计算
  → 情绪获得 +0.05 pleasure spike（"突破的喜悦"）
```

### 5.2 消极恢复：降低志向

```
SelfNarrative.reflect() 中：
  if competence_frustration > 0.8 && chronic_days > 90:
    → 可能触发 aspiration 放弃或降级
    → "也许我不适合做大师铁匠。做个好铁匠就够了。"
    → frustration 下降（因为设定点移动了）
    → 但同时 SelfConcept 可能受损（"放弃者"标签）
```

## 六、设计余量：v2.0 可能扩展

### 6.1 多领域挫折

v1.0：只追踪 SkillMastery 相关的 frustration
v2.0 可能：
- SocialCompetence frustration: "我不擅长社交"
- CreativeBlock frustration: "我创作不出新东西"
- 使用相同的 allostatic 框架，不同的数据源

### 6.2 社会比较加剧

v2.0 可能：
- 如果同行的技能远高于自己 → frustration 放大
- "不仅我自己进步慢，我的同龄人都在超越我"
- 需要 social comparison 查询（暂不存在）

### 6.3 教学缓解

v2.0 可能：
- 教导他人可以缓解自己的 competence_frustration
- "虽然我自己卡在 Lv.75，但我能教出 Lv.60 的弟子"
- 这桥接技能系统的教学机制 + IntrinsicGoal::LeaveLegacy
