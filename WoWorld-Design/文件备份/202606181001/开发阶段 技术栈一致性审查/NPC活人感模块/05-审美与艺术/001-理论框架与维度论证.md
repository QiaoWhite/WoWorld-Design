# 001 — 理论框架与维度论证

## 一、模块定位

### 1.1 审美系统的本质

WoWorld 中各模块处理不同领域的模拟：

| 模块 | 建模什么 | 核心结构 |
|------|---------|---------|
| 需求系统 | 稳态维持（homeostasis） | deficit → urgency → action → recovery |
| 权力系统 | 人际操作关系 | 有向多重图（PowerTopology） |
| 经济系统 | 交易对撮合 | 限价订单簿（LOB） |
| 文化系统 | 共享参数场 | 10 参数 + 派生树 |
| **审美系统** | **感知-评估-反应管道** | **Signal → Taste → Judgment pipeline** |

审美的本质与需求有根本不同：

- 需求是 **匮乏驱动**（deficit-driven）：饿了→找食物→吃饱→恢复
- 审美是 **吸引驱动**（attraction-driven）：看到美的东西→产生判断→体验本身就是终点

审美不需要"满足"，不存在"审美赤字"。这也是为什么进阶需求系统（025/001）中审美需求被明确拒绝——分析是正确的。

### 1.2 为什么需要独立模块

现有设计中，审美相关内容散落在多个模块：

- 文化系统 004：`artistry` 参数、`CulturalBeautyStandard`、`BuildingStylePreferences`、`ClothingPreferences`
- 物品系统 003：`AestheticProps`（13 StyleTag + ornamentation_level + color_palette…）
- NPC 02：`conventional_attractiveness()` 消费 CulturalBeautyStandard
- 生命系统 003：四层质量防线含美学评分（5 因子）
- 节日系统 008：Dance/Music/Theatre/Decoration 艺术活动
- 魔法 018："艺术范式"——魔法 = 意志通过美学的表达

**问题**：没有统一的审美"语言"。各模块各自定义了审美相关的概念，但彼此之间缺乏可互操作的接口。物品系统定义了 `AestheticProps`，但 NPC 如何对建筑产生审美判断？战斗系统如何让观战者产生审美体验？

**目标**：定义一套所有模块共享的审美"通用语"——`AestheticSignal`（任何可感知物的 6 维美学特征向量）和 `AestheticJudgment`（任何感知者的 4 维判断输出）。审美模块是**无状态引擎**——不存储数据，只定义类型、公式和事件处理。

## 二、理论根基：来自神经科学与心理学的支撑

### 2.1 预测处理与"流畅-惊喜"平衡

大脑是一台预测机器（预测编码理论）。审美体验产生于可预测模式（流畅）与可控意外（惊喜）之间的最佳平衡：

- **完美预测**（极高 fluency，极低 novelty）→ 流畅但乏味 → valence 低
- **高度预测失败**（极低 fluency，极高 novelty）→ 困惑 → valence 低
- **最佳点**（适中 fluency + 适中 novelty）→ "啊哈！"时刻 → valence 高

这直接决定了 **novelty → valence 必须是倒 U 型函数**，不能是线性的。

### 2.2 奖赏通路的多巴胺编码

欣赏或创作艺术时，伏隔核和腹侧被盖区（VTA）释放多巴胺。多巴胺不仅对奖赏本身做出反应，更对**奖赏预测误差**（"结果比预期更好"）做出强烈反应。这是 `prior_expectation` 字段的神经基础——期待值影响愉悦度。

### 2.3 岛叶与体内感觉

审美体验很大程度上是体内感觉（interoception）——"被击中""脊背发凉""胸口一暖"不是比喻，而是真实的神经身体事件。岛叶持续监测心跳、呼吸、肌肉紧张度，并将身体模式标记为情感感受。

这定义了 **`somatic_impact`** 的存在必要性——它不是信号维度之一，而是 `novelty × expressiveness` 的涌现产物。又新又动情 → 身体被"击中"。

### 2.4 默认模式网络的双重角色

- **强烈沉浸时 DMN 被抑制** → "忘我"状态（心流/审美眩晕） → 对应 arousal↑ + interest↑
- **意义建构时 DMN 被激活** → "回神"反思 → 对应 Ponder/记忆写入

### 2.5 镜像神经元与共情

观看表现痛苦或喜悦的艺术作品时，镜像神经元系统被激活，让我们能模拟他人的感觉和情绪。这解释了为什么 `expressiveness`（情感表现力）是独立的信号维度——它承载了从创作者到观者的情感传递通道。

### 2.6 进化心理学：求偶炫耀假说

艺术创作能力可能是诚实信号（honest signal）——精通复杂技艺暗示大脑功能健全、记忆力强、手眼协调。艺术才能可能在性选择中被视为有吸引力的特质。这解释了为什么 `virtuosity → respect → attraction` 的涌现链条是合理的——但不需要"求偶炫耀"硬编码。

### 2.7 社会同步效应

集体唱歌/跳舞/击鼓 → 内啡肽 + 催产素释放 → 增强社会连结和信任。这定义了 `SocialAestheticContext.audience_synchrony` 的作用——集体观看时个人判断被放大。

## 三、信号维度：为什么是 6 维而非更少或更多

### 3.1 推导过程

从神经科学的四个可分离通路反推：

1. **多巴胺奖赏通路**（伏隔核）→ 需要 `fluency`（流畅=省力=愉悦）和 `harmony`（和谐=协调=愉悦）
2. **交感神经激活通路** → 需要 `novelty`（新=激活）和 `expressiveness`（有情感=激活）
3. **认知投入通路**（前扣带回/背外侧前额叶） → 需要 `complexity`（信息密度=认知挑战）和 `novelty`（新信息=认知动机）
4. **社会评价通路**（内侧前额叶/颞顶联合区） → 需要 `virtuosity`（技艺=社会评价）

四个通路的输入需求：fluency, harmony, novelty, expressiveness, complexity, virtuosity ——恰好 6 个。

### 3.2 排除的候选维度

| 候选 | 为何不是独立维度 |
|------|-----------------|
| **somatic_impact**（身体冲击度） | = novelty × expressiveness 的涌现产物。又新又动情 → 被击中。非原子。 |
| **symbolic_depth**（象征深度） | = complexity × cultural_context 的涌现产物。非原子——依赖文化知识，由 `prior_expectation` + `familiarity` 承载。 |
| **beauty**（美） | = 判断输出（valence>0.7），非输入信号。美是从信号×taste 涌现的，不是信号本身。 |
| **originality**（原创性） | = novelty × virtuosity 的涌现。很新且技艺支持 → 是原创而非胡乱。 |
| **elegance**（优雅） | = fluency × harmony 的涌现 + simplicity（是 complexity 的反面）。复合判断。 |
| **sublimity**（崇高） | = somatic_impact + respect + 低 valence（mixed feeling）。是判断输出组合，非信号。 |

### 3.3 为什么是 6 而非 4 或 8

- **4 维不足**：如果只有 fluency/novelty/complexity/harmony → 缺少了社会评价维度（virtuosity）和情感传递维度（expressiveness），无法解释"我很敬佩这件作品"或"这作品让我感受到了创作者的痛苦"
- **8 维过多**：somatic_impact 和 symbolic_depth 是涌现属性，不独立存在（见上）

**6 维是系统完备且无冗余的最小集合。**

## 四、判断维度：为什么是 4 维

### 4.1 四个维度各自的不可归并性

| 维度 | 神经对应 | 行为后果 | 为何不可归并 |
|------|---------|---------|------------|
| **valence**（愉悦度） | 多巴胺奖赏通路 | 趋避行为：喜欢→靠近，厌恶→远离 | 一个人可以敬佩但不愉悦（"技术很强但我不喜欢"） |
| **arousal**（激活度） | 交感神经系统 | 行动强度：高→激动/紧张，低→平静/放松 | 一个人可以愉悦但不激动（"恬静的美"） |
| **interest**（兴趣度） | 前扣带回+认知投入 | 持续关注：高→细看/研究，低→略过 | 一个人可以愉悦但没兴趣（"好看，但也就那样"） |
| **respect**（敬佩度） | 社会评价通路 | 社会认可：高→尊敬/推崇，低→轻视 | 一个人可以不愉悦但很敬佩（"我不喜欢但必须承认厉害"） |

**关键涌现现象**：这 4 维的组合产生丰富的审美体验类别：
- 美 = 高 valence + 中低 arousal + 高 interest
- 震撼 = 高 valence + 高 arousal + 高 interest + 高 respect
- 崇高 = 中 valence + 高 arousal + 高 respect（混合了敬畏和微微不安）
- 无聊 = 所有四维都低
- 冒犯 = 负 valence + 高 arousal + 中 interest（让人不舒服但无法移开视线）

## 五、与现有文化系统参数的关系

### 5.1 `artistry` 的角色

`artistry` 是文化系统 10 个核心参数之一（0=功能优先，1=美感优先）。它不是审美的"开关"——低 artistry 文化中的 NPC 仍然有审美判断，只是以下差异：

- `artistry` 高 → `AestheticTaste.dimension_weights[expressiveness]` 高 → 对情感表现力敏感
- `artistry` 高 → `AestheticTaste.complexity_tolerance` 高 → 能欣赏复杂作品
- `artistry` 高 → 更频繁触发 Articulate（讨论审美话题）
- `artistry` 高 → 节日密度增加（文化系统 008 已有）

### 5.2 与其他文化参数的关系

| 文化参数 | 对 AestheticTaste 的影响 |
|---------|----------------------|
| `uncertainty_avoidance` → | `familiarity_bias` 偏高（偏好熟悉） |
| `openness_to_outsiders` → | `aesthetic_openness` 偏高（接受外来风格） |
| `individualism` → | `dimension_weights[novelty]` 偏高（重视独特性） |
| `power_distance` → | `dimension_weights[virtuosity]` 偏高（重视技艺等级） |
| `indulgence` → | `dimension_weights[harmony]` 偏高（追求感官愉悦） |
