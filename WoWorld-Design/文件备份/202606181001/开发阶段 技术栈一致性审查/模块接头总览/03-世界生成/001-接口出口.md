## 依赖注入 trait (供天气系统消费)
- **ElevationQuery**: 高程查询 → 地形雨/气温直减率
- **ClimateParamsQuery**: 温度基线/湿度基线/季节性幅度/纬度
- **BiomeMicroclimateQuery**: canopy_closure / albedo / ground_type / wetland_fraction / urban_fraction / distance_to_water_km
- **OceanCurrentQuery**: 洋流温度/方向 → 风暴生成
- **WorldBoundaryQuery**: is_in_storm_zone
- **类型**: Rust trait (woworld_core定义) → woworld_worldgen 实现
- **消费者**: [[../04-天气与季节/002-接口入口|天气]] 全部 5 个 Query
