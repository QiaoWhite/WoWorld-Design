# 语言表达模块 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 语言表达 v1.0-v1.1 + LLM增强层 (CHG-017/018/019)

## 影响链总览

```
语言表达系统变更
├── ExpressionRef 格式变更
│   ├── → NPC: 记忆存储
│   ├── → 历史: PhysicalBook 内容句柄
│   ├── → 魔法: grimoire/scroll 内容
│   └── → 物品: 铭文句柄
├── ContentResolver trait 变更
│   ├── → 历史: 书本解析
│   ├── → 魔法: 卷轴解析
│   └── → 物品: 铭文解析
├── LanguageId/ScriptId 编码变更
│   ├── → 历史: Work.language
│   ├── → 物品: 物品铭文
│   ├── → 魔法: 魔法文本
│   └── → NPC: 语言能力
├── ~430 片段组合模型变更
│   ├── → NPC: NPC 对话
│   ├── → 历史: 书本内容
│   ├── → 魔法: 卷轴文本
│   └── → UI: 显示文本
├── TurnMode 枚举变更
│   ├── → NPC: NPC 对话
│   └── → 文化: TurnMode::Ritual
├── PhaticLayer 五类应酬变更
│   └── → NPC: 社交行为
├── 五传播通道变更
│   ├── → NPC: 信息传递
│   └── → 历史: 谣言生命周期
├── 失真算子变更
│   └── → NPC: 信息失真模拟
├── DeceptionIntent 枚举变更
│   └── → NPC: 谎言检测
├── 谣言生命周期参数变更
│   └── → 历史: 谣言历史化
├── NonVerbalSignal 六类变更
│   ├── → NPC: 非语言行为
│   └── → Godot: 渲染
├── InputInterpreter 三层回落变更
│   ├── → Player: 输入处理
│   └── → NPC: NPC 理解玩家
├── LlmSceneConfig 场景增删
│   └── → 全部: LLM 参与度控制
├── LlmBackend trait 变更
│   └── → 语言表达内部
├── VoiceProfile 字段变更
│   ├── → 音频: TTS 消费
│   └── → NPC: 语音身份
├── VoiceEmotionModulation 变更
│   ├── → 音频: TtsEngine
│   └── → NPC: 情绪→语音
├── TtsEngine trait 变更
│   └── → 音频: 实现适配
├── VoicePriority 级别变更
│   └── → 音频: 调度逻辑
├── EffectiveAudibleRadius 因子变更
│   ├── → NPC
│   └── → 音频
└── 三层 Crate 架构边界变更
    └── → 全部: 导入依赖调整
```

| PhaticLayer 片段 | 🟢 低 | 1 | 内容扩展 |
| 三层 Crate 架构 | 🔴 高 | 全部 | 架构变更 |
