# 技术栈方案 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 技术栈 v4.0 (CHG-038 审计升级)
>
> 技术栈方案是全局架构文档——其变更影响全部21个模块。

## 引擎/语言版本变更
→ **全部模块**: Rust stable版本 / Godot 4.6 LTS → 所有GDExtension接口
→ **构建系统**: 编译目标/feature flag
→ **Mod系统**: TOML schema版本

## 架构拓扑变更
→ **全部模块**: Rust模拟核心 → GDExtension → Godot客户端 三层架构
→ **新增/移除层**: 所有模块的crate归属变更
→ **woworld_core 边界**: 核心类型提升/降级影响所有消费方

## 性能预算变更 ⚠️ 最高影响
→ **全部模块**: 帧预算16.7ms / VRAM 4.2GB / 内存1.4GB
→ **峰值互斥原则**: 各子系统预算重新分配 → 可能限制某些模块功能
→ **具体预算变更影响**:
  - CPU动画 ≤0.5ms → NPC数量/动画复杂度
  - Rust模拟 ≤4ms/帧 → 所有模拟系统 (GOAP/经济/物理/天气)
  - 音频 ≤0.17ms → 同时发声数/传播精度
  - 天气 ≤0.25ms → 天气轮询精度
  - 降水粒子 ≤0.4ms GPU → 粒子数量

## LOD 统一架构变更
→ **全部模块**: L1/L2/L3/L4 分桶策略
→ **NPC**: L1全AI→L2简化→L3统计
→ **音频**: L1全入→L2门控→L3≥120dB
→ **感知**: L1全入→L2主要实体→L3聚合
→ **物理**: L1全精度→L2简化碰撞→L3无碰撞
→ **体素**: 8级LOD, 3km Signed Heightfield 切换距离

## 技术栈方案自身格式变更
→ **TDI编号体系**: 237+126 TDI → 新增/重新编号影响引用
→ **CHG-038**: 技术栈全量审计 → 审计结论归档
→ **CHG-033**: 物理方案改为"仅玩家PhysicsServer3D" → 技术栈方案 §物理 完全重写

## 渲染管线变更
→ **Godot shader**: TABS cel (单diffuse, 无PBR) → 所有角色/建筑/道具 shader
→ **体素**: Transvoxel → ArrayMesh → Godot渲染
→ **天空/海洋**: 体积云shader / Gerstner波shader → 天气/海洋模块

## 数据存储变更
→ **LMDB → 替代方案**: 所有Chunk存储/NPC存档/历史记录
→ **双层记忆架构**: NPC记忆(热) + LMDB(冷)

## 平台支持变更
→ **Windows/Linux/macOS**: 各平台的渲染后端/文件系统/线程模型
→ **硬件目标**: GTX 1660 SUPER 6GB → 最低配置变更
