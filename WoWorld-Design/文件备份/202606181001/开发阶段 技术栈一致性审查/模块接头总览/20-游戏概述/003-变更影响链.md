# 游戏概述 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 游戏概述 v1.0
>
> 游戏概述变更是"宪法级"变更——影响所有模块的设计方向。

## 设计哲学变更 ⚠️ 最高影响
→ **全部模块**: "故事生成器不是故事容器" 原则变更 → 所有涌现式设计须重新审视
→ **NPC模块**: NPC设计深度 (记忆/情感/欲望) 的边界
→ **经济/权力/文化**: 涌现vs预设的平衡

## 核心竞争力变更
→ **对应模块**: 每个核心竞争力对应一组模块:
  - 涌现式交互 → NPC/经济/权力/文化/战斗
  - NPC完整人性 → NPC/生命/认知/审美/技能
  - 全球多元文化 → 文化/信仰/语言表达/世界生成
  - 冒险与生活平等 → 战斗/经济/技能/权力/载具
  - 程序化生成深度 → 世界生成/历史/文化/信仰
  - LLM增强 → 语言表达/NPC

## 世界规模变更
→ **世界生成**: 25万km² → 更大/更小 → 体素管线/Chunk存储/Voronoi文化区域
→ **性能优化**: VRAM/内存/CPU预算重新分配
→ **载具**: 旅行距离/交通网络规模

## NPC规模变更
→ **NPC模块**: 100k → N → SoA存储/rayon并行/GOAP调度
→ **性能优化**: CPU模拟预算 re-profile

## 海陆比例变更
→ **世界生成**: 海:陆≈7:3 → 海洋生成管线/海岸线
→ **载具**: 航海vs陆路交通比例
→ **生命**: 海洋生物vs陆地生物比例

## "完整的人" 维度变更
→ **NPC**: 记忆上限2000条 → 存储预算
→ **生命**: 生理模型复杂度
→ **认知**: MentalModel ≤20条 → 认知深度
