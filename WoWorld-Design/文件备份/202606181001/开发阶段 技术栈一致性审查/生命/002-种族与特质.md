## 一、核心原则：特质从历史涌现

**不预置**。"精灵敏捷+2、矮人力量+2"——这种手写表违背 WoWorld 的涌现哲学。

```rust
/// 种族特质——由种族历史+环境适应涌现
struct RaceTraits {
    physical_modifiers: PhysicalModifiers,
    elemental_affinities: [f32; 10],
    special_abilities: Vec<SpecialAbility>,
    lifespan_range: (f32, f32),
    size_range: (f32, f32),
    /// 这些特质的来源——用于解释和验证
    trait_origins: Vec<TraitOrigin>,
}

struct TraitOrigin {
    trait_name: String,           // 如 "耐火"
    source: TraitSource,          // 来源类型
    generations_to_emerge: u32,   // 经过多少代涌现
    causal_chain: String,         // 因果链："祖先在火山生活了300代→耐火"
}

enum TraitSource {
    Environmental { biome: BiomeType, pressure: String },
    Magical { magic_concentration: f32, element: Element },
    Selective { bred_by_race: Option<RaceId>, purpose: String },
    Divine { god: String },        // 仅A模式
    Catastrophic { event: String }, // 灾变后适应——大洪水/火山冬季/魔法灾难
}
```

## 三、初始智能种族

世界生成时预设少数"人类"种子（玩家可选的种族），其余智能种族从种子程序化涌现。

### 预设种子种族（所有世界都有——玩家可选）

| 种族 | 说明 | 生理倾向 |
|------|------|---------|
| **人类** | 基准种族——中性适应力 | 无极端倾向，学习速度快 |
| **精灵** | 长耳、长寿、自然亲和 | 灵元素容量偏高、木/灵元素亲和 |
| **矮人** | 矮小、矿脉直觉、耐火 | 力量高、耐久高、火/金元素亲和 |

> 这些倾向不是"手写死了"——它们是**该种族共性中的中位数**。个体差异仍然存在。

### 程序化涌现的新种族

除预设三种外，其余智能种族由生成器从群系+历史中涌现：

```
生成条件:
  - 一个聚落群域的种族在某种环境压力下隔离生活 >N 代
  - 环境适应压力连续施加在特定方向上
  - 该适应方向达到阈值 → 产生新的种族特质 → 称为新种族

例如:
  人类聚落群域在大沙漠中隔离生活 1200 代
  → 体温耐受范围上移、水分保持能力增强、夜行性涌现
  → 特质累积超过分化阈值 → "沙漠族"从人类中分化
```

## 五、种族特有标签

| 标签 | 说明 |
|------|------|
| `Humanoid` | 类人形态 |
| `LongLived` | 长寿（>200年） |
| `ShortLived` | 短寿（<30年） |
| `NightVision` | 夜视 |
| `HeatResistant` | 耐热 |
| `ColdResistant` | 耐寒 |
| `Amphibious` | 两栖 |
| `Arboreal` | 树栖适应 |
| `Burrowing` | 掘地适应 |
| `NaturalArmor` | 天然护甲（硬皮/鳞片/骨板） |
| `ElementalAffinity_<X>` | 某元素天生高亲和 |
| `MagicallyGifted` | 天生魔力运用能力 |
| `SpiritSensitive` | 对灵元素特别敏感 |
| `FastLearner` | 学习速度加成 |
| `SlowLearner` | 学习速度低但记忆持久 |
| `SolarPowered` | 光合/日光恢复体力 |
| `CaveAdapted` | 洞穴适应 |
