## 定位

本文件夹存放 WoWorld 世界生成系统的**正式开发规格**。世界生成是从一个 `uint64` 种子出发，经过 11 个阶段逐级细化的**确定性管线**（同一种子永远生成同一个世界）。

**核心原则**：
- **种子决定一切**。山脉、海洋、聚落、NPC、文化——全部由噪声函数和生成规则产生
- **体素 = 大地素材**。自建 Transvoxel（Rust 侧）→ Godot ArrayMesh
- **海洋是内容，不是空地**。海:陆 ≈ 7:3，航海/岛屿/跨洋发现是核心体验
- **涌现优先**。不写故事，只写规则

## 生成管线速览

```
种子(uint64)
  → P0: 全球大陆架构 (大陆板块噪声 → 海陆掩码 → 造山带/裂谷/热点)
  → P1: 海洋系统 (洋流 → 深度分级 → 海岸分类 → 可航行海域)
  → P2: 陆地自然基底 (高程图 → 水文模拟 → 气候带 → 生物群系)
  → P3: 资源分布 (矿产/岩料/木材/土沙/液体/有机/魔法——矿田生成)
  → P4: 聚落选址 (承载容量评分 → 选址 → 等级确定)
  → P5: 聚落内部布局 (政体/文化 → 功能区 → WFC 城市级填充)
  → P6: 建筑生成 (WFC 建筑级 → 房间布局 → 内饰 → 外装)
  → P7: 交通网络 (陆地MST道路 + 海上航线)
  → P8: NPC 初始化 (按承载容量分配人口 → 属性/关系初始化)
  → P9: 历史模拟 (快进 N 年 → 聚落兴衰/建筑老化/NPC关系演变)
  → P10: 世界边界处理 (自然海洋包裹 + 风暴带软边界)
  → 输出: ✅ 可游玩世界
```

## 关键参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| **世界面积** | 25 万 km²（≈英国，约 500×500km） | 初步版 |
| **海陆比** | 7:3（可调） | 默认基准，种子参数可自定义 |
| **高程范围** | ~1500m | 含 700m+ 山峦 |
| **体素单位** | 0.5m³/体素 | 地形精度 |
| **Chunk** | 垂直稀疏——水平 32m×32m，垂直动态 | 仅地表附近存体素 |
| **LOD** | 8 级 Clipmap（0-7）+ Signed Heightfield（3km+） | 最远 10km+ 远景 |
| **海洋** | Gerstner 程序化波，`OceanProvider` trait | 预留 FFT 升级接口 |
| **海平面** | 0.35（高程归一化值） | 低于此值为水域 |
| **山体阈值** | 0.72 | 山脉起始高程 |
| **聚落等级** | 村庄(60%) → 小镇(25%) → 城市(10%) → 城堡(5%) | 概率分布 |
| **矿田储量** | 一级(小型): 500-5,000 体素格 · 二级(中型): 5,000-50,000 · 三级(大型): 50,000-500,000+ | — |
