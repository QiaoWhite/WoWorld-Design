## 〇、注意力生命周期

```
                     ┌─────────────┐
      新奇/运动/威胁  │   Locked    │  注视锁——持续追踪
          ──────────→│  (HardLock) │←──────────
                     └──────┬──────┘            │
                            │ 目标消失/威胁解除  │
                            ↓                    │
                     ┌─────────────┐            │
                     │   SoftLock  │  软锁——仍在关注
                     └──────┬──────┘            │
                            │ 长时间无变化       │
                            ↓                    │
                     ┌─────────────┐            │
                     │   Residue   │  注意残留——短暂敏感
                     └──────┬──────┘            │
                            │ 残留期过           │
                            ↓                    │
                     ┌─────────────┐            │
                     │   Released  │  释放——重新竞争
                     └─────────────┘            │
                            └───────────────────┘
                              新刺激重新捕获
```

## 二、注意切换规则

```rust
fn should_switch_attention(current: &AttentionSlot, candidate: &PerceptEntry, npc: &NpcData) -> bool {
    match current.lock_type {
        HardLock => {
            // 只有更高优先级的Critical信号能打断
            candidate.priority >= Critical
        }
        SoftLock => {
            // 高优先级信号可以打断
            candidate.salience > current.salience * 1.5
        }
        NoLock => {
            // 任何更高显著性的新刺激
            candidate.salience > current.salience * 1.2
        }
    }
}
```

## 三、变化盲视——任务沉浸度

```rust
/// 任务沉浸度调制外周感知阈值
fn immersion_threshold(npc: &NpcData, current_action: &Action) -> f32 {
    let base = match current_action.category {
        Crafting | Smithing | Reading | Fishing => 0.7,  // 高沉浸活动
        Walking | Eating | BrowsingShop => 0.2,           // 低沉浸
        Guarding | Patrolling | Hunting => 0.1,           // 警觉活动——几乎不休
        Sleeping => 0.95,                                  // 几乎完全关闭
        _ => 0.4,
    };

    // 人格调制
    base * (0.8 + npc.personality.neuroticism * 0.4)  // 高神经质→更难沉浸
}
```

**涌现场景**：铁匠打铁→沉浸度0.7→外周感知阈值升高→玩家在门口晃五分钟才被注意到。

## 四、变化盲视——非注意盲视

专注于任务的 NPC 会对视野中的明显变化视而不见。不需要硬编码——从注意力负载自动涌现：

```rust
fn perceptual_threshold_multiplier(attention: &AttentionState) -> f32 {
    1.0 + attention.load * 2.0  // 高负载→阈值翻倍→更多刺激被过滤
}
```
