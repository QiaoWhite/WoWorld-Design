## 一、射线预算自适应公式

```rust
fn adaptive_ray_budget(scene: &VisualScene, attention: &AttentionState, combat: bool) -> u8 {
    let mut budget: i32 = 8;
    if scene.openness < 0.3 { budget += 4; }
    if scene.entity_density > 0.5 { budget += 2; }
    if combat { budget *= 2; }
    if attention.load > 0.7 { budget -= 3; }
    budget.clamp(1, 20) as u8
}
```

| 环境 | 预算 |
|------|------|
| 开阔平原，正常 | 8 |
| 密林/洞穴，正常 | 12 |
| 拥挤市场，正常 | 10 |
| 战斗，开阔 | 16 |
| 战斗，密林 | 24→clamp 20 |
| 高度专注(load>0.7) | 5 |
| 睡眠 | 1 |

## 二、射线分配优先级

```
分配顺序（用完budget即停）:
1. 注意锁定的实体 — 1条(HardLock) / 0条(SoftLock)
2. Motion/Threat信号 — 最多3条
3. GoalRelevant信号 — 最多2条
4. SociallySalient/Novel信号 — 最多1条
5. 随机扫视 — 剩余budget

随机扫视方向: facing ±15°内随机偏转。模拟无意识眼球运动(saccade)。
```

## 三、视线检测

```rust
fn line_of_sight_check(from, to, spatial, max_distance) -> bool {
    // 单次体素射线——利用体素场景的SDF或DDA
    // ~10µs 在稀疏体素上
    if distance(from, to) > max_distance { return false; }
    spatial.line_of_sight(from, to, max_distance)
}
```

透明实体（玻璃/水面/魔法屏障——`SpatialEntity.is_transparent=true`）不阻断视线。

## 四、天际线采样——自然奇观检测

```rust
struct HorizonSample {
    elevation_angle: f32,     // 天际线仰角（0=地平线）
    distance_km: f32,
    profile_roughness: f32,   // 0=光滑(海洋), 1=锯齿(山脉)
}

fn detect_natural_spectacle(pos, facing, spatial, weather) -> Option<NaturalSpectacle> {
    let azimuths = [-30, -15, 0, 15, 30].map(|d| facing_azimuth + d);
    let profiles: Vec<HorizonSample> = azimuths.map(|az| spatial.sample_horizon(pos, az)).collect();

    let max_elev = profiles.iter().map(|s| s.elevation_angle).max();
    let avg_elev = profiles.iter().map(|s| s.elevation_angle).sum::<f32>() / 5.0;

    // 山峰检测
    if max_elev - avg_elev > 0.15 && max_elev > 0.05 {
        return Some(MountainPeak { height: max_elev * profiles.max().distance_km * 1000.0 });
    }

    // 海洋检测
    if profiles.iter().all(|s| s.profile_roughness < 0.1 && s.elevation_angle.abs() < 0.02) {
        return Some(OceanHorizon);
    }

    None
}
```
