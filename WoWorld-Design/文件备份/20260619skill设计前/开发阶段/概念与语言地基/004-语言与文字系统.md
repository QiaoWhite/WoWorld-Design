## 二、语言 proficiency 的习得——从原子日志涌现

### 2.1 暴露计算

```rust
// woworld_core/src/language.rs  ← XD1修正: 移至woworld_core，纯函数
// TUNING常量定义在woworld_types::tuning中。
// NPC crate调用此纯函数，不依赖woworld_lang。

/// 从过去 24h 的原子执行日志中计算每种语言的暴露量
/// 不存在"路径"枚举——所有学习从 LISTEN/SPEAK/READ/WRITE/EXAMINE 原子涌现
pub fn compute_language_exposure(
    atom_log: &AtomExecutionLog,
    entity_index: &dyn EntityIndex,
) -> HashMap<LanguageId, f32> {
    let mut exposure: HashMap<LanguageId, f32> = HashMap::new();

    for entry in atom_log.last_24h() {
        match entry.atom_type {
            // LISTEN → 听觉语言输入
            AtomType::LISTEN => {
                if let Some(speech) = entry.perceived_speech() {
                    *exposure.entry(speech.language_id).or_default() +=
                        speech.clarity * speech.fraction_heard * LISTEN_WEIGHT; // 0.12
                }
            }
            // SPEAK → 主动语言实践（更强效果）
            AtomType::SPEAK => {
                if let Some(utterance) = entry.utterance() {
                    let interlocutor_prof = entry.interlocutor_proficiency(utterance.language);
                    let quality = 0.5 + interlocutor_prof * 0.5;
                    *exposure.entry(utterance.language).or_default() += quality * SPEAK_WEIGHT; // 0.20
                }
            }
            // READ → 文字语言暴露
            AtomType::READ => {
                if let Some(book) = entry.read_result() {
                    *exposure.entry(book.language).or_default() +=
                        book.comprehension() * READ_WEIGHT; // 0.15
                }
            }
            // EXAMINE → 微弱暴露
            AtomType::EXAMINE => {
                if let Some(lang) = entry.inscription_language() {
                    *exposure.entry(lang).or_default() += 0.03;
                }
            }
            // WRITE → 深度语言加工（最高权重）
            AtomType::WRITE => {
                if let Some(lang) = entry.writing_language() {
                    *exposure.entry(lang).or_default() += WRITE_WEIGHT; // 0.25
                }
            }
            _ => {}
        }
    }
    exposure
}
```

### 2.2 proficiency 更新

```rust
/// 每游戏日调用
pub fn update_language_proficiency(
    npc: &mut NpcData,
    atom_log: &AtomExecutionLog,
    entity_index: &dyn EntityIndex,
) {
    let exposure = compute_language_exposure(npc, atom_log, entity_index);

    for (lang, exp) in &exposure {
        let current = npc.language_proficiency(*lang);

        // 学习速率 = 暴露 × 认知弹性 × 语言学技能 × 语系迁移
        let rate = exp
            * cognitive_plasticity(npc)                         // 1 - cognitive_damping
            * linguistics_factor(npc)                           // 1 + Lv/100 * 0.5
            * family_transfer(npc, *lang)                       // 已知语言与目标语言的语系距离
            * developmental_window(npc);                        // 幼儿=更高，老年=更低

        // S 曲线——初期慢（啥也不懂），中期快（豁然开朗），末期慢（接近母语）
        let growth = rate * sigmoid_derivative(current) * PROFICIENCY_GROWTH_RATE; // [TUNING] 0.015

        npc.set_language_proficiency(*lang, (current + growth).min(1.0));
    }
}
```

### 2.3 涌现示例——为什么"师承"更快

```
NPC 今天的原子日志:
  3 次 LISTEN(北方语, clarity=0.7, fraction=0.6) → 3.6 × 0.12 = 0.432
  1 次 SPEAK(北方语, interlocutor_level=0.9)     → 0.95 × 0.20 = 0.190
  1 次 READ(北方语告示, comprehension=0.5)        → 0.5 × 0.15 = 0.075
  总暴露: 0.697

agent_snapshot:
  cognitive_plasticity = 0.85
  linguistics_factor   = 1.225  (linguistics Lv45)
  family_transfer      = 0.3    (中原语和北方语不同语系)
  developmental_window = 1.0    (年轻成年=峰值)

rate = 0.697 × 0.85 × 1.225 × 0.3 × 1.0 = 0.217

growth = 0.217 × sigmoid_derivative(0.0) × 0.015
       = 0.217 × 0.25 × 0.015 = 0.00081 / 天
→ 从 0 到 0.3 (基本交流): ~30天 ✓

关键: 没有一行代码说"师承=30天"。
30天是 LISTEN×3 + SPEAK×1(高质量) + READ×1 的自然涌现结果。
如果导师语言水平普通(interlocutor_level=0.5):
  SPEAK quality 降至 0.75 → 暴露降至 0.660 → ~37 天
```

## 三、语言衰减

```rust
/// 每游戏日调用
pub fn apply_language_decay(npc: &mut NpcData, atom_log: &AtomExecutionLog) {
    let exposure = compute_language_exposure(npc, atom_log, entity_index);

    for (lang, prof) in npc.known_languages.iter_mut() {
        if exposure.get(lang).copied().unwrap_or(0.0) < MAINTENANCE_THRESHOLD { // [TUNING] 0.01
            // 衰减: 每天 -0.0002 / proficiency
            let decay = DECAY_RATE / (*prof as f64).max(0.01); // [TUNING] 0.0002
            let floor = language_decay_floor(npc, *lang);
            *prof = (*prof - decay as f32).max(floor);
        }
    }
}

fn language_decay_floor(npc: &NpcData, lang: LanguageId) -> f32 {
    let age_when_learned = npc.language_acquisition_age(lang);
    if age_when_learned < 12.0 { 0.4 }          // 童年习得——永久保留
    else if age_when_learned < 18.0 { 0.25 }     // 青少年习得
    else { 0.05 }                                 // 成人习得——可能几乎完全忘记
}
```

**设计说明**: 语言有衰减——与技能系统的"只升不降"哲学不同。科学依据：语言是陈述性记忆（会衰减），技能是程序性记忆（永久保留）。

## 四、文字系统 proficiency

### 4.1 独立于口语 proficiency

`LanguageId` 管口语。`ScriptId` 管文字。`Proficiency` 两者独立。

```rust
pub struct ScriptProficiency {
    pub proficiency: f32,         // 0-1
    pub acquired_at: GameInstant,
    pub last_read_at: GameInstant,
}

// 文字暴露从原子日志涌现
fn compute_script_exposure(atom_log: &AtomExecutionLog) -> HashMap<ScriptId, f32> {
    let mut exposure: HashMap<ScriptId, f32> = HashMap::new();
    for entry in atom_log.last_24h() {
        match entry.atom_type {
            AtomType::READ => {
                if let Some(script) = entry.script() {
                    *exposure.entry(script).or_default() += entry.comprehension() * 0.15;
                }
            }
            AtomType::WRITE => {
                if let Some(script) = entry.script() {
                    *exposure.entry(script).or_default() += 0.25;  // 书写=深度加工
                }
            }
            AtomType::EXAMINE => {
                if let Some(script) = entry.script() {
                    *exposure.entry(script).or_default() += 0.03;
                }
            }
            _ => {}
        }
    }
    exposure
}
```

### 4.2 书籍理解——三维度

```rust
fn compute_reading_comprehension(book: &PhysicalBook, reader: &NpcData) -> f32 {
    let lang_prof = reader.language_proficiency(book.language);
    let script_prof = reader.script_proficiency(book.script);

    let concept_overlap = if let Some(encoding) = &book.source_concepts {
        let matches = classify_pattern(encoding.pattern_sig, &effective_concept_space(reader));
        matches.first().map(|(_, c)| *c).unwrap_or(0.0)
    } else { 0.5 };

    if lang_prof < 0.01 || script_prof < 0.01 { return 0.0; }
    lang_prof * script_prof * (0.5 + concept_overlap * 0.5)
}
```

## 五、概念习得

```rust
/// 每游戏日检查——NPC 是否学会新概念
pub fn update_concept_acquisition(npc: &mut NpcData, atom_log: &AtomExecutionLog) {
    for exposure in atom_log.concept_exposures() {
        if npc.concept_space_contains(exposure.id) { continue; }

        let accum = npc.concept_exposure_buffer.entry(exposure.id).or_default();
        *accum += exposure.salience;

        let threshold = concept_acquisition_threshold(npc);
        if *accum > threshold {
            npc.acquired_concepts.insert(exposure.id, AcquiredConcept {
                concept_id: exposure.id,
                source_culture: exposure.culture,
                acquired_at: GameInstant::now(),
                confidence: 0.3,  // 初始低置信——需要后续验证
                source: ConceptSource::Exposure,
            });
            npc.bump_concept_space_version();
        }
    }
    // 暴露缓冲区日常衰减
    for val in npc.concept_exposure_buffer.values_mut() {
        *val *= 1.0 - EXPOSURE_DECAY_PER_DAY;  // [TUNING] 0.05
    }
}

fn concept_acquisition_threshold(npc: &AgentSnapshot) -> f32 {
    let cognitive_mod = 1.0
        + (npc.cognitive_style.analytic_intuitive - 0.5) * 0.3   // 分析型更谨慎 → 阈值更高
        - (npc.cognitive_style.rigid_flexible - 0.5) * 0.4;       // 灵活型更开放 → 阈值更低
    CONCEPT_ACQUISITION_BASE_THRESHOLD * cognitive_mod  // [TUNING] 5.0
}
```

## 六、沟通可行性评估

```rust
/// 在生成 DialogueIntent 之前调用
pub fn communication_feasibility(speaker: &NpcData, listener: &NpcData) -> CommunicationFeasibility {
    let common_languages: Vec<(LanguageId, f32)> = speaker.known_languages.iter()
        .filter_map(|(lang, prof)| {
            let listener_prof = listener.language_proficiency(*lang);
            if listener_prof > 0.2 { Some((*lang, (prof.proficiency + listener_prof) / 2.0)) }
            else { None }
        })
        .collect();

    if common_languages.is_empty() {
        CommunicationFeasibility::Impossible {
            fallback: NonVerbalFallback::GestureAndPoint,  // → GESTURE 原子
        }
    } else {
        let best = common_languages.iter().max_by(|a, b| a.1.partial_cmp(&b.1).unwrap()).unwrap();
        CommunicationFeasibility::Possible { language: best.0, mutual_proficiency: best.1 }
    }
}
```
