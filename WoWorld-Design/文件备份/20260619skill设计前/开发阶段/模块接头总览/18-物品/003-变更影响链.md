# 物品系统 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 物品系统 v1.0 (CHG-014)
>
> 当物品系统发生变更时, 以下模块和概念会受影响.

## ItemDefId / ItemEntId 变更
→ **NPC模块**: 持有/交易/记忆 中的物品引用全部受影响
→ **战斗模块**: 武器/弹药/战利品 ID 格式
→ **魔法模块**: 法器/附魔物品 ID
→ **经济模块**: 交易/Contract 中的物品标识
→ **历史模块**: 铭文/遗物/墓碑中的物品引用
→ **载具模块**: 契书 VehicleDeed → ItemEntId 关联

## Quality × Rarity 变更
→ **经济**: 定价公式. Quality×Rarity → 价格基准
→ **NPC**: 物品偏好/贪婪度
→ **审美**: AestheticProps 映射

## 装备槽位 (BodyPlan派生) 变更
→ **NPC**: 装备管理/Outfit切换
→ **生命**: BodyPlan (woworld_core)
→ **模型物理**: 骨骼绑定点

## Assembly 组件树 变更
→ **战斗**: 武器模块化 slot_type 注册. ~20种组件类型
→ **魔法**: 法器组件树
→ **制造**: 配方组件组装

## Enchantment 双模式 变更
→ **魔法**: 附魔效果定义
→ **战斗**: 附魔战斗效果
→ **信仰**: DivineBlessingEnchantment → 通过附魔扩展点注册
→ **经济**: 卡槽附魔可撤换(日常经济) vs 直接附魔永久(历史锚点)

## ConsumableEffect 变更
→ **生命**: ingest_food() 参数
→ **NPC**: 进食决策 (need_action_match)

## 五层仓储体系 变更
→ **NPC**: 背包/库存管理
→ **载具**: 货舱 Container 接口

## WeaponPhysicalParams 映射表 变更 ⚠️ CHG-033
→ **模型物理**: 战斗轨迹适配器 (reach_scale/speed_scale/body_commitment)

## 制造配方 (CraftingRecipe) 变更
→ **NPC**: 制造行为
→ **技能**: crafting XP 获取
→ **文化**: TechnologyProfile → 可制造门槛

## 跨CHG影响
- **CHG-014**: 物品系统为权威规格
- **CHG-033**: BodyPlan 从 Life 提升至 woworld_core. 物品系统通过核心类型注册表引用
- **CHG-027**: ConsumableEffect schema 由 Life 定义, 物品只存不解
- **CHG-029**: 物品 AestheticProps → 审美 HasAestheticSignal impl
- **CHG-025**: 信仰物品扩展 (DivineRelicTemplate/FaithItemExtension) → 物品 tags
