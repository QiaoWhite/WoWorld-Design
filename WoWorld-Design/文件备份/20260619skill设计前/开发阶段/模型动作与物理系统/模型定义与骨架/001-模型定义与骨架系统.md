## 〇、核心架构

```
woworld_core（纯定义）:
  SkeletonDef, BoneDef, AnimationModules, AttachmentPoint
  ModelVariant enum, ModelDef, BodyPlan enum, PhysicalAppearance

woworld_model（启动时加载）:
  ModelRegistry: HashMap<ModelVariant, ModelDef>
  SkeletonRegistry: HashMap<SkeletonId, SkeletonDef>
  从 TOML 加载

Godot（渲染资产持有者）:
  GLB 文件→ArrayMesh RID→ModelRegistry 记录 RID 映射
  ModelVariant→RID 的静态 HashMap
```

Rust 说"渲染 ModelVariant::DwarfMale，这些是骨骼矩阵"。Godot 查映射表→取 RID→MultiMesh instance→GPU skinning shader。

### AnimationModules（骨架四分区）

```rust
pub struct AnimationModules {
    pub lower_body: Vec<usize>,   // pelvis→toes (10 bones)
    pub upper_body: Vec<usize>,   // spine_01→fingers (15 bones)
    pub head_neck: Vec<usize>,    // neck, head (2 bones)
    pub face_hands: Vec<usize>,   // jaw+brow+eye+thumb+index (L0: 9 bones, L1: 5)
}
```

每个 `ModulePose` 只操作一个模块。Lower 和 Upper 完全解耦。

## 二、Mesh LOD

| LOD | 距离 | 面数 | 骨骼 | 说明 |
|-----|------|------|------|------|
| **L0** | <10m | **1500** | 35 | 对话特写。全面部骨+3D眼珠。细节头/手 |
| **L1** | 10-50m | **800** | 33 | 主力。纹理面部。拳套手（拇指/食指分离） |
| **L2** | 50-150m | **300** | 28 | 无面骨/手指骨。四肢合并 |
| **L3** | 150-500m | **80** | 15 | 仅移动骨。粗略人形轮廓 |
| **L4** | 500m+ | **0** | 0 | Billboard/城市灯光统计锚点 |

L1 主力面数分解（800面）：头+发 180，颈 20，躯干 180，上臂 50×2，前臂 40×2，手 35×2，大腿 50×2，小腿 40×2，脚 25×2。

装备是**替换式**（非叠加）。头盔替换头发网格。胸甲替换躯干网格。全套重甲 ~600 面——比裸体更少。

## 三、面部——512² 纹理图集

### 为什么不用骨骼驱动

33 骨中面部只占 3-7 根。骨骼驱动的表情自由度受骨数限制。TABS 式低多边形 + flat/cel 渲染下，**纹理图集的表情丰富度远超骨骼**，且单张 512² 图集全部人形角色共享（~4MB VRAM）。

### 图集区域

| 区域 | 变体数 | 内容 |
|------|--------|------|
| 嘴型（Row 0-1） | 16 | 闭/微开/大笑/噘/咬/吐舌/5 phoneme/惊/怒 |
| 眉型（Row 2-3） | 16 | 平/抬/皱/八字/挑/压 |
| 眼态（Row 4） | 8 | 全开/半眯/闭/怒/惊 |
| 虹膜（Row 5） | 8 | 棕/蓝/绿/灰/金/红/白/异色 |
| 叠加（Row 6） | 64+ | 伤疤(32)+战纹(16)+纹身(16) |

### 运行时——6 字节/ NPC

```rust
#[repr(C)]
pub struct FacialExpression {
    pub mouth_shape: u8,      // 0-15 → mouth atlas row
    pub brow_shape: u8,       // 0-15 → brow atlas row
    pub eye_state: u8,        // 0-7 → eye atlas row
    pub iris_offset_x: i8,    // -127~127 → iris UV shift
    pub iris_offset_y: i8,    // -127~127
    pub blush_intensity: u8,  // 0-255 → vertex color red tint
}
// 塞入 MultiMesh INSTANCE_CUSTOM（16B vec4，绰绰有余）
```

Shader 采样：
```glsl
vec2 mouth_uv = base_uv + vec2(0.0, float(mouth_shape) * MOUTH_ROW_HEIGHT);
vec4 mouth_color = texture(facial_atlas, mouth_uv);
```

## 四、非人形骨架——自动生成

12 种非人形身体计划从 BodyPlan + SizeParams 程序化生成骨架。缓存：`HashMap<SkeletonId, Arc<SkeletonDef>>`。

| 身体计划 | 策略 |
|---------|------|
| Quadrupedal | +前肢，脊柱水平，脊椎波 |
| Serpentine | 长脊柱链，无肢体 |
| Avian | 翼骨（改造前肢），空心骨 |
| Centauroid | Bipedal upper + Quadrupedal lower，脊柱桥接 |
| Aquatic | 尾鳍驱动，身体波 |

骨架 ID = `BodyPlan × SizeParams.hash`。同物种个体共享骨架。

## 五、Blender→游戏管线

```
Blender（.blend）
  ├─ 建模 + 蒙皮（标准骨架骨骼名）
  ├─ vertex color（肤色基调）
  ├─ 自定义属性 "face_vertex"=true（标记面部顶点）
  └─ 导出 → GLB (binary glTF)

Python 导出脚本（~150 行 bpy）:
  ├─ 检查骨骼名匹配 SkeletonDef
  ├─ 4骨蒙皮→2骨蒙皮自动裁切（最小二乘拟合）
  └─ 自定义属性写入 GLB extras

Godot 启动:
  └─ GLTFDocument.load() → ArrayMesh RID → 缓存到 ModelRegistry
```

## 六、装备挂载

```rust
pub struct EquipmentMounting {
    pub replacements: Vec<BodyPartReplacement>,
    pub attachments: Vec<AttachmentTransform>,

pub enum ReplacementMode {
    Replace,   // 头盔=替换头发。胸甲=替换躯干
```

武器挂载点：weapon_r（R_hand 的子骨），shield_l（L_hand），back_slot（spine_02）。WeaponPhysicalParams（长度/重量/重心/握持位置）→驱动轨迹适配。

## 七、渲染风格

TABS 式：flat/cel shading，仅 diffuse pass。512² 纹理图集全人形共享。vertex color 提供肤色变化。**无 PBR**（无 normal/metallic/roughness/AO）。VRAM 角色纹理 <5MB（vs 原预算 300MB）。
