## 魔法核心原理

**两条公理**：
1. **二元构成原理**：万物由十元素（金木水火土风雷电血灵）构成
2. **意志投射原理**：施法 = 灵元素(意志)通过血元素桥梁 → 操控外界元素

**魔法方程式**：范围+对象+接触+元素+影响+时间+条件

## 与 NPC 活人感的接口

- NPC 的灵元素亲和度 = 人格+先天+训练 → 决定施法能力上限
- MagicUser 是 NPC 的一种职业路径（和其他职业平等）
- 法术熟练度 = 使用次数 → 自动化降级（战略→节奏→本能）
