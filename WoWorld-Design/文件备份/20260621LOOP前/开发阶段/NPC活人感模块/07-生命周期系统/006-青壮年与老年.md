## 一、青壮年：一生中最长的阶段

### 1.1 三个阶段合一的设计理由

YoungAdult + Adult + MiddleAge 合并为"青壮年"，不是因为它们在生物学上没有区别——而是因为在**玩家体验和游戏性**上，这是 NPC 能力的"满开"区间：

- YoungAdult（25-40% 寿命）：属性达 90-100%，开始独立生活、成家、选择职业
- Adult（40-70% 寿命）：100% 属性，生产力巅峰，大部分生育发生在此
- MiddleAge（70-85% 寿命）：属性微降 0-10%，经验/智慧继续增长，开始出现衰退征兆

玩家看到的世界应该"青春洋溢、朝气蓬勃"——这通过拉长青壮年区间和在世界生成时让这个区间的 NPC 占人口多数来实现。

### 1.2 参数演化——不是静止的平台期

这三个子阶段内，参数仍在演化——只是变化缓慢：

| 参数 | YoungAdult | Adult | MiddleAge |
|------|-----------|-------|-----------|
| 物理属性 | 90→100% | 100% | 100→90% |
| 生育能力 | 上升→峰值 | 峰值 | 峰值→开始衰退 |
| 技能XP | 快速积累 | 基础积累 | 积累→偶尔衰退 |
| 认知 plasticity | 从高下降 | 基线 | 继续轻微下降 |
| MentalModel 数量 | 快速增加 | 缓慢增加→稳定 | ≤20，偶尔替换 |
| 社会角色 | 建立 | 稳固 | 开始向后辈转移 |

## 二、老年：衰退但非静止

### 2.1 物理属性衰退

```rust
// Elder 阶段的属性乘数不是一条直线——是加速递减的 sigmoid 尾部
fn elder_attribute_multiplier(age_pct: f32) -> f32 {
    if age_pct < 0.85 { return 1.0; }
    let x = (age_pct - 0.85) / 0.15;  // 0→1 跨过 Elder 区间
    let decay = 1.0 / (1.0 + (x * 3.0).exp());  // sigmoid 从 1→~0.5
    decay * 0.9 + 0.1  // 最终不会降到 0 以下
}
```

不同属性的衰退速度不同：
- **力量和耐力**：衰退最快（线性下降）——老人搬不动重物
- **敏捷**：中度衰退——反应速度下降
- **感知**：感官退化——视觉/听觉阈值上升
- **体质**：衰退——更容易受伤、恢复更慢
- **智慧/判断**：不衰退（或继续增长直到很晚期）

### 2.2 认知老化三路径

```rust
pub enum CognitiveAgingPath {
    Healthy,        // ~85% 的 NPC。认知能力缓慢线性下降，主要在速度维度
    Pathological,   // ~5-8% 的 NPC。某几个维度急剧下降（记忆/方向感/识别）
    SuperAging,     // ~2-3% 的 NPC。几乎不下降。智慧继续累积到临终
}

fn derive_aging_path(cognitive_style: &CognitiveStyle, life_history: &LifeHistory) -> CognitiveAgingPath {
    let seed_baseline = cognitive_style.crystallized_factor();  // 0.3-0.7
    let activity_bonus = life_history.cognitive_engagement_score();  // 终身累积
    // cognitive_engagement_score 从以下派生：
    // - 一生中技能 XP 的总量和多样性
    // - MentalModel 的创建频率和质量
    // - 社交互动密度
    // - 是否从事认知密集型职业
    let adjusted = (seed_baseline + activity_bonus * 0.3).clamp(0.0, 1.0);
    
    // 概率抽样
    let r = rand::random::<f32>();
    if adjusted > 0.8 && r < 0.3 {
        CognitiveAgingPath::SuperAging
    } else if adjusted < 0.3 && r < 0.3 {
        CognitiveAgingPath::Pathological  
    } else {
        CognitiveAgingPath::Healthy
    }
}
```

**路径不由 mental_seed 固定**——终身认知活跃度是关键变量。一个终生读书、写作、辩论的老人，比一个终生做单调重复劳动的老人，更可能进入 SuperAging 路径。

### 2.3 老年人的 GOAP——从生产者到传承者

老年 NPC 的 GOAP 输出变化不由"老年模式"——由物理属性下降驱动动作选择的自然改变：

```
以前: stamina=0.9, strength=0.95
  → GOAP "砍一天柴"→可行
现在: stamina=0.4, strength=0.5
  → GOAP "砍一天柴"→成本太高、成功率太低→放弃
  → GOAP "在门口晒太阳"→可行
  → GOAP "给孙子讲故事"→可行（MentalModel 中有几十年的故事可以讲）
  → GOAP "教徒弟手艺"→可行（技能高 + 有时间）
```

**老年人"爱回忆"不是因为"年龄到了所以系统插入回忆行为"**——是因为：
1. 体能不能支持高强度活动→GOAP 自然选择低强度替代
2. MentalModel 已满 20 条且置信度高→新归纳越来越少
3. 已有 MentalModel 中沉淀了数十年经验→"讲故事"是把这些经验打包传递的最有效方式
4. LifeTheme 可能在 Legacy 阶段——主动追求"把东西传下去"

## 三、LifeTheme——叙事驱动的阶段感

### 3.1 现有 LifeTheme 兼容性

NPC ver2.0 的 SelfNarrative 已经定义了以下 LifeTheme 转换：

| Transition | 触发条件 | 维持 |
|-----------|---------|------|
| Exploration→Building | 任一 aspiration 进度 > 0.5 | 直到遇到危机或 stagnation |
| Building/Mastery→Crisis | stagnation>0.7 ∧ age>adult_age×1.5 | 中年危机的涌现——不是硬编码 45 岁 |
| Crisis→Exploration/Mastery | 取决于 openness | 重评后进入新方向 |
| →Decline | age>70%max ∧ body_decline>20% | 身体衰退驱动——叙事转向"我老了" |
| →Legacy | age>85%max ∨ closeness_to_death>0.8 | 叙事转向"我留下了什么" |

### 3.2 生命周期模块与 LifeTheme 的关系

生命周期模块不定义 LifeTheme——它提供 LifeTheme 的**输入信号**：

```
LifeStageTransition(Elder) → NPC 的 SelfNarrative 在下次 7 天反思中
  可能将 LifeTheme 评估为 Decline 或 Legacy
  （取决于 body_decline 程度 + closeness_to_death 认知）

closeness_to_death 不是生命模块给的——是 NPC 自己的认知系统
  从"同龄人接连去世" + "身体持续恶化" + "MentalModel 中关于寿命的认知"
  中自己归纳出的评估
```

两个同岁的老年 NPC——一个认为自己快死了（closeness_to_death=0.9），另一个还在兴致勃勃计划明年的菜园（closeness_to_death=0.1）。不是因为年龄不同——是因为认知加工不同。

## 四、无子女老人的晚年照料

### 4.1 照料是涌现的

一个无子女的独居老人：

```
物理能力：stamina<0.3, 无法远行寻找食物
情感：可能孤独，可能不在意

附近 NPC 的反应：
  if 文化中 elder_care_expectation 高 && 社区中有高 agreeableness NPC：
    → 该 NPC 的 GOAP 可能出现 "CareForElder" 目标
    → 送食物/帮忙修补房顶/陪说话
  if 文化个人主义主导 || 所有邻里都忙于自己的生存：
    → 没人来
    → 老人可能饿死、冻死、或孤独离世
```

**没有"老人保护系统"。** 被照顾或不被照顾取决于周围人的选择——这些选择又从这些人的 BigFive、关系、处境、文化规范中涌现。

### 4.2 文化中的老年照料规范

```rust
// 从 CultureCoreParams 派生——不新增独立参数
pub fn elder_care_expectation(culture: &CultureCoreParams) -> f32 {
    // 长期导向 → 重视代际关系 → 更可能照料老人
    // 个人主义 ← 相反
    culture.long_term_orientation * 0.7
    + (1.0 - culture.individualism) * 0.3
}
```

## 五、青少年期——社会身份重构

### 5.1 Adolescent 阶段的社会维度

Adolescent 阶段除了已有的审美结晶和认知初始化外，还有社会维度的变化：

```
Adolescent NPC 的 SocialIdentity 变化：
  peer_group_reference: family → peer_cohort（同龄群体）
  反叛因子（从 BigFive.openness × low_agreeableness 派生）：
    → 有意识地做和父母相反的事
    → 父母最看重的信念被主动降低采纳率
    → 使用风格上刻意和父母区分开来
  
  AttractionTemplate：
    absorb_culture() → 青春期特有的性/浪漫取向塑形
```

### 5.2 反叛因子泛化

现有的审美"反叛因子"应该泛化到所有 MentalModel 继承领域：

```rust
/// 青少年反叛——MentalModel 继承的负向版本
fn adolescent_rebellion_modifier(npc: &NpcData, parent_model: &MentalModel) -> f32 {
    // 反叛强度
    let rebel = npc.big_five.openness * 0.5 
              + (1.0 - npc.big_five.agreeableness) * 0.3
              + npc.rebellion_factor.strength * 0.2;
    
    // 如果父母的 MentalModel 被标记为"父母最看重的" →
    // 反叛者主动降低采纳率
    if parent_model.is_highly_valued_by_parent 
       && rebel > 0.5 
       && npc.life_stage == LifeStage::Adolescent 
    {
        -rebel * 0.5  // 负向调整——甚至可能采用相反信念
    } else {
        1.0  // 正常继承
    }
}
```

**反叛不是系统效果**——是 Adolescent 阶段特有的认知处理偏差。同一个 NPC 进入 YoungAdult 后，反叛因子自然下降。

## 六、更年期——生育能力的连续终结

```rust
/// 生育能力——独立于 libido 的连续衰退曲线
/// 一个 NPC 可以在生育能力接近 0 的时候仍然有正常的 libido
/// "想"和"能"是分开的信号
pub fn fertility_potential(age_pct: f32, race: &RaceTraits) -> f32 {
    if age_pct < race.life_stage_distribution.young_adult_start() {
        return 0.0;  // 青春期前不可
    }
    
    if age_pct < race.life_stage_distribution.adult_start() {
        // YoungAdult: 线性上升至峰值
        let range = race.life_stage_distribution.adult_start() 
                  - race.life_stage_distribution.young_adult_start();
        (age_pct - race.life_stage_distribution.young_adult_start()) / range
    } else if age_pct < race.life_stage_distribution.middle_age_start() {
        1.0  // Adult: 峰值平台
    } else {
        // MiddleAge→Elder: sigmoid 下降
        let x = (age_pct - race.life_stage_distribution.middle_age_start())
              / (1.0 - race.life_stage_distribution.middle_age_start());
        1.0 / (1.0 + (x * 8.0).exp())
    }
}
```

注意：fertility_potential 降到 ~0 时，libido 仍可能为正。一个 65 岁的人类老人的 libido 可能是 0.1-0.3，但 fertility_potential 已接近 0。这不是系统矛盾——这是人类生物学。"想"还在，"能"已经不在。

## 七、老年与创新的涌现关系

### 7.1 创新不拒绝老年人

创新管线（认知系统已有）不由年龄筛选。一个 80 岁的 Elder NPC 仍然可以产生 InnovationStage::Illumination：

- 如果他的 MentalModel 网络丰富且多领域交叉
- 如果他仍在从事认知密集型活动
- 如果他的 CognitiveAgingPath 是 SuperAging

他的创新可能比年轻时更"智慧"——不是更快的突破，而是更深的联系。这不是系统给予的"老人智慧 buff"——是 MentalModel 网络的长期沉淀质变。
