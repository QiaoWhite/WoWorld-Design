## 一、ResolvedAtmosphere —— 核心产出

本模块的单一职责：每帧产出一个 `ResolvedAtmosphere`，Godot 消费它以驱动全部渲染参数。

```rust
/// 完整氛围描述——17 个渲染参数
/// Copy 类型·68 bytes·栈传递
/// Rust 每帧计算一次·<0.02ms
#[derive(Copy, Clone)]
pub struct ResolvedAtmosphere {
    // ── 地面层 ──
    pub shadow_tint: [f32; 3],      // 暗面色调 RGB
    pub ambient_tint: [f32; 3],     // 环境光色调 RGB
    pub saturation_mod: f32,        // 饱和度乘数（1.0=不变）
    pub contrast_mod: f32,          // toon 对比度（1.0=不变, <1=柔和, >1=强烈）
    pub night_sat_r: f32,           // 夜间红色饱和度乘数
    pub night_sat_g: f32,           // 夜间绿色饱和度乘数
    pub night_sat_b: f32,           // 夜间蓝色饱和度乘数
    pub floor_brightness: f32,      // 全黑场景最小可读亮度

    // ── 尘土·雾层 ──
    pub dust_color: [f32; 3],
    pub dust_density: f32,
    pub dust_height: f32,
    pub fog_color: [f32; 3],
    pub fog_density_base: f32,
    pub fog_height_floor: f32,
    pub fog_height_ceiling: f32,
    pub fog_vertical_gradient: f32,  // 垂直密度梯度·负值=向下渐浓

    // ── 大气透视 ──
    pub aerial_tint: [f32; 3],      // 远景色调偏移
    pub aerial_blur: f32,           // 远景模糊（日式异世界=低模糊+强色调）

    // ── 天空层 ──
    pub sky_zenith_tint: [f32; 3],
    pub sky_horizon_tint: [f32; 3],
    pub night_sky_brightness: f32,
    pub night_sky_tint: [f32; 3],

    // ── 发光·轮廓 ──
    pub bloom_threshold: f32,       // bloom 触发亮度阈值
    pub rim_strength: f32,          // Fresnel rim 基础强度
    pub rim_color_source: RimColorSource,
}

pub enum RimColorSource { PrimaryLight, FixedWarm, FixedCool }
```

## 二、四层调制

```
ResolvedAtmosphere = biome_base × time_curve(hour) × weather_curve(weather) × season_curve(season)

× = 乘算叠加, 非替换。
每层是一个调制乘数集合——独立调制 saturation、contrast、fog_density 等参数。
```

### 2.1 层 1：biome_base（biome_atmosphere.toml）

每个群系一组 17 参数基线。玩家所在位置的群系从 World Gen 的连续参数场双线性插值——边界过渡平滑。

```toml
# biome_atmosphere.toml (摘录)
[biome.temperate_forest]
label = "温带阔叶林"
shadow_tint = [0.12, 0.18, 0.22]     # 暗部偏蓝绿
ambient_tint = [0.10, 0.18, 0.10]    # 环境光微绿
fog_color = [0.35, 0.48, 0.55]
fog_density = 0.15
fog_height_floor = 0.0
fog_height_ceiling = 60.0
saturation_mod = 1.05                 # 微增饱——森林鲜活
contrast_mod = 0.95                   # 软对比——树荫下硬影更少
night_sat_r = 0.55
night_sat_g = 0.35                    # 月光下绿色降得最多
night_sat_b = 0.75
aerial_tint = [0.38, 0.48, 0.62]     # 远景偏蓝
aerial_blur = 0.08
sky_zenith_tint = [0.30, 0.42, 0.65]
sky_horizon_tint = [0.48, 0.55, 0.68]
night_sky_brightness = 0.12
floor_brightness = 0.015
bloom_threshold = 0.7
rim_strength = 0.12
rim_color_source = "primary_light"
```

12 个群系全部在 TOML 中定义。未指定的群系回退 `[default]`。

### 2.2 层 2：time_curve（time_curve.toml）

6 个时间锚点——锚点间线性插值。控制日光的色温、强度和环境光变化。

```toml
# time_curve.toml (摘录)
[time.dawn]   # hour 5.83
hour = 5.83
sun_color = [0.9, 0.6, 0.35]          # 粉橙
sun_intensity = 0.3
ambient_color = [0.15, 0.08, 0.12]    # 淡紫
ambient_intensity = 0.3
shadow_strength = 0.5
saturation_mult = 0.7
contrast_mult = 0.85

[time.dusk]   # hour 17
hour = 17
sun_color = [1.0, 0.5, 0.25]          # 橙金——黄金时刻
sun_intensity = 0.6
ambient_color = [0.45, 0.3, 0.25]     # 暖粉
ambient_intensity = 0.5
shadow_strength = 0.9
saturation_mult = 1.05
contrast_mult = 1.1
```

### 2.3 层 3：weather_curve（weather_curve.toml）

天气条件→调制乘数。不替换——只乘算前两层的聚合值。

```toml
[weather.storm]
label = "暴风雨"
sun_mult = 0.05
ambient_mult = 0.7
shadow_strength_mult = 0.2
fog_density_mult = 2.5
saturation_mult = 0.6
contrast_mult = 0.5
aerial_blur_mult = 2.0
rim_strength_mult = 0.3
```

### 2.4 层 4：season_curve（season_modulation.toml）

四季循环——消费已有的 `WeatherQuery::current_season()`。已有系统定义 120 游戏天/年、均匀四季、春分开局。

```toml
[season.autumn]
saturation_mult = 0.90
ambient_warmth_add = 0.08     # 金秋
sun_intensity_mult = 0.75

[season.winter]
saturation_mult = 0.70
ambient_warmth_add = -0.05    # 冷白
sun_intensity_mult = 0.55
```

## 三、计算流程

```rust
/// 每帧一次——<0.02ms
fn resolve_atmosphere(
    pos: Vec3,
    biome_field: &dyn BiomeQuery,          // 已有——World Gen 产出的连续参数场
    weather: &dyn WeatherQuery,            // 已有 trait
    time: GameTime,
    sky_event: Option<&ResolvedSkyEvent>,   // SkyEvent 覆盖（可选）
    ocean: Option<&dyn OceanProvider>,      // 已有——用于水下判定
) -> ResolvedAtmosphere {
    // 1. 群系基值——双线性插值
    let biome = biome_field.sample_biome(pos);
    let base = biome_atmosphere_config.get(&biome.tag)
        .unwrap_or(&biome_atmosphere_config.default);

    // 2. 时间调制
    let hour = time.hour_fractional();
    let time_mod = time_curve.interpolate(hour);

    // 3. 天气调制
    let weather_state = weather.current_weather(pos, time);
    let weather_mod = weather_curve.get(&weather_state);

    // 4. 季节调制
    let season = weather.current_season(time);
    let season_mod = season_curve.get(&season);

    // 5. 四层乘算
    let mut resolved = base.clone();
    resolved.saturation_mod *= time_mod.saturation_mult * weather_mod.saturation_mult * season_mod.saturation_mult;
    resolved.contrast_mod *= time_mod.contrast_mult * weather_mod.contrast_mult * season_mod.contrast_mult;
    // ... 其余参数同理

    // 6. 水下覆盖（如果有海洋且相机在水下）
    if let Some(ocean) = ocean {
        let water_h = ocean.wave_height(pos.x, pos.z, time.as_secs_f64());
        if pos.y < water_h {
            let depth = water_h - pos.y;
            let underwater = resolve_underwater(depth);
            let t = (depth / 0.5).min(1.0);  // 0.5m 过渡带
            resolved = resolved.lerp(&underwater, t);
        }
    }

    // 7. SkyEvent 覆盖（如果有）
    if let Some(event) = sky_event {
        resolved = resolved.lerp(&event.params, event.blend_factor);
    }

    resolved
}
```

## 四、GDExtension 传输

```rust
#[gdextension]
fn get_atmosphere() -> PackedFloat32Array {
    let atm = ATMOSPHERE_MANAGER.resolve();  // <0.02ms
    PackedFloat32Array::from(atm.as_float_array())  // 17 floats
}
```

Godot 每帧调一次——不是 17 次独立调用。68 bytes。全局 shader uniform + WorldEnvironment 在一次调用中设置完毕。
