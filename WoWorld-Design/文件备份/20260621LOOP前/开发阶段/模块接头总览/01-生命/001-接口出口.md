## AgeClock (年龄时钟) ⚠️ 新增
- **概念**: advance_age(entity, delta_days, rng) → (LifeEntity, Vec<LifeEvent>). 纯函数。同输入+同种子→同输出。与调用频率无关
- **Owner**: [[../../../生命/013-生命周期时钟与事件|生命 013（新增）]]
- **类型**: Rust struct + impl, woworld_life
- **消费者**: LOD 调度器（各层调用） · [[../03-世界生成/002-接口入口|世界生成 P9]]
- **契约**: CHG-041 (本模块)

## LifeEvent (生命周期事件) ⚠️ 新增
- **概念**: StageTransition / Birth / Death / GestationEnded. 中性事实——不携带情感建议/行为期望/人际通知指令
- **Owner**: [[../../../生命/013-生命周期时钟与事件|生命 013（新增）]]
- **类型**: Rust enum, woworld_life
- **消费者**: EventBus → 各模块独立订阅（NPC/历史/权力/信仰/经济/模型物理/空间查询/战斗）
- **契约**: CHG-041 (本模块)

## GestationState / FetusBlueprint ⚠️ 新增
- **概念**: GestationState { progress, duration_days, fetuses: Vec<FetusBlueprint> }. FetusBlueprint { physical_seed, mental_seed, elemental_seed, mother_id, father_id, race, race_rule }. 受孕时确定种子——外观通过 age_factor 插值派生
- **Owner**: [[../../../生命/004-身体状态与生命过程|生命 004]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC 生命周期]] 产前影响积累 · 出生事件
- **契约**: CHG-041 (本模块)

## FertilityPotential (生育能力连续曲线) ⚠️ 替代 FertilityDrivers
- **概念**: fertility_potential(age_pct, race) → f32. 0→峰值→0 连续 sigmoid。替换旧 FertilityDrivers 集中式 struct。libido 和 fertility 是分离信号——"想"和"能"独立
- **Owner**: [[../../../生命/004-身体状态与生命过程|生命 004]]
- **消费者**: 受孕概率计算 · NPC 通用认知管线（生育欲望无专用通道）
- **⚠️ CHG-041**: FertilityDrivers (生命 012) 退役。生育欲望不再有专用公式——走通用认知管线

## InfantDependency (营养依赖状态) ⚠️ 新增
- **概念**: Nursing { mother_id, last_fed } → Weaning { mother_id, independence } → Weaned. 存储在 LifeEntity 上
- **Owner**: [[../../../生命/004-身体状态与生命过程|生命 004]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] L1 母亲 GOAP · [[../03-世界生成/002-接口入口|世界生成]] L3/L4 统计近似
- **契约**: CHG-041 (本模块)

## DeathSummary (死亡摘要) ⚠️ 新增

- **Owner**: [[../../../生命/004-身体状态与生命过程|生命 004]]
