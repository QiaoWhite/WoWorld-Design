# 历史模块 — 变更影响链

> **最后验证**: 2026-06-18 | **源文档版本**: 历史系统 v1.0 (CHG-013 基座 + 后续扩展)

## 影响链总览

```
历史系统变更
├── AetherImprint 结构变更
│   ├── → 生命: CachedImprintView 缓存
│   ├── → 魔法: 灵元素魔法
│   └── → NPC: 灵敏感知
├── AetherQuery trait 方法增删
│   ├── → 生命
│   ├── → 魔法
│   └── → NPC
├── HistoricalEvent 结构变更
│   ├── → 文化: CulturalShift 事件
│   ├── → 权力: Polity 事件
│   ├── → 信仰: 宗教事件
│   ├── → 世界生成: P9 引擎
│   └── → 经济: EconomicEventSink
├── CausalEdge 结构变更
│   ├── → 文化: CultureGenealogy 因果链
│   ├── → 信仰: FaithGenealogy 因果链
│   └── → 权力: Polity 兴衰因果
├── 双层历史边界变更
│   ├── → 世界生成: P9 管线
│   ├── → NPC: 历史知识/记忆
│   └── → 全部: HistoricalEvent 查询
├── 三层因果模型变更
│   ├── → 文化: 演变驱动
│   ├── → 权力: 政治演变
│   └── → 信仰: 宗教演变
├── PhysicalBook 结构变更
│   ├── → 语言表达: ExpressionRef 内容
│   ├── → 物品: 书本物品
│   ├── → NPC: 阅读行为
│   └── → 经济: 书本交易
├── LifeTrace 结构变更
│   ├── → 生命: DeathCause → 墓碑映射
│   ├── → NPC: 故人记忆
│   └── → 文化: 葬仪
├── HistorySimulationConfig 参数变更
│   ├── → 世界生成: P9 引擎
│   ├── → 文化: 四路径演变
│   └── → 天气: 气象异常查询
├── HistoricalWeatherAnomaly 消费变更
│   └── → 天气: 数据格式兼容性
├── 墓碑系统变更
│   ├── → 生命: 死亡触发
│   ├── → NPC: 祭祀/记忆
│   ├── → 信仰: FuneralCustoms
│   └── → 文化: honor_weight
└── 事件存储格式变更
    └── → 全部: 事件序列化兼容
```

## 变更风险等级

| 变更范围 | 风险等级 | 影响模块数 | 备注 |
|---------|---------|-----------|------|
| HistoricalEvent 结构变更 | 🔴 高 | 5+ | 所有历史事件消费者 |
| AetherImprint 格式变更 | 🟡 中 | 3 | 灵元素消费者 |
| 双层历史边界调整 | 🟡 中 | 3 | 世界生成 P9 管线 |
| 三层因果模型参数 | 🟢 低 | 3 | [TUNING] 参数 |
| PhysicalBook 字段变更 | 🟡 中 | 4 | 物品/语言/经济联动 |
| LifeTrace 字段变更 | 🟡 中 | 3 | 墓碑/记忆/葬仪 |
| AetherQuery 新增方法 | 🟢 低 | 3 | 默认实现可得 |
| HistorySimulationConfig | 🟢 低 | 3 | [TUNING] 参数 |

## 事件来源全景

| 事件来源模块 | 事件类型 | 历史消费方式 |
|-------------|---------|------------|
| 生命 | DeathCause / BirthEvent | HistoricalEvent 转化 |
| NPC | skills_snapshot / 重要人生节点 | LifeTrace 记录 |
| 权力 | Polity 涌现/变迁/消亡 | Political HistoricalEvent |
| 战斗 | BattleResultNotification | Military HistoricalEvent |
| 经济 | EconomicEventSink | Economic HistoricalEvent |
| 文化 | CulturalShift / CultureGenealogy | Cultural HistoricalEvent |
| 信仰 | FaithGenealogy 谱系边 | Religious HistoricalEvent |
| 天气 | HistoricalWeatherAnomaly | Weather HistoricalEvent |
