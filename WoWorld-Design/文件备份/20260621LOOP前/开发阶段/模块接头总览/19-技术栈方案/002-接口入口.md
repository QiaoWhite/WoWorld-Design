# 技术栈方案 — 接口入口

> **最后验证**: 2026-06-18 | **源文档版本**: 技术栈 v4.0 (CHG-038 审计升级)

## 基础设施消费
- **架构**: Rust模拟核心 → GDExtension → Godot客户端 (渲染/UI/音频/输入/玩家物理)
- **数据流**: woworld_core (共享类型) → woworld_spatial (空间管理) → 各crate
- **平台**: Windows / Linux / macOS
- **美术管线**: AI生成(Stable Diffusion) + 手动调整(Blender)
