## 三、woworld_core 新增/变更类型清单

### 3.1 EntityId — 统一实体标识符（变更）

```rust
/// bits [63]:  0=Item(ItemEntId), 1=NonItem
/// bits [62:60]: (NonItem) entity_kind
/// bits [59:0]:  local_id
pub struct EntityId(pub u64);

impl EntityId {
    pub const fn from_item(item_id: ItemEntId) -> Self { /* bit63=0 */ }
    pub fn is_item(&self) -> bool;
    pub fn entity_kind(&self) -> Option<EntityKind>;
}

pub enum EntityKind {
    Creature = 0,           // 动物/NPC/怪物 — 有 BodyState + AI
    BuildingComponent = 1,  // 建筑组件 — WFC 生成
    TerrainFeature = 2,     // 地质特征 — 矿脉/岩层/黏土/巨石/洞穴
    DroppedItem = 3,        // 掉落物 — 物理刚体
    Plant = 4,              // ★CHG-051: 植物个体 — T1可采集+T3培育+巨树/世界树
                            // Life 模块管理，通过 EntityIndex 暴露给采集配方表
}
```

### 3.4 Surface trait（新增）

```rust
pub trait Surface: Send + Sync {
    fn id(&self) -> SurfaceId;
    fn orientation(&self) -> SurfaceOrientation;
    fn normal(&self) -> Vec3;
    fn elevation_at(&self, local_pt: Vec2) -> f32;
    fn load_bearing_kpa(&self) -> f32;
    fn contains_footprint(&self, aabb: &Aabb2D) -> bool;
    fn usable_area(&self) -> (f32, f32);
}
```
