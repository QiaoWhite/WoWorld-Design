## 〇、总览

所有感官信道均注入受控的伪随机噪声，以产生认知偏差。噪声必须是**确定性可复现的**——同一存档、同一时刻、同一 NPC 加载时知觉完全相同。

---

## 一、确定性种子管理

```rust
pub struct PerceptualSeed {
    world_seed: u64,         // 世界种子——存档创建时确定
    npc_permutation: u64,    // NpcId hash——终身不变
    time_quantum: u32,       // 决策周期量子化——防止浮点时间不一致
}

impl PerceptualSeed {
    pub fn for_modality(&self, modality: Modality) -> impl Rng {
        Pcg64::seed_from_u64(hash_combine(
            self.world_seed,
            self.npc_permutation,
            self.time_quantum,
            modality as u64,       // 每种模态不同salt→噪声在模态间独立
        ))
    }
}
```

## 二、噪声函数族

| 噪声类型 | 数学形式 | 应用场景 |
|---------|---------|---------|
| 加性高斯 | `estimate + N(0, σ²)` | 距离估计、方向估计 |
| 乘性缩放 | `estimate × (1 + N(0, σ²))` | 尺寸估计、人群计数 |
| 分类混淆 | `P(误认B为A) ∝ f(相似度, 压力)` | 低清晰度下的分类 |
| 时间扭曲 | `duration × (1 + N(0, σ²))` | 目击时长估计 |
| 方向模糊 | `angle + N(0, σ²)` | 声源/气味源方向 |

## 三、各模态噪声参数

### 3.1 视觉

| 噪声 | σ 公式 | 调制来源 |
|------|--------|---------|
| 距离估计 | σ = 0.05 × distance_m + fear×0.08 | 距离 + 情绪 |
| 尺寸估计 | σ = 0.08 + fear×0.12 | 情绪（恐惧→威胁放大） |
| 方向 | σ = (1.0 - acuity_arcmin) × 0.3 | 视觉锐度 |
| 颜色（低光） | desaturate(light_level) | 光照水平 |
| 计数 | σ = 0.15 + darkness×0.1 | 光照 |

### 3.2 听觉

| 噪声 | σ 公式 | 调制来源 |
|------|--------|---------|
| 方向 | σ = (1.0 - directional_accuracy) × 0.5 | HearingModel |
| 距离 | 音频系统已处理传播衰减+遮挡 | — |
| 清晰度 | 音频系统已处理掩蔽→五档 clarity | — |

### 3.3 嗅觉

| 噪声 | σ 公式 | 调制来源 |
|------|--------|---------|
| 浓度 | intensity × (1 + N(0, 0.2)) | 无 |
| 方向（弱） | σ = 30° | 固定——人类嗅觉方向感极差 |

### 3.4 跨模态全局

| 噪声 | 调制来源 |
|------|---------|
| noise_mult × base_σ | PerceptualModifiers.noise_mult（醉酒→1.5倍噪声） |
| 时间感知 | PerceptualModifiers.time_dilation |

## 四、分类混淆矩阵

在低清晰度或高压下，实体可能被误认：

```rust
fn classification_confusion(
    percept: &PerceptEntry,
    knowledge: &Knowledge,
    pressure: f32,     // 0-1，来自情绪+状态
    darkness: f32,     // 0-1，来自光照
    rng: &mut impl Rng,
) -> CategoryTag {
    if percept.detail_level >= BlurredIdentity && pressure < 0.5 {
        return percept.classification.tag();  // 看得够清楚 → 不会误认
    }

    let confusion_prob = pressure * 0.3 + darkness * 0.2;

    if rng.gen::<f32>() < confusion_prob {
        // 查找视觉相似的其他类别
        let alternatives = knowledge.similar_categories(&percept.visual_features);
        if let Some(wrong) = alternatives.choose_weighted(rng) {
            return wrong;  // 误认——"把木头看成了蛇"
        }
    }

    percept.classification.tag()
}
```

**涌现场景**：醉酒+暗处→高压力+高黑暗→误认概率↑→NPC可能把路牌看成人影。
