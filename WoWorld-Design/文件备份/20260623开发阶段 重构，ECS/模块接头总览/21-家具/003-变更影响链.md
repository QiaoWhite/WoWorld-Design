# 家具与放置物品系统 — 变更影响链

> **最后验证**: 2026-06-20 | **源文档版本**: [[../../物品系统/家具与放置物品/README|家具与放置物品系统 v1.0]] | **状态**: ✅ v1.0

---

## 本模块变更的影响范围

### ItemPlacementProps / 家具族参数 变更
→ **制造系统**: 新家具类型→CraftingRecipe 的 `determines_material` 链路
→ **世界生成 Pass B**: 家具清单推导表 (FURNISH_TABLE TOML) 需刷新
→ **NPC GOAP**: 新 AffordanceSet 标记 → 行动候选的 precondition 需更新

### PlacementStore 数据结构 变更
→ **LMDB 存档格式**: chunk key 结构变更→版本号递增 + 迁移
→ **WorldEntityIndex 胶水代码**: 新 Store 注入需要重建启动组装逻辑
→ **EntityIndex trait 实现**: 新增方法 (affordances_of / material_of / try_acquire)
→ **ChunkStreamer**: 新的 chunk 键类型 → 流式加载路径新增

### PlacedEntry / PlacedCold 字段 变更
→ **序列化格式**: PackedPlacedEntry 字段变更→bincode version bump
→ **EntityIndex::bounds_of/materior_of 性能**: bounding_radius 预剔除依赖此字段

### SurfaceId 编码方案 变更
→ **建筑模块**: BuildingSurface 编码 (building_id:24 + component_id:16) 需锁定
→ **TerrainStore**: TerrainSurface chunk_key 编码对齐
→ **FeatureStore**: FeatureSurface 编码对齐

### CultureFurnishProfile 派生公式 变更
→ **文化系统**: 需确认 CultureCoreParams → CultureFurnishProfile 的派生不与其他系统冲突
→ **所有消费此 Profile 的 Pass B 产出的家具**: 重新生成后的 seed 子序列偏移

### AffordanceSet 位分配 变更
→ **NPC GOAP 规划器**: precondition 中的 affordance_mask 位掩码需重新计算
→ **物理原子层**: Graspability 判定链路（如果是新增标记影响判断）

### WorldEvent::ItemPlaced/ItemPickedUp/... 变更
→ **NPC MentalModel**: 新事件类型 → 订阅逻辑更新
→ **History 系统**: AetherImprint crawler 需消费新事件类型
→ **Economy 系统**: 房产估值因子可能需刷新
→ **Audio 系统**: 家具交互音效 → 新 AudioMaterial 类型

### CHIMNEY_FAMILY_ID (建筑模块) 变更
→ **BuildingGenerator WFC 求解器**: 新 tile 类型 → 权重表 + 邻接矩阵扩展
→ **Pass A 胶水代码**: 遍历 ConnectionFace 时需要识别 Chimney 的 usage_tag
→ **建筑模块持久化格式**: ComponentFamilyId 新常量 → 兼容性

### 0x44 (FurnitureChest) 废弃
→ **旧存档迁移**: 0x44 ItemDefId → 0x54_02 子类别重映射
→ **制造配方**: 所有引用 0x44 的配方 → 改为 0x54_02
→ **NPC 库存**: 旧存档中 NPC 持有的 0x44 物品 → 需迁移映射
