## 分类说明

| 分类 | 含义 |
|------|------|
| **实质设计问题** | 影响 Rust 代码架构或运行行为的真正设计缺陷 |
| **出口孤儿——文档脱节** | 提供者出口声称消费者，但消费者入口未列出。纯文档缺口 |
| **出口孤儿——未来预留** | 消费者模块尚未成型或接口对应功能未设计 |
| **出口孤儿——迁移残留** | 所有权已迁至其他模块，但原模块出口文件未清理 |
| **入口预留——文档脱节** | 消费者入口声称从某模块进口，但提供者出口未列出 |
