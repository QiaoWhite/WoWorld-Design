## 一、动物繁衍——种群动态模型

动物个体不逐帧模拟繁衍。物种层面用种群动态模型。仅在 L1（玩家邻近 50m）具体模拟个体。

```rust
fn tick_species_population(species: &Species, biome: &BiomeState, delta_days: f32, rng: &mut StdRng) -> PopulationChange {
    let current = species.population_in(biome);
    let carrying = biome.carrying_capacity_for(species);

    if current < carrying * 0.7 {
        // 种群不足——增长
        let growth_rate = species.reproduction.rate * (1.0 - current / carrying);
        let growth = (current as f32 * growth_rate * delta_days / 365.0) as u32;
        PopulationChange::Growth { new_count: current + growth }
    } else if current > carrying * 1.3 {
        // 种群过剩——衰减
        let decline = ((current - carrying) as f32 * 0.1 * delta_days / 30.0) as u32;
        PopulationChange::Decline { new_count: current - decline }
    } else {
        PopulationChange::Stable
    }
}
```

L1 个体模拟：玩家看到的怀孕母兽、幼崽跟随母兽、巢穴中的蛋、捕猎场景。

## 二、智能种族繁衍——社会关系驱动

### 2.1 与动物的根本区别

| | 动物 | 智能种族 |
|---|---|---|
| 繁殖驱动 | 种群密度 vs 承载容量 | 社会因素——不是本能 |
| 配偶选择 | 本能(发情期+择偶) | 文化制度+个人意愿 |
| 子女数量 | 物种固定范围 | 决策结果(想要几个+能养几个+生几个) |
| 后代属性 | 随机采样+变异 | 双亲混合+教育环境+文化影响 |
| 种群调控 | 食物链承载量 | 经济承载+战争+饥荒+瘟疫+政策+文化 |

### 2.2 个人生育决策因素

> ⚠️ **DEPRECATED — CHG-041**: `FertilityDrivers` + `fertility_desire()` 已退役，由 `FertilityPotential` 连续曲线 + 通用认知管线替代。以下内容仅作为设计演进参考保留，新代码不得依赖此结构体。

```rust
#[deprecated(since = "v2.0", note = "Use FertilityPotential continuous curve + generic cognitive pipeline instead")]
struct FertilityDrivers {
    // === 生理 ===
    health_weight: f32,           // 自身HP/伤病状态——伤病时降低意愿
    age_pressure_weight: f32,     // 接近繁殖年龄上限→"再不要就来不及了"

    // === 经济 ===
    economic_security_weight: f32, // 食物储存+住房空间+收入——"养得起吗？"

    // === 伴侣 ===
    partner_quality_weight: f32,   // 与伴侣的关系值/好感度——关系差=不想要

    // === 已有子女 ===
    existing_children_weight: f32, // 越多→越不想生(除非多生多育教义反转)

    // === 个人意愿 ===
    personal_desire_weight: f32,   // 由NPC的大五人格决定（养育型人格→高意愿）

    // === 稳定性 ===
    stability_weight: f32,         // 战争/饥荒/瘟疫→大幅降低

    // === 三层社会约束——乘数修正 ===
    religious_modifier: f32,       // 信仰教义修正
    legal_modifier: f32,           // 法律/政策修正
    cultural_modifier: f32,        // 文化习俗修正
}

/// 综合得分→转化为 GOAP 目标"想要孩子"
#[deprecated(since = "v2.0", note = "Use FertilityPotential instead")]
fn fertility_desire(drivers: &FertilityDrivers, npc: &NpcRef) -> f32 {
    let base = drivers.health_weight * 0.10
             + drivers.age_pressure_weight * 0.10
             + drivers.economic_security_weight * 0.25
             + drivers.partner_quality_weight * 0.15
             + drivers.existing_children_weight * 0.10
             + drivers.personal_desire_weight * 0.15
             + drivers.stability_weight * 0.15;

    // 宗教/法律/文化不是加数——是乘数
    base * drivers.religious_modifier
         * drivers.legal_modifier
         * drivers.cultural_modifier
}
```

```rust
struct LegalReproductionLaws {
```

### 3.3 文化习俗约束

```rust
struct CulturalReproductionCustoms {
    ideal_family_size: (u32, u32),           // 文化认为"理想"的子女数
    illegitimacy_stance: IllegitimacyStance, // 非婚生子女地位
    gender_preference: Option<ChildGenderPreference>, // 子女性别偏好
    adoption_accepted: bool,                 // 是否接受收养
    lineage_system: LineageSystem,           // 氏族/家族结构
    naming_tradition: NamingTradition,
}

enum IllegitimacyStance {
    Irrelevant,        // 婚生/非婚生无区别
    MildStigma,        // 轻微污名——社会地位略降
    SevereStigma { inheritance_barred: bool }, // 严重污名——剥夺继承权
    Taboo { punishment: SocialPunishment },    // 禁忌——可能被驱逐
}

enum ChildGenderPreference {
    Male { continue_until_count: u32 },   // "至少X个儿子"
    Female { continue_until_count: u32 }, // "至少X个女儿"
    Balanced,                             // "一男一女"
    None,
}

enum LineageSystem {
    Patrilineal,    // 父系
    Matrilineal,    // 母系
    Bilineal,       // 双系平等
    Ambilineal,     // 可选——新婚夫妇选从哪方
    ClanBased,      // 氏族——超越核心家庭
}
```

**文化习俗的社会压力——由 NPC 的心智系统感知**：

```rust
impl CulturalReproductionCustoms {
    fn social_pressure_on(&self, npc: &NpcRef) -> f32 {
        let mut pressure = 0.0;

        // 无子女 + 已达适育年龄 → 家族/邻里的社交压力
        if npc.children_count == 0 && npc.age > self.expected_first_child_age() {
            pressure += 0.3;  // GOAP 中出现"结婚"目标
        }

        // 有 X 个但性别偏好不满足 → "再试一个"
        if let Some(pref) = &self.gender_preference {
            if !pref.is_satisfied(npc.children()) {
                pressure += 0.4;  // "还需要一个儿子/女儿"
            }
        }

        // 实际子女 < 文化理想 → 温和压力
        if npc.children_count < self.ideal_family_size.0 {
            pressure += 0.1;
        }

        // 非婚生子女 + 此文化中为污名 → 压力（反向抑制生育）
        if npc.has_illegitimate_children && self.illegitimacy_stance >= MildStigma {
            pressure -= 0.5;  // 不敢再生非婚生的了
        }

        pressure
    }
}
```

## 四、综合示例

```
示例一：虔诚的农民女性，多生多育教义，鼓励生育政策，父系文化，需要儿子
  → 个人意愿 0.5
  → 宗教 ×1.5(多生多育) → "生育是神圣义务"
  → 法律 +0.2(补贴) → "每多一个孩子=减税+补贴"
  → 文化压力 +0.4(还没有儿子) → "必须有个儿子继承"
  → 综合: 极强——"生孩子"成为高频 GOAP 目标，直到生出至少1个儿子

示例二：自由城市富裕商人，无宗教，限制生育政策，均分继承
  → 个人意愿 0.3(想要但不多)
  → 宗教 —(无约束)
  → 法律 -0.4(超生罚款) + 均分继承(多生=每人少分)
  → 文化: 无性别偏好
  → 综合: 1-2个——"够了，再多孩子=财产稀释+罚款"

示例三：修道士，禁欲教义
  → 个人意愿 0.0(禁欲誓言)
  → 宗教 ×0.0(强制不生育) → 此生献给神
  → 综合: 不生育
```

## 五、配偶/伴侣制度——由文化决定

| 制度 | 说明 | 典型文化 |
|------|------|---------|
| `Monogamous` | 一夫一妻——单配偶 | 秩序型社会/宗教主导 |
| `Polygamous` | 多配偶——一男多女或一女多男 | 封建/部落 |
| `GroupMarriage` | 群婚——群体内多人为配偶 | 小型紧密社群 |
| `FreeUnion` | 自由——按个人意愿结成/解除伴侣关系 | 自由城市/个人主义 |
| `Arranged` | 包办——由家族/族长/媒人决定配偶 | 封建/宗族社会 |
| `Contractual` | 契约——固定期限的伴侣关系，到期可续 | 商业文明 |
| `Celibate` | 独身——无伴侣制度 | 特定宗教阶层 |
