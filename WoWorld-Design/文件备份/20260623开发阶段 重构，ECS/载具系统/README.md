## 定位

载具系统定义 WoWorld 中**无意志的移动平台**——船、火车、马车、飞行船。载具是"移动的微型聚落"：NPC 在载具上继续过日子（吃饭/睡觉/社交/工作）。

**核心区分**：
- **载具**（本模块管辖）= 无意志·无自身AI·由 NPC 操控
- **坐骑**（战斗系统+生命系统管辖）= 有意志·独立 AI（恐惧/脾气/疲劳）

**六条设计原则**：
1. **载具是社会的延伸**——移动的微型聚落
2. **只创造规则，不创造故事**——航线/运费/船员全部涌现
3. **日常全自动，灾害半自动**——NPC AI = 玩家 AI = 同一套代码
4. **只定义物理事实**——不碰价格/裁决/建造执行
5. **记忆优先，契书可选**——所有权在记忆中成立
6. **动力来源不约束航行**——自然力仅灾害级起效

## 载具分类

```
陆上载具: 马车·货运马车·战斗马车·篷车（游牧移动家园）
水上载具: 小船·帆船·商船·战舰·渡轮·渔船
空中载具: 魔力飞行船（仅高端魔力文明）
特殊: 跨域载具（水陆两用·可折叠便携）
```

## 关键参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| **VehicleId** | u32 | 载具唯一标识 |
| **VehicleQuery** | trait, 30+方法 | 唯一对外接口 |
| **移动参考系** | Rust 侧 | 载具内部 NPC 相对运动——非 PhysicsServer3D |
| **构造方式** | NPC 集体建造 | 三途径：编辑器/TOML导入/委托NPC（同建筑模块） |
| **CPU 预算** | ≤0.1ms/帧 | 空间查询类操作 |
| **内存** | ∼5MB | 载具实例全部数据 |
