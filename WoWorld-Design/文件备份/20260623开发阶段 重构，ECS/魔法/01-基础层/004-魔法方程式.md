# 魔法方程式

> **模块**: 魔法系统 · 魔法方程式  
> **状态**: 开发规格 · v1.0  
> **关联**: [[001-魔法总纲]] · [[002-十元素]] · [[003-元素交互]] · [[005-魔力体系]] · [[012-魔法设计]] · [[006-魔法类型]]

---

## 定义

魔法方程式是构成一个魔法的七个要素的完整参数组合。每一个已定型的魔法都有一套对应的方程式——它是该魔法的"源代码"，可以被阅读、复制、分析和优化。

---

## 七要素公式

```
魔法 = 范围 + 对象 + 接触 + 元素 + 影响 + 时间 + 条件
```

一个完整的魔法方程式必须为这七个要素中的每一个指定参数。参数之间的组合决定了魔法的效果、复杂度和魔力消耗。

---

## 各要素详解

### 1. 范围

定义魔法生效的空间形态：

| 空间形态 | 描述 | 典型应用 |
|----------|------|----------|
| **点** | 作用于一个精确的坐标 | 精确攻击、远程治疗、传送定位 |
| **线** | 沿一条直线或曲线生效 | 光束、雷击、元素射线 |
| **面** | 作用于一个二维平面 | 防护壁、力场墙、元素帷幕 |
| **体** | 填满一个三维空间 | 范围爆炸、区域治疗、领域魔法 |

范围的增大对魔力消耗的影响是**指数级**的——半径翻倍，魔力消耗约 ×4。

### 2. 对象

由四个子项共同决定魔法作用于"谁"或"什么"：

- **对象类型**：自己、友方、中立方、敌方、非生物，以及这五者的任意组合（共 2⁵ = 32 种离散组合，加上"全部"等混合选项，总计约 120 种可选配置）
- **对象数目**：单个、群体
- **对象命中**：针对（指定目标）、随机（范围内无差别）
- **对象指向**：位置、属性、心理、魔力、血量、精力等

对象数目的增加会引入额外的魔力系数——每个额外目标都需要独立的"链接消耗"。

### 3. 接触

定义魔法从施法者到目标的路径方式：

- **过程接触**：魔法存在一段从施法者到目标的"路径"。如飞行的火球、蔓延的冰霜、延伸的藤蔓。路径上的事物可能被波及，也可能被障碍阻挡。
- **凭空接触**：魔法没有中间过程，直接作用于目标。如对目标体内的**血**元素直接加热、在目标脚下瞬间生成岩刺。难以被常规手段拦截。

两种接触方式各有优劣，不是"凭空接触一定更好"——凭空接触虽然难以拦截，但魔力消耗更高（约 ×1.5-2），且某些元素（如土）天然不适合凭空接触。详见 [[003-元素交互]]。

### 4. 元素

魔法的元素属性。指定魔法方程式中涉及哪种或哪几种[[002-十元素|十元素]]。元素的选择受[[003-元素交互|亲和/排斥关系]]约束，并直接影响魔力消耗。

- **单一元素**：最基础。魔力消耗基准值 ×1.0
- **多元混合**：消耗随元素数量、亲和/排斥关系呈非线性增长

### 5. 影响

定义魔法对目标造成的具体改变：

**基本操作**（不可再分的原子操作）：
- **增加**：向目标增加某元素的量
- **减少**：从目标减少某元素的量
- **排斥**：使目标排斥某元素（远离、隔离）
- **吸收**：从目标吸收某元素到施法者或指定容器
- **释放**：向目标释放已储存的元素能量

**复合影响**（由基本操作组合而成）：
- **转移** = 从A吸收 + 向B释放
- **转化** = 对目标A减少(某元素) + 对目标B增加(另一元素)
- **链接** = 对A释放(标记) + 对B释放(共鸣)
- 以及更多上述操作的复杂组合

> 在公理层面（[[001-魔法总纲]] 意志投射原理），增加、减少、吸收、释放、排斥是五种基本变化，转化、转移、链接等是由基本变化组合而成的复合影响。

### 6. 时间

定义效果的持续时间：

| 时间类型 | 描述 | 魔力消耗特征 |
|----------|------|-------------|
| **瞬时** | 立即生效并结束 | 基准消耗 |
| **持续** | 效果维持一段时间 | 时长指数增长 |
| **触发** | 预设条件满足时自动生效（如陷阱、延迟爆发） | 比持续略低（休眠期间不消耗） |
| **永久** | 效果不可逆地固着 | 极高——通常需要仪式或法阵辅助 |

### 7. 条件

施展魔法必须满足的前提。条件越多、越苛刻，魔力消耗越低——因为条件从外部补足了施法者自身的不足：

- **自身条件**：施法者的**灵**元素强度、[[005-魔力体系|魔力储量]]、对元素理论的理解深度
- **环境条件**：特定地脉、星象、天气、元素浓度
- **物质条件**：[[009-触媒与材料|触媒]]、[[008-法阵系统|法阵]]、祭品等
- **仪式条件**：特定的动作、时间、方位——详见[[010-仪式魔法]]
- **契约条件**：与某存在订约后方可施展——详见[[011-契约魔法]]

## 要素间的约束关系

某些要素组合存在天然的配合或约束：

| 约束类型 | 举例 | 效果 |
|----------|------|------|
| 亲和配合 | 火元素 + 风元素 + 过程接触（风助火势沿路径蔓延） | 魔力消耗降低，效果增强 |
| 天然约束 | 土元素 + 凭空接触 | 土元素的惰性使其难以响应凭空接触——魔力消耗 ×2-3，稳定性下降 |
| 逻辑约束 | "永久"时间 + "破坏系"核心影响 | 破坏系通常不用于创造永久效果——需要在方程式中追加额外条件才能成立 |
| 伦理约束 | 惑心系 + "灵"元素 + 敌方 + 针对 | 在某些文化中此组合构成犯罪——详见[[017-魔法禁忌与风险]]和[[018-魔法与文化]] |

了解约束关系是魔法设计第二阶段（参数建模）的核心能力。

## 方程式的复杂度等级

| 等级 | 要素变化数 | 典型魔力消耗 | 代表法术 |
|------|-----------|-------------|----------|
| 基础 | 1-2 | 极低 | 点火（火元素 + 点范围 + 释放 + 瞬时） |
| 标准 | 3-4 | 低-中 | 火球术（火元素 + 体范围 + 过程接触 + 释放 + 瞬时） |
| 高阶 | 5-6 | 中-高 | 群体治疗（血元素 + 体范围 + 凭空接触 + 增加 + 持续 + 灵元素强度条件） |
| 传奇 | 全部7项精确指定 | 极高 | 城市级防护罩（全要素精确参数 + 仪式条件 + 永久时间） |

---

## 方程式的优化空间

同一个魔法效果可以用不同的方程式实现。例如，"对敌人造成火焰伤害"可以是：

- **方案A**：火 + 体范围 + 过程接触 + 释放 + 瞬时（火球术——经典方案）
- **方案B**：火 + 点范围 + 凭空接触 + 增加（直接在敌人体内加热其血元素——消耗更高但无法躲避）

两个方案的效果相似，但魔力消耗、可拦截性、风险完全不同。找到"最优方程式"——在给定约束下以最低消耗达到目标效果——是 [[012-魔法设计]] 中的核心挑战，也是"二次开发"（改进已有魔法）的技术基础。
