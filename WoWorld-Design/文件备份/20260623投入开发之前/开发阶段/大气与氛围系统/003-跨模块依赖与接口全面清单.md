# 003 — 跨模块依赖与接口全面清单

> **模块**: 大气与氛围系统
> **版本**: v1.0
> **创建日期**: 2026-06-21

---

## 一、本模块 OWN（权威定义）

| 类型 | 定义 |
|------|------|
| `ResolvedAtmosphere` | 17 参数渲染描述——68 bytes Copy 类型 |
| `BiomeAtmosphere` | 群系基值——TOML 加载。边界双线性插值 |
| `TimeCurve` / `WeatherCurve` / `SeasonCurve` | 调制曲线——TOML 加载。查表+插值 |
| `SkyEventManager` | SkyEvent 注册/注销/解析 |
| underwater.toml / sky_events.toml | 配置文件 |

## 三、消费本模块的接口

| 消费方 | 消费方式 |
|--------|---------|
| Godot toon shader | 全局 shader uniform —— 读 ResolvedAtmosphere 全部 17 参数 |
| Godot WorldEnvironment | GDExtension —— ambient_light_color, fog, bloom, sky |
| Godot AreaLight3DManager | 间接——读 BuildingQuery + 光源数据——非本模块接口 |
| 植被 shader | 同 toon shader——全局 shader uniform。不新增 VegetationProvider 接口 |
| NPC 感官系统 | 间接——fog_density → env_acoustic_modifier——[[../音频系统/001-音频系统总纲\|音频系统]] 已有 |

## 五、权威性声明

本模块文档是以下内容的**唯一权威来源**：

- `ResolvedAtmosphere` 的字段定义和语义
- 四层调制（biome × time × weather × season）的计算方式
- SkyEvent 的注册/解析机制

其他模块文档中引用氛围参数的部分，应以本文档为准。

**具体而言**：
- `音频系统/001-音频系统总纲` 中的 `env_acoustic_modifier` 应从本模块的 `fog_density` + `aerial_blur` + 地形开阔度派生——以本文档为准
- 植被模块的 shader 中的颜色参数应从本模块的全局 shader uniform 读取——以本文档为准
