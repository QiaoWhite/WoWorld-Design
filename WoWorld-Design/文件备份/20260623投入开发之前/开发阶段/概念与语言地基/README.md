## 模块定位

**概念与语言地基系统** 是 WoWorld 的概念表示层——连接"客观世界数据"和"自然语言表达"之间的空白地带。

WoWorld 当前有完整的数据层（权力拓扑、经济账本、关系图）和表达层（文本生成、语音输出），但缺少一个**概念中间层**——NPC 如何从原始数据中识别概念？如何用语言无关的方式思考？如何在跨文化接触中翻译？知识如何在代际间传递？

本模块填补这个空白。

## 核心架构

```
woworld_types/          ← 纯类型，零依赖
woworld_core/           ← 纯数学函数，仅依赖 types
woworld_lang/           ← 数据驱动渲染，依赖 types + TOML
业务模块                 ← 仅依赖 types，不知道概念系统存在
集成层                   ← 组装所有模块
```

详见 [[001-概念与语言地基总纲|001-总纲]]。

---

## 关键概念

- **PatternSignature**: 客观模式指纹——全球统一
- **ConceptLocalId**: 文化内的概念标识
- **CompactConceptEncoding**: 64B 压缩概念编码
- **Utterance**: 结构化话语——NPC 语言产出的统一格式
- **effective_concept_space()**: 混合概念空间的加权派生
- **translate_concepts()**: 跨文化概念翻译
