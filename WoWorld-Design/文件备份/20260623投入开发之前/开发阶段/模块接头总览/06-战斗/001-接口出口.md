# 战斗模块 — 接口出口

> **最后验证**: 2026-06-22 | **源文档版本**: 开发规格 v1.1 | **CHG-013 基座**

## 三层战斗模型
- **概念**: 本能层(每帧16ms·自动闪避/格挡/反击/本能法术/掩体本能) → 节奏层(0.5-1s·倾向选择: 前压/稳守/周旋/猛攻/脱战) → 战略层(5-30s+事件驱动·总目标/阶段目标/限定条件/自察). 玩家AI = NPC AI = 同套Rust代码
- **Owner**: [[../../../战斗/003-三层战斗模型|战斗 §003]] + [[../../../战斗/001-战斗系统总览|§001]]
- **类型**: Rust 分层架构, woworld_combat
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 决策引擎(高压版本) · [[../12-历史/002-接口入口|历史]] 战斗叙事 · [[../07-魔法/002-接口入口|魔法]] 魔法战斗中融入 · [[../17-模型动作与物理/002-接口入口|模型物理]] 骨骼蒙皮/动画
- **契约依据**: CHG-013

## CombatPerception (战斗感知 — 信息不对称核心)
- **概念**: visible_enemies[] (estimated_hp: Range<f32>, threat_level, observed_style, visible_weapon, casting_spell) + perceived_allies[] + environment (掩体/危险/脱战路径) + self_assessment (可能不准确—疲劳低估/恐惧高估). 6项不可见: HP/体力/魔力/技能等级/情绪/战斗风格—通过动画和行为推断. ★ v1.1 (CHG-057): EnrichedPerceptBatch消费→敌方情绪阅读(EmotionalCharge/GazeEstimate)提升威胁评估精度.
- **Owner**: [[../../../战斗/004-战斗信息与感知|战斗 §004]] + [[../../../战斗/001-战斗系统总览|§001 §六]]
- **类型**: Rust struct, woworld_combat
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 决策引擎 · [[../05-感官与知觉/002-接口入口|感官]] combat_intelligence.assess() (消费PerceptBatch)

- **概念**: 攻击轨迹+伤害类型+附加效果+消耗—部件拼装, 非手写技能列表. 流派从NPC群体的招式偏好中涌现. 战斗风格 = f(人格, 知识, 处境, 武器, 魔法, 环境)
- **Owner**: [[../../../战斗/005-招式积木与流派生成|战斗 §005]]
- **类型**: Rust, woworld_combat
- **消费者**: [[../08-技能/002-接口入口|技能]] 技能→招式映射 · [[../18-物品/002-接口入口|物品]] 武器招式库 · [[../17-模型动作与物理/002-接口入口|模型物理]] 动画姿态

## PostCombatTransition (战后过渡)
- **概念**: 战斗不"突然结束". 情绪残留(数分钟-数小时) + 惊跳反应 + 记忆编码(关键事件→长期记忆) + 伤势持久化 + 声誉传播(围观NPC→社交) + 创伤/成长(长期人格微调)
- **Owner**: [[../../../战斗/012-战后过渡与伤势|战斗 §012]]
- **类型**: Rust struct + 管线, woworld_combat
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 情绪/记忆写入 · [[../01-生命/002-接口入口|生命]] 伤势持久化 · [[../12-历史/002-接口入口|历史]] 大日志 · [[../13-语言表达/002-接口入口|语言]] 流言传播

## BodyPartDamage (部位伤害 — 渐进三档)
- **概念**: 轻伤(damage<30%)/中度伤(30-60%)/重伤(>60%). 仅影响数值(精度/速度/恢复率)—不影响攻击模组. 持械臂/非持械臂/躯干/腿/头/尾/翼/额外肢体. 独立存储 damage: f32(0.0-1.0)
- **Owner**: [[../../../战斗/012-战后过渡与伤势|战斗 §012]] (Combat权威) → Life消费
- **类型**: Rust struct, woworld_combat → Life引用
- **消费者**: [[../01-生命/002-接口入口|生命]] 部位损伤存储+恢复 · [[../02-NPC活人感/002-接口入口|NPC]] GOAP输入
- **契约依据**: CHG-013 (渐进三档软数值—不可二值)
