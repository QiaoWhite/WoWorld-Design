# 007 — NPC经济心智

## 一、核心原则

**经济系统不告诉NPC该买什么。** NPC的经济决策由他自己的全部心智模型驱动——经济系统只提供客观信息（价格、供给、自己钱包余额）。

两个有同样铜币的NPC，因为人格、情绪、记忆、认知不同——买不同的东西、付不同的价格、有不同的储蓄率。这就是涌现。

## 二、消费决策的三层驱动

```
生理层 → 触发GOAP安全网（饥饿/口渴/疲劳越过阈值 → 强制行动）
认知层 → 设定可行方案（搜索哪些市场、心理价位、满意标准）
人格情绪层 → 在可行方案中做情感加权（冲动vs耐心、从众vs独立）
```

三层叠加决定最终消费行为。没有一层单独"规定"NPC买什么。

## 三、行为经济学十概念 × NPC字段精准映射

以下十个行为经济学核心概念，全部可从NPC模块已有字段驱动。**不需要新增任何人格维度。**

### 3.1 损失厌恶 (Loss Aversion)

**经济学原理**：损失带来的负效用约为同等获利的2倍。Kahneman & Tversky中位数λ≈2.25。

**驱动字段**：`neuroticism` 放大损失感知（1.5-2.5×）；`wisdom` 理性校正（减少至0.6-1.0×）

```rust
fn loss_aversion_multiplier(npc: &NpcData) -> f32 {
    let base: f32 = 2.0;
    let neuroticism_amplification = 0.75 + npc.personality.neuroticism * 1.25;
    let wisdom_correction = 1.0 - npc.mental.wisdom * 0.4;
    base * neuroticism_amplification * wisdom_correction
}

/// 计算事件对NPC的主观效用
fn subjective_value(objective_delta: f32, npc: &NpcData) -> f32 {
    if objective_delta >= 0.0 {
        objective_delta  // 获利 = 1:1
    } else {
        objective_delta * loss_aversion_multiplier(npc)  // 损失被放大
    }
}
```

**玩家可观察表现**：高神经质NPC拒绝"打折出售传家宝"（禀赋效应叠加损失厌恶）。丢了10铜的反应比捡到10铜强烈得多——可能触发愤怒情绪、抱怨对话。

### 3.2 心理账户 (Mental Accounting)

**经济学原理**：人对"工资"、"意外之财"、"储蓄"的消费倾向不同。Thaler 1985。

**驱动字段**：收入来源标签（IncomeSource枚举——已在004中定义）；`conscientiousness` 调节跨账户转移意愿

```rust
/// NPC收到铜币时的心理标记
#[derive(Debug, Clone)]
enum MentalAccount {
    RegularIncome,     // 工资——倾向于预算消费
    Windfall,          // 意外之财——高概率冲动消费
    Savings,           // 储蓄——低消费倾向
    Gift,              // 礼物——社交消费倾向
}

fn mental_account_spending_propensity(
    account: MentalAccount,
    npc: &NpcData,
) -> f32 {
    match account {
        MentalAccount::RegularIncome => 0.7 - npc.personality.conscientiousness * 0.3,
        MentalAccount::Windfall => 0.9 - npc.personality.conscientiousness * 0.5,
        MentalAccount::Savings => 0.2,
        MentalAccount::Gift => 0.6 + npc.personality.extraversion * 0.2,
    }
}
```

**玩家可观察表现**：高尽责性NPC把意外之财存起来；低尽责性立刻花掉。同一笔钱，来源不同，花法不同。

### 3.3 锚定效应 (Anchoring)

**经济学原理**：第一次看到的价格成为认知参考锚点。后续价格判断相对于锚。

**驱动字段**：`MemoryStore` 中首个价格记忆；`wisdom` 调节挣脱锚定的能力

```rust
/// 查询NPC对某物品的价格锚
fn price_anchor(npc: &NpcData, item_def: ItemDefId, market_id: MarketId) -> Option<u32> {
    // 从known_prices中找最早的那条记录
    npc.economic_state.known_prices
        .get(&(market_id, item_def))
        .map(|k| k.known_price_copper)
}

/// 锚定效应——当前价格 vs 锚点价格的感知
fn anchored_price_perception(
    current_price: u32,
    anchor: u32,
    npc: &NpcData,
) -> PricePerception {
    let ratio = current_price as f32 / anchor as f32;
    let wisdom_adjustment = npc.mental.wisdom * 0.5; // 高智慧→能看穿锚定

    if ratio < 0.85 - wisdom_adjustment * 0.2 { PricePerception::VeryCheap }
    else if ratio < 0.95 - wisdom_adjustment * 0.1 { PricePerception::Cheap }
    else if ratio < 1.05 + wisdom_adjustment * 0.1 { PricePerception::Fair }
    else if ratio < 1.20 + wisdom_adjustment * 0.2 { PricePerception::Expensive }
    else { PricePerception::VeryExpensive }
}
```

**玩家可观察表现**：NPC第一次在某市场见到铁矿石50铜→成为锚。之后在其他市场见到30铜觉得"太便宜了"→可能大量买入。高智慧NPC能告诉自己"上次那个价只是运气"。

### 3.4 禀赋效应 (Endowment Effect)

**经济学原理**：拥有某物后，对其估值显著提高。卖出要价 > 买入愿意出价。

**驱动字段**：`openness` 低→禀赋效应强（不爱换东西）；持有时长也在记忆中体现

```rust
fn endowment_multiplier(npc: &NpcData, ownership_days: f32) -> f32 {
    let base = 1.0 + ownership_days.min(365.0) / 365.0 * 0.3; // 持有一年→+30%
    let openness_mod = (1.0 - npc.personality.openness) * 0.5; // 低开放性→+50%
    1.0 + (base - 1.0) * (1.0 + openness_mod)
}
```

**玩家可观察表现**：铁匠打的剑——自己觉得值500铜。隔壁店同样的剑——他觉得只值300铜。持有时间越长，估值越高。"这是我爷爷传下来的"→估值可能翻倍。

### 3.5 双曲贴现 (Hyperbolic Discounting)

**经济学原理**：人对当下 vs 未来的权衡是非理性的——当下诱惑权重过高。

**驱动字段**：`conscientiousness` 调节自制力；`wisdom` 调节远见；`EconomicCognition.time_preference_rate` 缓存结果

```rust
fn discount_future_value(
    future_value: f32,
    days_until: f32,
    npc: &NpcData,
) -> f32 {
    let rate = npc.economic_state.cognition.time_preference_rate;
    // 双曲贴现: V = FV / (1 + r × days)
    // 而非指数贴现: V = FV × e^(-r × days)
    future_value / (1.0 + rate * days_until / 365.0)
}
```

**玩家可观察表现**："今天花10铜喝酒" vs "存10铜下周买工具"——低尽责性选前者。高尽责性+高智慧选后者。"等三天能省5铜"→高时间偏好率的NPC不等。

### 3.6 羊群效应 (Herd Behavior)

**经济学原理**：看到别人买→自己也买。从众是快速决策的启发式。

**驱动字段**：`extraversion` 高→对社会信号更敏感；`agreeableness` 高→更愿跟随

```rust
fn social_proof_weight(
    npc: &NpcData,
    observed_buyers_count: u32,
    total_observers: u32,
) -> f32 {
    let base = observed_buyers_count as f32 / total_observers.max(1) as f32;
    let extraversion_mod = 0.5 + npc.personality.extraversion * 1.5;
    let agreeableness_mod = 0.5 + npc.personality.agreeableness * 1.5;
    (base * extraversion_mod * agreeableness_mod).min(1.0)
}
```

**玩家可观察表现**：市场恐慌→大家抢购粮食→更多人跟风抢→正反馈循环推高价格。高开放性低外向性NPC可能逆向操作（"别人都在卖的时候我买"）。

### 3.7 有限理性 / 满意原则 (Satisficing)

**经济学原理**：人不追求全局最优——找到"够好"的选项就停止搜索。Simon 1956。

**驱动字段**：`EconomicCognition.market_search_breadth`；`EconomicCognition.satisficing_threshold`

```rust
fn satisficing_check(
    npc: &NpcData,
    found_price: u32,
    best_possible_price: Option<u32>,
) -> bool {
    let threshold = npc.economic_state.cognition.satisficing_threshold;

    if let Some(best) = best_possible_price {
        // 是否足够接近已知的最低价？
        (best as f32 / found_price as f32) > (1.0 - threshold * 0.3)
    } else {
        // 没有参考——搜索了足够多的市场就停
        npc.economic_state.known_prices.len() as u8
            >= npc.economic_state.cognition.market_search_breadth
    }
}
```

**玩家可观察表现**：NPC不搜索全部市场找最低价——搜索1-3个最近的市场，找到"够好"的就买。高智慧NPC搜索更广但仍有上限。"货比三家"是智慧的表现，不是默认行为。

### 3.8 公平偏好 (Fairness)

**经济学原理**：人愿意惩罚不公平行为，即使自己受损。

**驱动字段**：`agreeableness` 高→强烈的公平感；情绪系统产生愤怒/感激

```rust
fn perceived_fairness(
    npc: &NpcData,
    price_paid: u32,
    reference_price: u32,
) -> f32 {
    let ratio = price_paid as f32 / reference_price as f32;
    let fairness_raw = if ratio > 1.3 { -0.5 }  // 被宰了
        else if ratio > 1.1 { -0.2 }
        else if ratio < 0.8 { 0.3 }  // 占了便宜
        else { 0.1 };  // 差不多

    // 高宜人性→对不公平更敏感
    fairness_raw * (0.5 + npc.personality.agreeableness * 1.0)
}
```

**玩家可观察表现**：高宜人性NPC发现被宰→不仅不再光顾，还可能散布坏话（语言表达系统的信息传播）。低宜人性NPC→"商业就是商业，下次我宰回来"。

### 3.9 过度自信 (Overconfidence)

**经济学原理**：低能力者高估自己的能力。

**驱动字段**：`intelligence` 低 + `extraversion` 高 → 高估议价能力

```rust
fn self_assessed_bargaining_skill(npc: &NpcData) -> f32 {
    let actual = estimate_actual_bargaining(npc); // 技能+经验
    let overconfidence = (1.0 - npc.mental.intelligence) * 0.3
        + npc.personality.extraversion * 0.2;
    (actual + overconfidence).min(1.0)
}
```

**玩家可观察表现**：低智慧高外向NPC经常在交易中吃亏，但自己察觉不到——"我做了笔好买卖！"（实际不是）。高智慧NPC能校准自我评估。

### 3.10 现状偏误 (Status Quo Bias)

**经济学原理**：人倾向于维持现状——不换供应商、不换品牌。

**驱动字段**：`openness` 低 + `conscientiousness` 高 → 不爱换

```rust
fn supplier_switching_resistance(npc: &NpcData, current_supplier_days: f32) -> f32 {
    let personality_resistance = (1.0 - npc.personality.openness) * 0.5
        + npc.personality.conscientiousness * 0.3;
    let habit_resistance = (current_supplier_days / 365.0).min(1.0) * 0.3;
    (personality_resistance + habit_resistance).min(1.0)
}
```

**玩家可观察表现**：老农坚持去同一个摊位买——即使隔壁新开的店更便宜。"老张家的东西我买了二十年了"——高habit_resistance。需要显著价差（>20%）才能克服。

## 四、情绪-经济反馈闭环

交易盈亏 → 情绪偏移 → 影响后续交易决策。这是"消费者信心"的微观基础。

```rust
/// 交易完成后调用——情绪反馈
fn post_trade_emotion_update(
    npc: &mut NpcData,
    trade: &TradeRecord,
    is_buyer: bool,
) {
    let perceived_value = evaluate_deal_quality(npc, trade, is_buyer);

    // 愉悦轴偏移
    npc.emotion.pleasure += perceived_value * 0.1;

    // 如果被严重宰了（perceived_value < -0.3）
    if perceived_value < -0.3 {
        // 愤怒上升
        npc.emotion.basic_emotions.anger += 0.2;
        // 如果是买方→对卖方的信任下降
        if is_buyer {
            if let Some(rel) = npc.relationships.get_mut(&trade.seller) {
                rel.trust -= 0.1;
            }
        }
    }

    // 如果占了便宜 → 愉快+可能再次光顾（记忆强化）
    if perceived_value > 0.2 {
        // 正面情绪→下次更愿意在此市场消费
        npc.economic_state.trade_preferences
            .preferred_markets.push(trade.market_id);
    }
}
```

**累积效应**：
- 连续几笔好交易 → 情绪愉悦 → 更愿意消费 → 经济活跃
- 连续被宰 → 愤怒/不信任 → 收缩消费 → 转向地下市场或更谨慎

## 五、消费决策权重函数——汇总

```rust
/// NPC的整体消费倾向——所有心智因素的综合输出
fn consumption_propensity(npc: &NpcData, item: ItemDefId) -> f32 {
    let mut weight = 0.5; // 中性基线

    // 人格层
    weight += (npc.personality.conscientiousness - 0.5) * -0.15; // 尽责→少冲动消费
    weight += (npc.personality.neuroticism - 0.5) * -0.10;       // 神经质→收缩
    weight += (npc.personality.extraversion - 0.5) * 0.10;        // 外向→多社交消费
    weight += (npc.personality.openness - 0.5) * 0.08;            // 开放→愿试新商品

    // 情绪层
    weight += npc.emotion.pleasure * 0.12;   // 开心→多花
    weight += npc.emotion.mood.mood_pleasure * 0.08; // 好心境→多花

    // 认知层
    weight += (npc.economic_state.cognition.financial_literacy - 0.5) * -0.05;

    // 习惯层
    if let Some(habit) = npc.habit_system.habits.get(&ActionId::Buy(item)) {
        weight += (habit.weight - 0.5) * 0.10;
    }

    // 生理层——饥饿直接触发GOAP，不走此路径

    weight.clamp(0.05, 0.95)
}
```

## 七、职业标签与心智联动

### 7.1 职业标签对经济行为的影响

职业标签不仅是社会身份——它们影响NPC的经济行为权重。详见[[001-经济系统总纲#3.4.4|001 §3.4.4 职业标签生命周期]]：

- **新标签涌现**（失业→成为铁匠学徒）→ 开始购买铁匠工具和原材料 + 收入来源类型切换 + 消费偏好改变
- **标签休眠**（退休）→ 停止购买职业相关物资 + 可能出售职业设备 + 收入来源从Wage/SelfEmployed变为储蓄
- **标签历史化**（伤残/死亡）→ 职业设备传给继承人或出售 + SelfNarrative.life_chapters记录 + 同行/客户记忆更新

### 7.2 职业标签的主观感知

NPC_B 如何认知 NPC_A 的职业——取决于主观观察和记忆：
- B亲眼看到A在打铁 → B知道A是铁匠（confidence高）
- B从C口中听说A是铁匠 → B的认知可能有误差（C可能记错了）
- B和A不熟 → B只知道A最显著的标签（"铁匠"），不知道A也是猎户

这种主观感知是社会阶级系统的输入——经济系统只提供职业标签的客观事实。
