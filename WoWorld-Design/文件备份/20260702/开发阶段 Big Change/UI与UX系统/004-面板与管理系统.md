## 物品栏面板（Tab / I）

### 网格背包
- 10 列 × 3 行 = 30 格基础槽位
- 每格 64×64px
- 物品按尺寸占格：1×1（小物品）到 2×3（大武器）
- 拖拽移动、右键菜单（使用/装备/丢弃/拆分/查看详情）
- 物品稀有度边框颜色：灰(Common) / 绿(Uncommon) / 蓝(Rare) / 紫(Epic) / 金(Legendary)

### 分类标签
- 全部 / 武器 / 防具 / 消耗品 / 材料 / 任务 / 书籍 / 杂项
- 搜索框：键入关键词实时过滤
- 排序：名称 / 稀有度 / 重量 / 价值 / 最近获取

### 装备槽（BodyPlan 可视化）
- 面板右侧：人物剪影 + 12 装备槽位
- 槽位：头 / 肩 / 胸 / 手 / 腿 / 足 / 主手 / 副手 / 戒指1-4
- 槽位从 BodyPlan（Rust 侧）自动派生——不预定义物种类型
- 拖拽物品到槽位 = 装备
- 双套 Outfit 切换按钮（Outfit A / Outfit B——由 NPC 自主决定切换逻辑）

### 容器展开
- 背包中的容器物品（背包/袋子/药水包）可点击展开
- 嵌套显示（最多五层——与物品系统 005 一致）
- 展开的容器在物品栏网格旁以子面板显示

### 物品详情悬浮
- 鼠标悬停物品 → 弹出详细面板
- 显示：属性列表、附魔信息、耐久度、描述文字、审美评分

## 技能面板（K）

### 树状图
- 5 大类技能树：战斗 / 魔法 / 工艺 / 学术 / 生存
- 22 子组分页
- 已解锁技能：亮色节点。未解锁：灰色节点 + 解锁条件提示
- 当前等级的技能：节点大小微增 + 经验条环

### 进度显示
- 每技能：当前等级 Lv.X（如 Lv.12）+ 进度条（XP / TotalXP）+ 天生倍率标注（如 ×1.15）
- 5 大类顶部显示总 XP / 总技能数
- 交叉训练效果：同组技能以连线标注

### 天赋层
- 技能面板左侧：天赋三层面板
  - MentalAccess: 智力限制（0.7-1.0）
  - 天生倍率: 每 NPC 每技能（0.7-1.3）
  - 交叉训练: 同组 bonus（0.04-0.20）
- 玩家 MentalAccess + PhysicalAccess 均固定 1.0

## 地图面板（M）

### 全屏地图
- 占据屏幕 80%（非全屏——保留四周游戏视野）
- 可拖拽平移 + 鼠标滚轮缩放
- 地形渲染：从 Rust 侧 `TerrainQuery::height_at` + `surface_material_at` 采样生成颜色纹理
- 当前坐标显示（顶部栏）

### 知识迷雾
- 探索过 = 永久可见
- 未探索 = 纯黑
- 从 NPC 购买的地图信息 = 该区域迷雾消散（以该 NPC 的知识为准——可能有错误/过时）

### 标记系统
- 右键放置标记点：可选择图标（营地/矿点/危险/宝箱/自定义）+ 输入标签
- 已发现位置自动标记：聚落(🏘️)、地标(🏰)、地下城入口(🕳️)
- 标记可按类型筛选显示

### 快速旅行
- 条件：已发现 + 安全路线 + 旅行资源（食物/金币/马匹/船只）
- 点击标记 → "快速旅行至此"（消耗时间 + 资源，不是瞬间移动）
- 旅行时间估算 + 中途遇敌概率显示

## 日志面板（J / L）

### 大日志浏览器
- 搜索框：支持全文搜索，筛选条件（时间范围/事件类型/NPC 名称/地点）
- 关系图谱视图：以 NPC/地点/事件为节点，连线显示关系
- 时间线视图：按时间倒序排列事件

### 内容类型
- 发现：新地点/新物种/新配方
- 对话摘要：与 NPC 的重要对话自动记录要点
- 任务追踪：不是"任务清单"——是玩家自己标注的"我想做"和 NPC 说的"你可以做"
- 知识：从书本/告示牌/碑文中学到的知识
- 交易记录：最近 N 次交易摘要

### 日志引用（对话中）
- 对话中选择"日志引用" → 打开日志浏览器 → 选择一条记录 → 自动组织为对话论据
- 如："我读过《橡木镇志》，上面说三百年前…"

## 蓝图编辑器（B）

### 用途
- 建筑 DIY：可视化编辑 Blueprint TOML
- 与 [[../../建筑模块/README|建筑模块]] 的 WFC 2.5D 生成器交互

### 操作
- 放置 ComponentFamily 组件块（9 核心 + Mod）
- WFC 求解器实时预览可能结果
- 保存为 Blueprint 物品（type 0x52）→ 存入背包
- 建造时：手持 Blueprint → 放置在地面 → 自动生成建筑
