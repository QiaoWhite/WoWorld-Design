## 场景一：跨文化概念翻译

```
初始条件:
  - 罗斯文化 NPC (概念: KHAD_AZ — "匠约")
  - 中原文化 NPC (概念: EMPLOYMENT — "雇佣", APPRENTICESHIP — "师徒")
  - 罗斯 NPC 说: Utterance { concepts: [KHAD_AZ], lang: rossian }
  - 中原 NPC proficiency(rossian) = 1.0

预期:
  - translate_concepts() 搜索中原概念空间:
    - KHAD_AZ 的 coverage: [Contract + Extract(labor) + Extract(wage) + Compel]
    - "雇佣": overlap 0.65 (缺少 Compel 维度)
    - "师徒": overlap 0.70 (都有 Compel + Contract)
    - 最佳匹配: "师徒" (0.70)
  - 渲染: "[他说他在铁匠铺有一个……大概是师徒关系？但语气里好像有敬意。]"

验证:
  - "师徒"而非"雇佣"是正确的——KHAD_AZ 带有 Compel(training)，雇佣不含
  - "敬意"注解正确——KHAD_AZ 有 honorific 维度，中原的师徒有，雇佣没有
```

## 场景二：语言 proficiency 的涌现增长

```
初始条件:
  - 中原 NPC, 北方语 proficiency = 0.0
  - 在北方城市住了 30 天
  - 每天: 5 次 LISTEN(北方语) + 2 次 SPEAK(北方语, interlocutor平均=0.6)

模拟:
  - 每日暴露: 5 × 0.7 × 0.6 × 0.12 + 2 × (0.5+0.6×0.5) × 0.20 = 0.252 + 0.320 = 0.572
  - rate = 0.572 × 0.85 × 1.0 × 0.3 × 1.0 = 0.146
  - Day 1: growth = 0.146 × 0.25 × 0.015 = 0.00055 → prof = 0.00055
  - Day 15: prof ≈ 0.12 (sigmoid 加速)
  - Day 30: prof ≈ 0.31 (sigmoid 峰值)
  - Day 60: prof ≈ 0.55 (开始减速)
  - Day 180: prof ≈ 0.82 (接近瓶颈)

验证:
  - 30天到 0.3 ✓ (基本交流)
  - 非线性 S 曲线 ✓
  - 无硬编码"师承=30天"——结果完全由原子日志驱动
```

## 场景三：演绎推理链

```
初始条件:
  - 父亲 NPC 有 MentalModel:
    - M1: "如果一个人有工作 → 他有收入" (confidence 0.9)
    - M2: "如果受益人有独立收入 → 资助者的支持义务解除" (confidence 0.85)
  - 新感知事实: "儿子在铁匠铺工作" (confidence 1.0)

模拟:
  - memory_encode_with_deduction():
    - depth 0: M1 应用于事实 → "儿子有收入" (confidence 0.9 × 1.0 = 0.9)
    - depth 1: M2 应用于 depth 0 结果 → "不用给儿子钱了" (confidence 0.85 × 0.9 / 1.15 = 0.67)
    - inference_conclusion: "不用给钱了" (confidence 0.67)

验证:
  - 推理链正确 ✓
  - 置信度逐层衰减 ✓
  - 推理结论存储为 inference_conclusion ✓
```

## 场景四：知识重新发现

```
初始条件:
  - 500 年前的《老铁锻经》抄本（copy_generation=3, physical_condition=0.4）
  - 记载了"分段淬火"概念（已被遗忘——世界中无 NPC 持有此 MentalModel）
  - 后世铁匠 "铁十" 阅读此书

模拟:
  - book.source_concepts → CompactConceptEncoding.expand()
  - classify_pattern("分段淬火" sig, 铁十的概念空间) → 匹配 success
  - 保真度: 0.95 × 0.98³ × 0.4 = 0.36
  - 初始 confidence: 0.36 × 0.6 = 0.22
  - 铁十创建 MentalModel: confidence 0.22, hypothesis=true, source=ReadFromBook
  - 铁十实验验证 → micro_digest → confidence 上升
  - 2 年后: confidence 0.65, hypothesis=false

验证:
  - 从书中恢复概念 ✓
  - 低初始置信 → 通过验证提升 ✓
  - 概念重新引入世界 ✓
```

## 场景五：概念空间的代际漂移

```
初始条件:
  - 中原文化概念 "雇佣"
  - 10 代（~300 年）间：行会制度兴起、学徒制转型
  - "雇佣"的使用频率从 0.8 降到 0.15
  - 外部文化接触（北方商业共和城邦）带来高频经济概念

模拟 (每代):
  - 代 1-3: 使用频率高 → granularity 稳定
  - 代 4-6: 行会制兴起 → "雇佣" frequency 下降 → 开始边缘化
  - 代 7-9: 北方商业文化接触 → "雇佣" coverage 向北方的 "employment" 概念漂移
  - 代 10: "雇佣" granularity 降低（边界模糊），但未完全消失
  - 同时: 新概念 "行会契约" 从 SettlementWisdomAggregate 涌现

验证:
  - 概念随社会变化而演变 ✓
  - 高频使用 → 强化；低频使用 → 边缘化 ✓
  - 外部文化接触导致概念漂移 ✓
  - 新概念从实际社会实践中涌现 ✓
```
