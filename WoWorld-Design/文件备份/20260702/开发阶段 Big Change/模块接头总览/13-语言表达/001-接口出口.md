# 语言表达模块 — 接口出口

> **最后验证**: 2026-06-22 | **源文档版本**: 语言表达 v1.0-v1.1 + LLM 增强层 (CHG-017/018/019) + ★ 修辞化渲染 (CHG-057)

## ExpressionRef — 可读物句柄
- **概念**: 8 字节 Copy 类型 `{ carrier_type: u8, local_id: [u8; 7] }`——对齐 ItemEntId 范式。不拥有内容——只是指针。可存入 NPC 记忆、大日志、对话锚点。
- **Owner**: [[../../语言表达/001-语言表达系统总纲|语言表达 001]] · [[../../语言表达/004-内容解析|语言表达 004]] | **类型**: Rust Copy struct, woworld_lang
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 记忆存储 · [[../12-历史/002-接口入口|历史]] PhysicalBook 内容句柄 · [[../07-魔法/002-接口入口|魔法]] grimoire/scroll 内容
- **契约**: CHG-017 | **待处理变更**: 无

## ContentResolver trait — 内容解析注册
- **概念**: `pub trait ContentResolver` — 各模块实现此 trait + 注册到 ExpressionRegistry。新载体类型零改动集成。LocatableReadable 扩展用于物理位置查询。
- **Owner**: [[../../语言表达/004-内容解析|语言表达 004]]（trait 定义 + ExpressionRegistry）
- **消费者**: [[../12-历史/002-接口入口|历史]] 书本内容 · [[../07-魔法/002-接口入口|魔法]] 卷轴/法阵 · [[../18-物品/002-接口入口|物品]] 铭文 · 全部可读物提供方
- **契约**: CHG-017

## LanguageId / ScriptId — 语言与文字标识
- **概念**: `LanguageId = u32` (高 16 位语系 + 低 16 位编号) / `ScriptId = u16`。此前为幽灵类型，CHG-017 正式定义。
- **Owner**: [[../../语言表达/002-语言与文字|语言表达 002]] | **类型**: Rust type alias, woworld_lang
- **消费者**: [[../12-历史/002-接口入口|历史]] Work.language · [[../18-物品/002-接口入口|物品]] 物品铭文 · [[../07-魔法/002-接口入口|魔法]] 魔法文本 · [[../02-NPC活人感/002-接口入口|NPC]] 语言能力
- **契约**: CHG-017

## Proficiency — 语言能力
- **概念**: `Proficiency = f32 (0.0-1.0)`——替代旧 bool literacy。NPC 语言学习由技能系统 linguistics 驱动。
- **Owner**: [[../../语言表达/002-语言与文字|语言表达 002]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 语言能力存储 · [[../08-技能/002-接口入口|技能]] linguistics 技能驱动
- **契约**: CHG-017

## 三层 Crate 架构
- **概念**: `woworld_lang_types` (零依赖类型层) → `woworld_lang_core` (文本生成+对话引擎) → `woworld_lang_full` (LLM+语音)。lang_expression 不依赖任何业务模块。
- **Owner**: [[../../语言表达/001-语言表达系统总纲|语言表达 001]]
- **消费者**: 全部模块——通过 ExpressionRegistry 和 DialogueContext 依赖注入
- **契约**: CHG-017

## 片段组合文本 — ~430 片段
- **概念**: 文本生成 = 片段组合模型（~430 片段 · ~86KB）——非单体模板。参数命名遵循标准常量 ~50 个。三种编码 (Text/Geometric/Audio) 统一产出文本。滤镜管道 (PersonalityFilter/RelationshipFilter) 后处理。
- **Owner**: [[../../语言表达/003-文本生成|语言表达 003]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] NPC 对话 · [[../12-历史/002-接口入口|历史]] 书本内容 · [[../07-魔法/002-接口入口|魔法]] 卷轴文本 · UI 显示
- **契约**: CHG-017

## 对话系统 — Conversation
- **概念**: 多参与者模型 (2→1000+)。TurnMode 四种 (FreeForm/Moderated/Speech/Ritual)。DialogueIntent 五种驱动 NPC 主动对话。ConversationTopic 三种来源 (预设/大日志/NLU)。DialogueContext 依赖注入——lang_expression 不反向查询任何模块。
- **Owner**: [[../../语言表达/005-对话系统|语言表达 005]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] NPC 间对话 / NPC-玩家对话 · [[../09-文化/002-接口入口|文化]] TurnMode::Ritual
- **契约**: CHG-017 + CHG-018

## PhaticLayer — 五类应酬
- **概念**: 五类应酬片段 (~210 片段)——开场/穿插/反应/收尾/自发。SocialField 群体动力学 (惯性/极化/从众)——听众=聚合统计非独立实体。1000 人演讲 ≈ 1 人文本成本 + 浮点计算。
- **Owner**: [[../../语言表达/006-社交层|语言表达 006]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 社交行为 · [[../09-文化/002-接口入口|文化]] CommunicationNorms 驱动
- **契约**: CHG-017

## 五传播通道 — PropagationPayload
- **概念**: Oral(面对面·衰减+失真)/Letter(跨远距·延迟=旅行时间)/Courier(信鸽·丢失风险)/Magical(即时·高门槛·魔耗)/Official(公告·一对多)。PropagationPayload 参数化传播数据包。
- **Owner**: [[../../语言表达/009-信息传播|语言表达 009]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 信息传递 · [[../12-历史/002-接口入口|历史]] 谣言生命周期
- **契约**: CHG-018

## 失真算子 — 五种
- **概念**: ScaleDistortion(数量膨胀/缩小)/DetailLoss(细节丢失)/EmotionalAmplification(情感放大)/AttributionShift(归因替换)/Moralization(道德评价附加)。每次传播应用——传播次数越多失真越严重。
- **Owner**: [[../../语言表达/009-信息传播|语言表达 009]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 信息失真模拟
- **契约**: CHG-018

## DeceptionIntent — 四种欺骗意图
- **概念**: Honest/Exaggeration/Omission/Fabrication。DeceptionDetection 检测概率 = (intelligence × 0.3) + (contradicting_info × 0.3) - (trust × 0.4)。
- **Owner**: [[../../语言表达/009-信息传播|语言表达 009]]
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 谎言检测
- **契约**: CHG-018

## 谣言生命周期 — 五阶段
- **概念**: Outbreak(24h) → Sustained(1-7d) → Decaying(7-30d) → Folklorized(30d+) → Dead。逻辑斯谛增长模型 `dR/dt = k·R·(1-R/N)`。
- **Owner**: [[../../语言表达/009-信息传播|语言表达 009]]
- **消费者**: [[../12-历史/002-接口入口|历史]] 谣言历史化
- **契约**: CHG-018

## NonVerbalSignal — 非语言表达
- **概念**: 六类信号 (面部/手势/姿态/视线/空间/触觉)。synthesize_nonverbal() 从对话内容 + 情绪 + 个性派生。
- **Owner**: [[../../语言表达/010-非语言表达|语言表达 010]] — 本模块保留 NonVerbalSignal 六类数据模型定义 + synthesize_nonverbal() 合成函数
- **已迁出**: GestureCultureMapping（手势→文化含义映射）→ [[../09-文化/001-接口出口|文化模块]] (CHG-024)。本模块通过 `woworld_culture::GestureCultureMapping` 消费此映射（见 002-接口入口）
- **消费者**: [[../02-NPC活人感/002-接口入口|NPC]] 非语言行为 · Godot 渲染
- **契约**: CHG-018 → CHG-024

## InputInterpreter — NLU 三层回落
- **概念**: L1 关键词 (永远可用, ~10µs) → L2 嵌入 (离线, ~5ms) → L3 LLM (可插拔, ~1-2s)。99% 对话走 L1+模板。PlayerInput 四种统一 (预设选项/搜索框/语音/打字)。
- **Owner**: [[../../语言表达/007-自然语言理解|语言表达 007]]
- **消费者**: Player 输入 · [[../02-NPC活人感/002-接口入口|NPC]] NPC 理解玩家
- **契约**: CHG-017

- **Owner**: [[../../语言表达/011-LLM增强层|语言表达 011]]

## VoiceProfile — 语音身份 ⚠️ 已迁移 (CHG-030)
- **概念**: 物理 (种族/性别/年龄 → 音高/音色/语速) + 个性 (外向性 → 表现力, 宜人性 → 粗砺度, 神经质 → 气声)。
- **迁移**: ~~语言表达 012~~ → **[[../16-音频/001-接口出口|音频系统 005]] (CHG-030 权威)**。VoiceProfile 是声学身份——属于声音层，非语言层。
- **残留**: 本模块保留 TtsEngine trait + VoicePacket 构建。语言表达拥有**内容**（说什么），音频拥有**声音**（怎么说）。
- **契约**: CHG-019 → CHG-030

## VoiceEmotionModulation — 语音情绪修饰 ⚠️ 已迁移 (CHG-030)
- **概念**: 五参数 (音高偏移/语速/音量/颤抖/气声) ← 当前情绪。
- **迁移**: ~~语言表达 012~~ → **[[../16-音频/001-接口出口|音频系统 005]] (CHG-030 权威)**。
- **残留**: 本模块在构建 VoicePacket 时消费 VoiceEmotionModulation（通过 AudioQuery trait，不拥有所有权）。
- **契约**: CHG-019 → CHG-030
