# 模型动作与物理 — 接口入口

> **最后验证**: 2026-06-18 | **源文档版本**: 模型动作与物理系统 v1.0 (CHG-033)
>
> 本模块**消费**以下外部接口。修改这些接口时需检查对模型/动画/物理系统的影响。

## 生命模块
- **BigFive → GaitStyle**: 步态9参数(步幅/步频/髋摆/膝高/脚外八/臂摆/重心起伏/头稳定/躯干旋转) 从 BigFive×EmotionState×疲劳度 数学派生 — [[../01-生命/001-接口出口|生命]]
- **BodyPlan → 骨架选择**: ⚠️ CHG-033: BodyPlan 已提升至 woworld_core — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]. 33骨(L1)/35骨(L0) 人形骨架
- **Vitals → 动画调制**: 疲劳度→步态, health→受伤跛行动画

## 物品模块
- **WeaponPhysicalParams → 战斗轨迹适配器**: 长度/重量/重心/握持位置 → reach_scale/speed_scale/body_commitment — [[../18-物品/001-接口出口|物品]]
- **装备 → 骨骼挂点**: BodyPlan 派生装备槽位→骨骼绑定 — [[../18-物品/001-接口出口|物品]]
- **CHG-033**: WeaponPhysicalParams 映射表定义在物品系统

## woworld_core
- **SkeletonDef**: 人形骨架33骨(L1)/35骨(L0), 4 AnimationModule分区 — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]
- **BodyPlan**: ⚠️ CHG-033: 从 Life crate 提升至 woworld_core — 跨模块基线
- **PhysicalAppearance**: 模型外观参数

## NPC模块
- **GOAP → 动画触发**: NPC GOAP 目标/动作 → 动画层栈触发信号 — [[../02-NPC活人感/001-接口出口|NPC]]
- **EmotionState → 面部表情**: 6字节面部 → 512²图集 UV 偏移 (16嘴×16眉×8眼)
- **Personality → GaitStyle**: BigFive → 步态参数派生

## 战斗模块
- **CombatStyleParams (8参数) → 战斗姿态涌现**: stance_width/guard_height/weight_distribution/rhythm_tempo/rhythm_syncopation/feint_frequency/commitment_level/recovery_style — [[../06-战斗/001-接口出口|战斗]]
- **攻击轨迹 → 动画适配**: WeaponPhysicalParams + CombatStyleParams → 轨迹调整

## 世界生成模块
- **密度场 → TerrainQuery**: height_at/normal_at/density_at/surface_material_at — 动画IK/脚步着地/物理碰撞 — [[../03-世界生成/001-接口出口|世界生成]]
- **CHG-033**: TerrainQuery 为空间查询四trait之一, world_gen实现

## 空间查询 (woworld_core) ⚠️ CHG-033 核心
- **TerrainQuery (9方法)**: height_at/normal_at/terrain_raycast/density_at/is_walkable/surface_material_at/medium_at/light_level_at/sample_horizon — 动画IK/脚步/COM抛物体 — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]
- **EntityIndex (6方法)**: entities_in_aabb/entity_aabb — 碰撞检测/双人交互 — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]
- **SpatialEventBus (3方法)**: push_event — 物理事件(撞击/着地) — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]
- **VisibilityQuery (2方法)**: line_of_sight — 视线/遮挡 — [[../00-全局基础设施/001-核心类型注册表|核心类型注册表]]

## 感官模块
- **medium_at() → 动画模式切换**: 水下/空气 → 游泳动画/步行动画切换 — [[../05-感官与知觉/001-接口出口|感官]]

## 天气模块
- **天气 → 动画调制**: 强风→站立平衡动画, 寒冷→颤抖动画 — [[../04-天气与季节/001-接口出口|天气]]

## 载具模块
- **T_V 变换矩阵 → 骑乘动画**: 骑手骨盆锚定马鞍/座位 transform — [[../15-载具/001-接口出口|载具]]
- **CHG-033**: 骑乘物理约束通过双人交互 TOML 模板表达
