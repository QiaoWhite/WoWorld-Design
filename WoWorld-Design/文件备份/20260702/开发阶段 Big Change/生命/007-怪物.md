## 一、定义

怪物**不是自然生态的一部分**——它们来自异常来源。与动物共享 12 维积木拼装，但额外叠加"异常特征层"。怪物的生态学过滤用**不同标准**——不必通过正常食物链检查。它们破坏正常生态——是"外来入侵物种"。

## 二、7种来源

```rust
enum MonsterOrigin {
    /// 魔法扭曲——正常生物被高浓度魔法/元素失衡永久改变
    Corrupted { source: CorruptionSource },
    /// 亡灵类——见 008-亡灵
    Undead,
    /// 异界渗入——从其他位面漏过来
    Extraplanar { plane: Plane },
    /// 人工创造——有人/有东西造了它
    Created { creator: CreatorType },
    /// 远古幸存——先于当前生态系统的古老物种
    Ancient { era: AncientEra },
    /// 自然变异——基于已生成的正常动物+异常特征叠加
    VariantOf { base_species_id: SpeciesId, anomaly: AnomalyOverlay },
    /// 混合——以上多种来源叠加
    Hybrid { origins: Vec<MonsterOrigin> },
}

enum CorruptionSource { HighMagicZone, ElementalRift, AlchemicalAccident, DivineCurse, BattlefieldResidue }
enum Plane { Abyss, ElementalPlane(Element), SpiritRealm, Feywild, Shadowfell }
enum CreatorType { MadWizard, AncientCivilization, God(A模式), Necromancer, Alchemist, Unknown }
```

## 三、自然动物变体

**关键创意**——怪物可从已生成的正常动物直接派生。同一群系中正常狼与魔狼(狼+骨刺增生+元素泄漏)共存。

```rust
struct AnomalyOverlay {
    features: Vec<AnomalyFeature>,
    severity: f32,  // 0=轻微异常(几乎看不出) ~ 1=严重扭曲(面目全非)
}

enum AnomalyFeature {
    BoneSpurs,           // 骨刺增生——攻击力+、外貌恐怖
    Gigantism,           // 体型巨大化——超出物种正常范围 ×1.5-3
    Dwarfism,            // 萎缩——但异常强化(浓缩=更强)
    ElementalLeak,       // 元素泄漏——身体散发元素能量(火焰鬃毛/冰冻气息/电光纹路)
    HyperRegeneration,   // 超速再生——远超正常——断肢数分钟内重生
    ChimericFusion,      // 多生物特征混合——合成兽——同时有狼+鹰+蛇的特征
    PlanarTaint,         // 位面特征——暗影/光晕/熔岩皮肤——来自其所属位面的物理特性
    PsychicAura,         // 精神光环——周围生物受到恐惧/疯狂/魅惑(被动)
    ConditionalImmortality, // 条件性不死——需特定方式才能永久杀死(银器/圣水/特定魔法/特定时间)
    CorruptedElement,    // 元素反转——火生物变为冰/水生物变为火——元素扭曲
    MindControl,         // 可以控制低智力生物
    PhaseShift,          // 短距离瞬移——部分时间不在实体位面
}
```

## 四、程序化生成流程

```
1. 确定来源——根据群系条件随机选择 MonsterOrigin
2. 如果是 VariantOf——选取群系中已有物种作为基底
3. 从 AnomalyOverlay 中随机选1-3个异常特征(severity控制程度)
4. 叠加异常特征到基底（修改体型/外皮/攻防/魔法等积木维度）
5. 独立怪物——用12维积木从头生成+不经过生态过滤中"食物"和"气候"检查
6. 分配稀有度——怪物通常稀有(0.1-0.5)——传说级极稀有(<0.01)
```

## 五、怪物的生态影响

- 怪物破坏正常食物链——掠食者不敢攻击怪物(异常特征=不可吃/危险)
- 怪物的存在可能扭曲周边环境——元素泄漏怪物→四周植被元素化
- 高浓度怪物区→NPC恐惧+聚落设防+冒险者聚集
- NPC传说中会积累"怪物目击"记录→影响当地文化

## 六、必备经典奇幻怪物

保底内容——每个世界都有：史莱姆(软体+酸性/分裂)、哥布林(小型二足+群居+狡猾——可能智能)、骷髅(→见亡灵)、幽灵(→见亡灵)。
