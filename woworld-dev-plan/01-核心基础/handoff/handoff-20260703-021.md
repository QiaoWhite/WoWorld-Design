## ⚠️ 重要：构建陷阱

**Godot 加载的 `.dll` 只在 `cargo build --workspace` 时更新。** `cargo test` 和 `cargo check` 不产出 cdylib。修改 Rust 代码后如果只跑 test，Godot 中看不到变化。验证流程必须包含 `cargo build --workspace`。

验证命令序列：
```bash
cd woworld
cargo check --workspace
cargo test --workspace          # 54/54
cargo clippy --workspace -- -D warnings
cargo build --workspace         # ← 关键！更新 .dll
```
