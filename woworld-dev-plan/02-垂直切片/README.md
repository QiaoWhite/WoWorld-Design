## §1 阶段目标

**一句话**：制作一个极窄但 100% 可玩的 Demo——一个群系 + 一个村庄 + 几个 NPC + 玩家能走路/对话/挖方块/造东西。证明 WoWorld 的核心体验成立。

**为什么是「垂直切片」而不是「继续铺系统」**：Phase 1 做完后所有 Layer 1 系统就位，但它们是孤立的。垂直切片迫使我们将这些系统**纵向打通**——从玩家输入到世界反馈的完整闭环。这是风险最高的步骤：如果核心循环不有趣，铺再多系统也没用。
