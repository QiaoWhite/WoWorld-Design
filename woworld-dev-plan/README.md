## 核心原则速查

> 详见 [`CONSTITUTION.md`](CONSTITUTION.md)。这里只列不可商量的。

1. **Rust 是唯一权威** — 所有游戏世界状态在 Rust 侧。Godot 只读。
2. **冲刺审批** — 用户审批冲刺目标。冲刺内任务 AI 自主。
3. **质量门不可跳过** — 机械门粘贴真实输出。功能门需人类签字。
4. **🧪 研究先行** — 🟠🔴 级新问题必须先搜索行业方案再编码。
5. **设计变更必响应** — 设计文档变了，执行手册同步更新。
6. **三击升级** — 连续失败不是"再试一次"，是升级。
