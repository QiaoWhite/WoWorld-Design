### 架构关键

- DensityStack 现在是 2 层: `TerrainBaseDensity(priority=0)` + `CaveDensity(priority=4)`
- CaveDensity 使用 Worley 3D |F₂−F₁| 噪声——低值 = 洞穴空洞，高值 = 固体
- 累加模式下洞穴层返回负密度 (-amplitude) 抵消地形正密度
- Transvoxel 等值面阈值 = 0.0（新体系：正值=实体，负值=空）
- 所有 MC 表格从 git 历史提取——纯数学常量，永久跨平台一致
