### 2. 本冲刺目标（≤3 条）

**目标 1: DensityField trait 补全（material_at + priority）**
→ 验收标准：`DensityField` trait 新增 `material_at(x, y, z) -> u8` 和 `priority() -> u8` 两个方法。`HeightfieldDensity` 实现这两个方法。Marching Cubes 网格生成改为从 `material_at` 获取材质索引（替代当前调用 `terrain.surface_material_at()` 的硬编码路径）。
→ 预计影响：`density.rs`（trait 签名）· `terrain.rs`（HeightfieldTerrain 实现）· `marching_cubes.rs`（消费方）· `terrain_mesh.rs`（消费方）

**目标 2: Seed u32 → u64 + stage/chunk hash 推导**
→ 验收标准：`WorldNoise::new(seed: u64)`。内部通过 `hash(seed, stage_id)` 派生每个管线阶段的种子；通过 `hash(stage_seed, chunk_coord)` 派生每个 Chunk 的种子。`noise::Perlin::new()` 接受 u32 的限制通过 split 高/低 32 位或两次 hash 解决。所有现有测试仍然通过（确定性保证）。
→ 预计影响：`noise_gen.rs`（seed 类型 + hash 函数）· 所有消费 `WorldNoise::new()` 的调用点
