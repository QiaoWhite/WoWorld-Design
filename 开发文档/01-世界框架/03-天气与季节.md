## Resource（不进 ECS Component）

```rust
struct WeatherState {
    current: WeatherEnum,  // Clear/PartlyCloudy/Overcast/LightPrecip/ModeratePrecip/HeavyStorm
    transition_progress: f32,  // 0→1 天气过渡
    temperature_regional: f32,
    temperature_ground: f32,
    humidity: f32,
    wind_speed: f32,
    wind_direction: f32,
    cloud_coverage: f32,
    precipitation_intensity: f32,
}

struct SeasonState {
    current: Season,  // Spring/Summer/Autumn/Winter
    day_of_year: u16,  // 0-119
    days_per_season: u16,  // 30
}
```

---

## System 定义

### WeatherDriverSystem

- **触发**: Phase 1, 每帧
- **读/写**: `&mut WeatherState` (Resource)
- **逻辑**: Markov 6-state 转移 + 温度/湿度/风演变
- **与其他 System 的关系**: 无

### SeasonSystem

- **触发**: Phase 1, 日边界（lazy）
- **读/写**: `&mut SeasonState` (Resource)
- **逻辑**: day_of_year → 季节判定

---

## 消费方式

任何 System 通过 Resource 读取天气——不需要 Entity 有 Weather Component：

```rust
fn movement_system(world: &hecs::World, weather: &WeatherState) {
    let speed_mod = match weather.current {
        WeatherEnum::HeavyStorm => 0.5,
        WeatherEnum::LightPrecip => 0.8,
        _ => 1.0,
    };
}
```
