## 三层需求模型

```
Layer 1 (生存/GOAP): 7 基本需求 → 直接触发 GOAP
Layer 2 (心理): esteem_deficit + competence_frustration → 概率引擎调制
Layer 3 (成长): 6 IntrinsicGoal → 长期愿望, GOAP 子目标自动展开
```

## System 定义

### EsteemEvaluationSystem
- **触发**: 决策周期
- **积累率**: esteem_deficit += 0.01/day
- **4 源赞赏检测**: 直接赞美×legitimacy×affection, 感激, 顺从/臣服, 被谈论
- **自我反思**: 成就记忆与自我概念对齐 → esteem_deficit 修正
- **文化调制**: `honor_weight_for_domain(domain)` — 8 个 RecognitionDomain

### CompetenceFrustrationSystem
- **公式**: `frustration = Σ(max(aspired_level - actual_level, 0) × weight)`
- **慢性放大**: chronic_days > 30 → frustration × 1.5 (饱和)
- **Personality 调制**: neuroticism 放大, conscientiousness 缓冲
- **恢复路径**: 技能突破(清零), 愿望降级(chronic_days > 90), 慢性衰减(-2/day)
- **边界**: 儿童无技能愿望(零 frustration), 老人重定向(远离技能成长)

### GoalResolutionSystem（扩展）
- **查询**: `(&Desire, &GrowthColumn, &IntrinsicGoal, &Personality)`
- **内在动力权重**: `intrinsic_weight = commitment × relevance(goal, action)`
- **6 种 IntrinsicGoal**: ExploreUnknown, MasterSkill, ActOnValue, CreateSomething, DeepenConnection, LeaveLegacy
- **目标生命周期**: Active → Dormant(需求压制) → Abandoned(放弃) → Achieved

## 跨层桥接

### survival_suppression() — 生存压制
```
if max(deficit_needs) > 0.7:
    suppression = sigmoid(max_deficit, k=10, x0=0.7)
    intrinsic_weight *= (1 - suppression)  // 只影响概率引擎, 不影响 GOAP
```

### frustration_regression() — ERG 回归
```
if competence_frustration > 0.5 && chronic_days > 15:
    // 临时调制 NeedSensitivity——不持久化
    modulated_social = social_sensitivity * (1 + frustration * 1.4)
    modulated_esteem = esteem_sensitivity * 1.8
    modulated_hunger = hunger_sensitivity * (1 + frustration * 0.3)
    modulated_libido = libido_sensitivity * (1 + frustration * 0.5)
```
