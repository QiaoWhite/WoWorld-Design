## System 定义

### CognitionUpdateSystem
- **触发**: 决策周期, `LodLevel.ai_lod <= 2`
- **读**: `&CognitiveStyle`, `&BigFive`, `&AgentSnapshot`
- **写**: `&mut CognitiveTide`
- **公式**:
  - `load = (active_goals / max_goals) × (percept_count / 64)`
  - `rumination = unsolved_contradictions × neuroticism`
  - `quietude = (1 - load) × (1 - rumination) × openness`
- **CommandBuffer**: 无

### MemoryConsolidationSystem
- **触发**: Entity 有 `PerceptBatch` + `MemoryHandle`
- **4 层压缩**:
  - L0 Hot (≤2000) → 最近事件, 全精度
  - L1 Cold (≤500) → 日级压缩, 情感标签保留
  - L2 Era (≤20) → 年级压缩, 主题抽取
  - L3 Life (≤5) → 生平摘要, 叙事化
- **Resource**: `&mut MemoryStorage`

### PatternRecognitionSystem
- **触发**: 低频（日级）
- **核心类型**: `PatternExpression` (~120B) — 因果步骤 + 结构 LSH + DomainSignature(u64)
- **DomainSignature**: 从 AtomTypeTag 组合涌现——非枚举
- **predict_outcome()**: 纯函数, 基于模式前向投影, O(log n) 单次查询, 零 World 引用
- **输出不确定性**: f32 [0, 1] — 模式匹配度越高, 不确定性越低

### SleepProcessingSystem
- **触发**: Entity 进入睡眠状态
- **逻辑**: Dreams = Dropout (随机抑制弱突触) + Synaptic Pruning (清除零使用连接)
- **自适应荒诞度**: cognitive_load 越高 → 梦境越荒诞
- **效果**: CognitiveTide 重置——load→0, rumination→0, quietude→1

### InnovationPipeline
- **6 阶段**: Observation → Pattern Recognition → Hypothesis → Formalize → Validate → Disseminate
- **输出**: AcademicWork (默认) / Invention / Artwork / SocialMovement (领域注册系统)
- **触发**: 低频——创新不是每帧事件
