## 核心哲学

**零年龄门禁。** 所有能力差异从 AgentSnapshot 的连续参数涌现——不是 `if age < X { cannot_do }`.

## System 定义

### AgeSystem
- **触发**: 日边界
- **逻辑**: `advance_age(entity, delta_days, rng) → Vec<LifeEvent>`
- **6 生命阶段**: Infant→Child→Adolescent→Adult→MiddleAge→Elder
- **Gompertz 衰老**: `mortality_risk(t) = base × e^(α×t)` — 初始低、后期指数增长

### ConceptionSystem
- **触发**: 低频（日级, 有 FertilityState 的 Entity）
- **FertilityPotential**: 连续 sigmoid 曲线——与 libido 分离
- **条件**: 双方 fertile + 关系存在 + 文化/信仰/法律 3 层约束
- **结果**: FertilityState{is_pregnant: true, fetus_seed}

### BirthSystem
- **触发**: `gestation_progress >= 1.0`
- **动作**: spawn 新生儿 Entity（含 InfantDependency + PrenatalAccumulator effect）
- **InfantDependency**: Nursing→Weaning→Weaned 状态机

### DeathTransitionSystem
- **触发**: hp=0（来自 DeathWatchSystem）
- **逻辑**: 30 种死因 6 大类——Violent(8)/Disease(5)/Senescence(5)/Environmental(4)/SpiritExhaustion(4)/Volition(4)
- **死亡摘要**: T+0 全量 → T+7 压缩 (1-3KB) → T+30 LMDB 冷归档
- **继承链**: 血亲→遗嘱→无人继承则消散

### CognitiveAgingSystem
- **触发**: 年级
- **3 种老化路径**: NormalDecline / CompensatedDecline / AcceleratedDecline
  — 由 CognitiveStyle + LifeTheme + chronic_intoxication 决定

| 玩家 | SapientMind + ControlMode | 跳过童年 | 夺舍/异世界两种入口 |
| 动物 | InstinctiveMind | 物种特定 | 行为树 |
| 怪物 | InstinctiveMind + Anomaly | 物种特定 | 异常特征叠加 |
| 不死族 | MinimalMind | 无年龄 | 灵元素残余 |
| 神性存在 | DivineMind | 不朽 | Mode A only |
