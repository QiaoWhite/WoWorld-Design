# 概念与语言 — ECS 架构

> **关联原文档**: [[开发阶段/概念与语言地基/README]]（CHG-044）
> **关联**: [[01-NPC总纲]] · [[../04-交互与表现/05-语言表达]]

---

## 核心哲学

NPC 的语言能力从 LISTEN/SPEAK/READ/WRITE/EXAMINE 原子日志中 S-curve 涌现——非预设。

---

## Component

| Component | 字段 | 内存 | 说明 |
|-----------|------|------|------|
| `PatternSignature` | u64 | 8B | 确定性 hash——全局客观 |
| `ConceptSpace` | concepts_handle:u64, count:u16 | 10B | → ConceptDictionary Resource |
| `LanguageProficiency` | language_id:u32, level:u8(0-100), scripts:[ScriptProficiency; 4] | 24B | S-curve: 初期慢→中期快→后期渐近 |
| `CompactConceptEncoding` | [u8; 64] | 64B | 自包含——不需要外部概念空间 |
| `DomainSignature` | u64 | 8B | 从 AtomTypeTag 聚类涌现——非枚举 |

## System

| System | 触发 | 动作 |
|--------|------|------|
| `ConceptAcquisitionSystem` | `(&ConceptSpace, &PerceptBatch)` | 新概念学习——从感知中提取 PatternSignature |
| `LanguageLearningSystem` | `(&LanguageProficiency, &PerceptBatch)` + `Resource<LanguageRegistry>` | S-curve 语言增长 |
| `ScriptLearningSystem` | `(&LanguageProficiency, &PerceptBatch)` | 文字熟练度 (独立于语言) |
| `DeductiveChainSystem` | 低频 | induction/analogy/deduction 3 种推理模式—wisdom_effective() 调制 |

## Resource

| Resource | 内容 | 大小 |
|----------|------|------|
| `ConceptDictionary` | PatternSignature(u64) → ConceptDef | ~5K 条目 |
| `LanguageRegistry` | LanguageId → {家族 DAG, 方言 sigmoid, 文字系统} | ~50KB |

## 跨模块接口

| 接口 | 方向 | 说明 |
|------|------|------|
| `wisdom_effective()` | → NPC | 综合 wisdom + experience + cognitive_style |
| `transmission_fidelity()` | ← Language | 6 路径独立保真度衰减率 |
| `translate_concepts()` | → Dialogue | 跨语言概念翻译 |

## 调优参数 (~8 个)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `LANGUAGE_S_CURVE_MIDPOINT` | 500 | S-curve 中点 (触数) |
| `CONCEPT_ACQUISITION_RATE` | 0.1/day | 概念学习速率 |
| … | | |
