## 核心哲学

**价格从交易中涌现, 不是从公式中计算。** 任何模块不得直接写入价格。

## System 定义

### MarketMatchingSystem
- **触发**: 区域模拟时钟 (90 real seconds)
- **定价**: 限价订单簿——价格×时间优先, 随机平局打破
- **5 种交易模式**: Barter / Currency / Auction / Credit / GiftExchange
- **中间商涌现**: 4 条件 (信息差 + 资本 + 风险容忍 + 物流) → NPC 自发套利

### PriceDiscoverySystem
- **触发**: 每交易批次后
- **原则**: 价格从实际交易中涌现——不设中央定价函数
- **更新**: `known_price = EMA(last_trade_price, α=0.3)` — 这是 NPC 的认知, 不是权威价格
- **传播**: 5 通道 (口头/书信/信使/魔法/官方公告) + 5 失真算子

### EconomicCognitionUpdateSystem
- **触发**: 低频
- **10 行为经济学概念**: 损失厌恶, 锚定效应, 心理账户, 双曲贴现, 从众, 框架效应, 禀赋效应, 沉没成本, 过度自信, 现状偏好
  — 全部从已有 NPC 字段映射, 不新增维度

### TradeExecutorSystem
- **两阶段提交**: Wallet +/- → Inventory +/-
- **原子保证**: 两步全部成功或全部回滚

## Resource

| Resource | 内容 |
|----------|------|
| `MarketRegistry` | 所有活跃市场的订单簿——非物理位置 |
| `MoneySupply` | 3 管道 (铸币/消费回收/跨区流动) + 5 自动稳定器 |
| `MarketRegulations` | 连续可调参数 + 4 预置 (自由市场/计划经济/军事/自给) |

## 市场权力

| 权力来源 | 经济效应 |
|---------|---------|
| 正式任命 | 税率设定 + 许可发放 |
| 购买力 | 价格影响 (大宗交易) |
| 事实垄断 | 供给控制 |
| 暴力 | 黑市/走私 |

## 约束检查清单

- [x] 无中央定价函数——价格从交易涌现
- [x] NPC 认知价格 ≠ 实际成交价格
- [x] 两阶段提交保证原子性
- [x] 惰性求值——经济系统不每帧全量运行
