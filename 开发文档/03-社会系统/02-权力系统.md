## 17 种通用权力原子

| 类别 | 原子 | u8 编码 | 说明 |
|------|------|---------|------|
| 结构 | Constitute, DefineMembership, Delegate, Relinquish | 0-3 | 创建/定义/委托/放弃 |
| 自我指涉 | Contract, Pledge | 4-5 | 合同(双边), 誓言(自约束) |
| 关系 | Constrain, Compel, Extract, Access | 6-9 | 限制/强制/提取/访问 |
| 规范 | ConferRank, Represent, PrescribeRule, Derogate | 10-13 | 授衔/代表/制定规则/减损 |
| 裁决 | Adjudicate, Sanction, Remit | 14-16 | 裁决/制裁/赦免 |

## System 定义

### PowerExerciseSystem
- **查询**: `(&PowerEdge)` where `valid_until.is_none() || valid_until > now`
- **逻辑**: 根据 atom_type 执行相应权力操作
- **事件输出**: exercise_power() → Vec<AtomEvent> → SpatialEventBus

### LegitimacyUpdateSystem
- **触发**: 低频（日级）
- **公式**: `legitimacy = Σ(source_i × weight_i) / Σ(weight_i)` — 5 因子加权平均
  - 5 来源: 传统/魅力/法理/绩效/强制
- **正向反馈阻尼**: legitimacy > 0.8 → 增长率×0.5

### PolityDetectionSystem
- **触发**: 低频（日级）
- **4 条件**: 领土连续性 + 统一权威 + 平均合法性≥0.30 + 持续≥365 天
- **结果**: `insert Polity` 或 `remove Polity`

### DiplomaticRelationSystem
- **触发**: 事件驱动 + 日级衰减
- **连续分**: -1.0 (敌对) ~ +1.0 (盟友)
- **离散状态**: neutral(> -0.3) / allied(> 0.5) / hostile(<-0.5) / war(<-0.8)
- **战争效果**: 贸易中断, 移动限制, NPC 敌意标记

### ContractSystem
- **查询**: `(&PowerEdge)` where `atom_type == Contract`
- **逻辑**: 双边合同——两个对称 PowerEdge 互相引用
- **违约**: ContractRecord 记录, 制裁塌缩链触发

### PledgeSystem
- **查询**: `(&PowerEdge)` where `atom_type == Pledge`
- **逻辑**: 自约束——违约触发合法性下降而非外部制裁

## 制裁塌缩链

```
holder 发现 target 违反约束
  → 找 willing enforcer (走 PowerTopology 出边)
  → 找到 → 制裁执行
  → 找不到 → holder legitimacy 下降
     → legitimacy < 0.2 → holder 的权力被质疑
        → 触发权力重分配
```

## 经济桥接

`PowerToEconomicBridge`: 权力→经济效应映射
- 收税权 (Extract atom) → Wallet 转账
- 市场准入 (Access atom) → Storefront 开关
- 价格管控 (PrescribeRule atom) → MarketRegulations 参数修改
