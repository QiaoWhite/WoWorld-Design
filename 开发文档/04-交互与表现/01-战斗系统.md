## 核心哲学

**玩家 AI = NPC AI = 同一套 Rust 代码。** 三层模型同时运行, 互相染色:
- 本能层 (per-frame ~16.7ms): 自动闪避/格挡/反击/本能法术
- 节奏层 (0.5-1s): 倾向选择——压进/稳健/游击/狂战/脱离
- 战略层 (5-30s + 事件驱动): 整体目标/阶段子目标/约束/自我评估

## Component 定义

| Component | 字段 | 内存 | 说明 |
|-----------|------|------|------|
| `CombatState` | stance:u8, target:Option<EntityId>, combo:u8, phase:u8(3层) | 16B | 战斗中临时 |
| `InCombat` | (tag) | 0B | 高频切换——用 tag |
| `IncomingDamage` | amount:u32, source:EntityId, damage_type:u8, body_part:u8 | 16B | 一帧存活——帧末清除 |
| `WoundSet` | wounds:[Option<Wound>; 8], count:u8 | ~128B | 渐进 3 档——轻/中/重伤 |
| `Stance` | aggression:f32, defense:f32, risk_tendency:f32, reaction_speed:f32, tactical_depth:f32, magic_preference:f32, team_tendency:f32, honor:f32 | 32B | 8 参数——无上限 |
| `CombatPerception` | visible_enemies:[EntityId; 16], perceived_allies:[EntityId; 16], self_assessment:f32 | ~140B | 6 项故意不可见: HP/stamina/mana/skill/emotion/combat_style |
| `PostCombatState` | emotional_residue:f32, startle_response:f32, trauma_growth:f32 | 12B | 战后心理 |

## System 定义

### CombatInitSystem
- **触发**: Entity 有 `CombatState` + `InCombat`
- **读 Resource**: `CombatStyleRegistry` (NPC 战斗风格参数表)
- **逻辑**: 3 层同时评估——本能 (闪/挡/反), 节奏 (倾向), 战略 (目标)
- **6 故意不可见**: HP/stamina/mana/skill_level/emotion/combat_style — 不在 CombatPerception 中

### DamageApplicationSystem
- **查询**: `(&IncomingDamage, &mut Vitals)`
- **部位伤害**: 3 档——轻(<30%), 中(30-60%), 重(>60%)—不影响攻击模组, 仅影响数值
- **CommandBuffer**: `remove IncomingDamage`

### MoraleSystem
- **查询**: `(&CombatState, &WoundSet, &AgentSnapshot, &EmotionalState)`
- **逻辑**: 伤势累积 + 目睹盟友倒下 → 士气下降; 首领鼓舞 → 士气上升

### PostCombatSystem
- **触发**: InCombat tag 被移除
- **效果**: emotional_residue (残留恐惧), startle_response (惊吓反应), memory encoding (战斗记忆), injury persistence (伤势持续), reputation propagation (名声传播), trauma/growth (创伤或成长)

## 遭遇响应 (4 选项)

| 选项 | 触发条件 |
|------|---------|
| Flee | 自评估<0.3 × 威胁 |
| Compromise (外交) | agreeableness>0.6 |
| Avoid (绕路/潜行) | perception 先发现对方 |
| Fight | 以上都不满足 |

## Resource

| Resource | 内容 |
|----------|------|
| `CombatStyleRegistry` | NPC 战斗风格参数 (8 dims) |
| `DamageTypeTable` | 伤害类型→效果映射 |
| `MaterialProperties` | 力×硬度差 = 变形 (24KB L2 缓存常驻) |

## 性能预算

| 层 | 预算 |
|----|------|
| 本能层 | ≤0.5ms |
| 节奏层 | ≤1.0ms (共享) |
| 战略层 | ≤2.0ms (共享) |
| 战斗 AI 总计 | ≤1.5ms (与其他 GOAP 互斥) |

## 约束检查清单

- [x] 玩家和 NPC 共享同一套战斗代码
- [x] 部位伤害是数值模型——不影响攻击模组
- [x] CombatPerception 故意不暴露 6 项关键信息
- [x] MaterialProperties 只读——24KB L2 常驻
