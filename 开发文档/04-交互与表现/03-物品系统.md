# 物品系统 — ECS 架构

> **关联原文档**: [[开发阶段/物品系统/README]]

---

## Component

| Component | 字段 | 内存 |
|-----------|------|------|
| `ItemDefId` | u64 (newtype) | 8B |
| `ItemEntId` | u64 (newtype) | 8B |
| `EquipmentSlots` | slots:[Option<ItemEntId>; 12] | ~96B |
| `InventoryHandle` | storage_key:u64, count:u16 | 10B |
| `Pickupable` | (tag) | 0B |
| `CraftingTask` | recipe_id:u32, progress:f32, tool_tags:[u8; 4] | 16B |

## System

| System | 查询 | 动作 |
|--------|------|------|
| `PickupSystem` | `(&Pickupable, &Position)` + `PlayerNear` | 装 InventoryHandle |
| `EquipmentSystem` | `(&EquipmentSlots, &InventoryHandle)` | 装备/卸下物品 |
| `CraftingSystem` | `(&CraftingTask, &SkillHandle, &Position)` | 推进锻造进度 |

## Resource

| Resource | 内容 |
|----------|------|
| `ItemRegistry` | ItemDefId → ItemProperties |
| `CraftingRecipeRegistry` | TOML 配方表 |
| `InventoryStorage` | EntityId → Vec<ItemEntId> |
