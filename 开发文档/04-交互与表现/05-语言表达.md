## Component 定义

| Component | 字段 | 内存 | 说明 |
|-----------|------|------|------|
| `ExpressionRef` | carrier_type:u8, local_id:u64 | 8B | Copy——全局唯一表达引用 |
| `LanguageProficiency` | language_id:u32, level:u8(0-100), scripts:[ScriptProficiency; 4] | 24B | S-curve 增长 |
| `Conversation` | participants:[EntityId; 8], count:u8, turn_mode:u8(4种), topic_hash:u64 | ~72B | 对话 Entity 组件 |

## System 定义

### DialogueSystem
- **查询**: `(&Conversation, &LanguageProficiency, &Personality)`
- **4 种 TurnMode**: FreeForm / Moderated / Speech / Ritual
- **5 种 DialogueIntent**: Inform / Inquire / Persuade / Express / Ritual
- **寒暄层**: 5 类 ~210 个模板片段

### TextGenerationSystem
- **读 Resource**: `TextGenerator` (~430 片段, ~86KB)
- **3 种模式**: render_utterance (对话) / generate (生成) / render_thought (独白)
- **修辞渲染**: 明喻/隐喻/类比解释

### PropagationSystem
- **查询**: `(&Utterance)` + `Resource<SpatialEventBus>`
- **5 通道**: 口头/书信/信使/魔法/官方公告
- **5 失真算子**: ScaleDistortion / DetailLoss / EmotionalAmplification / AttributionShift / Moralization
- **谣言生命周期**: Outbreak(24h) → Sustained(1-7d) → Decaying(7-30d) → Folklorized(30d+) → Dead

### InputInterpreterSystem
- **3 层回退**: L1 关键词匹配 (~10μs) → L2 向量嵌入 (~5ms) → L3 LLM (~1-2s)
- **LLM 增强**: 可选——19 场景, 5 优先级, 12 取消触发器
- **静默模型**: 无合适回应时, NPC 自然保持沉默

### NonVerbalSystem
- **查询**: `(&NonVerbalSignal, &EmotionalState, &CultureCoreParams)`
- **文化差异映射**: GestureCultureMapping——同手势在不同文化中含义不同

## Resource

| Resource | 内容 | 大小 |
|----------|------|------|
| `TextGenerator` | 模板片段 + 标准参数 | ~86KB |
| `LanguageRegistry` | LanguageId → LanguageDef (家族 DAG, 方言 sigmoid) | ~50KB |
| `ExpressionRegistry` | carrier_type 范围分配 | ~10KB |

## 信息传播（谣言涌现）

```
NPC A 产生 Utterance
  ↓ B 感知 (audio_lod 决定保真度)
NPC B 转述给 C (5 失真算子之一随机应用)
  ↓ C 感知并转述给 D (进一步失真)
谣言生命周期: logistic增长 dR/dt = k×R×(1-R/N)
```

**这不是 bug——这是游戏核心驱动力。**

## 约束检查清单

- [x] Utterance 是结构化数据——非裸 String
