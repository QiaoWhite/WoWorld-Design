## Component 定义

| Component | 字段 | 内存 | 说明 |
|-----------|------|------|------|
| `PerceptBatch` | entities:[EntityId; 64], count:u8, visual_scene:u64, auditory_scene:u64, salient_entity:Option<EntityId> | ~280B | 每决策周期重建，不持久化 |
| `ScentSource` | scent_type:u8, intensity:f32, decay_rate:f32, source_entity:Option<EntityId> | 16B | Entity 组件——气味来源 |
| `VisionParams` | fov_radians:f32, range:f32, dark_adaptation:f32, racial_template:u8 | 16B | 种族差异 |

| `AttentionState` | locked_entity:Option<EntityId>, lock_remaining:f32, habituation:[(EntityId,f32); 8] | ~80B | 注意力生命周期 |

## System 定义

### VisionSystem
- **查询**: `(&Position, &Eyes, &VisionParams)` + `Resource<EntityIndex>` + `Resource<VisibilityQuery>`
- **逻辑**: 实体分类（背景/信号）+ 自适应射线预算 + 天空线采样
- **暗适应**: 亮→暗 慢 (30min), 暗→亮 快 (30s), 指数松弛
- **运动检测**: 双路径（中心凹高精度 + 周边快速检测）
- **输出**: `PerceptBatch.visual_scene` 更新

### HearingSystem
- **查询**: `(&Position, &Ears)` + `Resource<SpatialEventBus>`
- **输出**: `PerceptBatch.auditory_scene` 更新

### OlfactionSystem
- **查询**: `(&Position)` + `Resource<SpatialEventBus>` (scent_sources_in)
- **输出**: `PerceptBatch` 气味条目

### SalienceEvaluationSystem
- **触发**: Entity 有 PerceptBatch + AttentionState
- **分类**: 每个被感知实体 → background / signal
- **显著性评分**: novelty × intensity × threat × personal_relevance
- **注意切换**: 高显著性新实体抢占注意
- **变化盲视**: 低显著性变化可能被忽略
- **CommandBuffer**: 更新 `AttentionState`

### AttentionLifecycleSystem
- **状态机**: Lock (锁定中) → Residue (残留) → Habituation (习惯化)
- **Lock**: 高显著性实体锁定 2-5s
- **Residue**: 解锁后 0.5-1s 残留——相同实体无法立即重新锁定
- **Habituation**: 反复出现的低显著性实体逐渐习惯化——显著性阈值上升

## Resource

| Resource | 内容 |
|----------|------|
| `VisibilityQuery` (trait) | line_of_sight, line_of_sight_hit — 由 worldgen 实现 |

## 感知噪声与偏差

- **噪声函数族**: Gaussian (感知误差), Uniform (完全随机), Pink (持续性偏差)
- **Seed 确定性**: 同 seed + 同 Entity + 同 tick → 同噪声值
- **情绪调制**: high arousal → 感知范围+20%, 准确度-10%; low valence → 威胁检测阈值-30%

## EnrichedPerceptBatch（派生）

由高层认知 System 消费：
- `gaze_estimates`: 被感知实体的视线方向估计
- `emotional_readings`: 被感知实体的情绪状态 (VAD 3D)
- `bound_entities`: 跨模态绑定 (同一声源同时被看到和听到)
- `crowd_emotional_field`: 人群平均情绪 (L1 规范)

## 性能预算

| 层级 | 内容 | 预算/Entity |
|------|------|-----------|
| L1 | 射线 + 显著性 + 注意力 | <0.1ms (10 rays) |
| L2 | 仅听觉 + 嗅觉 | <0.01ms |
| L3+ | 跳过 | 0 |
