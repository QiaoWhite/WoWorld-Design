## Hybrid 架构

Rust 侧: 音频传播计算 + 事件生成。Godot 侧: AudioServer 实际播放。

---

## Component (Rust 侧)

| Component | 字段 | 内存 | 说明 |
|-----------|------|------|------|
| `SoundFootprint` | sound_type:u8(5类), intensity:f32, frequency_band:u8, distance_attenuation:f32 | 16B | 物理声学快照 |
| `AudioEmitter` | emitter_handle:u64, voice_priority:u8(5级) | 12B | Godot AudioServer 句柄 |
| `HearingModel` | hearing_range:f32, frequency_sensitivity:[f32; 4] | 20B | 种族+感知+年龄+人格 |
| `VoiceProfile` | pitch:f32, timbre:f32, accent:u8, emotion_modulation:f32 | 16B | 从 Language Expression 转移 |

## System (Rust 侧)

| System | 查询 | 动作 |
|--------|------|------|
| `AudioPropagationSystem` | `(&SoundFootprint, &Position)` + `Resource<SpatialEventBus>` | 衰减×吸收×风×温度×遮挡→SpatialEventBus |
| `AudioMaskingSystem` | `(&PerceptBatch, &HearingModel)` | 频段分割×清晰度×环境噪声→可听度 |
| `VoiceManagerSystem` | `(&VoiceProfile, &EmotionalState)` | 5 级 VoicePriority 队列 + 中断仲裁 |

## System (Godot 侧, Phase 2)

| System | 动作 |
|--------|------|
| `AudioDispatchSystem` | Rust AudioEvent → Godot AudioServer.play() |

## Resource

| Resource | 内容 |
|----------|------|
| `AudioQuery` (trait) | 30 方法——零分配 |
| `AudioRegistry` | AudioMaterial (15 种) + SurfaceMaterial (21 种)→声学属性 |
| `ReverbProfile` | AcousticSpace→混响参数 |

## 5 类声音

| 类别 | 例 | 优先级 |
|------|-----|--------|
| WorldSound | 脚步声/环境音 | 低 |
| Voice | NPC 对话 | 高 (VoiceManager 管理) |
| Music | 背景音乐 | 中 |
| Notification | UI 提示 | 高 |
| Accessibility | 无障碍提示 | 最高 |
