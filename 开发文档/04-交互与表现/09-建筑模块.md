# 建筑模块 — ECS 架构

> **关联原文档**: [[开发阶段/建筑模块/README]]

---

## Component

| Component | 字段 |
|-----------|------|
| `BuildingComponent` | family_id:u16, variant:u8, material_ids:[u8; 4] |
| `Blueprint` | component_graph_handle:u64 |
| `ConstructionSite` | progress:f32, required_materials:[MaterialRequirement; 8] |

## System

| System | 查询 | 动作 |
|--------|------|------|
| `BuildingSystem` | `(&BuildingComponent)` | 建筑功能（生产/存储/居住） |
| `ConstructionSystem` | `(&ConstructionSite, &Position)` + 附近 NPC | 推进建造进度 |
| `WFCSolverSystem` | Blueprint 初始化时 | 2.5D WFC 求解→BuildingComponent |

## Resource

| Resource | 内容 |
|----------|------|
| `ComponentRegistry` | ComponentFamily 定义库 |
