# 类型迁移表

> **关联**: [[001-原模块到ECS映射]]

---

## 已有 Rust 类型的 ECS 角色

| 当前类型 | 位置 | ECS 角色 | 处理方式 |
|---------|------|---------|---------|
| `EntityId(u64)` | `woworld_core::types` | Entity 标识符 | 保留，与 hecs::Entity 互转 |
| `EntityKind` | `woworld_core::types` | Tag Component | **保留在 core**——godot crate 消费此类型。ECS 侧 `entity_kind.rs` 为 newtype 包装 |
| `SpatialEntity` | `woworld_core::types` | Component Bundle | 拆为 `Position` + `Rotation` + `Velocity` + `Collider` + `LayerMask` |
| `WorldPos` | `woworld_core::types` | 基础类型 | 保留，用于 `Position` Component |
| `Aabb` | `woworld_core::types` | 基础类型 | 保留，用于 `Collider` Component |
| `TerrainHit` | `woworld_core::types` | 查询结果 | 保留 |
| `SpatialEvent` | `woworld_core::types` | Event | 保留，作为 Resource 中的事件 |
| `ScentSource` | `woworld_core::types` | 基础类型 | **保留在 core**——`SpatialEventBus::scent_sources_in()` trait 返回类型，移入 ecs 造成循环依赖 |
| `AcousticTag` | `woworld_core::types` | 基础类型 | **保留在 core**——`TerrainQuery::acoustic_tag_at()` trait 返回类型，移入 ecs 造成循环依赖 |
| `WorldTime` | `woworld_core::time` | Resource 内的数据 | 保留 |
| `WorldClock` | `woworld_core::time` | Resource | 保留 |
| `LodPrescription` | `woworld_core::lod` | Component | 移入 `components/lod.rs`，重命名为 `LodLevel` |
| `CameraState` | `woworld_core::lod` | Resource 内的数据 | 保留 |
| `FrameBudget` | `woworld_core::lod` | Resource 内的数据 | 保留 |
| `VramPressure` | `woworld_core::lod` | Resource 内的数据 | 保留 |
| `EntityLodInput` | `woworld_core::lod` | System 输入（临时） | 保留 |
| `InteractionIntent` | `woworld_core::lod` | System 输入 | 保留 |
| `WeatherState` | `woworld_core::weather_types` | Resource 内的数据 | 保留 |
| `Season` | `woworld_core::weather_types` | Resource 内的数据 | 保留 |
| `SurfaceMaterial` | `woworld_core::material` | 基础类型 | 保留 |
| `ItemDefId` | `woworld_core::id` | Component | 保留 |
| `ItemEntId` | `woworld_core::id` | Entity 关联 | 保留 |
| `SkillId` | `woworld_core::id` | Component 内的字段 | 保留 |
| `ProfessionTagId` | `woworld_core::id` | Component 内的字段 | 保留 |
| `SpeciesId` | `woworld_core::id` | Component 内的字段 | 保留 |
| `ChunkCoord` | `woworld_core::id` | 空间索引 key | 保留 |

## 不进 ECS 的类型（保留原样）

| 类型 | 原因 |
|------|------|
| `HeightfieldTerrain` | 地形——世界级单例 |
| `HeightfieldOcean` | 海洋——世界级单例 |
| `WorldNoise` | 噪声——共享计算资源 |
| `BiomeClassifier` | 群系——共享计算资源 |
| `CaveDensity` | 洞穴——共享计算资源 |
| `AtmosphereSynthesizer` | 大气——纯计算 |
| `AtmosCurve` | 大气曲线——纯数据 |
| `SimpleWeatherDriver` | 天气——全局状态机 |
| `SimpleSeasonProvider` | 季节——纯函数 |
| `TerrainMeshData` | 网格数据——引擎无关传输 |
| `GerstnerWave` | 海洋波——计算参数 |
| 所有 GodotClass (`WorldDriver`/`VoxelChunk`/`OceanPlane`) | Godot 渲染节点 |
| `EditDensitySnapshot` / `EditDensityBuilder` | ★CHG-065 地形修改——CoW 3D 密度（ECS Resource 桥接） |
| `EditHeightfieldSnapshot` / `EditHeightfieldBuilder` | ★CHG-065 地形修改——CoW 2D 高度投影（ECS Resource 桥接） |
| `EditTerrainSnapshot` / `EditTerrainBuilder` | ★CHG-065 地形修改——组合快照（ECS Resource 桥接） |
| `ModificationBatch` / `DirtyChunkQueue` | ★CHG-065 地形修改——批量积累+脏标记（ECS Resource） |
| `EditDensityLayer` | ★CHG-065 地形修改——DensityProvider 桥接实现（不进 ECS·插入 DensityStack） |
