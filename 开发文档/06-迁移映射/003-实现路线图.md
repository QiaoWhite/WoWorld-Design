## 阶段 0: ECS 基础设施（当前可执行）

### Step 0.1: 添加 hecs 依赖

```toml
# woworld_core/Cargo.toml
[dependencies]
hecs = "0.10"
```

预计依赖增量：hecs + ahash + hashbrown ≈ 3 个轻量 crate。

### Step 0.2: 定义核心 Component

在 `woworld_core/src/components/` 下创建：
- `transform.rs` → `Position`, `Rotation`, `Velocity`
- `entity_kind.rs` → `EntityKind` tag
- `lod.rs` → `LodLevel`（迁移已有 `LodPrescription`）

### Step 0.3: WorldDriver 集成

```rust
// woworld_godot/src/terrain_chunk.rs
pub struct WorldDriver {
    ecs: hecs::World,  // ★ 新增
    // 现有字段全部保留
}
```

### Step 0.4: 第一个 ECS System

将现有 `LodCoordinator::compute_lod()` 封装为 `LodCoordinatorSystem`：
- 写入 `LodLevel` Component 到 Player Entity
- 验证：107 现有测试全绿 + 新 ECS 测试
